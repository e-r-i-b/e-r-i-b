Инструкция по сборке стенда контрольной компиляции.

Структура каталогов.

<root>
	api - исходный код вспомогательных библиотек;
	PhizIC - директория с исходным кодом ИКФЛ;
	Toolkit - инструменты необходимые для сборки;
	tools - исходный код, вспомогательных средств;
	build.bat - командный файл стенда;
	buildExternal.cmd - вспомогательный командный файл;
	externalTools.txt - содержит описание библиотек сторонних производителей;
	readme.txt - этот файл;

Описание директории Toolkit можно найти в файле Toolkit\readme.txt.

Документацию по развертыванию и настройке ИКФЛ можно найти по пути  root\PhizIC\userdoc.

Перед запуском необходимо убедиться, что на машине установлен microsoft .net framework. 
Если это не так, то для установки необходимо запустить файл инсталляции 
Toolkit\dotNetFramework\dotnetfx.exe. Framework отсутствует, 
если в процессе сборки возникает ошибка, что не найден файл mscore.dll.

Для запуска процесса сборки необходимо запустить командный файл build.bat в корне СКК. 

Результаты работы стенда в каталоге Results, который имеет следующую структуру:
<results>
	PhizIC.image - дистрибутив ИКФЛ.
	RSAlarm - дистрибутив RS-Alarm.
	RSBridge - дистрибутив RS-Bridge.
