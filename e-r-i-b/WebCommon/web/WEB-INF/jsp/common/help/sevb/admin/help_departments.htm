Регистрация нового подразделения

Пункт меню Подразделения (подразделения АС "Сбербанк ОнЛ@йн") позволяет добавлять, удалять и редактировать сведения о подразделениях, а также управлять списком уполномоченных лиц, от имени которых заключается договор на обслуживание в системе «Сбербанк ОнЛ@йн».

Подразделения в справочнике имеют иерархическую структуру: они делятся на головные подразделения и подчиненные – внутренние структурные подразделения (ВСП).

Список подразделений

Список подразделений открывается автоматически в главном меню в пункте Подразделения. В нем содержится список подразделений банка, которые подключены или не подключены к системе ИКФЛ "Сбербанк ОнЛ@йн". Все сотрудники и клиенты банка привязываются к одному из данных подразделений.

Создание подразделения

Если Вы хотите создать в системе новое подразделение, щелкните по кнопке Добавить, откроется страница редактирования подразделения "Сбербанк ОнЛ@йн". Порядок добавления нового подразделения см. в разделе контекстной справки Регистрация нового подразделения.

Редактирование подразделения

Для того чтобы отредактировать сведения о подразделении, в списке выберите подразделение, установите флажок напротив наименования интересующего Вас подразделения и щелкните по кнопке Изменить . Откроется страница редактирования подразделения банка, на которой измените значения интересующих Вас полей, затем щелкните по кнопке Сохранить , и система покажет данное подразделение в списке в измененном виде.

Репликация подразделения

Для того чтобы загрузить из внешней системы внутренние структурные подразделения, выберите в списке интересующие подразделения, установив напротив них флажки, и щелкните по кнопке Репликация . После загрузки подразделений система Вам покажет протокол, в котором содержится информация о количестве загруженных, измененных подразделений.

Примечание: репликация внутренних структурных подразделений доступна только для тех подразделений, для которых установлен признак головного.

Удаление подразделения

Если необходимо удалить запись о подразделении, установите флажок напротив интересующего Вас подразделения и щелкните по кнопке Удалить . Система попросит Вас подтвердить операцию, Вы подтверждаете удаление, и подразделение будет удалено из списка.

Примечание: подразделения, за которыми закреплены сотрудники или клиенты, не могут быть удалены из системы.

Общие сведения

Вы можете добавить в справочник подразделения разных уровней иерархии.

Примечание: сотрудник банка может работать с системой на определенном уровне иерархии отделения Сбербанка. Настройка уровня иерархии подразделений в системе «Сбербанк ОнЛ@йн» задается встроенным администратором в пункте меню Настройки – Параметры системы.

Для того чтобы добавить подразделение верхнего уровня, нажмите на кнопку Добавить.

Для создания подчиненного подразделения (ВСП) установите флажок напротив интересующего Вас подразделения верхнего уровня и щелкните по кнопке Добавить. Откроется страница редактирования подразделения банка, на которой в полях формы необходимо указать сведения о добавляемом подразделении. Система экспортирует поля данной формы в договор на предоставление услуг с использованием системы «Сбербанк ОнЛ@йн». Для того чтобы создать подразделение, заполните следующие поля формы:

  • В поле "Наименование" укажите название нового подразделения.

  • Если создаваемое подразделение будет обслуживаться в системе ИКФЛ «Сбербанк ОнЛ@йн», то установите флажок в строке «Подключено к Сбербанк ОнЛ@йн».

    Примечание: сотрудники банка могут быть закреплены за любым подразделением, а клиенты только за теми, которые обслуживаются в системе ИКФЛ «Сбербанк ОнЛ@йн».

  • При создании головного подразделения установите флажок в поле «Головное». Данный признак означает, что можно реплицировать внутренние подразделения из внешней системы (RS-Retail).

    Примечание: для внешней системы может быть только одно головное подразделение.

  • Если создаваемое подразделение будет подключено к системе «Сбербанк ОнЛ@йн», то Вам необходимо указать внешнюю систему (RS-Retail), которая автоматизирует данное подразделение. Для этого в поле «Внешняя система» щелкните по кнопке (...), откроется страница со списком внешних систем, выберите из списка интересующую систему, установив напротив нее галочку и щелкнув по кнопке Выбрать . На данной странице можно воспользоваться фильтром.

    Фильтр/поиск внешних систем

    Для поиска интересующих Вас внешних систем можно воспользоваться фильтром. Для этого щелкните по кнопке Показать фильтр и задайте критерии поиска:

    • "ID" - поиск по идентификатору внешней системы в системе «Сбербанк ОнЛ@йн»;

    • "Идентификатор" - поиск по уникальному идентификатору внешней системы;

    • "Наименование" - поиск по наименованию шлюза:

    • "Адрес шлюза" - поиск по адресу шлюза.

    Вы можете осуществлять поиск по подстроке в текстовых полях фильтра. Например, можно ввести в поля фильтра только несколько первых букв наименования внешней системы.

    Можно отфильтровать список по любой комбинации полей. Например, можно отфильтровать все внешние системы по наименованию и адресу шлюза.

    После того как критерии поиска заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас внешних систем.

    Если подразделение не будет обслуживаться в системе ИКФЛ «Сбербанк ОнЛ@йн», то указывать внешнюю систему не нужно.

  • В поле «Наименование» укажите название соответствующего подразделения во внешней системе (RS-Retail). Для этого щелкните по кнопке (...), откроется страница со справочником подразделений RS-Retail, на которой выберите название подразделения, установив напротив него флажок, и щелкните по кнопке Выбрать. На данной странице Вы можете воспользоваться фильтром.

    Фильтр подразделений

    Для поиска интересующих Вас подразделений щелкните по кнопке Показать фильтр и задайте критерии поиска:

    • "ТБ" - поиск по номеру территориального банка СБРФ;

    • "ОСБ" - поиск по номеру отделения СБРФ;

    • "Филиал" - поиск по номеру филиала СБРФ;

    • "Наименование" - поиск по названию подразделения.

    Вы можете осуществлять поиск по подстроке в текстовых полях фильтра. Например, можно ввести в поля фильтра только несколько первых букв наименования подразделения RS-Retail.

    После того как параметры фильтра заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас подразделений.

    После выбора из справочника подразделения RS-Retail автоматически заполнятся поля «Почтовый адрес», «Местонахождение», «Телефон».

  • Далее введите операционное время работы подразделения или можно использовать настройку вышестоящего подразделения, для этого установите флажок напротив строки «Использовать настройку вышестоящего», в результате поле станет недоступно для редактирования.

  • В поле «Тариф за организацию расчетного обслуживания» укажите тариф или задайте настройку тарифа вышестоящего подразделения.

  • В поле «Ежемесячная плата» введите размер оплаты в месяц за использование услуг системы «Сбербанк ОнЛ@йн», также можно использовать настройку ежемесячной оплаты вышестоящего подразделения.

После того как все сведения о подразделении указаны, щелкните по кнопке Сохранить , и подразделение будет добавлено в справочник.

Уполномоченные лица

С каждым подразделением банка связан список уполномоченных лиц. Для каждого подразделения можно задать одно или несколько уполномоченных лиц, которые могут заключать договор на подключение клиентов банка к системе "Сбербанк ОнЛ@йн". Эта информация подставляется в форму Договора на предоставление услуг с использованием системы "Сбербанк ОнЛ@йн".

Добавление уполномоченного лица

Если Вы хотите добавить уполномоченное лицо, щелкните по кнопке Добавить и заполните все поля формы: ФИО, должность и документ, удостоверяющий полномочия. После заполнения всех полей формы щелкните по кнопке Сохранить , и система покажет новое уполномоченное лицо в списке уполномоченных лиц данного подразделения банка.

Редактирование уполномоченного лица

Если Вам необходимо отредактировать сведения об уполномоченном лице, щелкните по его фамилии, имени, отчеству или установите флажок напротив имени уполномоченного лица и нажмите на кнопку Изменить , откроется форма редактирования сведений об уполномоченном лице. В форме Вам нужно изменить интересующие Вас сведения и щелкнуть по кнопке Сохранить .

Удаление уполномоченного лица

Если Вы хотите удалить из списка уполномоченное лицо, установите флажок напротив его имени и щелкните по кнопке Удалить . Система попросит Вас подтвердить операцию, Вы подтверждаете, и выбранное Вами уполномоченное лицо будет удалено из списка.
