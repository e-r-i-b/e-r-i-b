Смена пароля

Пункт меню Сервис позволяет Вам выполнять различные сервисные операции, например, менять свой пароль для входа в систему и т.д.

Смена пароля

Чтобы изменить пароль для входа в систему, Вам необходимо:

  1. в поле "Введите старый пароль" указать старый пароль для входа в систему,

  2. в поле "Введите новый пароль" вписать Ваш новый пароль,

  3. подтвердить новый пароль, заполнив поле "Повторите новый пароль" той же комбинацией символов.

  4. щелкнуть по кнопке Изменить , Вам откроется окно, в котором будет написано "пароль на вход в систему изменен".

В результате при следующем входе в систему Вам нужно будет указать в поле "Пароль" Ваш новый пароль.

Система контролирует качество паролей, поэтому при смене пароля может попросить Вас ввести более сложный пароль в соответствии с настройками системы безопасности.
