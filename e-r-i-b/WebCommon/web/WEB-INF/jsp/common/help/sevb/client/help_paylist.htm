В журнале операций содержатся все выполненные Вами платежные операции, а также незавершенные платежи.

Все платежи

В данном пункте меню содержится список всех выполненных Вами платежей. По каждому платежу в списке отображается дата и время операции, дата приема к исполнению, номер, вид операции, счет списания, наименование получателя платежа, сумма платежа, сумма комиссии, сумма операции, валюта и статус документа в системе.

Вы можете выполнить следующие операции:

  • Просмотр документа

    Вы можете просмотреть любой документ, для этого нужно щелкнуть по номеру интересующего Вас документа в списке.

  • Редактирование документа

    Если Вы хотите отредактировать документ, установите галочку напротив интересующего платежа и щелкните по кнопке Редактировать .

    Примечание: возможность редактирования платежа доступна только для документов со статусом "Введен".

  • Создание копии документа

    Вы можете создать копию платежа, отметив документ, который служит оригиналом в списке, и щелкнув по кнопке Создать копию . В результате Вы перейдете к форме ввода нового документа, причем в нее будут скопированы все сведения из документа, выбранного Вами в качестве оригинала. Данная возможность полезна при создании нескольких однотипных документов.

  • Печать

    Вы можете распечатать интересующий документ, установив напротив него галочку и щелкнув по кнопке Печать .

  • Фильтр платежей

    Для того чтобы воспользоваться фильтром, щелкните по кнопке Показать фильтр и заполните поля фильтра:

    • "Номер" - поиск по номеру документа в системе;

    • "Вид операции" - поиск по виду платежной операции, например, покупка иностранной валюты и др.;

    • "Счет" - поиск по номеру счета, по которому совершен платеж;

    • "Статус" - поиск по статусу документа в системе: "Введен" (платеж создан, но не отправлен на обработку в банк), "Обрабатывается"(платеж подтвержден и отправлен на обработку в банк), "Исполнен" (средства переведены с одного счета на другой), "Отказан" (Вам отказано в операции);

    • "Дата совершения операции" - поиск по дате и времени создания документа в системе;

    • "Сумма" - поиск по сумме, на которую совершен платеж.

    • "Дата приема к исполнению" - поиск по дате и времени приема документа к исполнению;

    • "Наименование получателя" - поиск по названию получателя платежа;

    Вы можете фильтровать список по любой комбинации полей. Например, можно отобрать все документы, созданные в один день со статусом "Исполнен".

    После того как параметры поиска заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас платежей.
