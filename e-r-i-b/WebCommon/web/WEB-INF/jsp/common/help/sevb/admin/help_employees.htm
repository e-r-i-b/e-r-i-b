Пункт меню Сотрудники предназначен для управления учетными записями сотрудников банка, которые работают с системой "Сбербанк ОнЛ@йн". Он позволяет регистрировать новых сотрудников, удалять, блокировать и разблокировать сотрудников, управлять доступом сотрудников к системе "Сбербанк ОнЛ@йн".

В данном пункте меню автоматически открывается список сотрудников банка подразделений системы "Сбербанк ОнЛ@йн". Для каждого сотрудника в списке отображается фамилия, имя, отчество, подразделение, в котором зарегистрирован сотрудник, дополнительная информация, логин и схема прав доступа к операциям в системе.

Добавление сотрудника

Для того чтобы добавить нового сотрудника в список, щелкните по кнопке Добавить , откроется анкета сотрудника. Порядок добавления сотрудника см. в разделе контекстной справки "Регистрация сотрудников".

Редактирование сведений о сотруднике

Если Вы хотите отредактировать сведения о сотруднике, щелкните по его фамилии, имени, отчеству, откроется анкета для редактирования, в которой измените нужные сведения и нажмите на кнопку Сохранить .

Удаление сотрудника

Если Вам необходимо удалить сотрудника из списка, установите галочку напротив интересующего Вас сотрудника и нажмите на кнопку Удалить , система попросит подтвердить операцию, и после подтверждения сотрудник будет удален из списка.

Фильтр/поиск сотрудников

Для поиска всех интересующих Вас сотрудников банка щелкните по кнопке Показать фильтр и задайте критерии поиска:

  • "ФИО"- поиск по имени и фамилии сотрудника;

  • "Номер отделения"- поиск по номеру отделения банка, в котором зарегистрирован сотрудник;

  • "Номер филиала"- поиск по номеру филиала банка, в котором числится сотрудник;

  • "Идентификатор пользователя (ID)"- поиск по идентификатору пользователя в системе;

  • "Дополн. информация"- поиск по дополнительной информации, которая известна о сотруднике;

  • "Статус"- поиск по статусу сотрудника в системе, например: "Активный", "Заблокирован".

Вы можете осуществлять поиск по подстроке в текстовых полях фильтра. Например, можно ввести в поля фильтра только несколько первых букв фамилии, имени или отчества сотрудника банка.

Также Вы можете фильтровать список по любой комбинации полей. Например, можно отобрать всех активных сотрудников с фамилией на букву "А" из одного отделения банка.

После того как параметры фильтра заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас сотрудников банка.

Назначение схемы прав

Назначение схемы прав доступа в системе выполняется двумя способами:

  1. В списке сотрудников банка. В списке определите сотрудников, которым будет назначаться схема прав доступа, установив флажки напротив фамилий интересующих Вас сотрудников. В поле "Схема прав" выберите ту схему прав, которая будет назначена выбранным сотрудникам, и щелкните по кнопке Назначить права. Также Вы можете щелкнуть по схеме прав доступа выбранного сотрудника в списке, откроется вкладка анкеты Права доступа, на которой измените схему прав и нажмите на кнопку Сохранить .

  2. Можно также назначить схему прав в анкете сотрудника банка: во вкладке анкеты Права доступа.

Общие сведения

В анкете сотрудника Вам нужно правильно заполнить все поля, отмеченные звездочкой:

  • Подразделение банка, за которым будет закреплен новый сотрудник. Чтобы заполнить поле "Подразделение", щелкните по кнопке (...), откроется страница со списком подразделений "Сбербанк ОнЛ@йн", на которой выберите интересующее Вас подразделение банка и щелкните по кнопке Выбрать . На данной странице Вы также можете воспользоваться фильтром.

    Фильтр подразделений

    Для просмотра списка интересующих подразделений нажмите кнопку Показать фильтр и заполните поле "Наименование" - название подразделения, затем щелкните по кнопке Применить, и система покажет список интересующих Вас подразделений банка.

    Примечание: поле анкеты "Подразделение" должно быть заполнено обязательно, чтобы новый сотрудник имел доступ к информации о клиентах, закрепленных за данным подразделением.

  • Фамилию, имя и отчество нового сотрудника;

  • Логин, пароль, подтверждение пароля, новый сотрудник должен будет указать логин и пароль при входе в систему. При первом входе система потребует сменить пароль.

Вы можете сохранить частично заполненную анкету сотрудника банка, щелкнув по кнопке Сохранить .

Права доступа

Во вкладке анкеты Права доступа Вам необходимо указать права доступа для нового сотрудника к операциям в системе "Сбербанк ОнЛ@йн". Права доступа Вы можете выбрать двумя способами:

  1. С помощью назначения сотруднику схемы прав. В системе есть 2 группы схем прав:

    • если Вам нужно назначить схему прав "Для администраторов", то в поле "Схема прав" выберите из списка схему администратора. После этого в списке операций будут доступны для редактирования только те операции, которые предназначены для администратора системы. Затем щелкните по кнопке Сохранить , в результате данному сотруднику будет назначена выбранная схема прав.

    • если Вы хотите назначить схему прав "Для сотрудников", то в поле "Схема прав" выберите из списка схему сотрудника. После этого в списке будут доступны для редактирования только те операции, которые предназначены для сотрудников банка. Затем щелкните по кнопке Сохранить , в результате данному сотруднику будет назначена выбранная схема прав.

  2. С помощью назначения индивидуальных прав.

    Если Вам нужно назначить сотруднику банка индивидуальные права, т.е. права доступа к операциям в системе, не связанным ни с одной схемой прав, то для данного пользователя сначала выберите из выпадающего списка для какой категории сотрудников банка Вы хотите назначить индивидуальные права, а затем поставьте галочки напротив соответствующих (доступных для этой категории) операций. После того как список прав сформирован, щелкните по кнопке Сохранить .

Если Вы хотите выйти из пункта меню создания схемы прав доступа сотрудника, щелкните по кнопке Список .

Доступ к подразделениям

Данный раздел доступен только для сотрудников банка, которым назначены права администратора.

Для того чтобы сотруднику задать доступ к подразделениям банка, выберите вкладку Доступ к подразделениям в анкете сотрудника. Откроется справочник подразделений определенного уровня иерархии, в котором в поле «Доступ» установите флажки напротив выбранных подразделений и щелкните по кнопке Сохранить .

Примечание: сотрудник банка может работать с системой на определенном уровне иерархии отделения Сбербанка. Настройка уровня иерархии подразделений в системе «Сбербанк ОнЛ@йн» задается встроенным администратором в пункте меню Настройки – Параметры системы.

При создании администратора ему будут доступны все подразделения, к которым имеет доступ создавший его администратор.

В пределах одного подразделения количество сотрудников с правами администратора ограничено. Если данное количество превышено, то разрешить доступ к подразделению невозможно, и система покажет соответствующее сообщение.

Примечание: количество сотрудников с правами администратора задается встроенным администратором в пункте меню Настройки – Параметры системы.

Настройки оповещений

Каждому сотруднику можно задать настройки оповещений. Оповещения отсылаются системой сотруднику в виде SMS или письма о том, что в системе произошли события, например, о том, что клиент банка совершил платежную операцию и т.д.

По умолчанию настройки оповещений в системе отключены.

Чтобы настроить сотруднику банка рассылку оповещений, необходимо выполнить следующие действия:

  • В поле «Присылать сообщения по электронной почте на адрес (E-mail)» впишите адрес электронной почты сотрудника, которому будут рассылаться оповещения.

  • В поле «Присылать сообщения SMS на номер» введите номер мобильного телефона сотрудника банка.

  • Укажите формат SMS-сообщений, установив галочку напротив поля "Транслитерация".

Из списка возможных оповещений выберите интересующие Вас виды рассылок. Рядом с каждой рассылкой находятся два флажка: SMS и E-mail.

Чтобы указать рассылки, установите флажки напротив каждого выбранного оповещения и щелкните по кнопке Сохранить . Для подписки на оповещения о совершении платежей и заявок клиентами банка Вы можете щелкнуть по кнопке (…), откроется список платежей и заявок, из которого Вы можете выбрать рассылку на определенные платежи и заявки, установив флажок напротив интересующего Вас события и щелкнув по кнопке Выбрать .

  • Если ни один из флажков не установлен, то данное оповещение сотруднику не отправляется.

  • Если установлен флажок SMS, то в случае возникновения данного события сотруднику будет отправлено SMS на указанный в его анкете номер мобильного телефона.

  • Если установлен флажок E-mail, то в случае возникновения данного события сотруднику будет отправлено электронное письмо на указанный в его анкете адрес.

  • Можно устанавливать оба флажка - в этом случае оповещения будут отправляться по обоим каналам.

После того как указаны параметры оповещений, щелкните по кнопке Сохранить .

Блокировка и разблокировка сотрудников

Учетные записи сотрудников можно блокировать. Это позволяет запретить сотруднику доступ в систему (например, на время отпуска), не удаляя его учетную запись.

Блокировка выполняется двумя способами:

  1. В главном меню системы щелкните по пункту Сотрудники. На экран будет выведен список всех зарегистрированных в системе сотрудников.

    В списке выберите тех сотрудников, учетные записи которых будут заблокированы, установив флажки, расположенные рядом с именами сотрудников.

    Щелкните по кнопке Заблокировать. Откроется окно, в котором необходимо указать срок и причину блокировки, и снова нажать на кнопку Заблокировать.

  2. Также блокировку сотрудника можно выполнить из анкеты сотрудников, для этого нужно открыть анкету выбранного сотрудника и нажать на кнопку Заблокировать.

Разблокировка учетных записей выполняется точно также, только отмечать надо заблокированные учетные записи и щелкнуть по кнопке Снять блокировку.

Заблокированных сотрудников в списке можно отфильтровать с помощью Фильтра/поиска сотрудников. Для этого в фильтре надо задать статус сотрудника "Заблокирован", и система покажет в списке только заблокированных сотрудников банка.

Визуально отличить заблокированного сотрудника можно по пиктограмме , которая появляется рядом с фамилией заблокированного сотрудника банка.
