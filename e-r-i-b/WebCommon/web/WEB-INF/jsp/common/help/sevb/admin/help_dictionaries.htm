В системе предусмотрены справочники, которые необходимы для работы с системой ИКФЛ «Сбербанк ОнЛ@йн».

Справочники делятся на два вида: общесистемные и справочники, относящиеся к конкретному подразделению. К общесистемным справочникам относятся справочник банков, валют, видов валютных операций, кодов КБК, они облегчают заполнение документов и позволяют избежать возникновения опечаток в реквизитах платежных документов. К частным справочникам относится справочник видов вкладов, доступных в ИКФЛ «Сбербанк ОнЛ@йн». Общесистемные справочники загружаются в систему «Сбербанк ОнЛ@йн» из RS-Retail ОПЕРУ, а частные – из системы RS-Retail филиалов.

Справочник внешних систем заполняется администратором системы вручную при первом входе в систему. Порядок создания внешних систем смотрите в разделе контекстной справки Справочник внешних систем.

Справочник банков

Справочник банков содержит реквизиты банков получателей, облегчает заполнение документов и позволяет избежать возникновения опечаток в реквизитах платежных документов. Справочник банков автоматически заполняется при установке системы.

Для каждого банка в справочнике хранятся его код БИК, наименование, номер корреспондентского счета и краткое наименование банка. При выборе банка из справочника все эти сведения автоматически подставляются в документ. Справочник банков предоставляется уже заполненным, но Вы можете добавлять в него новые записи или редактировать информацию о банках.

Создание банка

Чтобы добавить в список реквизиты нового банка, щелкните по кнопке Добавить, откроется страница редактирования сведений о банке, на которой заполните следующие поля: в поле "Наименование" впишите название банка, в поле "БИК" укажите банковский идентификационный код, в поле "Корр. счет" введите корреспондентский счет банка, при необходимости заполните остальные поля и щелкните по кнопке Сохранить . В результате созданный Вами банк появится в списке банков.

Редактирование банка

Для редактирования сведений о банке установите напротив него галочку и щелкните по кнопке Редактировать или щелкните на наименовании банка, откроется страница редактирования, на которой внесите необходимые изменения и щелкните по кнопке Сохранить .

Удаление банка

Если Вы хотите удалить из списка банк или несколько банков, установите напротив него/них галочку и щелкните по кнопке Удалить .

Для поиска интересующего Вас банка в системе предусмотрен фильтр.

Для того чтобы воспользоваться фильтром, щелкните по кнопке Показать фильтр и задайте критерии поиска:

  • "Наименование" - поиск по названию банка;

  • "БИК" - поиск по банковскому идентификационному коду:

  • "Город" - поиск по названию города, в котором зарегистрирован банк.

Вы можете фильтровать список по любой комбинации полей. Например, можно отобрать все банки, название которых начинается с буквы "А", и из одного города.

После того как параметры поиска заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас банков.

Справочник внешних систем

Справочник внешних систем содержит подробную информацию о копиях САРБУ RS-Retail филиалов банка («базах» RS-Retail филиалов). Для каждой внешней системы создается шлюз для взаимодействия с ней, который представляет собой отдельное приложение на сервере.

Создание внешней системы

Для того чтобы добавить внешнюю систему, щелкните по кнопке Добавить, откроется страница редактирования, на которой заполните следующие поля:

  • В поле «Название» введите название внешней системы;

  • В поле «Адрес шлюза» укажите адрес шлюза для взаимодействия внешней системы (RS-Retail) с системой «Сбербанк ОнЛ@йн». Например, http://phizic-server:8888/PhizGate63/services, где phizic-server – имя или IP-адрес сервера, на котором развернуто приложение шлюза, 8888 – порт приложения, который формируется при установке сервера приложений, PhizGate63 – название шлюза (наименование, которое указывается при заполнении поля Application Name перед запуском приложения шлюза).

    Примечание: подробнее о настройках адреса шлюза смотрите в руководстве «Интернет-клиент для физических лиц. Установка и настройка».

    Поле «Идентификатор» заполнится автоматически и будет недоступно для редактирования.

  • В поле «Номер агентской карты «Город»» укажите номер карты данного филиала банка для взаимодействия «Сбербанк ОнЛ@йн» с системой ИСППН «Город».

  • В поле «Пароль карты» впишите пароль карты «Город».

После того как все поля заполнены, щелкните по кнопке Сохранить .

Фильтр/поиск внешних систем

Для поиска интересующих Вас внешних систем можно воспользоваться фильтром. Для этого щелкните по кнопке Показать фильтр и задайте критерии поиска:

  • "ID" - поиск по идентификатору внешней системы в системе «Сбербанк ОнЛ@йн»;

  • "Идентификатор" - поиск по уникальному идентификатору внешней системы;

  • "Наименование" - поиск по наименованию шлюза:

  • "Адрес шлюза" - поиск по адресу шлюза.

Вы можете осуществлять поиск по подстроке в текстовых полях фильтра. Например, можно ввести в поля фильтра только несколько первых букв наименования внешней системы.

Можно отфильтровать список по любой комбинации полей. Например, можно отфильтровать все внешние системы по наименованию и адресу шлюза.

После того как критерии поиска заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас внешних систем.

Редактирование внешней системы

Для замены шлюза необходимо остановить приложение шлюза на сервере приложений, установить новую версию и запустить шлюз снова.

Чтобы изменить сведения о внешней системе, установите флажок напротив выбранной внешней системы и щелкните по кнопке Редактировать . Откроется страница для редактирования, на которой внесите необходимые изменения и щелкните по кнопке Сохранить .

Удаление внешней системы

Для того чтобы удалить внешнюю систему, установите флажок напротив выбранной внешней системы и щелкните по кнопке Удалить .

Примечание: если с внешней системой связаны подразделения, то удаление данной внешней системы невозможно.

Загрузка справочников

Данное меню предназначено для загрузки в систему АС «Сбербанк ОнЛ@йн» из системы RS-Retail справочника валют, справочника банков, справочника видов валютных операций, совершаемых физическими лицами, и справочника КБК.

Справочник банков содержит реквизиты банков. Справочник видов валютных операций содержит коды видов валютных операций, совершаемых физическими лицами. Справочник КБК используется при заполнении печатных форм налоговых документов.

Чтобы загрузить справочники, необходимо выделить галочкой нужный справочник или несколько справочников и щелкнуть по кнопке Загрузить .
