Список депозитных продуктов

Пункт меню Депозиты предназначен для управления депозитными продуктами банка, которые будут доступны для клиентов в системе ИКФЛ «Сбербанк ОнЛ@йн». При выборе данного пункта меню для каждого подразделения, доступного сотруднику банка, открывается список депозитных продуктов банка.

В списке депозитных продуктов по каждому из них показывается следующая информация:

  • Наименование вклада. Наименование депозитного продукта банка.

  • Сумма вклада. Отображается перечень диапазонов суммы вклада в зависимости от валюты и срока размещения средств.

  • Валюта вклада. Отображается список валют, в которых данный депозитный продукт предоставляется клиентам банка.

  • Срок вклада. Отображается перечень возможных сроков вклада для данного депозитного продукта.

  • Процентная ставка по вкладу. Отображается процентная ставка (% годовых) для данного депозитного продукта (вида вклада) в зависимости от валюты, срока и суммы размещаемых денежных средств.

Фильтр/поиск вкладов

На странице со списком депозитных продуктов банка Вы можете воспользоваться фильтром/поиском вкладов банка. Для этого щелкните по кнопке Показать фильтр и задайте критерии поиска.

В поле "Подразделение" щелкните по кнопке (...), откроется окно со справочником подразделений "Сбербанк ОнЛ@йн", в котором установите флажок напротив интересующего Вас подразделения и щелкните по кнопке Выбрать .

После того как критерии заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас вкладов банка.

Управление депозитными продуктами

Для работы со списком депозитных продуктов банка Вам необходимо их завести в системе ИКФЛ «Сбербанк ОнЛ@йн». Существует два способа заведения депозитных продуктов:

  1. Автоматически, т.е. можно реплицировать депозитные продукты из системы RS-Retail. Для этого щелкните по кнопке Репликация , откроется страница со справочником подразделений «Сбербанк ОнЛ@йн», доступных данному сотруднику. На этой странице выберите из справочника подразделения, для которых необходимо реплицировать список вкладов, и нажмите на кнопку Реплицировать . В результате в систему ИКФЛ «Сбербанк ОнЛ@йн» реплицируется из системы RS-Retail информация о всех вкладах банка для указанных подразделений.

    Примечание: В систему ИКФЛ "Сбербанк ОнЛ@йн" из САРБУ RS-Retail загружаются только виды вкладов с установленным пользовательским типом, например, «I» (Интернет), т.е. вклады, для которых разрешена операция безналичного открытия. Карточные виды вкладов, виды вкладов со спец. доступом, а также с установленным пользовательским типом «Щ» (счет типа «Т») не загружаются в систему.

  2. Ручное добавление депозитных продуктов, т.е. Вы самостоятельно вводите все депозитные продукты банка при помощи кнопки Добавить . Вы сами определяете, какие виды вкладов входят в состав депозитных продуктов банка.

В списке Вы можете управлять депозитными продуктами банка следующим образом:

  • Вы можете добавить в список депозитный продукт. Для этого щелкните по кнопке Добавить , откроется страница со справочником подразделений «Сбербанк ОнЛ@йн», доступных данному сотруднику. На этой странице выберите из справочника подразделение, для которого хотите создать депозитный продукт, и щелкните по кнопке Выбрать . Откроется страница редактирования депозитного продукта, на которой в поле "Имя депозитного продукта" впишите название создаваемого депозитного продукта, в поле «Описание депозитного продукта» укажите его описание. В поле «Подразделение» отображается информация о подразделении, для которого Вы создаете депозитный продукт, оно недоступно для редактирования.

    Ниже в форме редактирования депозитных продуктов содержится список вкладов, в котором по каждому вкладу в системе отображается идентификатор (уникальное имя депозитного продукта в системе), название (название депозитного продукта), валюта (валюта, в которой производится вклад). Вам необходимо указать перечень видов вкладов, входящих в состав создаваемого депозитного продукта, установив флажки напротив интересующих Вас видов вкладов. После заполнения формы щелкните по кнопке Сохранить , и новый депозитный продукт появится в списке депозитных продуктов банка.

  • Если Вам необходимо отредактировать депозитный продукт банка, в списке продуктов выберите интересующий Вас депозитный продукт, установите напротив него флажок и щелкните по кнопке Редактировать . Откроется страница редактирования депозитного продукта банка, на которой Вы можете изменить название продукта и перечень видов вкладов, входящих в состав депозитного продукта, затем щелкните по кнопке Cохранить , и депозитный продукт появится в списке в измененном виде.

  • Если Вам нужно удалить депозитный продукт банка из списка, установите флажок напротив выбранного депозитного продукта и щелкните по кнопке Удалить .
