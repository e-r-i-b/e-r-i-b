Назначение заявки

Данный пункт меню предназначен для подачи в банк заявки об утрате сберегательной книжки.

Для того чтобы подать в банк заявку, щелкните по ссылке Заявка об утрате сберегательной книжки или на иконке заявки, откроется страница с формой создания заявки. Создать заявку можно также с главной страницы, установив галочку напротив интересующего Вас счета и щелкнув по кнопке Заблокировать счет.

Заполнение заявки

В форме заявки в поле "вклад" выберите из списка номер вклада, на который была оформлена сберегательная книжка. Далее Вы можете выбрать:

  • если Вы хотите расторгнуть договор и закрыть счет, то установите флажок напротив строки "Расторгнуть договор с закрытием счета по вкладу". Остаток денежных средств на данном счете Вы можете либо получить наличными, выбрав соответствующее значение в списке, либо указать в списке счет, на который нужно перечислить оставшиеся средства.

  • если Вы хотите получить дубликат сберегательной книжки, то установите флажок напротив строки "Выдать дубликат сберегательной книжки".

После того как все поля заполнены, щелкните по кнопке Сохранить. Если Вы хотите вернуться в список заявок без сохранения изменений, щелкните по кнопке Список. Если Вам нужно вернуться на страницу видов платежей, нажмите кнопку Платежи.

Подтверждение заявки

После сохранения откроется страница подтверждения заявки. Вам нужно проверить правильность заполнения всех полей и нажать кнопку Подтвердить. После подтверждения на форме заявки появится штамп "Принято к исполнению", означающий, что заявка отправлена на обработку в банк.

Для того чтобы отредактировать созданную заявку об утрате сберегательной книжки, щелкните по кнопке Редактировать. Откроется страница редактирования заявки, на которой можно изменить необходимые сведения и щелкнуть по кнопке Сохранить.

На данной странице Вы можете создать новую заявку об утрате сберегательной книжки, нажав кнопку Создать, в результате Вы перейдете на страницу создания заявки.

Если Вы хотите перейти в список заявок без сохранения изменений, щелкните по кнопке Список.

Для того чтобы вернуться на страницу видов платежей щелкните по кнопке Платежи.

Список заявок

Фильтр заявок

В список заявок об утрате сберегательной книжки можно зайти со страницы создания заявки, щелкнув по кнопке Список, или из пункта главного меню Платежи, щелкните по ссылке История операций на изображении заявки об утрате сберегательной книжки.

В данном списке Вы можете посмотреть все созданные заявки об утрате сберегательной книжки. По каждой заявке в списке отображается дата создания заявки, номер заявки, номер вклада и статус операции.

Вы можете выполнить следующие операции:

  • Просмотр документа

    Вы можете просмотреть любой документ, для этого нужно щелкнуть по номеру интересующего Вас документа в списке. Вы можете отправить в банк заявки со статусом "Введена". Для этого щелкните по номеру счета, откроется страница подтверждения, на которой нажмите кнопку Подтвердить.

  • Создание копии

    Вы можете создать копию заявки, щелкнув по кнопке Создать копию . В результате Вы перейдете к форме создания копии документа. Данная возможность полезна при создании нескольких однотипных заявок.

Фильтр заявок

Вы можете воспользоваться фильтром заявок.

Для того чтобы найти интересующую Вас заявку в списке, щелкните по кнопке Показать фильтр и задайте критерии поиска:

  • "Статус" - поиск по статусу заявки, например, "Введена" (заявка создана, но не отправлена в банк), "Обрабатывается" (заявка отправлена в банк на обработку), "Исполнена" (заявка исполнена), "Отказана" (отказано в выполнении операций по заявке).

  • "Период" - поиск по времени создания заявки;

  • "Номер вклада" - поиск по номеру вклада, по которому создана заявка.

Вы можете фильтровать список по любой комбинации полей. Например, можно отобрать все документы, созданные в один день со статусом "Исполнен".

После того как параметры поиска заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас заявок.
