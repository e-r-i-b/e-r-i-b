Платежи

Данная страница предназначена для выбора платежной операции, которую Вы хотите совершить. Платежи объединены в несколько групп: платежные операции, операции по обмену валюты и печать платежного документа. Также с помощью данного пункта меню можно подать в банк различные виды заявок.

Для того чтобы перейти к заполнению формы платежа, Вам необходимо щелкнуть по значку платежа либо по его названию.

По каждому платежу/заявке можно просмотреть список документов, которые Вы создали в системе "Сбербанк ОнЛ@йн" по данному виду платежа/заявки, для этого нажмите кнопку История операций.
