Список клиентов

Фильтр/поиск клиентов

Пункт меню Клиенты предназначен для управления пользователями системы "Сбербанк ОнЛ@йн". Он позволяет регистрировать новых клиентов, удалять, блокировать и разблокировать клиентов, управлять доступом клиентов к счетам и картам, назначать клиентам схемы прав, определяющие доступные им операции.

В данном пункте меню автоматически открывается список всех клиентов, закрепленных за определенным подразделением банка. Для каждого клиента в списке отображается следующая информация: фамилия, имя, отчество, номер договора, схема пра доступа к операциям в системе и статус клиента.

Добавление клиента

Для того чтобы добавить в список нового клиента банка, щелкните по кнопке Добавить , откроется анкета клиента. Процедура регистрации клиентов банка описана в разделе контекстной справки "Регистрация клиентов".

Редактирование клиента

В случае, если изменились сведения о клиенте или нужно подключить новые счета пользователя, Вы можете отредактировать сведения о клиенте. Для этого щелкните на имени клиента, откроется анкета клиента банка, внесите изменения на интересующих Вас вкладках анкеты и нажмите кнопку Сохранить . Затем распечатайте Дополнительное соглашение, щелкните по кнопке Печать дополнительного соглашения, нажмите кнопку Зарегистрировать клиента , для того чтобы изменения к Договору были приняты системой. Если возникли проблемы, например, на счете, с которого будет взиматься плата за предоставление услуги, недостаточно средств, система покажет соответствующее сообщение. Вам необходимо исправить все неточности, снова сохранить изменения и нажать кнопку Зарегистрировать клиента .

В списке клиентов банка Вы можете изменить схему прав доступа к операциям в системе для любого клиента. Для этого установите галочку напротив интересующего Вас клиента, в поле "Схема прав" выберите из списка схему прав доступа и щелкните по кнопке Назначить права .

Удаление клиента/расторжение договора

Вы можете расторгнуть договор с клиентом, который имеет статус "Активный", это можно сделать из списка клиентов банка с помощью кнопки Удалить и из анкеты клиента.

Чтобы расторгнуть договор с данным клиентом на предоставление услуг с использованием системы "Сбербанк ОнЛ@йн", в анкете клиента банка введите в поле "Дата расторжения договора" дату расторжения, в поле "Причина расторжения договора" выберите причину из списка и щелкните по кнопке Сохранить . Затем щелкните по кнопке Зарегистрировать клиента для внесения изменений в договор, распечатайте заявление о расторжении договора, щелкнув по кнопке Расторжение договора. Далее нажмите кнопку Удалить . В результате клиент будет удален из списка клиентов банка и появится в списке архива клиентов банка.

Вы можете полностью удалить из системы клиента, имеющего статус "Подписание договора" и "Подключение". Для этого установите галочку напротив интересующего Вас клиента и щелкните по кнопке Удалить .

Фильтр/поиск клиентов

Для поиска интересующего Вас клиента в системе предусмотрен фильтр. Чтобы воспользоваться фильтром, щелкните по кнопке Показать фильтр и задайте критерии поиска:

  • "ФИО клиента" - поиск по фамилии, имени и отчеству клиента;

  • "Вид документа" - поиск по виду документа, удостоверяющего личность клиента;

  • "Серия"- поиск по серии паспорта клиента банка;

  • "Номер" - поиск по номеру паспорта клиента;

  • "Логин" - поиск по логину клиента;

  • "Номер договора" - поиск по номеру договора клиента банка;

  • "Статус" - поиск по статусу клиента банка в системе: "Активный" (клиент имеет доступ к системе ИКФЛ "Сбербанк ОнЛ@йн"), "Заблокирован" (клиент заблокирован по какой-либо причине), "Подключение" (клиент заведен в системе, с ним подписан договор на предоставление услуг, но он не подключен), "Подписание договора" (клиент заведен в системе ИКФЛ "Сбербанк ОнЛ@йн", но с ним еще не подписан договор).

Вы можете осуществлять поиск по подстроке в текстовых полях фильтра. Например, можно ввести в поля фильтра только несколько первых букв фамилии, имени или отчества клиента.

Также Вы можете фильтровать список по любой комбинации полей. Например, можно отобрать всех клиентов с фамилией на букву "А" с паспортом.

После того как параметры фильтра заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас клиентов банка.

Подключение клиента

Чтобы зарегистрировать нового клиента Вы должны выполнить следующие действия:

  1. В главном меню системы щелкните по пункту Клиенты. На экран будет выведен список уже зарегистрированных в системе клиентов.

  2. Щелкните по кнопке Добавить. На экран будет выведен список клиентов, зарегистрированных в системе автоматизации розничных банковских услуг. В данном списке Вы можете осуществить поиск клиентов банка, сведения о которых хранятся в системе RS-Retail.

    Установите флажок напротив клиента, информацию о котором Вы хотите импортировать и щелкните по кнопке Импортировать .

  3. Откроется анкета импортированного клиента, которому система автоматически присвоит статус "Подписание договора". В анкете обязательны для заполнения все поля, отмеченные звездочкой.

  4. Затем Вам необходимо последовательно заполнить все разделы анкеты клиента банка:

    5. Общие сведения
    Счета и карты
    Права доступа
    Представители
    Получатели (физ.лица)
    Получатели (юр.лица)

    Детальную информацию по данным вкладкам анкеты клиента банка смотрите ниже в соответствующих разделах контекстной справки.

  5. Далее Вам нужно подключить клиента к системе. Для этого перейдите на вкладку анкеты клиента Общие сведения и выполните следующие действия:

  • Распечатайте документы для подключения клиента к системе.

    Примечание: чтобы распечатать документы, Вам необходимо настроить параметры печати один раз перед первой печатью. Для этого в верхнем меню щелкните по кнопке Файл, откройте пункт "Параметры страницы", выберите вкладку "Поля и колонтитулы" для отступов полей задайте значение "0", уберите все колонтитулы, сохраните изменения.

    Далее нажмите кнопку Печать анкеты, распечатайте анкету клиента банка; щелкните по кнопке Печать договора и распечатайте Договор для предоставления услуг системы "Сбербанк ОнЛ@йн" и все приложения к нему, Дополнительное соглашение на открытие счета по вкладу и др.

  • Проверьте клиента по АС "Стоп-лист" на включение его в "блокирующие" списки. Для этого нажмите кнопку Стоп-лист , чтобы отправить запрос на проверку клиента. Если АС "Стоп-лист" даст в качестве ответа информацию о том, что клиент содержится в каком-либо из "блокирующих" списков, то система "Сбербанк ОнЛ@йн" не разрешит подключить клиента, если клиент содержится в предупредительном списке, то подключение такого клиента осуществляется на усмотрение сотрудника банка. Если проверка прошла успешно, то Вы можете зарегистрировать клиента в системе.

  • После подписания с клиентом договора о предоставлении услуг с использованием системы ИКФЛ "Сбербанк ОнЛ@йн" во вкладке анкеты клиента Общие сведения щелкните по кнопке Договор подписан . Система попросит Вас подтвердить операцию, и после подтверждения Вы сможете подключить клиента. После подтверждения клиенту будет присвоен статус «Подключение», и Вы сможете подключить клиента. Если возникли какие-либо изменения в анкете клиента, можно нажать на кнопку Редактировать , и Вы вернетесь на стадию подписания договора. Далее внесите все необходимые изменения, снова щелкните по кнопке Договор подписан и подтвердите операцию.

  • Подключите клиента к системе, щелкнув по кнопке Подключить клиента , если все поля анкеты заполнены верно, клиенту будет присвоен статус "Активный"; если не выполнены какие-либо действия или не заполнены поля, система покажет сообщение об этом, выполните требования системы и снова нажмите кнопку Подключить клиента .

Общие сведения

В данной вкладке анкеты вводятся сведения о клиенте. Основная информация о клиенте Вы импортируется из САРБУ RS-Retail. Остальные сведения введите самостоятельно. Обязательны для заполнения все поля, отмеченные звездочкой: ФИО, дата рождения, гражданство, вид документа, удостоверяющего личность клиента, серия, номер, дата выдачи, кем выдан документ, номер мобильного телефона, мобильный оператор, кодовое слово и дата начала обслуживания клиента.

Вы имеете возможность сохранить частично заполненную анкету. Для этого щелкните по кнопке Сохранить .

Если Вы хотите выйти из анкеты клиента, щелкните по кнопке Список.

Счета и карты

Помимо персональной информации для каждого клиента необходимо создать список счетов и пластиковых карт, к которым он будет иметь доступ. Также Вам необходимо указать, с какого счета (счетов) будет взиматься ежемесячная абонентская плата за использование услуг "Сбербанк ОнЛ@йн".

Чтобы добавить сведения о счетах и картах клиента банка, выполните следующие действия:

  • В анкете клиента выберите пункт Счета и карты, на экран будет выведен список счетов и карт, к которым клиент имеет доступ. Для того чтобы добавить счет/карту, щелкните по кнопке Добавить счет (для добавления карты нажмите на кнопку Добавить карту), установите флажок напротив выбранного счета/карты и щелкните по кнопке Добавить. В результате счет/карта отобразится в списке счетов и карт клиента.

  • Для указания счета (счетов), с которого будут сниматься денежные средства за предоставление услуг, установите галочки в графе "оплата" напротив интересующих Вас рублевых счетов и щелкните по кнопке Сохранить .

    Примечание: оплата услуги ИКФЛ "Сбербанк ОнЛ@йн" осуществляется только с рублевых счетов клиента банка.

  • Если Вы хотите удалить счет или карту клиента из списка, установите галочку напротив счета (счетов) и щелкните по кнопке Удалить , система попросит Вас подтвердить операцию, после подтверждения счет будет удален из списка.

Права доступа

В данной вкладке анкеты Вам необходимо задать схему прав доступа клиента банка к операциям в системе. Схема прав доступа определяет перечень операций, доступных клиенту при входе в систему.

Для того чтобы задать клиенту определенную схему прав, выполните следующие действия:

  1. Установите галочку в поле "Политика доступа клиентов в "Сбербанк ОнЛ@йн"".

  2. В поле "Подтверждение входа" выберите из списка, каким образом данный клиент сможет подтвердить вход в систему ИКФЛ "Сбербанк ОнЛ@йн":

    • без подтверждения (только с помощью логина и пароля);

    • с подтверждением SMS-паролем.

      Примечание: SMS-пароль - это пароль, который приходит клиенту банка в виде sms на номер, указанный в анкете. Если Вы назначите клиенту подтверждение входа SMS-паролем, то данному клиенту необходимо будет ввести на странице регистрации логин, нажать на кнопку Продолжить, затем ввести пароль, полученный при первом входе, и щелкнуть по кнопке Продолжить. Система ИКФЛ сформирует одноразовый SMS-пароль и отправит его клиенту на номер мобильного телефона, указанный в анкете. В результате откроется специальная страница, на которой клиенту необходимо ввести одноразовый SMS-пароль.

  3. В поле "Подтверждение операций" укажите, каким способом в системе будет подтверждаться совершение операций клиентом банка:

    • с помощью SMS-пароля;

      Примечание: если Вы выбрали данный способ подтверждения операций в системе, то после сохранения расчетного документа на странице подтверждения внизу отображается текст «Проверьте правильность заполнения полей платежа. Для того чтобы отправить в банк платеж на исполнение необходимо ввести SMS-пароль». Для получения SMS-пароля клиенту необходимо нажать на кнопку Получить SMS-пароль. Система сформирует и отправит на мобильный телефон клиента SMS-пароль для подтверждения операции. Затем клиенту на странице подтверждения необходимо ввести полученный SMS-пароль и нажать на кнопку Подтвердить. Система проверит правильность SMS-пароля. Если SMS-пароль введен верно, документ будет принят к исполнению.

    • с помощью АСП:

      Примечание: АСП (аналог собственноручной подписи клиента) может быть использована для разрешения конфликтных ситуаций с клиентами. С помощью проверки АСП можно просмотреть по каждой платежной операции, совершенной клиентом, какие поля были подписаны, и проконтролировать совпадение АСП.

  4. В поле "Схема прав" выберите из списка интересующую Вас схему доступа или задайте индивидуальные права, установив флажки у тех операций, которые будет осуществлять данный клиент, и щелкните по кнопке Сохранить .

Если Вы хотите вернуться в список клиентов банка, щелкните по кнопке Список .

Список представителей

Во вкладке анкеты Представители Вы можете просмотреть, добавить, изменить и удалить по желанию клиента представителей, которые будут иметь доступ к выбранным счетам, картам и операциям клиента в системе.

Если Вам нужно просмотреть анкету с подробной информацией о представителе клиента, кликните на его имени, и откроется анкета представителя.

Регистрация представителя клиента

Вы можете подключить нового представителя клиенту банка во вкладке анкеты клиента Представители, щелкнув по кнопке Добавить.

Откроется анкета создания нового представителя, в которой Вам необходимо заполнить все поля, отмеченные звездочкой: ФИО, дату рождения, гражданство, серию и номер документа, удостоверяющего личность представителя клиента, дату выдачи, кем выдан документ, номер мобильного телефона и дату начала обслуживания.

Вам нужно выбрать доступные представителю счета и карты клиента банка, для этого поставьте флажок напротив пункта Перечень доступных счетов или Перечень доступных карт.

Вам необходимо также указать список доступных данному представителю операций в системе:

  1. Установите галочку в поле "Политика доступа клиентов в "Сбербанк ОнЛ@йн"".

  2. В поле "Подтверждение входа" выберите из списка, каким образом данный представитель клиента сможет подтвердить вход в систему ИКФЛ "Сбербанк ОнЛ@йн":

    • без подтверждения (только с помощью логина и пароля);

    • с подтверждением SMS-паролем.

      Примечание: SMS-пароль – это одноразовый пароль, формируемый системой ИКФЛ "Сбербанк ОнЛ@йн". Если Вы назначите представителю клиента подтверждение входа SMS-паролем, то ему необходимо будет ввести на странице регистрации логин, нажать на кнопку Продолжить, ввести пароль, полученный при первом входе, и нажать кнопку Продолжить. Система ИКФЛ сформирует одноразовый SMS-пароль и отправит его представителю клиента на номер мобильного телефона, указанный в анкете. В результате откроется специальная страница, на которой представителю клиента необходимо ввести одноразовый SMS-пароль.

  3. В поле "Подтверждение операций" укажите, каким способом в системе будет подтверждаться совершение операций клиентом банка:

    • с помощью SMS-пароля;

    • с помощью АСП.

  4. Затем установите флажки у тех операций, которые будет осуществлять данный пользователь, и щелкните по кнопке Сохранить .

Примечание: должен быть разрешен доступ представителя клиента минимум к одному счету и одной операции в системе.

Затем распечатайте анкету представителя для проверки информации о представителе клиента, для этого нажмите кнопку Печать анкеты.

Чтобы активировать представителя клиента банка, представитель должен придти в отделение банка, подписать все необходимые документы, после чего Вы в анкете представителя щелкните по кнопке Подключить представителя . Если все обязательные поля анкеты заполнены правильно, то представителю клиента система присвоит статус "Активный", и он получит доступ в систему "Сбербанк ОнЛ@йн".

Если Вы хотите вернуться в список представителей клиента, щелкните по кнопке Список представителей .

Редактирование сведений о представителе

Для того чтобы отредактировать сведения о представителе, установите галочку напротив интересующего Вас представителя клиента и нажмите на кнопку Изменить или щелкните по фамилии, имени, отчеству представителя клиента. Откроется анкета представителя, в которой отредактируйте значения нужных полей и щелкните по кнопке Сохранить .

Удаление представителя

Для того чтобы удалить из списка представителя клиента, установите галочку напротив интересующего Вас представителя и щелкните по кнопке Удалить , затем подтвердите удаление, и представитель клиента будет удален из системы.

Блокировка представителя

Если Вам необходимо заблокировать представителя, щелкните на имени представителя, откроется анкета представителя клиента, затем нажмите кнопку Заблокировать , откроется окно, в котором укажите срок и причину блокировки, снова щелкните по кнопке Заблокировать , и представитель будет заблокирован. Система присвоит такому представителя статус "Заблокирован", и рядом с его именем появится пиктограмма . Разблокировка представителя клиента осуществляется аналогично, только нужно щелкнуть по кнопке Снять блокировку и подтвердить операцию.

Удаление получателя

Получатели (юр. лица)

Пункт меню Получатели (юр. лица) используется, для того чтобы завести получателей, в адрес которых клиент сможет совершать платежные операции.

В системе ИКФЛ "Сбербанк ОнЛ@йн" предусмотрены два вида получателей платежей:

  1. Получатели (недоговорные) – юридические лица с заданными банковскими реквизитами (организации, с которыми не заключен договор Сбербанком России). Реквизиты данных получателей задает клиент при заключении договора на предоставление услуг системы "Сбербанк ОнЛ@йн" или вводит самостоятельно в Справочник получателей.

  2. Получатели (договорные) – юридические лица, с которыми заключен договор с системой ИСППН «Город».

Примечание: с помощью системы ИСППН «Город» можно оплачивать коммунальные и иные подобные платежи в адрес юридических лиц. В системе ИКФЛ "Сбербанк ОнЛ@йн" клиент банка через ИСППН «Город» может осуществлять оплату различных товаров и услуг. Перечень возможных видов платежей (вместе с их получателями) запрашивается из системы «Город» в реальном времени.

В данном пункте меню Вы можете добавить, изменить и удалить получателя первой группы. Для каждого получателя первого вида в списке отображаются следующие сведения: порядковый номер, обозначение, наименование, ИНН получателя, счет получателя, банк получателя, БИК, корр.счет.

Создание получателя

Добавление недоговорных и договорных получателей в системе осуществляется разными способами.

Добавление получателя, с которым не заключен договор

Для того чтобы добавить нового получателя, нажмите кнопку Добавить , откроется анкета получателя. В анкете заполните все поля со звездочкой. В поле "Обозначение" введите обозначение получателя, которое будет отображаться в форме создания платежа, в поле "Наименование" впишите название получателя (организации), в поле "ИНН" укажите идентификационный номер налогоплательщика, в поле "Счет" введите счет получателя, далее заполните реквизиты банка получателя платежей: в поле "Наименование" щелкните по кнопке (...), откроется страница со списком банков получателей.

На данной странице Вы можете воспользоваться фильтром.

Фильтр банков получателей

Для того чтобы воспользоваться фильтром, задайте в полях фильтра критерии поиска:

  • "Наименование" - наименование банка получателя;

  • "БИК" - банковский идентификационный код банка получателя;

  • "Город" - название города, в котором зарегистрирован банк получателя.

    Вы можете осуществлять поиск по подстроке в текстовых полях фильтра. Например, можно ввести только несколько первых букв названия банка получателя.

    Можно фильтровать список по любой комбинации полей.

    После того как параметры фильтра заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас банков получателей.

Для того чтобы ввести реквизиты банка получателя, установите галочку напротив интересующего Вас банка и щелкните по кнопке Выбрать , поля "БИК" (банковский идентификационный код банка получателя), "Корр.счет"(корреспондентский счет банка получателя) будут заполнены автоматически. После того как все поля заполнены, щелкните по кнопке Сохранить , в результате новый получатель появится в списке.

Если Вам нужно вернуться на страницу со списком получателей, щелкните по кнопке Список получателей , и система вернет Вас на страницу списка получателей.

Редактирование получателя

Если Вам необходимо изменить реквизиты получателя, щелкните на наименовании получателя или установите галочку напротив интересующего Вас получателя и щелкните по кнопке Изменить . Откроется форма для редактирования, на которой измените значения в интересующих Вас полях, и щелкните по кнопке Сохранить .

Удаление получателя

Если Вы хотите удалить получателя из списка, установите галочку напротив интересующего Вас получателя и щелкните по кнопке Удалить .

Добавление получателей, с которыми заключен договор

Для добавления получателя, у которого заключен договор с системой ИСППН «Город» выполните следующие действия:

  1. Во вкладке анкеты клиента Получатели (юр. лица) прикрепите клиенту карту «Город». Для привязки к системе ИКФЛ "Сбербанк ОнЛ@йн" карты «Город» введите номер карты в поле «номер карты Город (ПАН)» и нажмите на кнопку Добавить карту .

  2. Откроется страница с информацией по данной карте, на которой щелкните по кнопке Сохранить .

В результате карта внешней системы ИСППН «Город» будет добавлена в список получателей. В списке получателей по карте отображается номер карты «Город» (ПАН), данные о владельце и адрес владельца.

Для того чтобы вернуться на страницу со списком получателей, щелкните по кнопке Список , и система вернет Вас на страницу списка получателей.

Вы можете удалить карту «Город» из системы ИКФЛ "Сбербанк ОнЛ@йн", щелкнув по кнопке Удалить карту Города .

Журнал оплаты

С помощью пункта меню Журнал оплаты Вы имеете возможность просмотреть все операции по списанию платы за обслуживание.

По каждой операции в журнале отображается следующая информация:

  • Дата и время совершения операции. Для неуспешных операций отображается дата последней попытки списания платы.

  • Номер документа. Отображается номер документа, который был ему присвоен при его создании.

  • Вид операции, например, «Единовременная» или «Абонентская».

  • Сумма операции.

  • Период оплаты. Отображаются даты начала и окончания периода обслуживания, за который взимается плата. Для единовременной платы отображается дата ее списания (дата подключения клиента или его представителя).

  • Статус операции, например, «Проведена», «Недостаточно средств».

  • Количество неудачных попыток оплаты. Выводится количество неуспешных попыток взимания оплаты.

Вы можете воспользоваться фильтром для поиска интересующих Вас операций. Для этого щелкните по кнопке Показать фильтр и задайте критерии поиска:

  • "Дата операции" - поиск по дате совершения операции;

  • "Статус операции" - поиск по статусу операции в системе;

  • "Максимальное количество записей" - количество записей, выводимых на странице.

Вы можете фильтровать список по любой комбинации полей.

После того как параметры фильтра заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас операций.

Размер платы за подключение и абонентской платы можно посмотреть и изменить в пункте главного меню Подразделения (поля «Тариф за организацию расчетного обслуживания» и «Ежемесячная плата»).

Блокировка и разблокировка клиентов

Учетные записи клиентов можно блокировать. Это позволяет запретить клиенту доступ в систему (например, из-за того, что данная услуга не была своевременно оплачена), не удаляя его учетную запись.

Блокировка выполняется следующим образом:

  1. В главном меню системы щелкните по пункту Клиенты. На экран будет выведен список всех зарегистрированных в системе клиентов.

  2. В списке выберите тех клиентов, учетные записи которых будут заблокированы, установив флажки, расположенные рядом с их именами.

  3. Щелкните по кнопке Заблокировать. Откроется окно, в котором необходимо указать срок и причину блокировки, и снова нажать на кнопку Заблокировать. Система присвоит такому клиенту статус "Заблокирован".

Разблокировка учетных записей выполняется точно также, только отмечаются заблокированные учетные записи и нажимается кнопка Снять блокировку.

Заблокированных клиентов в списке можно отфильтровать с помощью Фильтра/поиска клиентов. Для этого в фильтре надо задать статус клиента "Заблокирован", и система покажет в списке только заблокированных клиентов.

Визуально отличить заблокированного клиента можно по пиктограмме , которая появляется рядом с фамилией заблокированного клиента банка.
