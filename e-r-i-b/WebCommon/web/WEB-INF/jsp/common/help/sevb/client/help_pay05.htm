Назначение платежа

Данный документ позволяет Вам продавать иностранную валюту за рубли через систему "Сбербанк ОнЛ@йн". Для того чтобы создать платеж в главном меню выберите пункт Платежи, а в открывшемся списке пункт Продажа иностранной валюты.

Заполнение документа

Для продажи иностранной валюты за рубли в форме платежа заполните следующие поля: в поле "Списать со счета" выберите из списка валютный счет, с которого будут сниматься денежные средства для продажи валюты, в поле "Зачислить на счет" выберите из списка рублевый счет для зачисления денежных средств, в поле "Курс банка покупки валюты" введите курс, по которому будет произведена продажа валюты. Также Вам нужно заполнить одно из полей:

  • если Вы хотите продать определенную сумму иностранной валюты, то в поле "Сумма в иностранной валюте" впишите размер суммы;

  • если Вы хотите продать валюту на определенную сумму рублей, то в поле "Сумма в рублях" введите размер суммы.

    Примечание: Вы можете заполнить только одно из данных полей.

После того как все поля заполнены, щелкните по кнопке Сохранить .

Если Вы решили не делать платеж, с данной страницы можно вернуться к списку выполненных Вами платежей по продаже иностранной валюты, щелкнув по кнопке Список , или на страницу выбора типа платежа, щелкнув по кнопке Платежи .

Подтверждение платежа

После сохранения заполненного документа Вы попадаете на страницу подтверждения. Здесь Вам будет показана заполненная форма платежа, в которой Вам нужно проверить правильность указанных сведений, после чего выполнить одно из следующих действий:

  • Подтверждение платежа

    Для подтверждения платежа щелкните по кнопке Подтвердить , чтобы документ был отправлен в банк на обработку. Отследить состояние выполненной операции Вы сможете по статусу платежа в списке платежей (см. раздел справки Список платежей по продаже иностранной валюты).

    Примечание: процедура подтверждения зависит от используемого Вами способа регистрации в системе. Например, если Вы используете SMS-пароль для подтверждения, то на странице подтверждения сначала для получения SMS-пароля щелкните по кнопке Получить пароль, затем введите пароль из полученного SMS-сообщения в соответствующее поле.

  • Редактирование платежа

    Если Вы хотите отредактировать созданный документ, щелкните по кнопке Редактировать , откроется страница для редактирования, на которой внесите необходимые изменения и щелкните по кнопке Сохранить .

  • Создание платежа

    На странице подтверждения Вы можете перейти к форме создания нового платежа, щелкнув по кнопке Создать.

  • Печать документа

    Если Вы хотите распечатать данный документ, щелкните по кнопке Печать .

  • Возврат в список платежей

    Вы можете вернуться на страницу со списком платежей по продаже иностранной валюты, щелкнув по кнопке Список .

Просмотр документа

После подтверждения платежа Вы попадаете на страницу просмотра, где отображается заполненный документ. О том, что документ передан в банк, свидетельствует отображаемый на форме документа штамп "Принято к исполнению". На страницу просмотра документа Вы можете также попасть из списка документов, щелкнув по номеру счета списания в списке. На странице просмотра Вы можете выполнить следующие действия:

  • Создание нового документа

    Вы можете создать еще один документ данного типа, щелкнув по кнопке Создать . В результате Вы перейдете к форме ввода нового документа. Данная возможность полезна при создании нескольких однотипных документов.

  • Отзыв документа

    Вы можете отозвать ошибочно отправленный документ. Для этого щелкните по кнопке Отозвать , в результате Вы перейдете к форме отзыва перевода. Информация в полях документа недоступна для редактирования. Чтобы отозвать перевод, щелкните по кнопке Сохранить .

  • Печать документа

    Если Вы хотите распечатать данный документ, щелкните по кнопке Печать .

  • Возврат к списку платежей

    Можно вернуться к списку выполненных Вами платежей по продаже иностранной валюты, щелкнув по кнопке Список .

Список платежей по продаже иностранной валюты

В данном списке показываются все выполненные Вами платежи по продаже иностранной валюты. Для каждого платежа отображаются дата, номер документа, номер счета списания, сумма списания, номер счета зачисления, сумма зачисления и статус, отражающий текущее состояние операции. Из этого списка Вы можете узнать, сколько средств было списано в валюте с Вашего счета при продаже валюты за определенную сумму рублей (в поле "Сумма списания") или сколько средств было зачислено на Ваш счет при продаже конкретной суммы валюты (в поле "Сумма зачисления"). На данной странице Вы можете выполнить следующие операции:

  • Просмотр документа

    Вы можете просмотреть любой документ, для этого нужно щелкнуть по номеру счета списания платежа.

  • Отзыв документа

    Вы можете отозвать ошибочно отправленный документ. Для этого щелкните по кнопке Отозвать , в результате Вы перейдете к форме отзыва перевода. Информация в полях документа недоступна для редактирования. Чтобы отозвать перевод, щелкните по кнопке Сохранить .

  • Создание копии документа

    Вы можете создать копию документа, отметив галочкой документ, который служит оригиналом в списке, и щелкнув по кнопке Создать копию . В результате Вы перейдете к форме ввода нового документа, причем в нее будут скопированы все сведения из документа, выбранного Вами в качестве оригинала. Данная возможность полезна при создании нескольких однотипных документов.

  • Удаление платежа

    Вы можете удалить из списка платеж со статусом "Введен", для этого установите галочку напротив интересующего документа и щелкните по кнопке Удалить .

  • Фильтр платежей

    Для того чтобы воспользоваться фильтром, щелкните по кнопке Показать фильтр и заполните поля фильтра:

    • "Статус" - поиск по статусу документа в системе: "Введен" (платеж создан, но не отправлен на обработку в банк), "Обрабатывается" (платеж подтвержден и отправлен на обработку в банк), "Исполнен" (операция выполнена), "Отказан" (Вам отказано в операции);

    • "Период" - поиск по времени и дате создания документа в системе;

    • "Сумма" - поиск по размеру суммы, на которую совершен платеж;

    • "Счет списания" - поиск по номеру счета, с которого переведены денежные средства;

    • "Счет зачисления" - поиск по номеру Вашего счета, на который переведены денежные средства.

    Вы можете фильтровать список по любой комбинации полей. Например, можно отобрать все документы, созданные в один день со статусом "Исполнен".

    После того как параметры поиска заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас платежей.
