Параметры системы

С помощью пункта меню Настройки задаются параметры системы. Вы можете задать настройки, определяющие функционирование системы.

Клиенты и сотрудники

Пункт меню Клиенты и сотрудники предназначен для управления настройками системы, которые регулируют работу клиентов и сотрудников банка в системе. В данном пункте меню Вы можете управлять следующими настройками системы:

  1. Разрешенные символы для пароля

    Данная настройка предназначена для управления символами, которые создаются в системе для ввода пароля. Чтобы изменить допустимые символы, используемые для создания паролей, в поле "Разрешенные символы для пароля" выберите из списка нужное значение и нажмите на кнопку Сохранить . Например, если Вы выбираете значение "Только цифры", то при создании пароля система генерирует пароль, состоящий только из цифр. После сохранения настройка будет изменена в системе и вступит в действие.

    Примечание: помимо настройки пароля, есть возможность контроля за качеством пароля в системе в конфигурационных файлах, т.е. администратор системы может задать более строгие критерии для создания пароля.

  2. Разрешенные символы для логина

    Данная настройка используется для изменения допустимых символов, используемых при создании логинов. Чтобы изменить разрешенные символы для логина, в поле "Разрешенные символы для логина" выберите из списка необходимое значение и нажмите на кнопку Сохранить . Например, если Вы выбираете значение "Цифры и заглавные буквы", то при создании логина система будет генерировать логины для сотрудников и клиентов, состоящие только из цифр и заглавных букв. После сохранения настройка будет изменена в системе и вступит в действие.

  3. Настройка длины пароля

    Данная настройка предназначена для определения, какой длины будут создаваемые системой пароли, и для общего ограничения длины паролей, вводимых клиентами и сотрудниками банка. Если Вы хотите изменить длину создаваемых паролей, введите количество допустимых символов в поле "Длина пароля" и щелкните по кнопке Сохранить . После сохранения настройка вступит в действие.

    Примечание: помимо настройки пароля, есть возможность контроля за качеством пароля в системе в конфигурационных файлах, т.е. администратор системы может задать более строгие критерии для создания пароля.

  4. Количество попыток при регистрации

    Данная настройка используется для изменения количества попыток для ввода пароля до автоматической блокировки доступа в систему. Чтобы изменить количество попыток, введите в поле "Количество попыток (регистрация)" число попыток и щелкните по кнопке Сохранить . После сохранения настройка вступит в действие.

  5. Период блокировки

    Данная настройка предназначена для изменения времени, на которое блокируется доступ к системе при многократном неверном вводе пароля. Для изменения периода блокировки введите в поле "Период блокировки" числовое значение (период блокировки исчисляется в секундах) и щелкните по кнопке Сохранить .

    Если Вы меняете только одну настройку, то после ее изменения нажимаете по кнопке Сохранить , если Вы меняете сразу несколько настроек, то Вы вводите изменение в каждое изменяемое поле и потом нажимаете Сохранить .

    Эти параметры действуют для всех зарегистрированных сотрудников и клиентов и вступают в силу сразу же после сохранения изменений.

Права доступа сотрудника

Пункт Права доступа сотрудника позволяет задать схему прав, которая автоматически назначается при создании нового сотрудника банка в системе.

В данном пункте меню в поле "Схема прав сотрудника" выберите из списка схему прав (для администраторов или для сотрудников банка), которая будет по умолчанию назначена сотрудникам банка, после чего щелкните по кнопке Сохранить . Настройка вступит в действие после сохранения.

Права доступа клиента

Настройка Права доступа клиента предназначена для указания способа подтверждения клиентами банка входа в систему, подтверждения операций, назначения схемы прав, задаваемых системой по умолчанию. Для изменения данной настройки выполните следующие действия:

  1. Установите галочку в поле "Политика доступа клиентов в "Сбербанк ОнЛ@йн"".

  2. В поле "Подтверждение входа" выберите из списка, каким образом данный клиент сможет подтвердить вход в систему ИКФЛ "Сбербанк ОнЛ@йн":

    • без подтверждения (только с помощью логина и пароля);

    • с подтверждением SMS-паролем.

    Примечание: SMS-пароль - это пароль, который приходит клиенту банка в виде SMS на номер, указанный в анкете.

  3. В поле "Подтверждение операций" укажите, каким способом в системе будет подтверждаться совершение операций клиентом банка:

    • с помощью SMS-пароля;

    • с помощью АСП:

    Примечание: АСП (аналог собственноручной подписи клиента) может быть использован для разрешения конфликтных ситуаций с клиентами. С помощью проверки АСП можно просмотреть по каждой платежной операции, совершенной клиентом, какие поля были подписаны, и проконтролировать совпадение АСП.

  4. В поле "Схема прав" выберите из списка интересующую Вас схему доступа.

После заполнения всех полей щелкните по кнопке Сохранить , и настройки вступят в действие.

SMS-пароли

Пункт меню SMS-пароли предназначен для управления настройками SMS-паролей, используемых для подтверждения входа в систему и для подтверждения операций клиентами банка. Данные настройки при создании SMS-паролей задаются в системе по умолчанию.

В системе ИКФЛ "Сбербанк ОнЛ@йн" Вы можете изменить и сохранить следующие параметры формирования SMS-паролей:

  1. Длина сообщения

    Данная настройка предназначена для определения длины SMS-сообщения, которое будет приходить клиенту банка на номер мобильного телефона, указанный в анкете. Чтобы изменить длину пароля, в поле «Длина сообщения» укажите количество символов для формирования SMS-пароля. После сохранения настройка вступит в действие.

  2. Количество попыток ввода SMS-пароля

    Данная настройка используется для изменения количества попыток ввода SMS-пароля. Для изменения количества попыток в поле «Количество попыток ввода» укажите, сколько раз клиент может ввести SMS-пароль до автоматической блокировки. После сохранения настройка вступит в действие.

  3. Время жизни пароля

    Данная настройка предназначена для изменения времени, в течение которого клиент сможет ввести пароль для подтверждения входа или операций в системе. Чтобы изменить настройку, в поле «Время жизни (в секундах)» введите время действия SMS-пароля. После сохранения настройка вступит в действие.

  4. Используемые символы

    Данная настройка предназначена для управления символами, которые создаются системой для ввода паролей. Для того чтобы изменить используемые символы для SMS-пароля, в поле «Используемые символы» выберите из списка символы, которые будут использоваться при создании SMS-пароля. После сохранения настройка вступит в действие.

После заполнения всех полей щелкните по кнопке Сохранить , и настройки вступят в действие.

Справочники

В пункте меню Справочники необходимо указать внешнюю систему, из которой по умолчанию будут реплицироваться общесистемные справочники.

Для этого в поле «Внешняя система для загрузки справочников» щелкните по кнопке (…), откроется справочник внешних систем. Затем выберите из справочника интересующую внешнюю систему, установив напротив нее флажок, и щелкните по кнопке Выбрать . На данной странице Вы можете воспльзоваться фильтром внешних систем.

Фильтр/поиск внешних систем

Для поиска интересующих Вас внешних систем можно воспользоваться фильтром. Для этого щелкните по кнопке Показать фильтр и задайте критерии поиска:

  • "ID" - поиск по идентификатору внешней системы в системе «Сбербанк ОнЛ@йн»;

  • "Идентификатор" - поиск по уникальному идентификатору внешней системы;

  • "Наименование" - поиск по наименованию шлюза:

  • "Адрес шлюза" - поиск по адресу шлюза.

Вы можете осуществлять поиск по подстроке в текстовых полях фильтра. Например, можно ввести в поля фильтра только несколько первых букв наименования внешней системы.

Можно отфильтровать список по любой комбинации полей. Например, можно отфильтровать все внешние системы по наименованию и адресу шлюза.

После того как критерии поиска заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас внешних систем.

Для того чтобы настройка вступила в действие, нажмите на кнопку Сохранить .

В данном пункте меню можно настроить виды периодичности оповещений, которые будут доступны сотруднику банка для подписки. В списке расписаний по каждому оповещению есть возможность настроить параметры, задающие его периодичность.

Список расписаний сотрудников

В списке расписаний по каждому оповещению отображаются:

  1. Название оповещения;

  2. Описание, характеризующее периодичность срабатывания оповещения.

  3. Выражение.

Вид оповещения может быть одним из следующих:

  • "Совершение платежной операции или ввод заявки" - через заданный промежуток времени система проверяет количество совершенных платежей или ввод заявок клиентами банка, если кто-либо из клиентов совершил операцию, то сотруднику банка придет сообщение.

  • "Блокировка пользователя" - через заданный промежуток времени система проверяет, изменился ли статус пользователей на "Заблокирован", если статус изменился, то сотруднику банка придет сообщение о блокировке пользователей.

  • "Ошибки заданного уровня важности" - через определенный промежуток времени система проверяет, произошли ли ошибки заданного уровня важности, если ошибки произошли, то сотруднику банка придет оповещение.

Добавление расписания

В список расписаний Вы можете добавить новое расписание. Для этого щелкните по кнопке Добавить, откроется форма создания рассылки, в которой в поле "Рассылка" выберите из справочника интересующий Вас вид рассылки сообщений, в поле "Выражение" введите то значение, которое в системе соответствует данной рассылке, в поле "Описание" введите содержание описания новой рассылки и щелкните по кнопке Сохранить , и созданная Вами рассылка появится в списке расписаний. Подробную информацию по созданию и изменению регулярного выражения Вы можете прочитать в "Руководстве сотрудника банка".

Редактирование расписания

Список расписаний также можно отредактировать. Для этого установите галочку напротив интересующего Вас вида рассылки, нажмите кнопку Редактировать . Система покажет Вам форму редактирования рассылки, в которой Вы можете изменить значение в поле "Выражение" и в поле "Описание". После изменения параметров нажмите кнопку Сохранить , и данная рассылка появится в списке рассылок в измененном виде.

Удаление расписания

Если Вы хотите удалить расписание оповещения, установите галочку напротив интересующей Вас рассылки и щелкните по кнопке Удалить . Система попросит Вас подтвердить удаление выбранной рассылки, после подтверждения данная рассылка будет удалена из списка расписаний.

Параметры системы

Данная настройка предназначена для управления уровнем иерархии подразделений и управления количеством администраторов, которые имеют право работать с подразделением. Данные настройки задаются в системе по умолчанию.

Для изменения уровня иерархии подразделений в поле «Уровень иерархии подразделений» впишите нужное значение. По умолчанию задается второй уровень иерархии, означающий, что в системе ИКФЛ «Сбербанк ОнЛ@йн» существуют подразделения двух уровней: верхнего и нижнего.

Для изменения количества администраторов, работающих с одним подразделением, в поле «Количество администраторов на подразделение» введите интересующее Вас значение.

После изменения параметров настройки щелкните по кнопке Сохранить , и настройки вступят в действие.
