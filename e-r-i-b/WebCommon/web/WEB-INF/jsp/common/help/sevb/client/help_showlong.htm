Длительные поручения

Длительное поручение (Долгосрочное платёжное поручение) - метод платежа, который позволяет Вам проводить регулярные денежные перечисления одному и тому же получателю на одну и ту же сумму. С помощью системы «Электронная Сберкасса» можно создать и распечатать отчеты об исполнении длительных поручений, которые были переданы Вами на исполнение в отделениях Сбербанка России.

Отчет об исполнении

Данный документ позволяет Вам создать и распечатать отчет об исполнении длительных поручений через систему "Электронная Сберкасса".

Для печати отчета выполните следующие действия:

  1. Выберите из списка номер счета, по которому хотите получить отчет,

  2. Укажите интересующий Вас период, за который Вам нужен отчет об исполнении длительного поручения. Максимальный срок, за который Вы можете получить отчет об исполнении, составляет 6 месяцев.

  3. В поле тип отчета выберите из списка нужный тип отчета, например, на перевод (перечисление) сумм на другие счета физических лиц в рублях.

После того как все поля заполнены, щелкните по кнопке Печать.
