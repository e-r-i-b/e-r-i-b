Список счетов и карт

В пункте главного меню Счета и карты Вы можете просмотреть список Ваших счетов, доступ к которым разрешен в системе "Сбербанк ОнЛ@йн", список выпущенных для Вас пластиковых карт, получить выписку по счету или справку о состоянии вкладов, а также подать в банк заявку об утрате сберегательной книжки и заявку на блокировку пластиковой карты.

Данная страница является главной в системе, она отображается сразу после регистрации, и Вы всегда можете вернуться к ней, щелкнув по пункту Счета и карты в главном меню. На данной странице отображаются курсы валют ЦБ, установленные на сегодняшний день.

Для каждого счета отображаются номер счета, тип счета (текущий, до востребования и т.д.), остаток (сколько средств в данный момент у Вас на счете), максимальная сумма для снятия и валюта счета.

Под списком счетов располагается список банковских карт, в котором для каждой карты показывается ее номер, тип карты, остаток (сколько средств в данный момент у Вас на счете), лимит получения наличных, лимит покупок, валюта счета, статус карты (заблокирована она или нет) и срок действия карты.

Также на данной странице открывается список последних операций, которые Вы совершили в системе. В этом списке отображается десять последних операций, по каждой операции Вы можете увидеть номер записи, дату и время совершения операции и наименование операции.

По умолчанию в списке отображаются только открытые счета и карты. Если Вы хотите просмотреть информацию обо всех счетах и картах, установите галочку напротив строки Показывать неактивные счета/карты. Закрытые и заблокированные счета и карты будут отмечаться в списке значком .

Выписка по счетам

Просмотреть выписку (список совершенных операций) по счету Вы можете двумя способами:

  1. Отметьте флажком интересующий Вас счет в списке (можно выбрать несколько счетов) и нажмите на кнопку Выписка или щелкните по номеру счета. Вы перейдете к окну просмотра выписки, где с помощью фильтра сможете изменить ее параметры (например, период, за который распечатывается выписка, он установлен в системе по умолчанию с 1 числа текущего месяца по текущий операционный день банка, но можно выбрать любой интересующий период). Затем нажмите кнопку Печать выписки, система сформирует и отобразит выписку, которую можно распечатать на принтере.

  2. В левом меню щелкните по ссылке Выписка по счетам, система сформирует выписку по всем Вашим счетам.

    Если Вам нужна выписка по одному или нескольким счетам, то можно воспользоваться фильтром. Для этого в фильтре выберите из списка нужный счет, задайте период, за который хотите получить выписку, и щелкните по кнопке Применить. Затем нажмите кнопку Печать выписки, система сформирует и отобразит выписку, которую можно распечатать на принтере.

Чтобы вернуться на главную страницу, где отображается список счетов и пластиковых карт, щелкните по кнопке Список счетов/карт .

Выписка по картам

Просмотреть выписку (список совершенных операций) по карте Вы можете двумя способами:

  1. Отметьте флажком интересующую Вас карту в списке (можно выбрать несколько карт) и нажмите на кнопку Выписка или щелкните по номеру карты. Вы перейдете на страницу просмотра выписки, на которой сможете изменить ее параметры (например, период, за который распечатывается выписка, он установлен в системе по умолчанию с 1 числа текущего месяца по текущий операционный день банка, но можно выбрать любой интересующий период). Затем нажмите кнопку Печать выписки, система сформирует и отобразит выписку, которую можно распечатать.

  2. В левом меню щелкните по ссылке Выписка по картам, система сформирует выписку по всем Вашим картам.

    Если Вы хотите составить выписку по одной или нескольким картам, то можно воспользоваться фильтром. Для этого в фильтре выберите из списка нужную карту, задайте период, за который хотите получить выписку по карте, и щелкните по кнопке Применить. Затем нажмите кнопку Печать выписки, система сформирует и отобразит выписку, которую можно распечатать.

    В выписке по пластиковой карте содержатся следующие сведения:

    • Номер банковской карты;

    • Держатель карты;

    • Общая сумма доступных средств;

    • Сумма лимита покупок карты;

    • Сумма наличного лимита карты;

    • Таблица операций по карте (10 последних операций), содержащая следующие поля: дата операции, сумма операции, валюта операции и тип операции.

Чтобы вернуться на главную страницу, где отображается список счетов и пластиковых карт, щелкните по кнопке Список счетов/карт .

Справка о состоянии

В системе "Сбербанк ОнЛ@йн" можно получить справку о состоянии любого вклада несколькими способами:

  1. Выберите из списка номер счета по вкладу и щелкните по кнопке Справка о состоянии .

    Примечание: по текущим счетам данная справка не составляется.

    В результате Вы перейдете на страницу просмотра справки о состоянии вклада, где сможете изменить ее параметры (например, период, за который распечатывается справка, он установлен в системе по умолчанию с 1 числа текущего месяца по текущий операционный день банка, но можно выбрать любой интересующий период). Затем нажмите кнопку Печать справки о состоянии, система сформирует и отобразит справку о состоянии, которую Вы сможете распечатать на принтере.

  2. Щелкните на странице Счета и карты по пункту левого меню Справка о состоянии. Система сформирует справку по всем счетам, кроме текущих счетов. Вы можете распечатать на принтере справку о состоянии вклада, для этого щелкните по кнопке Печать справки о состоянии.

    Если Вам нужна справка по одному или нескольким счетам, то можно воспользоваться фильтром. Для этого в фильтре выберите из списка нужный счет, задайте период, за который хотите получить справку, и щелкните по кнопке Применить.

Чтобы вернуться на главную страницу, где отображается список счетов и карт, щелкните по кнопке Список счетов/карт .
