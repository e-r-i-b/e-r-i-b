Справочник получателей

В системе предусмотрен справочник получателей, который облегчает заполнение документов и позволяет избежать возникновения опечаток в реквизитах документов.

В данном справочнике хранятся реквизиты получателей, которые Вы можете использовать при заполнении различных переводов, например, при оплате товаров и услуг. Для каждого получателя в справочнике хранятся его номер, условное обозначение, наименование получателя, ИНН, номер счета получателя, наименование банка получателя, БИК банка получателя, корр.счет банка получателя и статус получателя в системе. При выборе получателя из справочника все эти сведения автоматически подставляются в документ. Справочник является индивидуальным для каждого пользователя системы.

Вы вручную формируете свой справочник получателей. На данной странице Вы можете выполнять следующие действия:

  • Добавление получателя

    Чтобы добавить получателя в справочник, щелкните по кнопке Добавить. В результате Вы перейдете к форме ввода нового получателя. В данной форме в поле "Тип получателя" выберите, какой тип получателя Вы заводите (юридическое или физическое лицо), установив напротив интересующего типа галочку; в поле "Обозначение" введите обозначение получателя, в поле «Наименование (ФИО)» впишите название организации для юридического лица или ФИО получателя для физического лица; в поле "ИНН" укажите ИНН получателя, в поле "счет" укажите счет получателя, затем заполните реквизиты банка, в котором открыт счет получателя. Для заполнения реквизитов банка в поле "Наименование" щелкните по кнопке (...), откроется страница со справочником банков. На данной странице установите галочку напротив интересующего Вас банка и нажмите кнопку Выбрать. Также на этой странице Вы можете воспользоваться фильтром.

    Фильтр банков получателей

    Для того чтобы воспользоваться фильтром, щелкните по кнопке Показать фильтр и заполните поля фильтра:

    "Наименование" - поиск по названию банка получателя;

    "БИК" - поиск по банковскому идентификационному коду;

    "Город" - поиск по городу, в котором зарегистрирован банк.

    Вы можете фильтровать список по любой комбинации полей, например, можно отобрать все банки, название которых начинается на одну и ту же букву и из одного города.

    После того как критерии поиска заданы, щелкните по кнопке Построить список, и система покажет список интересующих Вас банков.

    После выбора банка получателя все реквизиты банка будут скопированы в создаваемый документ.

    После заполнения полей щелкните по кнопке Сохранить .

  • Редактирование получателя

    Вы можете отредактировать любую запись справочника, для этого установите галочку напротив интересующего Вас получателя в списке и нажмите на кнопку Изменить или щелкните по наименованию получателя. В результате Вы перейдете к форме редактирования записи получателя, в которой измените необходимые сведения и щелкните по кнопке Сохранить .

Для поиска интересующего Вас получателя в системе предусмотрен фильтр. Чтобы воспользоваться фильтром, щелкните по кнопке Показать фильтр и задайте критерии поиска:

Фильтр/поиск получателей:

  • "Наименование" - поиск по наименованию получателя;

  • "Счет" - поиск по номеру счета получателя;

  • "Банк" - поиск по названию банка, в котором открыт счет получателя.

Вы можете фильтровать список по любой комбинации полей. Например, можно отобрать всех получателей, наименование которых начинается с буквы "В" и счета которых открыты в одном банке.

После того как параметры поиска заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас получателей.
