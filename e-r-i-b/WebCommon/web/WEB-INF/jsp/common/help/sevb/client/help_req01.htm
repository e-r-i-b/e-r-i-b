Назначение заявки

С помощью данного пункта меню Вы можете подать в банк заявку на открытие счета по вкладу. Для этого в главном меню выберите пункт Платежи, а в открывшемся списке пункт Заявка на открытие счета. После заполнения заявки система попросит проверить правильность заполнения полей и подтвердить созданную заявку, после чего Ваша заявка будет отправлена в банк. Ход обработки заявки можно отслеживать в списке заявок по статусу.

Заполнение заявки на открытие счета

Для заполнения формы заявки на открытие счета по вкладу Вам необходимо заполнить обязательные поля:

  • В поле "Вид вклада" выберите из выпадающего списка интересующий Вас вид вклада.

  • В поле "Валюта вклада" выберите валюту, в которой желаете открыть вклад.

  • В поле "Срок вклада" введите срок, на который хотите открыть счет.

  • В поле "Сумма вклада" укажите сумму, которую Вы планируете разместить на депозитном счете.

  • В поле "Списать со счета" выберите из выпадающего списка счет, с которого Вы хотите списать денежные средства.

    Примечание: валюта счета списания должна совпадать с выбранной Вами валютой вклада.

  • В поле "Действие по окончании срока вклада" необходимо выбрать одно из двух значений:

    1. "Автоматически пролонгировать" - система автоматически продлит действие договора вклада после завершения срока его действия;

    2. "Перевести на счет" - выберите из выпадающего списка счет, на который будут переведены денежные средства по окончании срока вклада.

    После заполнения полей формы нажмите на кнопку Сохранить . Заявка приобретет статус "Введена". Если Вы решили не делать заявку, с данной страницы можно вернуться к списку выполненных Вами заявок на открытие счета, щелкнув по кнопке Список , или на страницу выбора типа платежа/заявки, щелкнув по кнопке Платежи .

    Подтверждение заявки

    После сохранения заполненного документа Вы попадаете на страницу подтверждения. Здесь Вам будет показана заполненная форма заявки на открытие счета по вкладу, в которой Вы должны проверить правильность указанных сведений, после чего выполнить одно из следующих действий:

    • Подтверждение заявки

      Если все сведения указаны правильно, щелкните по кнопке Подтвердить , и документ будет отправлен в банк на обработку.

      Примечание: процедура подтверждения зависит от используемого Вами способа регистрации в системе. Например, если Вы используете SMS-пароль для подтверждения, то на странице подтверждения сначала для получения SMS-пароля щелкните по кнопке Получить пароль, затем введите пароль из полученного SMS-сообщения в соответствующее поле.

    • Возврат к редактированию

      Если Вы хотите изменить какие-либо реквизиты заявки, щелкните по кнопке Редактировать , и Вы вернетесь к форме редактирования документа.

    • Создание нового документа

      Вы можете создать еще один документ данного типа, щелкнув по кнопке Создать . В результате Вы перейдете к форме ввода нового документа. Данная возможность полезна при создании нескольких однотипных документов.

    • Печать документа

      Если Вы хотите распечатать данный документ, щелкните по кнопке Печать .

    • Отказ от формирования заявки

      Если Вы передумали делать заявку, можно вернуться к списку сделанных Вами заявок на открытие счета по вкладу, щелкнув по кнопке Список .

    Просмотр документа

    После подтверждения заявки Вы попадаете на страницу просмотра, где отображается заполненный документ. О том, что документ передан в банк, свидетельствует отображаемый на форме документа штамп «Принято к исполнению». На страницу просмотра документа Вы можете также попасть из списка документов, щелкнув по номеру того документа, который хотите просмотреть. На странице просмотра Вы можете выполнить следующие действия:

    • Создание нового документа

      Вы можете создать еще один документ данного типа, щелкнув по кнопке Создать . В результате Вы перейдете к форме ввода нового документа. Данная возможность полезна при создании нескольких однотипных документов.

    • Отзыв документа

      Вы можете отозвать ошибочно отправленный документ. Для этого щелкните по кнопке Отозвать , в результате Вы перейдете к форме отзыва перевода. Информация в полях документа недоступна для редактирования. Чтобы отозвать заявку, щелкните по кнопке Сохранить .

    • Печать документа

      Если Вы хотите распечатать данный документ, щелкните по кнопке Печать .

    • Возврат к списку заявок

      Можно вернуться к списку сделанных Вами заявок на открытие счета по вкладу, щелкнув по кнопке Список .

    Список заявок

    Фильтр/поиск заявок

    В данном списке отображаются все сделанные Вами заявки на открытие счета по вкладу. Вы можете зайти в список со страницы выбора типа заявки, щелкнув на выбранной заявке по ссылке История операций, или при создании заявки, щелкнув по кнопке Список . Для каждой заявки в списке отображаются дата создания, номер заявки, номер счета и статус, по которому можно отследить состояние заявки в системе, и другие сведения. Вы можете выполнить следующие операции:

    • Просмотреть заявку

      Для этого щелкните по номеру интересующей Вас заявки в списке и Вы попадете на страницу просмотра заполненной формы заявки.

    • Отозвать документ

      Вы можете отозвать ошибочно отправленный документ. Для этого щелкните по кнопке Отозвать , в результате Вы перейдете к форме отзыва перевода. Информация в полях документа недоступна для редактирования. Чтобы отозвать заявку, щелкните по кнопке Сохранить .

    • Создание копии документа

      Вы можете создать копию документа, для этого отметьте документ, который служит оригиналом в списке, и щелкните по кнопке Создать копию . В результате Вы перейдете к форме ввода нового документа, причем в нее будут скопированы все данные из документа, выбранного Вами в качестве оригинала. Данная возможность полезна при создании нескольких однотипных документов.

    • Удалить заявку

      Для этого установите флажок напротив заявки, которую Вы хотите удалить, и нажмите на кнопку Удалить .

    • Фильтр заявок

      Для поиска в системе интересующих Вас заявок предусмотрен фильтр, который позволяет осуществлять поиск по любому из полей либо их комбинации. Щелкните по кнопке Показать фильтр. Откроется страница с формой, в которой необходимо задать критерии поиска:

      • В поле "Статус" выберите из выпадающего списка статус Вашей заявки на открытие счета/вклада: "Введена", "Исполнена", "Отказана" и др.;

      • В поле "Период" введите период времени, в течение которого была создана заявка на открытие счета по вкладу.

      Вы можете фильтровать список по любой комбинации полей, например, можно отобрать все заявки на открытие счета по вкладу, созданные в один период, со статусом "Принята".

      После того как критерии поиска заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас заявок.
