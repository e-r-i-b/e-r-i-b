Пункт главного меню Карты ключей предназначен для работы с картами ключей (создания, активации, блокировки и удаления). Карты ключей назначаются клиентам банка и используются ими для подтверждения платежных операций. Также возможно дополнительное подтверждение входа клиента в систему.

Карта ключей или Карта разовых паролей - это таблица, которая содержит заданное количество одноразовых паролей (ключей). Каждый ключ в карточке имеет свой порядковый номер.

Список карт ключей

В пункте меню Список карт ключей Вы увидите все карты ключей, созданные в системе. По каждой карте можно просмотреть номер карты, дату создания и количество ключей на карте.

Создать карту ключей можно типографским способом и через утилиту печати.

Если Вам необходимо удалить карту ключей клиента банка, установите флажок напротив той карты ключей, которую хотите удалить, и щелкните по кнопке Удалить. Удалять можно только карты ключей, непривязанные клиентам банка.

Выданные карты ключей клиент банка использует для дополнительного (опционального) подтверждения входа в систему и для подтверждения операций (отправка заявок, подтверждение платежей и т.д.).

Фильтр/поиск карт ключей

Для просмотра списка всех интересующих Вас карт ключей нужно нажать кнопку Показать фильтр и правильно заполнить поля фильтра:

  • "Номер" - поиск по номеру карты,

  • "Количество ключей" - поиск по количеству ключей в карте,

  • "Создана с.. по.." - поиск по дате создания карты.

Осуществлять поиск карт ключей Вы можете по любой комбинации полей, например, можно задать поиск по количеству ключей в карте и по дате создания карты.

После того как искомые параметры заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас карт ключей.

Генерация карт

Данный пункт меню предназначен для создания карт ключей типографским способом (т.е. с помощью печати скретч-карт, когда пароли карты ключей наносятся под защитный слой скретч-карты).

При нажатии на пункт меню Генерация карт открывается форма создания карт ключей, в данной форме укажите количество карт и количество ключей на карте, щелкните по кнопке Создать. Далее сохраните файл на диске, который будет содержать всю необходимую информацию для печати скретч-карты.

Примечание: существует альтернативный способ печати карт ключей - через утилиту печати. См. пункты контекстной справки Заявка на печать карт ключей и Загрузка обработанной заявки.

Заявка на печать карт ключей

Чтобы создать запрос на печать карт ключей, необходимо в форму для создания запроса на печать ввести количество карт в запросе, количество ключей в карте и подразделение банка, для которого изготавливаются карты ключей по данному запросу, и щелкнуть по кнопке Выгрузить. Откроется стандартное окно Windows Сохранить. Вы должны сохранить файл на сменном носителе (флешке, диске), отнести на сервер, где установлена утилита печати пин-конвертов, скопировать файл в каталог XML_in, запустить утилиту и распечатать карту ключей.

Загрузка обработанной заявки

Чтобы загрузить обработанную заявку в систему, Вы должны на сервере скопировать файл с ответом из утилиты печати из каталога XML_out на сменный носитель, с которого копируете файл ответа на компьютер, где установлена система ИКФЛ, открываете пункт Карты ключей - Загрузка обработанной заявки, нажимаете кнопку Обзор, выбираете путь до файла с ответом и щелкаете по кнопке Выполнить импорт. После выполнения процедуры импорта Вы можете привязывать карты ключей клиентам банка. Для привязки карты ключей Вы должны указать номер этой карты в анкете того клиента, которому она будет выдана. Откройте анкету выбранного Вами клиента, выберите вкладку Карты ключей и нажмите кнопку Добавить. Откроется окно, в которое нужно ввести номер карты и нажать кнопку Добавить.

Статусы карт ключей

Карты паролей могут иметь в системе следующие статусы:

  • Неактивна - карта выдана пользователю и "закреплена" за ним, но еще не активирована, т.е. действует предыдущая карта;

  • Активна - карта действительна, на ней есть неиспользованные пароли;

  • Использована - все пароли на карте израсходованы, карта далее использоваться не может.

  • Заблокирована - карта заблокирована по запросу клиента (утеря, порча и пр.) или автоматически в случае неверного указания пароля с карты. Карта с таким статусом не может быть использована клиентом для работы в системе.

Активация карт ключей

Активация карты может выполняться двумя способами:

  • При первоначальной выдаче клиенту карты или комплекта карт активацию выполняет сотрудник банка.

    Вы можете активировать карту ключей клиента. Для этого нужно выбрать клиента, кликнуть на его имени, откроется анкета клиента банка, выберите вкладку анкеты Карты ключей, в ней найдите карту, которую необходимо активировать, и щелкните по кнопке Активировать.

  • Когда у клиента остается последний ключ на карте, система предлагает ему активировать новую карту (если у клиента есть зарегистрированные и неактивированные карты). Для активации очередной карты клиент должен ввести ее серийный номер и подтвердить его последним паролем с предыдущей активированной карты.

Блокировка карт ключей

Блокировка карты может выполняться следующим образом:

  • Карта автоматически блокируется при неверном вводе пароля определенное количество раз подряд. Количество попыток для ввода пароля до автоматической блокировки доступа задается с помощью пункта меню Настройки - Клиенты и сотрудники. См. соответствующие разделы контекстной справки.

  • Карта может быть заблокирована сотрудником банка по телефонному или личному обращению клиента (например, в случае утери карты). Идентификация клиента сотрудником банка в этом случае может осуществляться, например, по кодовому слову, которое клиент должен сообщить сотруднику банка. Кодовое слово содержится в анкете клиента банка.

    Для блокировки карт ключей необходимо выбрать клиента, кликнуть на его имени, откроется анкета клиента банка, выберите вкладку анкеты Карты ключей, в ней найдите карту, которую необходимо заблокировать, и щелкните по кнопке Заблокировать.

Разблокировка карт ключей

Вы можете снять блокировку с карты ключей клиента. Разблокировка карты выполняется в случае, если карта была заблокирована автоматически, но карта осталась при клиенте и может использоваться по назначению дальше. В этом случае, обращаясь в банк по телефону или лично, клиент должен назвать сотруднику банка очередной пароль из карты. Сотрудник банка вводит этот пароль в системе, и, если пароль верный, карта разблокируется. Для этого нужно выбрать клиента, кликнуть на его имени, откроется анкета клиента банка, выберите вкладку анкеты Карты ключей, в ней найдите карту, которую необходимо разблокировать, и щелкните по кнопке Снять блокировку.
