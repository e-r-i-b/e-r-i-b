Фильтр/поиск платежей

В журнале платежей отображаются все выполненные Вами операции. В данном журнале Вы можете выполнить следующие действия:

  • Просмотр документа

    Вы можете просмотреть любой документ, для чего достаточно просто щелкнуть по номеру интересующего Вас документа в списке.

  • Отзыв перевода

    Вы можете отозвать переданный на обработку в банк перевод. В этом случае переведенные средства (за вычетом комиссии банка) будут возвращены на Ваш счет. Чтобы отозвать документ, отметьте его в списке, установив напротив него галочку, и щелкните по по кнопке Отозвать , расположенной в верхней части страницы. На экран будет выведена форма отзыва перевода, в которой Вам необходимо щелкнуть по кнопке Сохранить .

  • Редактирование документа

    Если документ еще не передан на обработку в банк, Вы можете отредактировать его. Для этого отметьте редактируемый документ в списке, установив напротив него галочку, и щелкните по по кнопке Редактировать . На экран будет выведена форма ввода документа, в которой измените значения интересующих Вас полей и сохраните изменения.

  • Создание копии документа

    Вы можете создать копию документа, отметив документ, который служит оригиналом в списке, и щелкнув по кнопке Создать копию , расположенной в верхней части страницы. В результате Вы перейдете к форме ввода нового документа, причем в данную форму будут скопированы все сведения из документа, выбранного Вами в качестве оригинала. Данная возможность полезна при создании нескольких однотипных документов.

Фильтр/поиск платежей

Также на этой странице Вы можете воспользоваться фильтром для поиска интересующих Вас операций.

Для того чтобы воспользоваться фильтром, щелкните по кнопке Показать фильтр и заполните поля фильтра:

"Номер операции" - поиск по номеру платежа в системе;

"Вид операции" - поиск по виду операции: например, "Пополнение карты", "Отзыв перевода", "Перевод между счетами", "Перевод рублей РФ";

"Счет" - поиск по номеру счета;

"Статус" - поиск по статусу операции в системе: "Введен", "Обрабатывается", "Исполнен", "Отозван", "Отказан".

"Дата совершения операции" - поиск по дате совершения операции;

"Сумма" - поиск по сумме, на которую совершен перевод;

"Дата приема к исполнению" - поиск по дате, когда перевод принят к исполнению;

"Наименование получателя" - поиск названию получателя, которому переведены денежные средства.

Вы можете фильтровать список по любой комбинации полей, например, можно отобрать все операции, совершенные в один день со статусом "Обрабатывается".

После того как критерии поиска заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас операций.
