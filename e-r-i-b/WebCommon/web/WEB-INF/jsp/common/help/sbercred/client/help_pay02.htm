Назначение перевода

Данный перевод позволяет Вам перевести средства со своего рублевого счета на счет другого юридического или физического лица.

Заполнение документа

В форме перевода рублей РФ Вам надо указать дату (в это поле автоматически будет подставлена текущая дата), выбрать из списка доступных Вам счетов счет списания (счет, с которого будут сняты деньги) и указать сумму переводимых средств.

Далее необходимо указать, кому Вы переводите средства. Сначала укажите является получатель юридическим или физическим лицом, установив галочку напротив соответствующего получателя. После этого необходимо заполнить реквизиты получателя: название (для юридического лица) или ФИО (для физического лица), коды ИНН и КПП (для физического лица КПП не нужно указывать), номер счета, на который будут перечислены средства, и реквизиты банка, в котором этот счет открыт. Не забудьте указать назначение платежа.

Для быстрого заполнения реквизитов можно воспользоваться предусмотренными в системе справочником получателей и справочником банков. Для доступа к справочнику получателей щелкните по кнопке (...), расположенной рядом с полем Название организации. Откроется страница со справочником получателей, на которой надо выбрать необходимого получателя, после чего щелкнуть по кнопке Выбрать. На данной странице Вы можете воспользоваться фильтром.

Фильтр получателей

Для того чтобы воспользоваться фильтром, щелкните по кнопке Показать фильтр и заполните поля фильтра:

"Наименование" - поиск по названию получателя;

"Счет" - поиск по номеру счета получателя;

"Банк" - поиск по названию банка получателя.

Вы можете фильтровать список по любой комбинации полей, например, можно отобрать всех получателей, наименование которых начинается на одну и ту же букву и счета которых открыты в одном банке.

После того как критерии поиска заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас получателей.

После выбора получателя все сведения будут скопированы в создаваемый документ.

Для доступа к справочнику банков щелкните по кнопке (...), расположенной рядом с полем БИК. Откроется страница со справочником банков, на которой надо выбрать банк получателя, после чего щелкнуть по кнопке Выбрать. Также на этой странице Вы можете воспользоваться фильтром.

Фильтр банков получателей

Для того чтобы воспользоваться фильтром, щелкните по кнопке Показать фильтр и заполните поля фильтра:

"Название" - поиск по названию банка получателя;

"БИК" - поиск по банковскому идентификационному коду;

"Город" - поиск по городу, в котором зарегистрирован банк.

Вы можете фильтровать список по любой комбинации полей, например, можно отобрать все банки, название которых начинается на одну и ту же букву и из одного города.

После того как критерии поиска заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас банков.

После выбора банка получателя все реквизиты банка будут скопированы в создаваемый документ.

После того как все поля заполнены, щелкните по кнопке Сохранить . Вы также можете добавить частично заполненный документ в свой список шаблонов, щелкнув по кнопке Сохранить как шаблон , чтобы в дальнейшем использовать его для быстрого формирования аналогичных документов.

Если Вы решили не делать перевод, с данной страницы можно вернуться к списку выполненных Вами переводов, щелкнув по кнопке Список , или на страницу выбора типа платежа, щелкнув по кнопке Платежи .

Подтверждение перевода

После сохранения заполненного документа Вы попадаете на страницу подтверждения. Здесь Вам будет показана заполненная форма перевода, в которой Вы должны проверить правильность указанных данных, после чего выполнить одно из следующих действий:

  • Подтверждение перевода

    Процедура подтверждения зависит от используемого Вами способа регистрации в системе.

    • Если Вы используете карту одноразовых паролей, то Вам необходимо будет ввести пароль с указанным номером с Вашей карты одноразовых паролей.

    • Если Вы используете сертификат системы криптозащиты, то Вам необходимо будет указать, какой сертификат использовать для подписания документа ЭЦП.

    Когда все данные указаны, щелкните по кнопке Подтвердить , чтобы документ был отправлен в банк на обработку.

  • Возврат к редактированию

    Если Вы хотите изменить какие-либо реквизиты перевода, щелкните по кнопке Редактировать , расположенной в верхней части страницы, чтобы вернуться к форме ввода документа для его редактирования.

  • Создание нового документа

    Вы можете создать еще один документ данного типа, щелкнув по кнопке Создать , расположенной в верхней части страницы. В результате Вы перейдете к форме ввода нового документа. Данная возможность полезна при создании нескольких однотипных документов.

  • Печать документа

    Если Вы хотите распечатать данный документ, щелкните по кнопке Печать .

  • Отказ от выполнения перевода

    Если Вы передумали осуществлять рублевый перевод, можно вернуться к списку выполненных Вами переводов, щелкнув по кнопке Список , или на страницу выбора типа платежа, щелкнув по кнопке Платежи .

Просмотр документа

После подтверждения платежа Вы попадаете на страницу просмотра, где отображается заполненный документ. О том, что документ передан в банк, свидетельствует отображаемый на форме документа штамп «Принято к исполнению». На страницу просмотра документа Вы можете также попасть из списка документов, щелкнув по номеру того документа, который хотите просмотреть. На странице просмотра Вы можете выполнить следующие действия:

  • Отзыв документа

    Вы можете отозвать ошибочно отправленный документ. Для этого щелкните по кнопке Отозвать , в результате Вы перейдете к форме отзыва перевода. Информация в полях документа недоступна для редактирования. Чтобы отозвать перевод, щелкните по кнопке Сохранить .

  • Создание нового документа

    Вы можете создать еще один документ данного типа, щелкнув по кнопке Создать , расположенной в верхней части страницы. В результате Вы перейдете к форме ввода нового документа. Данная возможность полезна при создании нескольких однотипных документов.

  • Печать документа

    Если Вы хотите распечатать данный документ, щелкните по кнопке Печать .

  • Возврат к списку переводов или выбору типа платежа

    Можно вернуться к списку выполненных Вами рублевых переводов, щелкнув по кнопке Список , или на страницу выбора типа платежа, щелкнув по кнопке Платежи .

Список рублевых переводов

В данном списке отображаются все выполненные Вами рублевые переводы на счета юридических или физических лиц. В списке для каждого перевода рублей РФ отображается его дата, номер документа, счета списания и зачисления, сумма перевода и статус, отображающий текущее состояние операции. Вы можете выполнить следующие операции:

  • Просмотр документа

    Вы можете просмотреть любой документ, для чего достаточно просто щелкнуть по номеру интересующего Вас документа в списке.

  • Отзыв документа

    Вы можете отозвать ошибочно отправленный документ. Для этого щелкните по кнопке Отозвать , в результате Вы перейдете к форме отзыва перевода. Информация в полях документа недоступна для редактирования. Чтобы отозвать перевод, щелкните по кнопке Сохранить .

  • Создание копии документа

    Вы можете создать копию документа, для этого отметьте документ, который служит оригиналом в списке, и щелкните по кнопке Создать копию , расположенной в верхней части страницы. В результате Вы перейдете к форме ввода нового документа, причем в нее будут скопированы все данные из документа, выбранного Вами в качестве оригинала. Данная возможность полезна при создании нескольких однотипных документов.

  • Фильтр переводов

    Для того чтобы воспользоваться фильтром, щелкните по кнопке Показать фильтр и заполните поля фильтра:

    "Статус" - поиск по статусу перевода в системе: "Введен", "На исполнении", "Исполнен", "Отказан", "Отозван";

    "Период" - поиск по времени создания перевода;

    "Сумма" - поиск по размеру суммы перевода.

    "Счет списания" - поиск по номеру счета, с которого списаны средства.

    "Счет получателя" - поиск по номеру счета получателя перевода.

    Вы можете фильтровать список по любой комбинации полей, например, можно отобрать все рублевые переводы на одну сумму со статусом "Исполнен".

    После того как критерии поиска заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас переводов.
