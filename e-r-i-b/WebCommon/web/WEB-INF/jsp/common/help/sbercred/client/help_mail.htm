Список отправленных писем

Система позволяет Вам отправлять в банк и получать из банка сообщения свободного формата — письма. Для работы с письмами выберите в главном меню пункт Почта. На экран будет выведен список полученных Вами писем.

Список полученных писем

Фильтр полученных писем

В пункте меню Список полученных писем Вы можете выполнить следующие действия:

  • Вы можете создать новое письмо. Для этого щелкните по кнопке Создать и заполните поля формы:

    • Значения в полях "дата и время отправки сообщения", "номер сообщения" проставляются системой автоматически;

    • В поле "тема сообщения" введите тему письма.

    • В поле "текст сообщения" впишите полный текст письма.

    После заполнения всех полей формы создания письма щелкните по кнопке Отправить, и письмо будет отправлено в банк. Если Вы не хотите отправлять письмо, щелкните по кнопке Отмена .

  • Можно просмотреть текст любого письма. Для этого установите флажок напротив интересующего Вас письма и щелкните по кнопке Просмотр.

  • Если Вы хотите ответить на письмо сотрудника банка, установите флажок напротив выбранного письма и щелкните по кнопке Ответить . Откроется страница редактирования письма. Поля "дата", "номер", "время" и "тип" письма автоматически заполняются системой. Вам нужно заполнить поля "тема" (можно оставить тему письма сотрудника банка), "текст" сообщения и щелкнуть по кнопке Отправить. Если Вы не хотите отвечать на письмо, щелкните по кнопке Отмена.

Кроме того, Вы можете осуществлять фильтрацию сообщений в списке.

Фильтр полученных писем

Для просмотра списка всех интересующих Вас полученных писем нужно щелкнуть по кнопке Показать фильтр и заполнить поля фильтра:

  • "Тема письма" - поиск по тематике письма;

  • "Период с.. по.." - поиск по дате и времени получения сообщения,

  • "Обязательно для прочтения" - поиск по признаку обязательности, обязательные для прочтения письма отмечены с помощью специальной пиктограммы ;

  • "Прочтено" - поиск по признаку, прочтено письмо или нет;

Вы можете фильтровать список по любой комбинации полей. Например, можно отобрать письма одного периода с одинаковой тематикой письма.

После того как параметры поиска заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас писем.

Список отправленных писем

Фильтр отправленных писем

В пункте меню Список отправленных писем Вы можете выполнить следующие действия:

  • Отправить сообщение в банк. Для этого щелкните по кнопке Создать и заполните поля формы:

    • Значения в полях "номер сообщения", "дата и время отправки сообщения" проставляются системой автоматически;

    • В поле "тема сообщения" введите тему письма.

    • В поле "текст сообщения" впишите полный текст письма.

    После заполнения всех полей формы создания письма щелкните по кнопке Отправить, и письмо будет отправлено в банк. Если Вы не хотите отправлять письмо, щелкните по кнопке Отмена .

  • Просмотреть список отправленных Вами в банк писем. Для этого установите галочку напротив интересующего Вас письма и щелкните по кнопке Просмотр.

Фильтр отправленных писем

Для просмотра списка всех интересующих Вас отправленных писем нужно щелкнуть по кнопке Показать фильтр и заполнить все поля фильтра:

  • "Тема письма" - поиск по тематике письма;

  • "Тип" - поиск по типу письма: "Письмо", "Вопрос", "Жалоба", "Проблема";

  • "Период с.. по.." - поиск по дате и времени отправления письма в банк.

Вы можете фильтровать список по любой комбинации полей. Например, можно отобрать письма одного периода с одинаковой тематикой письма.

После того как параметры поиска заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас писем.
