Назначение заявки

Вы можете подать в банк заявку на закрытие счета или вклада. Для этого в главном меню выберите пункт Заявки, а затем пункт Закрытие счета/вклада. Ход обработки заявки можно отслеживать в списке заявок по статусу.

Заполнение заявки

Заполнение заявки на закрытие счета
Заполнение заявки на закрытие вклада

Сначала Вам необходимо выбрать, что Вы собираетесь закрыть (счет или вклад), так как от данного выбора зависит набор полей в заявке. Для этого в поле "закрыть" укажите из списка, что Вы закрываете.

Заполнение заявки на закрытие счета

В заявке на закрытие счета в поле "счет для закрытия" выберите счет, который следует закрыть.

  • Если Вы хотите закрыть валютный счет, то в форме заявки появятся дополнительные поля для перевода иностранной валюты. В поле "название получателя платежа" введите наименование получателя, в поле "код страны получателя" укажите код, для этого щелкните по кнопке (...), откроется страница со справочником стран-получателей, в котором выберите интересующий Вас код, установив напротив него галочку, щелкните по кнопке Выбрать. На данной странице Вы можете воспользоваться фильтром.

    Фильтр/поиск стран

    Для того чтобы воспользоваться фильтром, щелкните по кнопке Показать фильтр и заполните поля фильтра:

    "Код" - поиск по буквенному коду страны;

    "Числовой код" - поиск по числовому коду страны;

    "Наименование" - поиск по названию страны.

    После того как критерии поиска заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас стран.

    Затем нужно ввести счет получателя, город, адрес получателя. Далее нужно ввести реквизиты банка получателя. Для это в поле в поле "SWIFT" щелкните по кнопке (...), откроется страница со справочником банков, на которой выберите SWIFT банка, установив напротив него галочку, щелкните по кнопке Выбрать. На данной странице Вы можете воспользоваться фильтром/поиском банков.

    Фильтр банков получателей

    Для того чтобы воспользоваться фильтром, щелкните по кнопке Показать фильтр и заполните поля фильтра:

    "SWIFT" - поиск по коду SWIFT банка;

    "Наименование" - поиск по названию банка получателя;

    "Город" - поиск по городу, в котором зарегистрирован банк.

    Вы можете фильтровать список по любой комбинации полей, например, можно отобрать все банки, название которых начинается на одну и ту же букву и из одного города.

    После того как критерии поиска заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас банков.

    После выбора банка получателя все реквизиты банка: город, код страны, адрес будут скопированы в создаваемый документ.

    Далее нужно ввести корр. счет банка получателя и указать условия перевода.

  • Если Вам надо перечислить оставшиеся средства на счет, который открыт в другом банке, то в поле "остаток средств перевести на счет" выберите соответствующее значение. В результате в форме ввода отобразятся дополнительные поля формы перевода рублей РФ.

    В форме перевода рублей РФ необходимо указать, кому Вы переводите средства. Сначала укажите является получатель юридическим или физическим лицом, установив галочку напротив соответствующего получателя. После этого необходимо заполнить реквизиты получателя: название (для юридического лица) или ФИО (для физического лица), коды ИНН и КПП (для физического лица КПП не нужно указывать), номер счета, на который будут перечислены средства, и реквизиты банка, в котором этот счет открыт.

    Для быстрого заполнения реквизитов можно воспользоваться предусмотренными в системе справочником получателей и справочником банков. Для доступа к справочнику получателей щелкните по кнопке (...), расположенной рядом с полем Название организации. Откроется страница со справочником получателей, на которой надо выбрать необходимого получателя, после чего щелкнуть по кнопке Выбрать. На данной странице Вы можете воспользоваться фильтром.

    Фильтр получателей

    Для того чтобы воспользоваться фильтром, щелкните по кнопке Показать фильтр и заполните поля фильтра:

    "Наименование" - поиск по названию получателя;

    "Счет" - поиск по номеру счета получателя;

    "Банк" - поиск по названию банка получателя.

    Вы можете фильтровать список по любой комбинации полей, например, можно отобрать всех получателей, наименование которых начинается на одну и ту же букву и счета которых открыты в одном банке.

    После того как критерии поиска заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас получателей.

    После выбора получателя все сведения будут скопированы в создаваемый документ.

    Для доступа к справочнику банков щелкните по кнопке (...), расположенной рядом с полем БИК. Откроется страница со справочником банков, на которой надо выбрать банк получателя, после чего щелкнуть по кнопке Выбрать. Также на этой странице Вы можете воспользоваться фильтром.

    Фильтр банков получателей

    Для того чтобы воспользоваться фильтром, щелкните по кнопке Показать фильтр и заполните поля фильтра:

    "Название" - поиск по названию банка получателя;

    "БИК" - поиск по банковскому идентификационному коду;

    "Город" - поиск по городу, в котором зарегистрирован банк.

    Вы можете фильтровать список по любой комбинации полей, например, можно отобрать все банки, название которых начинается на одну и ту же букву и из одного города.

    После того как критерии поиска заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас банков.

    После выбора банка получателя все реквизиты банка будут скопированы в создаваемый документ.

После того как все поля заполнены, щелкните по кнопке Сохранить . Если Вы решили не делать заявку, с данной страницы можно вернуться к списку выполненных Вами заявок на закрытие счета или вклада, щелкнув по кнопке Список , или на страницу выбора типа заявки, щелкнув по кнопке Заявки .

Заполнение заявки на закрытие вклада

В заявке на закрытие вклада Вам надо выбрать из списка своих вкладов тот, который необходимо закрыть. Все остальные поля заявки будут заполнены автоматически.

После того как все поля заполнены, щелкните по кнопке Сохранить . Если Вы решили не делать заявку, с данной страницы можно вернуться к списку выполненных Вами заявок на закрытие счета или вклада, щелкнув по кнопке Список , или на страницу выбора типа заявки, щелкнув по кнопке Заявки .

Подтверждение заявки

После сохранения заполненного документа Вы попадаете на страницу подтверждения. Здесь Вам будет показана заполненная форма заявки на закрытие счета или вклада, в которой Вы должны проверить правильность указанных данных, после чего выполнить одно из следующих действий:

  • Подтверждение заявки

    Процедура подтверждения зависит от используемого Вами способа регистрации в системе.

    • Если Вы используете карту одноразовых паролей, то Вам необходимо будет ввести пароль с указанным номером с Вашей карты одноразовых паролей.

    • Если Вы используете сертификат системы криптозащиты, то Вам необходимо будет указать, какой сертификат использовать для подписания документа ЭЦП.

    Когда все сведения указаны, щелкните по кнопке Подтвердить , и документ был отправлен в банк на обработку.

  • Печать документа

    Если Вы хотите распечатать данный документ, щелкните по кнопке Печать .

  • Отказ от формирования заявки

    Если Вы передумали делать заявку, можно вернуться к списку сделанных Вами заявок на закрытие счета или вклада, щелкнув по кнопке Список , или на страницу выбора типа заявки, щелкнув по кнопке Заявки .

Просмотр документа

После подтверждения заявки Вы попадаете на страницу просмотра, где отображается заполненный документ. О том, что документ передан в банк, свидетельствует отображаемый на форме документа штамп «Принято к исполнению». На страницу просмотра документа Вы можете также попасть из списка документов, щелкнув по номеру того документа, который хотите просмотреть. На странице просмотра Вы можете выполнить следующие действия:

  • Создание нового документа

    Вы можете создать еще один документ данного типа, щелкнув по кнопке Создать , расположенной в верхней части страницы. В результате Вы перейдете к форме ввода нового документа. Данная возможность полезна при создании нескольких однотипных документов.

  • Печать документа

    Если Вы хотите распечатать данный документ, щелкните по кнопке Печать .

  • Возврат к списку заявок или выбору типа заявки

    Можно вернуться к списку сделанных Вами заявок на закрытие счета/вклада, щелкнув по кнопке Список , или на страницу выбора типа заявки, щелкнув по кнопке Заявки .

Список заявок на закрытие счета/вклада

В данном списке отображаются все сделанные Вами заявки на закрытие счета или вклада. Для каждой заявки в списке система показывает дату создания заявки, номер заявки, тип заявки и статус, по которому можно отследить состояние Вашей заявки. Вы можете выполнить следующие операции:

  • Просмотр документа

    Вы можете просмотреть любой документ, щелкнув по номеру интересующего Вас документа в списке.

  • Создание новой заявки

    Вы можете создать новую заявку на закрытие счета или вклада, щелкнув по кнопке Создать , расположенной в верхней части страницы. В результате Вы перейдете к форме ввода нового документа.

  • Фильтр заявок

    Для того чтобы воспользоваться фильтром, щелкните по кнопке Показать фильтр и заполните поля фильтра:

    "Статус" - поиск по статусу заявки в системе: "Введена", "Одобрена", "Исполнена", "Отказана";

    "Период" - поиск по времени создания заявки;

    "Тип заявки" - поиск по типу заявки: заявка на открытие счета или заявка на открытие вклада.

    Вы можете фильтровать список по любой комбинации полей, например, можно отобрать все заявки на закрытие вклада со статусом "Принята".

    После того как критерии поиска заданы, щелкните по кнопке Построить список, и система покажет список интересующих Вас заявок.
