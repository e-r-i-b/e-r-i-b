Список счетов, карт и вкладов

В пункте главного меню Счета и карты Вы можете просмотреть список Ваших счетов, с которыми можно работать в системе "Интернет-клиент для физических лиц", список выпущенных для Вас пластиковых карт и список открытых Вами срочных вкладов, получить выписку по счету, карте, вкладу, а также можно подать в банк заявку на открытие или закрытие счета/вклада, выполнить перевод денежных средств между Вашими счетами, пополнить пластиковую карту, просмотреть информацию по кредиту, полученному в банке, и просмотреть новости банка.

Данная страница является главной в системе, она отображается сразу после регистрации и Вы всегда можете вернуться к ней, щелкнув по пункту Счета и карты в главном меню. На данной странице отображаются курсы валют ЦБ, установленные на сегодняшний день.

Для каждого счета отображаются номер счета, тип счета (текущий, до востребования и т.д.), остаток (сколько средств в данный момент у Вас на счете), валюта счета и дата последней операции.

Под списком счетов располагается список банковских карт, в котором для каждой выпущенной для Вас банковской карты отображаются ее номер (обратите внимание, что в целях безопасности отображается только первые четыре и четыре последних цифры номера пластиковой карты), тип карты, остаток (сколько средств в данный момент у Вас на счете), валюта счета, статус карты (заблокирована она или нет) и срок действия карты.

Для каждого вклада отображаются вид вклада, сумма вклада, валюта вклада, срок вклада в днях, процентная ставка (% годовых), дата открытия вклада и дата окончания.

В том случае, если Вы получили в банке кредит или несколько кредитов, на главной странице для каждого кредита отображается номер договора, кредитный продукт, дата окончания, сумма кредита, валюта, дата погашения, сумма погашения, сумма просрочки.

По умолчанию в списке отображаются только открытые счета, карты и вклады. Если Вы хотите просмотреть информацию обо всех счетах и вкладах, установите галочку напротив строки Показывать неактивные счета/карты/вклады. Закрытые и заблокированные счета, карты и вклады будут отмечаться в списке значком .

Также на главной странице можно посмотреть новости системы. Для просмотра детальной информации по каждой новости щелкните на названии новости или нажмите кнопку Подробнее, и система покажет страницу с подробной информацией по выбранной новости.

Просмотр информации по счету

Вы можете получить подробную информацию по любому счету из списка счетов на главной странице. Для этого необходимо щелкнуть по номеру того счета, о котором Вы хотите просмотреть подробные сведения. Информация по счету включает в себя номер счета, текущий остаток по счету, валюту счета, текущее состояние (открыт или заблокирован), дату открытия, номер договора. В окне просмотра сведений о счете Вы можете выполнить следующие действия:

  • Печать карточки счета

    Чтобы распечатать карточку счета, щелкните по кнопке Печать .

  • Возврат к списку счетов, карт и вкладов

    Чтобы вернуться на главную страницу, где отображается список счетов и вкладов щелкните по кнопке На главную .

Просмотр информации по карте

Вы можете получить подробную информацию по Вашим пластиковым картам. Для этого необходимо щелкнуть по номеру карты, по которой Вы хотите просмотреть подробные сведения. Информация по карте включает в себя ФИО держателя карты, тип карты, номер и дату договора, номер карты, срок действия карты, номер спецкарточного счета, лимит задолженности, доступные средства, текущую задолженность, валюту и дату последней операции по карте. В окне просмотра сведений по карте Вы можете выполнить следующие действия:

  • Печать сведений по пластиковой карте

    Чтобы распечатать сведения по Вашей пластиковой карте, щелкните по кнопке Печать .

  • Возврат к списку счетов, карт и вкладов

    Чтобы вернуться на главную страницу, где отображается список счетов, карт и вкладов, щелкните по кнопке На главную .

Просмотр информации по вкладу

Вы можете получить подробную информацию по любому вкладу из списка Ваших вкладов на главной странице. Для этого необходимо щелкнуть по названию того вклада, о котором Вы хотите получить подробную информацию. Сведения по вкладу включают в себя название вида вклада, сумму вклада, валюту вклада, срок вклада, процентную ставку, номер депозитного счета, признак пролонгации вклада на следующий срок (возможность его продлить), сумму неснижаемого остатка вклада (размер суммы, которую нельзя снять до окончания срока вклада), номер счета для перечисления причисленных процентов (на какой счет Вам буду перечислены проценты по вкладу), номер счета для перечисления средств по истечении срока вклада, текущее состояние (открыт или заблокирован), дату открытия, дату закрытия и номер договора на предоставление услуг системы "Интернет-клиент для физических лиц". В окне просмотра сведений о вкладе Вы можете выполнить следующие действия:

  • Печать сведений о вкладе

    Чтобы распечатать сведения о вкладе, щелкните по кнопке Печать .

  • Возврат к списку счетов, карт и вкладов

    Чтобы вернуться на главную страницу, где отображается список счетов, карт и вкладов, щелкните по кнопке На главную .

Просмотр информации по кредиту

Вы можете получить подробную информацию по кредитам, выданным Вам банком. Чтобы получить подробные сведения по кредиту, щелкните на названии кредитного продукта. Информация по кредиту включает в себя текущее состояние кредита (открыт или закрыт), номер договора, название кредитного продукта, дату начала, дату окончания кредита, сумму кредита, валюту, процентную ставку, срок кредита, сумму ежемесячного платежа по кредиту, сумму остатка по кредиту, дату погашения, сумму погашения, сумму просроченной задолженности по кредиту, дату последнего платежа, сумму последнего платежа, сумму для полного досрочного погашения. Далее располагается график погашения кредита. В окне просмотра сведений по кредиту Вы можете выполнить следующие действия:

  • Печать сведений по кредиту

    Чтобы распечатать сведения по кредиту, щелкните по кнопке Печать .

  • Возврат к списку счетов, карт и вкладов

    Чтобы вернуться на главную страницу, где отображается список счетов, карт и вкладов, щелкните по кнопке На главную .

Выписка по счетам

Просмотреть выписку (список совершенных операций) по счету Вы можете несколькими способами:

  1. Отметьте флажком интересующий Вас счет в списке (можно выбрать несколько счетов) и щелкните по кнопке Выписка. Затем нажмите кнопку Выписка, система сформирует и отобразит выписку, которую Вы сможете распечатать.

  2. Если Вы хотите просмотреть выписку по одному счету, щелкните в списке счетов на сумме остатка средств на счете, и Вы перейдете к окну просмотра выписки по выбранному счету.

  3. В левом меню щелкните по ссылке Выписка по счетам, выберите из списка нужный счет или карту, задайте дату и время, за которое хотите получить выписку по счету. Затем нажмите кнопку Печать, система сформирует и отобразит выписку, которую Вы сможете распечатать.

На странице просмотра выписки Вы можете выполнить следующие действия:

  • Фильтр выписки

    В окне просмотра выписки Вы можете выбрать счет, изменить параметры выписки (например, период, за который распечатывается выписка, он установлен в системе по умолчанию за последние сутки, но Вы можете выбрать любой интересующий период и дату), затем нажать кнопку Выписка, и система сформирует выписку по выбранному счету.

  • Печать выписки

    Чтобы распечатать выписку по счетам, щелкните по кнопке Печать выписки .

  • Возврат к списку счетов, карт и вкладов

    Чтобы вернуться на главную страницу, где отображается список счетов, вкладов и пластиковых карт, щелкните по кнопке Список счетов/карт , расположенной в верхней части страницы.
