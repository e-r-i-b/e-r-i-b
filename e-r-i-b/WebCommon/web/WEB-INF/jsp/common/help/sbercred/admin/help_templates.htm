Шаблоны платежей

Пункт меню Шаблоны платежей позволяет создавать, удалять, редактировать и копировать шаблоны платежей, которые будут доступны всем пользователям системы.

Для упрощения ввода платежей в системе предусмотрены шаблоны платежей. Шаблоны могут создаваться как клиентами, так и сотрудниками банка в системе администрирования. Шаблоны платежей, созданные сотрудниками банка в системе администрирования, называются шаблонами банка. Данный пункт меню является средством управления шаблонами банка.

Шаблоны банка могут быть использованы клиентами только для создания по ним своих шаблонов и платежей. Клиенты не могут удалять или изменять шаблоны банка.

Для того чтобы создать шаблон платежа в пункте меню Шаблоны платежей, щелкните по кнопке Создать, выберите вид платежа, установите галочку напротив интересующего Вас вида платежа, шаблон для которого Вы создаете, и нажмите кнопку Продолжить. Откроется страница Ввод/редактирование платежа, на которой в строке "название шаблона" Вам нужно ввести название шаблона платежа. Затем в форме шаблона заполните все необходимые поля и щелкните по кнопке Сохранить. Если поля формы заполнены правильно, то шаблон платежа появится в списке шаблонов.

Если Вам необходимо изменить шаблон платежа, установите флажок напротив инересующего Вас шаблона и щелкните по кнопке Изменить. Откроется страница редактирования шаблона платежа, на которой Вы можете изменить значение любых полей шаблона, и щелкните по кнопке Сохранить. Если Вы не хотите менять шаблон платежа, щелкните по кнопке Список, и система вернет Вас на страницу списка шаблонов платежей.

Если Вы хотите удалить шаблон из списка, установите флажок напротив интересующего Вас шаблона платежа и щелкните по кнопке Удалить. Система попросит Вас подтвердить операцию, Вы подтверждаете, и шаблон будет удален из списка.

Для того чтобы создать копию выбранного шаблона платежа, установите флажок напротив интересующего Вас шаблона и щелкните по кнопке Копировать.
