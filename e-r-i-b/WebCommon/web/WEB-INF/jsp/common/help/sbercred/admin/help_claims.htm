Пункт меню Заявки предназначен для работы с поступающими от клиентов заявками.

Список заявок

В пункте меню Заявки автоматически открывается список всех заявок, поступивших от клиентов банка.

В системе ИКФЛ предусмотрены следующие типы заявок:

  • заявка на открытие счета/вклада;

  • заявка на закрытие счета/вклада;

  • заявка на пополнение вклада;

  • заявка на списание средств со вклада;

В списке заявок Вы можете просмотреть и обработать заявку клиента банка.

Чтобы просмотреть заявку, щелкните по ее номеру в списке заявок. Открывается страница с подробной информацией о данной заявке.

В зависимости от способа обработки заявки можно разделить на две группы:

  1. Заявки, которые попадают в RS-Retail после того как Вы нажмете на кнопку Исполнить. Такая процедура предусмотрена только для заявки на открытие счета/вклада в соответствии с законодательством.

  2. Заявки, которые автоматически попадают в RS-Retail (список отложенных документов). К данной группе относятся все заявки, кроме заявки на открытие счета/вклада. Заявки данной группы Вы не можете обработать в системе ИКФЛ.

Заявки первой группы Вы можете обработать следующим образом:

  • Если Вам необходимо отклонить заявку, щелкните на номере интересующей Вас заявки, откроется страница просмотра заявки, на которой в поле "причина отказа заявки" выберите причину отказа из специального справочника или введите в ручную другую причину и щелкните по кнопке Отклонить. В результате система присвоит заявке статус "Отказана".

  • Если Вы хотите одобрить заявку, щелкните на номере интересующей Вас заявки, откроется страница просмотра заявки, и щелкните по кнопке Одобрить. В результате система присвоит заявке статус "Одобрена", т.е. данная заяка может быть исполнена.

  • Чтобы исполнить заявку, т.е. отправить заявку на обработку в RS-Retail (только для заявок второй группы), щелкните на номере интересующей Вас заявки, откроется страница просмотра заявки, и щелкните по кнопке Исполнить. В результате система присвоит заявке статус "Исполнена".

При необходимости Вы можете распечатать документы по любой заявке. Для этого кликните на номере выбранной заявки, откроется страница с подробной информацией о данной заявке, щелкните по кнопке Печать.

Также если Вам нужно закрыть заявку, щелкните по номеру заявки в списке, откроется страница просмотра заявки, и щелкните по кнопке Закрыть. Закрытие заявки подразумевает, что Вы с данной заявкой больше работать не планируете, т.к. запрос клиента по данной заявке обработан.

Фильтр/поиск заявок

Вы можете осуществлять отбор заявок в списке с помощью фильтра, щелкнув по кнопке Показать фильтр, по любому из указанных полей:

  • "Номер заявки" - поиск по номеру заявки,

  • "Статус" - поиск по текущему статусу заявки.

  • "Тип заявки " - поиск по типу заявки.

  • "Передана в банк с.. по.. " - поиск по дате и времени передачи заявки клиентом в банк.

Вы можете осуществлять поиск по подстроке в текстовых полях фильтра. Например, можно выбрать из справочника и указать только тип заявки.

Также Вы можете фильтровать список по любой комбинации полей. Например, можно отобрать все одобренные заявки, созданные за последние три дня.

После того как параметры поиска заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас заявок.
