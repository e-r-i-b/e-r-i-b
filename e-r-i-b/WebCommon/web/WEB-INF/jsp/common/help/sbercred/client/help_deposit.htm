Фильтр/поиск вкладов

На данной странице Вы можете просмотреть список открытых Вами вкладов, а также подать в банк заявку на открытие нового вклада, заявку на пополнение и закрытие вклада. Для каждого вклада отображаются вид вклада, сумма вклада, валюта вклада, срок вклада в днях, процентная ставка, дата открытия вклада, дата окончания, статус и номер договора.

Для того чтобы подать в банк заявку на открытие вклада, щелкните по кнопке Открыть вклад, и Вам откроется форма для заполнения заявки. Порядок заполнения заявки описан в разделе контекстной справки Заявка на открытие вклада.

Если Вам нужно пополнить вклад, щелкните по кнопке Пополнить вклад, и Вам откроется форма для заполнения заявки. Процедуру заполнения заявки см. в разделе контекстной справки Пополнение вклада.

Для того чтобы подать в банк заявку на закрытие вклада, щелкните по кнопке Закрыть вклад, и Вам откроется форма для заполнения заявки. Порядок заполнения заявки описан в разделе контекстной справки Заявка на закрытие вклада.

Если Вы хотите перейти к списку депозитных продуктов банка, щелкните по кнопке Вклады.

Фильтр/поиск вкладов

Также на этой странице Вы можете воспользоваться фильтром для поиска интересующих Вас вкладов.

Для того чтобы воспользоваться фильтром, щелкните по кнопке Показать фильтр и заполните поля фильтра:

"Статус" - поиск по статусу вклада: "Открытый", "Закрытый".

"Вид вклада" - поиск по виду вклада: например, "Накопительный до 180";

"Срок вклада" - поиск по периоду, на который вклад открыт, например "30 дней";

"Сумма" - поиск по сумме, на которую открыт вклад;

"Валюта" - поиск по валюте, в которой открыт вклад;

"Дата открытия" - поиск по дате открытия вклада;

"Дата закрытия" - поиск по дате закрытия вклада;

Вы можете фильтровать список по любой комбинации полей, например, можно отобрать все вклады, открытые на один срок в рублях.

После того как критерии поиска заданы, щелкните по кнопке Поиск, и система покажет список интересующих Вас вкладов.

Депозитные продукты

На данной странице приведен полный список предлагаемых Вам банком депозитных продуктов. Для каждого депозитного продукта в списке отображается срок вклада, сумма вклада, процентная ставка (% годовых), периодичность выплаты процентов, возможность пополнения вклада, возможность снятия части вклада и описание вклада. Если Вы хотите получить более подробную информацию о каком-нибудь из перечисленных видов депозитов, щелкните по кнопке Подробнее. Откроется страница с детальной информацией по выбранному депозитному продукту, на которой Вы можете оценить доходность депозита по размеру вклада при помощи калькулятора вкладов. Чтобы им воспользоваться щелкните по кнопке Калькулятор. Если Вы хотите вернуться на страницу вкладов банка, нажмите кнопку Список .

Калькулятор вкладов

Депозитный калькулятор позволяет Вам быстро оценить доходность депозитов и выбрать подходящий для Вас вклад банка. Чтобы воспользоваться калькулятором вкладов, в поле "вид вклада" выберите интересующий Вас вид вклада, в поле "валюта вклада" укажите валюту (список валют зависит от выбранного вида вклада) и укажите срок вклада. В поле "сумма вклада" укажите сумму, которую Вы планируете разместить на депозитном счете. Затем щелкните по кнопке Рассчитать, и калькулятор покажет Вам сумму, которая будет на депозитном счете по окончании срока вклада, сумму полученного дохода и процентную ставку по выбранному виду вклада.

Вы можете вернуться на страницу Ваших вкладов, для этого щелкните по кнопке Список .
