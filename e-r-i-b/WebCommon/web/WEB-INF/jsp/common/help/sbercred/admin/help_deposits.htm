Список депозитных продуктов

Пункт меню Депозиты предназначен для управления депозитными продуктами банка, которые будут доступны для клиентов в системе ИКФЛ. При выборе данного пункта меню открывается список депозитных продуктов банка.

В списке депозитных продуктов по каждому из них показывается следующая информация:

  • Наименование вклада. Наименование депозитного продукта банка.

  • Сумма вклада. Отображается перечень диапазонов суммы вклада в зависимости от валюты и срока размещения средств.

  • Валюта вклада. Отображается список валют, в которых данный депозитный продукт предоставляется клиентам банка.

  • Срок вклада. Отображается перечень возможных сроков вклада для данного депозитного продукта.

  • Процентная ставка по вкладу. Отображается процентная ставка (% годовых) для данного депозитного продукта (вида вклада) в зависимости от валюты, срока и суммы размещаемых денежных средств.

Управление депозитными продуктами

Вы можете управлять депозитными продуктами банка следующим образом:

  • Добавление депозитного продукта

    Вы можете добавить в список депозитный продукт. Для этого щелкните по кнопке Добавить, откроется страница редактирования депозитного продукта. На данной странице в поле "имя депозитного продукта" впишите название создаваемого депозитного продукта, в поле "описание" введите комментарий депозитного продукта. В форме ввода депозитных продуктов по каждому вкладу в системе показывается: идентификатор (уникальное имя депозитного продукта в системе), название (название депозитного продукта), валюта (валюта, в которой производится вклад). Затем укажите перечень видов вкладов, входящих в состав депозитного продукта, установите флажки напротив интересующих Вас видов вкладов, щелкните по кнопке Сохранить, и новый депозитный продукт появится в списке депозитных продуктов банка.

  • Редактирование депозитного продукта

    Если Вам необходимо отредактировать депозитный продукт банка, в списке продуктов выберите интересующий Вас депозитный продукт, установите напротив него флажок и щелкните по кнопке Изменить. Откроется страница редактирования депозитного продукта банка, на которой Вы можете изменить название продукта и перечень видов вкладов, входящих в состав депозитного продукта, затем щелкните по кнопке Сохранить, и депозитный продукт появится в списке в измененном виде.

  • Удаление депозитного продукта

    Если Вам нужно удалить депозитный продукт банка из списка, установите флажок напротив выбранного депозитного продукта и щелкните по кнопке Удалить.
