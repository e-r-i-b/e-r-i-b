Пункт меню Кредиты предназначен для работы с заявками, поступившими от клиентов банка на предоставление кредита, для управления кредитными продуктами, видами кредитов и кредитными офисами.

Заявки

Фильтр/поиск заявок

Пункт меню Заявки предназначен для обработки поступивших от клиентов банка заявок на кредит. С помощью данного пункта меню Вы можете просмотреть, отправить на обработку в RS-Loans, отказать в предоставлении кредита, отправить заявку на редактирование клиенту банка (например, в том случае, если форма заявки заполнена с опечатками), переместить заявки в архив и распечатать все документы на кредит. Заявки на предоставление кредита могут поступать как от анонимных клиентов банка (любых пользователей Интернета, которые зашли на сайт банка и подали заявку на кредит), так и от клиентов, зарегистрированных в системе ИКФЛ. Заявки от анонимных клиентов выделяются в списке с помощью специальной пиктограммы.

В списке кредитных заявок по каждой заявке отображается следующая информация:

  • "Номер заявки" - порядковый номер заявки;

  • "Номер заявки в системе RS-Loans" - номер заявки во внешней системе RS-Loans;

  • "Дата" - дата создания заявки;

  • "ФИО клиента" - фамилия, имя и отчество клиента банка;

  • "Сумма запрашиваемого клиентом кредита" - размер суммы, которую клиент хочет получить в кредит;

  • "Срок запрашиваемого клиентом кредита" - срок, на который клиент банка хочет получить кредит;

  • "Сумма утвержденного банком кредита" - размер суммы, которую банк может предоставить данному клиенту в кредит;

  • "Срок утвержденного банком кредита" - срок, в течение которого клиент банка обязан выплатить кредит;

  • "Валюта кредита" - валюта, в которой предоставляется кредит;

  • "Статус заявки" - статус заявки на кредит в системе ("Принята", "Введена", "В рассмотрении", "Требуется доработка", "Кредит выдан", "Утверждена", "Отказана").

Просмотр заявки

Для того чтобы просмотреть заявку клиента банка, кликните на номере заявки, откроется страница просмотра заявки на кредит. Если по заявке есть поручитель, то форма анкеты поручителя отображается на той же странице под формой заявки заёмщика.

Отказ в предоставлении кредита

Вы можете отказать клиенту банка в предоставлении кредита, если заявка не удовлетворяет правилам заполнения, принятым в банке. Для этого кликните на номере заявки и щелкните по кнопке Отказать. Затем подтвердите отказ, и система присвоит выбранной заявке статус "Отказана".

Обработка заявки

Вы можете отправить заявку на обработку в RS-Loans, если заявка удовлетворяет правилам заполнения, принятым в банке. Для этого нужно поставить галочку напротив нужной заявки и нажать кнопку Отправить. Кнопка Отправить также доступна и при просмотре заявки. При этом заявка получает статус "В рассмотрении". В RS-Loans можно отправить заявки, которые имеют статус "Принята".

Редактирование заявки

Вы можете отправить заявку на редактирование клиенту банка (в том случае, если, например, клиент банка некорректно, с опечатками заполнил поля заявки или если не хватает сведений для оформления документов на кредит). Для этого поставьте галочку напротив интересующей Вас заявки и нажмите кнопку Требуется доработка. Кнопка Требуется доработка также доступна и при просмотре заявки. Затем подтвердите операцию, и система присвоит заявке статус "Требуется доработка". Этот статус можно присвоить только тем заявкам, у которых был статус "Принята".

Перемещение заявки в архив

Вы можете переместить в архив заявки, с которыми не будете работать. Перемещать можно только заявки со статусом "Отказана", "Утверждена" и "Кредит выдан". Для этого нужно поставить галочку напротив нужных заявок и нажать кнопку В архив. Кнопка В архив также доступна и при просмотре заявки. Заявки, переведенные в архив, видны только в списке архивных заявок.

Печать документов на кредит

Чтобы распечатать документы на кредит, кликните на номере заявки, откроется окно просмотра заявки на кредит, и щелкните по кнопке Распечатать документы. К документам на кредит относится пакет документов, необходимых при оформлении кредита (кредитный договор, договор поручительства, договор залога, акт сдачи/приемки залога и т.п.). Для настройки пакета документов используется пункт меню Сервис, см. ссылки "Шаблоны документов", "Пакеты шаблонов".

Фильтр/поиск заявок

Для поиска интересующей Вас заявки в системе предусмотрен фильтр. Чтобы воспользоваться фильтром, щелкните по кнопке Показать фильтр и задайте критерии поиска:

  • "Тип клиента" - поиск по типу клиента: анонимный или зарегистрированный клиент;

  • "ФИО клиента" - поиск по имени клиента;

  • "Серия и номер паспорта (документа)" - поиск по серии и номеру паспорта или другого документа клиента банка;

  • "Номер заявки" - поиск по номеру заявки клиента банка;

  • "Статус заявки" - поиск по статусу заявки на кредит в системе ("Принята", "Введена", "В рассмотрении", "Требуется доработка", "Кредит выдан", "Утверждена", "Отказана").

  • "Вид кредита" - поиск по виду кредита (типу кредитования, например, "Автокредит", "Потребительский" и др.);

  • "Офис банка" - поиск по офису банка, в котором клиенту банка предоставляется кредит;

  • "Период" - поиск по дате и времени поступления заявки от клиента банка;

  • "Сумма запрашиваемого клиентом кредита" - поиск по размеру суммы, которую клиент банка хочет получить в кредит;

Вы можете осуществлять поиск по подстроке в текстовых полях фильтра. Например, можно ввести в поля фильтра только несколько первых букв фамилии, имени или отчества клиента.

Также Вы можете фильтровать список по любой комбинации полей. Например, можно отобрать заявки всех анонимных клиентов со статусом "В рассмотрении".

После того как параметры поиска заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас заявок.

Архив заявок

Фильтр/поиск заявок

Пункт меню Архив заявок предназначен для истории полученных заявок от клиентов банка на предоставление кредита. В данном пункте меню содержится список обработанных сотрудником банка заявок.

В списке архивных заявок отображаются заявки со статусами "Введена", "Отказана", "Утверждена" и "Выдан кредит".

Вы можете просмотреть архивные заявки. Для этого кликните на номере заявки, и откроется страница просмотра заявки.

Кредитные продукты

С помощью данного пункта Вы можете управлять кредитными продуктами банка в системе: создавать, редактировать, удалять кредитные продукты. Кредитный продукт - это конкретное предложение (продукт) в рамках определенного вида кредита (типа кредитования). Например, для вида кредита "Автокредит" продуктами будут являться "Эконом", "Экспресс", "Классический".

  • Создание кредитного продукта

    Для того чтобы создать новый кредитный продукт, щелкните по кнопке Добавить. Откроется страница создания нового кредитного продукта, на которой заполните все поля. В поле "имя кредитного продукта" введите название кредита, которое будет отображаться пользователям системы; в поле "вид кредита" укажите из списка вид кредита (тип кредитования) для создаваемого Вами кредитного продукта; в поле "анонимная заявка" укажите, будет ли доступен кредитный продукт для анонимных клиентов; в поле "количество поручителей" введите количество поручителей, необходимых для данного кредитного продукта; в поле "пакет документов" выберите из списка название пакета документов, необходимых для оформления кредита (пакет документов определяет перечень шаблонов документов, которые печатаются и подписываются клиентом при оформлении кредита); в поле "описание" введите дополнительную информацию по кредитному продукту.

    Далее для каждого офиса, для которого доступен данный продукт, нужно указать вид кредита, соответствующий виду кредита в системе RS-Loans. Для этого напротив каждого кредитного офиса выберите в графе "вид кредита" нужный вид кредита.

    После заполнения полей нажмите кнопку Сохранить, и кредитный продукт будет создан в системе.

  • Редактирование кредитного продукта

    Для того чтобы отредактировать кредитный продукт, выберите кредитный продукт из списка, установите напротив него галочку и нажмите кнопку Редактировать. Откроется страница редактирования кредитного продукта, на которой Вам необходимо изменить интересующую Вас информацию о кредитном продукте и щелкнуть по кнопке Сохранить. Изменение сведений о кредитном продукте осуществляется аналогично заведению нового кредитного продукта, см. раздел справки "Создание кредитного продукта".

  • Удаление кредитного продукта

    Если Вы хотите удалить кредитный продукт из списка, отметьте галочкой те кредитные продукты, которые Вам нужно удалить, и нажмите кнопку Удалить. Удаление кредитного продукта подразумевает невозможность дальнейшей работы с ним в системе: он исчезает из списка кредитных продуктов банка, кредитного калькулятора, а также прочих списков. При этом, если у клиента есть действующие кредиты по данному продукту, то информация по нему (продукту) в рамках текущего кредита будет доступна пользователю.

Виды кредитов

Пункт меню Виды кредитов используется для управления видами кредитов банка. Вид кредита - это тип кредитования: например, "Потребительский", "Кредит на неотложные нужды", "Автокредит" и т.п. Вид кредита указывается для каждого кредитного продукта, например, для кредитного продукта "Эконом" вид кредита будет "Автокредит". Вид кредита определяет поля анкеты на кредит, общие для всех входящих в него кредитных продуктов.

В данном пункте меню автоматически открывается страница со списком видов кредитов. По каждому виду в списке отображается наименование вида кредита и его описание. Вы можете добавить, отредактировать и удалить виды кредитов банка.

  • Создание вида кредита

    Для того чтобы добавить новый вид кредита, щелкните по кнопке Добавить, откроется страница редактирования вида кредита. На данной странице введите в поле "наименование" название вида кредита, в поле "описание" укажите сведения о кредите, в поле "описание заявки" щелкните по кнопке Обзор, выберите из списка необходимый xml-файл с описанием заявки, в поле "XSTL шаблон детальной информации" щелкните по кнопке Обзор, укажите из списка XSTL-файл с подробной информацией по данному виду кредита.

    Вы также можете настроить отображение кредитного калькулятора по данному виду кредита: для того чтобы создаваемый вид кредита отобразился в кредитном калькуляторе, поставьте галочку напротив строки "кредитный калькулятор"; если данный вид кредита предназначен для покупки товара, установите галочку напротив строки "целевой кредит", и у клиента появится поле "стоимость товара" в кредитном калькуляторе для расчета суммы кредита.

    После того как все сведения введены, нажмите кнопку Сохранить, и система покажет новый вид кредита в списке.

  • Редактирование вида кредита

    Для редактирования вида кредита кликните на его названии, откроется страница для редактирования. На данной странице при необходимости измените значения в полях "наименование", "описание", "описание заявки" и "XSTL шаблон детальной информации".

    Вы имеете возможность предварительно сохранить старый xml-файл и одновременно выгрузить его из базы, для этого щелкните по кнопке Выгрузить описание заявки, xml-файл выгрузится из базы, а затем нажмите кнопку Сохранить.

    Для того чтобы отредактировать поля "описание заявки" и "XSTL шаблон детальной информации", в поле "описание заявки" щелкните по кнопке Обзор, выберите из списка xml-файл с описанием заявки, в поле "XSTL шаблон детальной информации" щелкните по кнопке Обзор, укажите из списка XSTL-файл с подробной информацией по данному виду кредита, и нажмите кнопку Сохранить.

  • Удаление вида кредита

    Если Вы хотите удалить вид кредита из списка, установите галочку напротив интересующего Вас вида кредита и щелкните по кнопке Удалить.

Кредитные офисы

Пункт меню Кредитные офисы предназначен для того, чтобы завести кредитные офисы, к которым будут привязаны кредитные продукты.

В данном пункте открывается список кредитных офисов банка. По каждому офису в списке отображается его название, адрес, телефон и дополнительная информация.

При первоначальной настройке системы справочник кредитных офисов реплицируется из RS-Loans, репликация справочников осуществляется в пункте меню Сервис - Загрузка справочников. При последующих запусках репликации этого справочника новые офисы добавляются, а старые остаются без изменений.

Вы можете изменить информацию о кредитных офисах, удалить офис из списка и указать головной офис.

  • Для того чтобы указать головной офис в списке, установите напротив интересующего Вас офиса галочку, нажмите на кнопку Головной офис, и офис будет головным (в списке он выделяется жирным шрифтом). Сотрудник, который привязан к головному офису, сможет видеть заявки всех кредитных офисов банка.

  • Если Вы хотите отредактировать информацию о кредитном офисе, кликните на названии кредитного офиса в списке. Откроется форма редактирования кредитного офиса, в которой можно редактировать все поля, кроме идентификатора. После того как все сведения изменены, щелкните по кнопке Сохранить.

  • Если Вы хотите удалить кредитный офис (офисы), выберите в списке все офисы, которые нужно удалить, установив напротив них галочки, и щелкните по кнопке Удалить.
