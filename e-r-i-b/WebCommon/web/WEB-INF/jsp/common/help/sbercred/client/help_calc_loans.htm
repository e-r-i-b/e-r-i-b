Кредитный калькулятор

Кредитный калькулятор позволит вам получить информацию по любому виду кредита. Он позволяет рассчитать сумму ежемесячных выплат и комиссий, а также эффективную процентную ставку по определенному кредитному продукту. Для доступа к калькулятору выберите в главном меню пункт Кредиты, а затем щелкните по пункту Кредитный калькулятор в левом меню.

В форме кредитного калькулятора вам надо указать следующие сведения:

  • Сначала в поле Офис из предоставляемого списка выберите тот офис банка, где вы собираетесь получать кредит.

  • В поле Вид кредита из предоставляемого системой списка выберите интересующий вас кредит.

  • В поле Кредитный продукт из предлагаемого списка кредитных продуктов надо выбрать тот, для которого будет производиться расчет.

  • В поле Валюта кредита выбирается валюта кредита.

  • В поле Срок кредита выбирается один из возможных сроков для выбранного кредитного продукта.

  • В поле Процентная ставка по кредиту отображается годовая процентная ставка соответствующая выбранному кредитному продукту. Это поле заполняется автоматически и вам ничего вводить здесь не нужно.

  • В поле Сумма первоначального взноса по умолчанию подставляется минимальная сумма взноса для выбранного кредитного продукта. Вы можете увеличить сумму первоначального взноса по кредиту.

  • Поле Сумма кредита отображается только если вы выбрали кредит наличными. В нем надо указать сумму, которую вы расчитываете получить.

Заполнив необходимые поля, щелкните по кнопке Рассчитать. В результате на экран выводятся следующие данные:

  • Сумма кредита (получаемая в кредит сумма без комиссий и процентов).

  • Сумма ежемесячного платежа по кредиту, которая складывается из выплат по основному долгу, выплат по процентам, а так же комиссий за ведение ссудного счета.

  • Комиссия за ведение ссудного счета и способ расчета комиссии.

  • Полная стоимость кредита (разница между суммой всех плановых платежей, включая комиссии, и суммой кредита).

  • Разница между стоимостью товара, приобретенного за наличные и стоимостью товара приобретенного в кредит.

  • Эффективная процентная ставка по кредиту, которая вычисляется на основе введённых данных.

      • Дальше располагается таблица платежей по месяцам.
