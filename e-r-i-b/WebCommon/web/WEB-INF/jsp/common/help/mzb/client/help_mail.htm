Письма

Система позволяет Вам отправлять в банк и получать из банка сообщения свободного формата — письма. Для работы с письмами выберите в главном меню пункт Почта. На экран будет выведен список полученных вами писем. В нем вы можете выполнить следующие действия:

  • Отправить сообщение в банк. Для этого щелкните по кнопке Создать и заполните поля предложенной формы.

  • Просмотреть полученное из банка письмо. Для того, чтобы увидеть полный текст полученного из банка письма, щелкните по отображаемой в списке теме письма.

  • Ответить на полученное из банка письмо. Для этого отметьте флажком то письмо, на которое вы хотите ответить, щелкните по расположенной над списком кнопке Ответить и заполните поля предложенной формы.

  • Просмотреть список отправленных вами в банк писем. Для этого щелкните по расположенной в левом меню команде Список отправленных писем. Любое письмо из списка отправленных можно просмотреть полностью, щелкнув по его номеру.
