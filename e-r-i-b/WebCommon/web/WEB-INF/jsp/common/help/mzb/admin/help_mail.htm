Пункт меню Письма предназначен для работы с письмами: для просмотра входящих писем от клиентов банка и отправки им писем.

Список входящих писем

Фильтр входящих писем

При выборе пункта меню Письма на экран выводится список писем, полученных от клиентов банка. Вы можете создать новое письмо, просмотреть и ответить на письмо клиента банка. По каждому письму в нём отображаются: "номер", "отправитель", "тема", "дата", "тип письма".

Вы можете просмотреть текст любого письма. Для этого установите флажок напротив интересующего Вас письма от клиента банка и щелкните по кнопке Просмотр или кликните на номере письма.

Если вы хотите ответить на письмо клиента банка, установите флажок напротив выбранного письма или кликните на номере письма и щелкните по кнопке Ответить. Откроется страница редактирования письма. Поля "номер", "дата", "время" и "получатель" письма автоматически заполняются системой. Вы должны заполнить поля "тема" (можно оставить тему письма клиента банка), "текст" сообщения и щелкнуть по кнопке Отправить. Если Вы не хотите отвечать на письмо, щелкните по кнопке Отмена.

При необходимости Вы можете вернуться к списку входящих писем, для этого щелкните по кнопке Список.

Кроме того, Вы можете осуществлять фильтрацию сообщений в списке.

Фильтр входящих писем

Для просмотра списка всех интересующих Вас входящих писем нужно щелкнуть по кнопке Показать фильтр и правильно заполнить все поля фильтра:

  • "Фамилия, имя, отчество" - поиск по имени клиента банка, отправившего письмо;

  • "Период с.. по.." - поиск по дате и времени получения сообщения сотрудником банка,

  • "Тема письма" - поиск по тематике письма;

  • "Тип сообщения " - поиск по типу сообщения;

  • "Номер письма " - поиск по номеру письма;

Вы можете осуществлять поиск по подстроке в текстовых полях фильтра. Например, можно ввести в поля фильтра только несколько первых букв фамилии, имени или отчества клиента банка.

Также Вы можете фильтровать список по любой комбинации полей. Например, можно отобрать письма всех клиентов с фамилией на букву "В" с одинаковой тематикой письма.

После того как искомые параметры поиска заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас писем.

Список отправленных писем

Фильтр отправленных писем

Сотрудник банка может создавать и отправлять новые письма, адресованные определенному клиенту или группе клиентов.

Вы можете просмотреть текст любого письма. Для этого установите флажок напротив интересующего Вас письма, отправленного клиенту или группе клиентов банка, и щелкните по кнопке Просмотр или кликните на номере письма.

Чтобы отправить клиенту банка новое письмо, щелкните по кнопке Создать, откроется форма редактирования сообщения, в которой Вы должны заполнить все поля:

  • Значения в полях "номер сообщения", "дата и время отправки сообщения" проставляются системой автоматически;

  • В поле "получатель" Вы должны указать клиента или группу клиентов, которым хотите отправить сообщение. Для этого установите галочку напротив выбранного вида получателя и щелкните по кнопке Продолжить. Откроется страница со списком клиентов или групп клиентов, на которой Вы должны установить галочку напротив интересующего Вас клиента или группы клиентов, и щелкнуть по кнопке Выбрать.

  • В поле "тема сообщения" введите тему письма.

  • В поле "текст сообщения" впишите полный текст письма.

После заполнения всех полей формы создания письма щелкните по кнопке Отправить, и письмо будет отправлено выбранному Вами получателю. Если Вы не хотите отправлять письмо клиенту или группе клиентов банка, щелкните по кнопке Отмена.

Фильтр отправленных писем

Для просмотра списка всех интересующих Вас отправленных писем нужно щелкнуть по кнопке Показать фильтр и правильно заполнить все поля фильтра:

  • "Фамилия, имя, отчество" - поиск по имени клиента банка, которому отправлено письмо;

  • "Имя группы " - поиск по по названию группы, в которой состоит клиент банка;

  • "Тема письма" - поиск по тематике письма;

  • "Тип получателя" - поиск по типу получателя (по ФИО клиента или по названию группы клиентов);

  • "Период с.. по.." - поиск по дате и времени получения сообщения сотрудником банка.

Вы можете осуществлять поиск по подстроке в текстовых полях фильтра. Например, можно ввести в поля фильтра только несколько первых букв фамилии, имени или отчества клиента банка.

Также Вы можете фильтровать список по любой комбинации полей. Например, можно отобрать письма всех клиентов с фамилией на букву "В" с одинаковой тематикой письма.

После того как искомые параметры поиска заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас писем.
