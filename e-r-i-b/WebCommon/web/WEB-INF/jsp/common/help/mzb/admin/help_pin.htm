С помощью пункта меню Пин-конверты Вы можете выгрузить запрос на генерацию ПИН-конверта, содержащего выдаваемые клиентам для регистрации в системе логин и пароль и распечатать пин-конверт через утилиту печати.

Заявка на печать пин-конвертов

Для того чтобы создать запрос на печать пин-конвертов, заполните поля формы создания запроса на печать. В поле "количество пин-конвертов в запросе" введите количество пин-конвертов, в поле "подразделение" выберите из справочника подразделение банка, для которого создаются пин-конверты. Для этого на странице списка подразделений установите галочку напротив интересующего Вас подразделения и нажмите кнопку Выбрать.

Причем на странице списка подразделений Вы можете отфильтровать интересующие Вас подразделения банка по названию. Чтобы осуществить поиск, нажмите кнопку Показать фильтр, в поле "наименование" введите полностью или первые буквы названия подразделения банка, щелкните по кнопке Применить, система покажет интересующие Вас подразделения.

После заполнения полей формы создания запроса щелкните по кнопке Выгрузить. Откроется стандартное окно Windows Сохранить. Сохраните файл на сменный носитель (диск, "флешку"), отнесите его на сервер, где установлена утилита печати пин-конвертов, скопируйте файл в каталог XML_in, запустите утилиту печати и распечатайте пин-конверты.

Загрузка паролей клиентов

Чтобы загрузить ответ из утилиты печати, Вам необходимо на сервере, где установлена утилита печати, скопировать из каталога XML_out файл ответа на сменный носитель, отнести его на компьютер, где запущена система ИКФЛ. Для того чтобы загрузить пароли клиента в систему, необходимо в пункте Пин-конверты - Загрузка паролей клиентов щелкнуть по кнопке Обзор, выбрать путь до файла с ответом и нажать кнопку Выполнить импорт. После чего пароли клиентов банка будут загружены в систему.
