Журнал операций

Журнал содержит информацию по всем, совершенным Вами платежным операциям, независимо от их вида (перевод средств, оплата товаров и услуг и пр.).

В зависимости от выбранного пункта меню вы можете просмотреть либо журнал всех операций, либо журнал незавершенных операций. В первом случае в журнале отображаются все платежи клиента за текущую дату, во втором — все незавершенные платежи, т.е. не имеющие статусов «Исполнен», «Отозван» и «Отказан».

По каждой платежной операции в журнале выводится следующая информация:

  • Дата и время совершения операции. Соответствует дате и времени отправки пользователем платежного документа на обработку в банк.

  • Дата приема к исполнению документа банком. Дата может отличаться от даты совершения операции в случае отправки документа по окончании операционного дня.

  • Номер документа. Отображается номер документа, который был ему присвоен при его создании.

  • Вид операции. Выводится наименование вида платежной операции, например: «Пополнение вклада», «Перевод рублей РФ», «Оплата товаров/услуг за...» и т.п.

  • Счет списания. Счет клиента, с которого списываются средства для совершения платежной операции.

  • Валюта платежа. Отображается символьный код валюты платежа.

  • Сумма платежа. Сумма платежной операции без учета комиссии банка за её совершение.

  • Сумма комиссии. Сумма комиссии банка за совершение данной платежной операции.

  • Сумма операции. Итоговая сумма платежной операции с учетом комиссии банка.

  • Статус операции. Поле определяет текущее состояние процесса обработки платежного документа.

  • Наименование получателя платежа.

Находясь в данном списке вы можете выполнять следующие операции:

  • Просмотр документа

    Вы можете просмотреть любой документ, для чего достаточно просто щелкнуть по номеру интересующего Вас документа в таблице.

  • Отзыв документа

    Вы можете отозвать ошибочно отправленный документ. Для этого отметьте флажком отзываемый документ и щелкните по кнопке Отозвать , расположенной в верхней части страницы. В результате вы перейдете к форме отзыва перевода, где для формирования отзыва необходимо щелкнуть по кнопке Сохранить . Информация в полях отзыва для редактирования недоступна.

  • Редактирование документа

    Если документ еще не передан на обработку в банк, вы можете отредактировать его. Для этого отметьте редактируемый документ в списке и щелкните по по кнопке Редактировать , расположенной в верхней части страницы. На экран будет выведена форма ввода документа, в которой вы можете менять его содержимое.

  • Создание копии документа

    Вы можете создать копию документа, отметив документ, который служит оригиналом в списке, и щелкнув по кнопке Создать копию , расположенной в верхней части страницы. В результате вы перейдете к форме ввода нового документа, причем в нее будут скопированы все данные из документа, выбранного Вами в качестве оригинала. Данная возможность полезна при создании нескольких однотипных документов.

По умолчанию в таблице отображаются только документы за текущий день. Если документов слишком много, то в таблице будут отображены только первые 20 документов. Для поиска необходимых документов в системе предусмотрен фильтр. Чтобы воспользоваться им щелкните по ссылке Показать фильтр, расположенной над таблицей с документами. Можно применять следующие фильтры:

  • Фильтр по номеру документа. Для данного вида фильтра укажите фрагмент номера в поле Номер. В результате будут показаны все документы, в номере которых есть указанный фрагмент. Например, если Вы введете в поле Номер цифру 1, то в результате применения фильтра будут выбраны все документы в номере которых есть цифра 1.

  • Фильтр по счету. В поле Счет из предоставляемого системой списка Вы можете выбрать отдельный счет. В результате в таблицу попадут только те документы, в которых данный счет указан как счет списания.

  • Фильтр по дате совершения операции. В данном фильтре вы задаете диапазон дат. В формируемую таблицу попадут все документы, дата совершения операции которых попадает в заданный диапазон.

  • Фильтр по дате приема к исполнению. Фильтр аналогичен предыдущему. Единственное отличие в том, что выборка производится по дате приема документа к исполнению, а не по дате совершения операции.

  • Фильтр по наименованию получателя. Для данного вида фильтра укажите в поле фрагмент наименования получателя. В результате будут показаны все документы, где в наименовании получателя есть указанный фрагмент. Например, если Вы введете в поле слово «банк», то в результате применения фильтра будут выбраны все документы в наименовании получателя которых есть фрагмент «банк» (этот фрагмент может быть как отдельным словом, так и частью другого, более длинного слова). Обратите внимание, что при поиске учитывается регистр символов.

  • Фильтр по виду операции. В поле Тип операции из предоставляемого системой списка Вы можете выбрать отдельный тип операции. В результате в таблицу попадут только документы выбранного типа.

  • Фильтр по статусу документа. В поле Статус из предоставляемого системой списка Вы можете выбрать конкретный статус документа. В результате в таблицу попадут только документы с указанным статусом.

  • Фильтр по сумме операции. В данном фильтре вы задаете диапазон сумм. В формируемую таблицу попадут все документы, сумма списания в которых попадает в заданный диапазон.

Вы можете произвольным образом комбинировать эти фильтры, чтобы ужесточить критерии отбора и сократить список проходящих фильтр документов. Фильтр устанавливается после щелчка по кнопке Применить. Если вы хотите снять фильтр, надо открыть панель фильтра, щелкнуть по кнопке Очистить, а затем щелкнуть по кнопке Применить.
