Список депозитных продуктов

Пункт меню Депозиты предназначен для управления депозитными продуктами банка, которые будут доступны для клиентов в системе ИКФЛ. При выборе данного пункта меню открывается список депозитных продуктов банка.

В списке депозитных продуктов по каждому из них показывается следующая информация:

  • Наименование вклада. Наименование депозитного продукта банка.

  • Сумма вклада. Отображается перечень диапазонов суммы вклада в зависимости от валюты и срока размещения средств.

  • Срок вклада. Отображается перечень возможных сроков вклада для данного депозитного продукта.

  • Процентная ставка по вкладу. Отображается процентная ставка (% годовых) для данного депозитного продукта (вида вклада) в зависимости от валюты, срока и суммы размещаемых денежных средств.

Управление депозитными продуктами

Для работы со списком депозитных продуктов банка Вам необходимо их завести в системе ИКФЛ. Существует два способа заведения депозитных продуктов:

  1. Автоматически, т.е. можно реплицировать депозитные продукты из системы RS-Retail. Для этого щелкните по кнопке Репликация, и в систему ИКФЛ реплицируется из системы RS-Retail информация о всех вкладах банка.

  2. Ручное добавление депозитных продуктов. Для этого нажмите кнопку Добавить, откроется страница редактирования депозитного продукта. В поле "имя депозитного продукта" Вы должны написать название создаваемого депозитного продукта. Затем укажите перечень видов вкладов, входящих в состав депозитного продукта, установив флажки напротив интересующих Вас видов вкладов и щелкнув по кнопке Сохранить, и новый депозитный продукт появится в списке депозитных продуктов банка. Сотрудник банка сам определяет, какие виды вкладов входят в состав депозитного продукта банка.

На странице Редактирования депозитных продуктов банка Вы можете управлять депозитными продуктами банка следующим образом:

  • Вы можете добавить в список депозитный продукт. Для этого щелкните по кнопке Добавить, откроется страница редактирования депозитного продукта. В поле "имя депозитного продукта" Вы должны написать название создаваемого депозитного продукта. В форме ввода депозитных продуктов по каждому вкладу в системе показывается: идентификатор (уникальное имя депозитного продукта в системе), название (название депозитного продукта), валюта (валюта, в которой производится вклад). Затем укажите перечень видов вкладов, входящих в состав депозитного продукта, установите флажки напротив интересующих Вас видов вкладов и щелкните по кнопке Сохранить, и новый депозитный продукт появится в списке депозитных продуктов банка.

  • Если Вам необходимо отредактировать депозитный продукт банка, в списке продуктов выберите интересующий Вас депозитный продукт, установите напротив него флажок и щелкните по кнопке Изменить. Откроется страница редактирования депозитного продукта банка, на которой Вы можете изменить название продукта и перечень видов вкладов, входящих в состав депозитного продукта, затем щелкните по кнопке Cохранить, и депозитный продукт появится в списке в измененном виде.

  • Если Вам нужно удалить депозитный продукт банка из списка, установите флажок напротив выбранного депозитного продукта и щелкните по кнопке Удалить.
