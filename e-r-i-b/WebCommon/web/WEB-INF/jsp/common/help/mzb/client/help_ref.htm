Справочники

В системе предусмотрены справочники, которые облегчают заполнение документов и позволяют избежать возникновения опечаток в реквизитах документов.

Справочник получателей

В данном справочнике хранятся реквизиты получателей, которые Вы можете использовать при заполнении рублевых переводов и налоговых платежей. Для каждого получателя в справочнике хранятся его название (или ФИО для физического лица), коды ИНН и КПП, адрес, номер счета и реквизиты банка получателя, в котором открыт данный счет. При выборе получателя из справочника все эти сведения автоматически подставляются в документ.

Справочник банков

В данном справочнике хранятся реквизиты банков, которые Вы можете использовать при заполнении рублевых переводов и налоговых платежей. Для каждого банка в справочнике хранятся его название, адрес, код БИК и номер корреспондентского счета. При выборе банка из справочника все эти сведения автоматически подставляются в документ. Справочник банков предоставляется вам уже заполненным, вы не можете добавлять в него новые записи или редактировать имеющиеся.
