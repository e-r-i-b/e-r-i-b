С помощью пункта меню Настройки задаются параметры системы. Вы можете задать настройки, определяющие функционирование системы.

Клиенты и сотрудники

Пункт меню Клиенты и сотрудники предназначен для управления настройками системы, которые регулируют работу клиентов и сотрудников банка в системе. В данном пункте меню Вы можете управлять следующими настройками системы:

  1. Разрешенные символы для пароля

    Данная настройка предназначена для управления символами, которые создаются в системе для ввода пароля. Чтобы изменить допустимые символы, используемые для создания паролей, в поле "разрешенные символы для пароля" выберите из списка нужное значение и нажмите кнопку Сохранить. Например, если Вы выбираете значение "только цифры", то при создании пароля утилита печати генерирует пароль, состоящий только из цифр. После сохранения настройка будет изменена в системе и вступит в действие.

  2. Разрешенные символы для логина

    Данная настройка используется для изменения допустимых символов, используемых при создании логинов. Чтобы изменить разрешенные символы для логина, в поле "разрешенные символы для логина" выберите из списка необходимое значение и нажмите кнопку Сохранить. Например, если Вы выбираете значение "цифры и заглавные буквы", то при система будет генерировать логины для сотрудников и клиентов, состоящие только из цифр и заглавных букв. После сохранения настройка будет изменена в системе и вступит в действие.

  3. Настройка длины пароля

    Данная настройка предназначена для определения, какой длины будут создаваемые утилитой печати пароли, и для общего ограничения длины паролей, вводимых клиентами и сотрудниками банка. Если Вы хотите изменить длину создаваемых паролей, введите количество допустимых символов в поле "длина пароля" и щелкните по кнопке Сохранить. После сохранения настройка вступит в действие.

  4. Количество попыток при регистрации

    Данная настройка используется для изменения количества попыток для ввода пароля до автоматической блокировки доступа в систему. Чтобы изменить количество попыток, введите в поле "количество попыток (регистрация)" число попыток и щелкните по кнопке Сохранить. После сохранения настройка вступит в действие.

  5. Период блокировки

    Данная настройка предназначена для изменения времени, на которое блокируется доступ к системе при многократном неверном вводе пароля. Для изменения периода блокировки введите в поле "период блокировки" числовое значение (период блокировки исчисляется в секундах) и щелкните по кнопке Сохранить.

    Если Вы меняете только одну настройку, то после ее изменения нажимаете по кнопке Сохранить, если Вы меняете сразу несколько настроек, то Вы вводите изменение в каждое изменяемое поле и потом нажимаете Сохранить.

    Эти параметры действуют для всех зарегистрированных сотрудников и клиентов и вступают в силу сразу же после сохранения изменений.

Права доступа сотрудника

Пункт Права доступа сотрудника позволяет задать схему прав, которая автоматически назначается при создании нового сотрудника банка в системе.

В данном пункте меню в поле "схема прав сотрудника" выберите из списка схему прав, которая будет по умолчанию назначена сотрудникам банка, после чего щелкните по кнопке Сохранить. Настройка вступит в действие после сохранения.

Права доступа клиента

Пункт Права доступа клиента позволяет задать способ аутентификации клиентов банка по умолчанию, для каждого способа задается схема прав, которая автоматически назначается при создании нового клиента банка в системе, а также способ подтверждения входа и способ подтверждения операций клиентами банка.

Упрощенный доступ

Данный тип доступа используется при работе клиентов банка в системе ИКФЛ без сертификата. Для того чтобы задать упрощенный доступ для создаваемых в системе клиентов банка, установите флажок напротив строки "упрощенный доступ". При данном типе доступа Вы можете управлять следующими настройками системы:

  1. Подтверждение входа

    Настройка Подтверждение входа предназначена для указания того, каким образом клиент банка будет подтверждать вход в систему. Для изменения данной настройки выберите интересующее Вас значение в поле "подтверждение входа":

    "Без подтверждения" - при клиент банка входит в систему только при помощи логина и пароля;

    "Одноразовый пароль" - при входе в систему клиент банка должен ввести не только логин, пароль, а также требуемый системой ключ из карты ключей.

  2. Подтверждение операций

    Настройка Подтверждение операций используется для определения того, необходимо ли клиенту банка подтверждать совершенные в системе операции одноразовым паролем с карты ключей. Для изменения данной настройки выберите в поле "подтверждение операций" одно из значений:

    "Без подтверждения" - клиент банка совершает операции в системе без дополнительного подтверждения;

    "Одноразовый пароль" - при совершении операции клиент банка должен ввести требуемый системой ключ из карты ключей.

  3. Схема прав

    Настройка Схема прав предназначена для указания схемы прав, назначаемой системой для упрощенного доступа по умолчанию. Для изменения данной настройки в поле "схема прав" задайте из списка интересующую Вас схему прав и нажмите кнопку Сохранить.

После заполнения всех полей щелкните по кнопке Сохранить.

Защищенный доступ

Данный тип доступа предназначен для входа клиентов банка в систему в безопасном режиме по сертификату СКЗИ, для подтверждения операций и подписания документов электронно-цифровой подписью (ЭЦП). Для того чтобы задать защищенный доступ для создаваемых в системе клиентов банка, установите флажок напротив строки "защищенный доступ". При данном типе прав доступа Вы можете управлять следующими настройками системы:

  1. Подтверждение входа

    Данная настройка определяет способ подтверждения входа клиентов банка в систему. По умолчанию указывается значение "ЭЦП".

  2. Подтверждение операций

    Данная настройка определяет способ подтверждения входа клиентов банка в систему. По умолчанию указывается значение "ЭЦП".

  3. Схема прав

    Данная настройка регулирует, какая схема прав будет назначаться клиентам банка по умолчанию при аутентификации по защищенному доступу. Чтобы задать схему прав доступа для клиентов банка, в поле "схема прав" задайте из списка интересующую Вас схему прав и нажмите кнопку Сохранить.

    После заполнения всех полей щелкните по кнопке Сохранить, и настройка вступит в действие.

    Карты ключей

    В пункте меню Карты ключей можно указать, сколько карт ключей будет генерироваться одновременно, количество ключей на каждой из создаваемых карт, набор символов, из которых будут состоять создаваемые ключи, длину ключей в карте. Вы можете управлять следующими настройками генерации карт ключей в системе:

    1. Создание карт

      Данная настройка предназначена для управления количеством карт и ключей на карте при создании карт ключей. Для изменения настроек при создании карт Вам нужно:

      • в поле "создание карт" укажите количество карт, создаваемых за одну процедуру генерации, для этого введите в данное поле числовое значение от 1 до 500;

      • в поле "карт по" введите числовое значение от 1 до 500, определяющее количество ключей на карте, и щелкните по кнопке Сохранить. После сохранения настройка вступит в действие.

    2. Разрешенные символы

      Данная настройка позволяет управлять символами, используемыми системой при генерации карт ключей. Для изменения разрешенных символов в поле "разрешенные символы" выберите из списка интересующее Вас значение и щелкните по кнопке Сохранить. Настройка вступит в действие после сохранения.

    3. Длина ключа

      Данная настройка позволяет Вам регулировать длину ключей на карте, генерируемых системой. Для изменения длины пароля в поле "длина пароля" введите числовое значение от 1 до 20, затем щелкните по кнопке Сохранить.

      Если Вы меняете только одну настройку, то после ее изменения нажимаете по кнопке Сохранить, если Вы меняете сразу несколько настроек, то Вы вводите изменение в каждое изменяемое поле и потом нажимаете Сохранить.

    Указанные здесь настройки генерации карт ключей будут использоваться по умолчанию, но при генерации карт в пункте главного меню Карты ключей - Генерация карт их можно изменить.

    В данном пункте меню можно настроить виды периодичности оповещений, которые будут доступны сотруднику банка для подписки. В списке расписаний по каждому оповещению есть возможность настроить параметры, задающие его периодичность.

    Список расписаний сотрудников

    В списке расписаний по каждому оповещению отображаются:

    1. Вид настройки;

    2. Периодичность срабатывания оповещения;

    3. Название оповещения.

    Вид оповещения может быть одним из следующих:

    • "Пополнение пластиковой карты" - через заданный промежуток времени система проверяет, была ли такая операция совершена кем-либо из клиентов банка, в случае пополнения, сотруднику банка придет сообщение о том, что клиенты совершили пополнение карты.

    • "Блокировка пользователя" - через заданный промежуток времени система проверяет, изменился ли статус пользователей на "Заблокирован", если статус изменился, то сотруднику банка придет сообщение о блокировке пользователей.

    • "Окончание срока действия сертификата" - через заданный промежуток времени система проверяет срок действия сертификатов клиентов банка, если срок действия истек, то сотруднику банка придет оповещение.

    • "Блокировка карты ключей" - через заданный промежуток времени система проверяет, изменился ли статус карт ключей клиентов банка на "Заблокирована", если статус изменился, то сотруднику банка придет сообщение.

    • "Совершение платежей и ввод заявок" - через заданный промежуток времени система проверяет количество совершенных платежей и ввод заявок клиентами банка, если кто-либо из клиентов совершил операцию, то сотруднику банка придет сообщение.

    • "Оповещения о системных событиях" - система через заданный промежуток времени информирует сотрудника банка о системных событиях.

    Добавление расписания

    В список расписаний Вы можете добавить новое расписание. Для этого щелкните по кнопке Добавить, откроется форма создания рассылки, в которой в поле "рассылка" выберите из справочника интересующий Вас вид рассылки сообщений, в поле "выражение" введите то значение, которое в системе соответствует данной рассылке, в поле "описание" введите содержание описания новой рассылки и щелкните по кнопке Сохранить, и созданная Вами рассылка появится в списке расписаний. Подробную информацию по созданию и изменению регулярного выражения Вы можете прочитать в "Руководстве сотрудника банка".

    Редактирование расписания

    Список расписаний также можно отредактировать. Для этого установите галочку напротив интересующего Вас вида рассылки, нажмите кнопку Редактировать. Система покажет Вам форму редактирования рассылки, в которой Вы можете изменить значение в поле "выражение" и в поле "описание". После изменения параметров нажмите кнопку Сохранить, и данная рассылка появится в списке рассылок в измененном виде.

    Удаление расписания

    Если Вы хотите удалить расписание оповещения, установите галочку напротив интересующей Вас рассылки и щелкните по кнопке Удалить. Система попросит Вас подтвердить удаление выбранной рассылки, после подтверждения данная рассылка будет удалена из списка расписаний.

    Расписания клиентов

    Список расписаний клиентов

    Удаление расписания

    В данном пункте меню можно настроить виды периодичности оповещений, которые будут рассылаться клиентам банка. В списке расписаний по каждому оповещению есть возможность настроить параметры, задающие его периодичность.

    Список расписаний клиентов

    В списке расписаний по каждому оповещению отображаются:

    1. Вид настройки;

    2. Периодичность срабатывания оповещения;

    3. Название оповещения.

    Вид оповещения может быть одним из следующих:

    • "Окончание срока действия сертификата" - система через заданный промежуток времени проверяет срок действия сертификатов клиентов банка, если срок истек, то клиентам придет сообщение на e-mail или sms-сообщение.

    • "Блокировка карты ключей" - система через заданный промежуток времени проверяет изменение статуса карт ключей клиентов банка на "Заблокирована", если статус карт изменился, то клиентам придет сообщение на e-mail или sms-сообщение.

    • "Окончание срока действия депозита" - система через заданный промежуток времени проверяет приближение даты окончания депозитов клиентов банка, при приближении срока клиентам придет оповещение на e-mail или sms-сообщение.

    • "Выписка по счетам и картам" - система с заданной периодичностью высылает клиентам банка на e-mail оповещение, содержащее выписку по счетам и картам клиентов банка.

    • "Информация об остатках" - система с заданной периодичностью высылает клиентам банка на e-mail или в виде sms оповещение, содержащее информацию о текущих остатках средств на счетах клиентов банка.

    • "Остаток меньше заданного значения" - система через заданный промежуток времени проверяет остаток на счетах клиентов банка, если он меньше заданного значения, то клиентам приходит оповещение на e-mail или sms-сообщение.

    • "Регистрация в системе" - система через заданный промежуток времени проверяет регистрацию новых клиентов, если новые клиенты банка зарегистрированы в системе, то клиентам придет оповещение об успешном входе в систему на e-mail или sms-сообщение.

    • "Блокировка пользователя" - система через заданный промежуток времени проверяет статус клиентов банка, если статус изменился на "Заблокирован", то клиентам придет оповещение на e-mail или sms-сообщение.

    • "Изменение статуса платежей" - через заданный промежуток времени система проверяет изменение статусов платежей клиентов банка, при изменении состояния платежей клиентам придет оповещение на e-mail или sms-сообщение.

    • "Изменение остатка на счете" - система через заданный промежуток времени проверяет изменение остатка на счетах в результате проведения операций клиентами банка, при изменении остатка на счетах клиентам придет оповещение на e-mail или sms-сообщение.

    Добавление расписания

    В список расписаний Вы можете добавить новое расписание. Для этого щелкните по кнопке Добавить и в поле "рассылка" выберите из справочника интересующий Вас вид рассылки сообщений, в поле "выражение" введите то значение, которое в системе соответствует данной рассылке, в поле "описание" введите содержание описания новой рассылки и щелкните по кнопке Сохранить. Подробную информацию по созданию и изменению регулярного выражения Вы можете прочитать в "Руководстве сотрудника банка".

    Редактирование расписания

    Список расписаний также можно отредактировать. Для этого установите галочку напротив интересующего Вас сообщения, нажмите кнопку Редактировать. Система покажет Вам параметры рассылки, среди которых Вы можете изменить значение в поле "выражение" и в поле "описание". После изменения параметров нажмите кнопку Сохранить и данная рассылка появится в списке рассылок в измененном виде.

    Удаление расписания

    Если Вы хотите удалить расписание сообщения, установите галочку напротив интересующей Вас рассылки и щелкните по кнопке Удалить. Система попросит Вас подтвердить удаление выбранной рассылки, после подтверждения данная рассылка будет удалена из списка расписаний.
