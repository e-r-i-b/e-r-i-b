Пункт меню Подразделения позволяет управлять списком подразделений банка (добавлять, удалять и редактировать сведения о подразделениях), а также управлять списком работающих в подразделениях уполномоченных лиц.

Список подразделений

Список подразделений открывается автоматически в главном меню в пункте Подразделения. В нем содержится список подразделений банка, в которых предоставляется услуга Интернет-банкинга. Все сотрудники и клиенты банка привязываются к одному из данных подразделений.

Для того чтобы отредактировать сведения о подразделении, в списке подразделений выберите подразделение, кликните на названии подразделения или установите флажок напротив названия интересующего Вас подразделения и щелкните по кнопке Изменить. Откроется страница редактирования подразделения банка, на которой нужно изменить значения интересующих Вас полей, затем щелкните по кнопке Сохранить, и система покажет данное подразделение в списке в измененном виде.

Если необходимо удалить запись о подразделении, установите флажок напротив интересующего Вас подразделения и щелкните по кнопке Удалить. Система попросит Вас подтвердить операцию, Вы подтверждаете удаление, и, если к выбранному подразделению на привязаны сотрудники и клиенты, подразделение будет удалено из списка.

Общие сведения

Вы можете добавить новое подразделение в список. Для этого щелкните по кнопке Добавить подразделение. На экран будет выведена форма, в полях которой необходимо указать сведения о добавляемом подразделении. Система экспортирует все поля данного документа в договор на предоставление услуг Интернет-банкинга. В поле "наименование" Вам нужно указать название подразделения. Для этого щелкните по кнопке <...>(для вызова списка подразделений) и выберите из списка интересующее Вас подразделение банка. При выборе подразделения автоматически заполняются поля "местонахождение" и "телефон". Остальные поля формы необязательны для заполнения.

После того как все сведения о подразделении указаны, щелкните по кнопке Сохранить.

Уполномоченные лица

С каждым подразделением банка связан список уполномоченных лиц. Для каждого подразделения можно задать одно или несколько уполномоченных лиц, которые могут заключать договор на подключение клиентов банка к системе. Эта информация подставляется в форму договора.

Если Вы хотите добавить уполномоченное лицо, щелкните по кнопке Добавить и заполните все поля формы: ФИО, должность и документ, удостоверяющий полномочия. После заполнения всех полей формы щелкните по кнопке Сохранить, и система покажет новое уполномоченное лицо в списке уполномоченных лиц данного подразделения банка.

Если Вам необходимо отредактировать сведения об уполномоченном лице, кликните на его имени или установите флажок напротив имени уполномоченного лица и щелкните по кнопке Изменить, откроется форма редактирования списка уполномоченных лиц. В форме Вам нужно изменить интересующие Вас сведения и щелкнуть по кнопке Сохранить.

Если Вы хотите удалить из списка уполномоченное лицо, установите флажок напротив его имени и щелкните по кнопке Удалить. Система попросит Вас подтвердить операцию, Вы подтверждаете, и выбранное Вами уполномоченное лицо будет удалено из списка.
