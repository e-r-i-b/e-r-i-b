Шаблоны документов

Для быстрого осуществления часто совершаемых платежных операций Вы можете использовать шаблоны платежей. Каждый шаблон представляет собой форму некоторого платежа с частично заполненными реквизитами.

Чтобы добавить частично заполненный документ в свой список шаблонов, щелкните по кнопке Сохранить как шаблон , расположенной в верхней части формы ввода документа. После этого система предложит Вам ввести название шабона под которым он будет сохранен в библиотеке.

Вы также можете добавить новый шаблон и из списка шаблонов. Для этого откройте список имеющихся шаблонов, выбрав в главном меню пункт Переводы, а затем щелкнув по значку Шаблоны платежей. На экран будет выведен список имеющихся шаблонов. Щелкните по расположенной над списком кнопке Создать, выберите тип документа и вы перейдете к созданию шаблона в форме ввода документа выбранного типа.

Для создания платежа по шаблону выберите в главном меню пункт Переводы, а затем щелкните по значку Шаблоны платежей. Вам будет предоставлен список имеющихся шаблонов, из которых Вы выбираете нужный, установив находящийся рядом с ним флажок, а затем щелкаете по расположенной над списком кнопке Создать платеж.
