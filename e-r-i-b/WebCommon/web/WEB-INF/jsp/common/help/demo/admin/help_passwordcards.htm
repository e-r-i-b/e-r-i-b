Пункт главного меню Карты ключей предназначен для работы с картами одноразовых паролей. Карты ключей назначаются клиентам банка и используются ими для подтверждения платежных операций. Также возможно дополнительное подтверждение входа клиента в систему.

Карта ключей или Карта разовых паролей - это таблица, которая содержит заданное количество одноразовых паролей (ключей). Каждый ключ в карточке имеет свой порядковый номер.

Список карт

В пункте меню Список карт отображается список всех созданных в системе карт ключей, не привязанных ни к одному клиенту. По каждой карте можно просмотреть номер карты, дату создания и количество ключей на карте.

Создать карту ключей можно типографским способом и через утилиту печати.

Если Вам необходимо удалить карту ключей клиента банка, установите флажок напротив той карты ключей, которую хотите удалить, и щелкните по кнопке Удалить.

Выданные карты ключей клиент банка использует для дополнительного (опционального) подтверждения входа в систему и для подтверждения операций (отправка заявок, подтверждение платежей и т.д.).

Фильтр/поиск карт ключей

Для просмотра списка всех интересующих Вас карт ключей нужно нажать кнопку Показать фильтр и правильно заполнить поля фильтра:

  • "Номер" - поиск по номеру карты,

  • "Количество ключей" - поиск по количеству ключей в карте,

  • "Создана с.. по.." - поиск по дате создания карты.

Осуществлять поиск карт ключей Вы можете по любой комбинации полей, например, можно задать поиск по количеству ключей в карте и по дате создания карты.

После того как искомые параметры заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас карт ключей.

Генерация карт ключей

Данный пункт меню предназначен для создания карт ключей типографским способом (т.е. с помощью печати скретч-карт, когда пароли карты ключей наносятся под защитный слой скретч-карты).

При нажатии на пункт меню Генерация карт ключей открывается форма создания карт ключей, в данной форме укажите количество карт и количество ключей на карте, щелкните по кнопке Создать. Далее сохраните файл на диске, который будет содержать всю необходимую информацию для печати скретч-карты.

Примечание: существует альтернативный способ печати карт ключей - через утилиту печати. См. пункты контекстной справки Заявка на печать карт ключей и Загрузка обработанной заявки.

Заявка на печать карт ключей

Чтобы создать запрос на печать карт ключей, необходимо в форму для создания запроса на печать ввести количество карт в запросе, количество ключей в карте и подразделение банка, для которого изготавливаются карты ключей по данному запросу. Для выбора подразделения щелкните по кнопке (...), откроется справочник подразделений, в котором установите галочку напротив интересующего Вас подразделения и щелкните по кнопке Выбрать. После того как поля формы заполнены, нажмите на кнопку Выгрузить. Откроется стандартное окно Windows Сохранить. Далее Вам необходимо сохранить файл на сменном носителе ("флешке", диске), отнести на сервер, где установлена утилита печати ПИН-конвертов, скопировать файл в каталог XML_in, запустить утилиту и распечатать карту ключей.

Загрузка обработанной заявки

Чтобы загрузить обработанную заявку в систему, Вам нужно на сервере скопировать файл с ответом из утилиты печати из каталога XML_out на сменный носитель, с которого затем скопировать файл ответа на компьютер, где установлена система ИКФЛ. Далее откройте пункт меню Карты ключей - Загрузка обработанной заявки, нажмите на кнопку Обзор, укажите путь до файла с ответом и щелкните по кнопке Выполнить импорт. После выполнения процедуры импорта Вы можете привязывать карты ключей клиентам банка. Для привязки карты ключей Вам необходимо указать номер этой карты в анкете того клиента, которому она будет выдана. Откройте анкету выбранного Вами клиента, выберите вкладку Карты ключей и нажмите на кнопку Добавить. Откроется окно, в котором введите номер карты и нажмите на кнопку Добавить.

Статусы карт ключей

Карты паролей могут иметь в системе следующие статусы:

  • Неактивна - карта выдана пользователю и "закреплена" за ним, но еще не активирована, т.е. действует предыдущая карта;

  • Активна - карта действительна, на ней есть неиспользованные пароли;

  • Использована - все пароли на карте израсходованы, карта далее использоваться не может;ы

  • Заблокирована - карта заблокирована по запросу клиента (потеря, порча и пр.) или автоматически в случае неверного указания пароля с карты. Карта с таким статусом не может быть использована клиентом для работы в системе.
