Обмен валюты

С помощью пункта меню Обмен валюты Вы можете выполнить следующие платежные операции:

  1. Покупка/продажа иностранной валюты - покупка и продажа иностранной валюты за рубли;

  2. Обменять валюту - покупка, продажа иностранной валюты за рубли и конверсия одной иностранной валюты в другую.

Назначение платежа

Данный документ позволяет Вам продать или приобрести иностранную валюту. Чтобы создать перевод, в главном меню выберите пункт Платежи, а затем пункт левого меню Обмен валюты - Покупка/продажа иностранной валюты.

Заполнение документа

В форме покупки и продажи иностранной валюты Вам надо указать дату (в это поле автоматически будет подставлена текущая дата), и выбрать вид выполняемой операции (покупка или продажа):

  • Если Вы хотите купить валюту, нужно указать вид валюты, сумму приобретаемой валюты, счет, с которого будут списаны средства для приобретения валюты, и счет, на который будет зачислена приобретенная валюта.

  • Если Вы хотите продать валюту, надо указать вид валюты, сумму продаваемой валюты, счет для списания средств на продажу валюты и счет для зачисления средств, полученных от продажи валюты.

После того как все поля заполнены, щелкните по кнопке Сохранить. Вы также можете добавить частично заполненный документ в свой список шаблонов, щелкнув по кнопке Сохранить как шаблон, чтобы в дальнейшем использовать его для быстрого формирования аналогичных документов.

Если Вы решили не делать перевод, с данной страницы можно вернуться к списку выполненных Вами операций, щелкнув по кнопке Список, или на страницу типов платежей, щелкнув по кнопке Платежи.

Подтверждение перевода

После сохранения заполненного документа Вы попадаете на страницу подтверждения. Здесь Вам будет показана заполненная форма покупки/продажи иностранной валюты, в которой Вы должны проверить правильность указанных сведений, после чего выполнить одно из следующих действий:

  • Подтверждение перевода

    Процедура подтверждения зависит от используемого Вами способа регистрации в системе. Например, если Вы используете карту одноразовых паролей, то Вам необходимо будет ввести пароль с указанным номером с Вашей карты одноразовых паролей.

    • Когда все сведения указаны, щелкните по кнопке Подтвердить, и документ будет отправлен в банк на обработку.

  • Возврат к редактированию

    Если Вы хотите изменить какие-либо реквизиты перевода, щелкните по кнопке Редактировать, и Вы вернетесь к форме ввода документа для его редактирования.

  • Печать документа

    Если Вы хотите распечатать данный документ, щелкните по кнопке Печать.

  • Создание нового документа

    Вы можете создать еще один документ данного типа, щелкнув по кнопке Создать. В результате Вы перейдете к форме ввода нового документа. Данная возможность полезна при создании нескольких однотипных документов.

  • Отказ от выполнения перевода

    Если Вы передумали осуществлять обмен валюты, можно вернуться к списку выполненных Вами операций, щелкнув по кнопке Список.

Просмотр документа

После подтверждения обменной операции Вы попадаете на страницу просмотра, где отображается заполненный документ. О том, что документ передан в банк, свидетельствует отображаемый на форме документа штамп «Принято к исполнению». На страницу просмотра документа Вы можете также попасть из списка документов, щелкнув по номеру того документа, который хотите просмотреть. На странице просмотра Вы можете выполнить следующие действия:

  • Создание нового документа

    Вы можете создать еще один документ данного типа, щелкнув по кнопке Создать. В результате Вы перейдете к форме ввода нового документа. Данная возможность полезна при создании нескольких однотипных документов.

  • Отзыв документа

    Вы можете отозвать ошибочно отправленный документ. Для этого щелкните по кнопке Отозвать, в результате Вы перейдете к форме отзыва перевода. На экран будет выведена подробная информация по отзываемому документу. При необходимости измените значение в поле "Дата отзыва", информация в остальных полях документа недоступна для редактирования. Для отзыва перевода щелкните по кнопке Сохранить. Откроется страница, на которой проверьте правильность заполнения полей документа и нажмите на кнопку Подтвердить, чтобы передать документ в банк на исполнение.

  • Печать документа

    Если Вы хотите распечатать данный документ, щелкните по кнопке Печать.

  • Возврат к списку операций

    Можно вернуться к списку выполненных Вами обменных операций, щелкнув по кнопке Список.

Список валютообменных операций

В данном списке отображаются все выполненные Вами операции по покупке и продаже иностранной валюты. Вы можете зайти в список операций со страницы выбора типа платежа, щелкнув на выбранном платеже по кнопке История операций, или при создании платежа, щелкнув по кнопке Список. Для каждой операции в списке отображаются вид операции (покупка или продажа), дата, номер документа, счета списания и зачисления, суммы списания и зачисления и статус, отображающий текущее состояние операции. Вы можете выполнить следующие операции:

  • Просмотр документа

    Вы можете просмотреть любой документ, для этого нужно щелкнуть по номеру интересующего Вас документа в списке.

  • Отзыв документа

    Вы можете отозвать ошибочно отправленный документ. Для этого щелкните по кнопке Отозвать, в результате Вы перейдете к форме отзыва перевода. На экран будет выведена подробная информация по отзываемому документу. При необходимости измените значение в поле "Дата отзыва", информация в остальных полях документа недоступна для редактирования. Для отзыва перевода щелкните по кнопке Сохранить. Откроется страница, на которой проверьте правильность заполнения полей документа и нажмите на кнопку Подтвердить, чтобы передать документ в банк на исполнение.

  • Создание копии документа

    Вы можете создать копию документа, отметив документ, который служит оригиналом в списке, щелкнув по кнопке Создать копию. В результате Вы перейдете к форме ввода нового документа, причем в нее будут скопированы все данные из документа, выбранного Вами в качестве оригинала. Данная возможность полезна при создании нескольких однотипных документов.

  • Удаление платежа

    Вы можете удалить перевод из списка, для этого установите галочку напротив интересующего документа и щелкните по кнопке Удалить.

  • Фильтр переводов

    Для того чтобы найти интересующий Вас перевод в списке, щелкните по кнопке Показать фильтр и заполните поля фильтра:

    "Статус" - поиск по статусу перевода в системе: "Введен", "На исполнении", "Исполнен", "Отказан", "Отозван";

    "Период" - поиск по времени создания перевода;

    "Сумма" - поиск по размеру суммы перевода;

    "Счет списания" - поиск по номеру счета, с которого списаны средства;

    "Счет зачисления" - поиск по номеру счета, на который зачислены средства.

    Вы можете фильтровать список по любой комбинации полей, например, можно отобрать все переводы на одну сумму со статусом "Исполнен".

    После того как критерии поиска заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас переводов.

Статусы платежа

Ход обработки платежа Вы можете наблюдать в списке платежей или в журнале операций по статусу. Для обмена валюты в системе предусмотрены следующие статусы:

  • Введен - платеж введен, сохранен в системе, но не подтвержден;

  • Обрабатывается - платеж полностью заполнен клиентом банка, подтвержден и передан на обработку в банк;

  • Ожидание обработки - ожидается передача документа в банк или получение информации о результате его исполнения;

  • Отозван - платеж передан в банк на обработку, но затем Вами отозван по какой-либо причине;

  • Исполнен - платеж проведен в банке: денежные средства со счета клиента поступили на указанный в форме счет зачисления;

  • Отказан - платеж отказан банком с указанием причины, например, по причине нехватки денежных средств на счете клиента на момент их списания.
