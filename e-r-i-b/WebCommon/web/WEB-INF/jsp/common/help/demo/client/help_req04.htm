Назначение заявки

Вы можете оформить заявку на перевыпуск пластиковой карты. Для этого в главном меню выберите пункт Заявки, а в открывшемся списке пункт Перевыпуск карты. На экран будет выведена форма заявки, которую необходимо заполнить. Ход обработки заявки можно отслеживать в списке заявок по статусу.

Заполнение заявки

В заявке на перевыпуск пластиковой карты в поле "ФИО" автоматически подставляется фамилия, имя и отчество владельца карты, данное поле недоступно для редактирования. Чтобы подать заявку, выберите из списка карт ту, которую необходимо перевыпустить, и из выпадающего списка укажите причину перевыпуска пластиковой карты.

После того как все поля заполнены, щелкните по кнопке Сохранить. Если Вы решили не делать заявку, с данной страницы можно вернуться к списку выполненных Вами заявок на перевыпуск пластиковой карты, щелкнув по кнопке Список, или на страницу выбора типа заявки, щелкнув по кнопке Заявки.

Подтверждение заявки

После сохранения заполненного документа Вы попадаете на страницу подтверждения. Здесь Вам будет показана заполненная форма заявки на перевыпуск пластиковой карты, в которой Вам нужно проверить правильность указанных сведений, после чего выполнить одно из следующих действий:

  • Подтверждение заявки

    Процедура подтверждения зависит от используемого Вами способа регистрации в системе. Например, если Вы используете карту одноразовых паролей, то Вам необходимо будет ввести пароль с указанным номером с Вашей карты одноразовых паролей.

    • Когда все сведения указаны, щелкните по кнопке Подтвердить, и документ будет отправлен в банк на обработку.

  • Печать документа

    Если Вы хотите распечатать данный документ, щелкните по кнопке Печать.

  • Отказ от формирования заявки

    Если Вы передумали делать заявку, можно вернуться к списку сделанных Вами заявок на перевыпуск пластиковой карты, щелкнув по кнопке Список, или на страницу выбора типа заявки, щелкнув по кнопке Заявки.

Просмотр документа

После подтверждения заявки Вы попадаете на страницу просмотра, где отображается заполненный документ. О том, что документ передан в банк, свидетельствует отображаемый на форме документа штамп «Принято к исполнению». На страницу просмотра документа Вы можете также попасть из списка документов, щелкнув по номеру того документа, который хотите просмотреть. На странице просмотра Вы можете выполнить следующие действия:

  • Создание нового документа

    Вы можете создать еще один документ данного типа, щелкнув по кнопке Создать. В результате Вы перейдете к форме ввода нового документа. Данная возможность полезна при создании нескольких однотипных документов.

  • Печать документа

    Если Вы хотите распечатать данный документ, щелкните по кнопке Печать.

  • Возврат к списку заявок или выбору типа заявки

    Можно вернуться к списку сделанных Вами заявок на перевыпуск пластиковой карты, щелкнув по кнопке Список, или на страницу выбора типа заявки, щелкнув по кнопке Заявки.

Список заявок

В данном списке отображаются все сделанные Вами заявки на перевыпуск пластиковой карты. Вы можете зайти в список со страницы выбора типа заявки, щелкнув на выбранной заявке по кнопке История операций, или при создании заявки, щелкнув по кнопке Список. Для каждой заявки в списке отображаются дата создания, номер заявки, номер карты, причина перевыпуска карты и статус, по которому можно отследить состояние заявки в системе. Вы можете выполнить следующие операции:

  • Просмотр документа

    Вы можете просмотреть любой документ, для этого нужно щелкнуть по номеру интересующего Вас документа в списке.

  • Создание новой заявки

    Вы можете создать новую заявку на перевыпуск пластиковой карты, щелкнув по кнопке Создать. В результате Вы перейдете к форме ввода нового документа.

  • Удаление заявки

    Вы можете удалить заявку из списка, для этого установите галочку напротив интересующего документа и щелкните по кнопке Удалить.

  • Фильтр заявок

    Для того чтобы найти интересующую Вас заявку, щелкните по кнопке Показать фильтр и заполните поля фильтра:

    "Карта" - поиск по номеру и типу карты;

    "Номер" - поиск по номеру заявки;

    "Причина перевыпуска" - поиск названию причины перевыпуска карты;

    "Статус" - поиск по статусу заявки в системе: "Введена", "Одобрена", "Исполнена", "Отказана", "Принята" и т.д.;

    "Период" - поиск по времени создания заявки.

    Вы можете фильтровать список по любой комбинации полей, например, можно отобрать все заявки на перевыпуск карты, созданные в один период, со статусом "Исполнена".

    После того как критерии поиска заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас заявок.

Статусы заявок

Ход обработки заявок Вы можете наблюдать в списке заявок по статусу. Для заявлений по перевыпуску пластиковой карты в системе предусмотрены следующие статусы:

  • Введен - документ введен, сохранен в системе, но не подтвержден;

  • Принят - документ полностью заполнен клиентом банка, подтвержден и передан на обработку в банк;

  • Одобрен - документ одобрен сотрудником банка, но не исполнен;

  • Исполнен - документ исполнен банком;

  • Отказан - документ отказан банком с указанием причины.
