Назначение заявки

С помощью данного пункта меню Вы можете подать в банк заявку на открытие счета или вклада. Для этого в главном меню выберите пункт Заявки, а в открывшемся списке пункт Открытие счета/вклада, или пункт меню Вклады - Заявка на открытие вклада. После заполнения заявки система попросит проверить правильность заполнения полей и подтвердить созданную заявку, после чего Ваша заявка будет отправлена в банк. Ход обработки заявки можно отслеживать в списке заявок по статусу.

Заполнение заявки

Заполнение заявки на открытие счета
Заполнение заявки на открытие вклада

Для заполнения заявки необходимо определить, какой тип заявки Вы собираетесь открыть. Для этого в поле "Открыть" выберите из выпадающего списка интересующий Вас тип заявки: счет или вклад. От этого зависит набор полей в заявке.

Заполнение заявки на открытие счета

Для заполнения формы заявки на открытие счета Вам необходимо заполнить обязательные поля:

  • В поле "Валюта счета" выберите валюту, в которой желаете открыть счет.

  • В поле "Тип счета" выберите из выпадающего списка интересующий Вас тип счета: например, "Текущий счет".

  • В поле "Дата визита в отделение банка" введите дату, когда Вы сможете прийти в отделение банка.

  • В поле "Офис банка" выберите из выпадающего списка нужный офис.

  • Поле "Ф.И.О." заполняется автоматически, но его можно отредактировать.

  • В поле "Гражданство" укажите Ваше гражданство.

  • В поле "Контактный телефон" укажите номер телефона, по которому с Вами может связаться сотрудник банка.

Вы можете просмотреть и скачать для ознакомления типовой договор по интересующему Вас типу счета. Для этого щелкните по ссылке Посмотреть договор.

После заполнения полей формы нажмите на кнопку Сохранить. Заявка приобретет статус "Введена". Если Вы решили не делать заявку, с данной страницы можно вернуться к списку выполненных Вами заявок на открытие счета/вклада, щелкнув по кнопке Список, или на страницу выбора типа заявки, щелкнув по кнопке Заявки.

Заполнение заявки на открытие вклада

Для заполнения формы заявки на открытие вклада Вам необходимо заполнить обязательные поля:

  • В поле "Вид вклада" выберите из выпадающего списка интересующий Вас вид вклада.

  • В поле "Валюта вклада" выберите валюту, в которой желаете открыть вклад.

  • В поле "Срок вклада" выберите из выпадающего списка срок, на который Вы хотите открыть вклад.

  • В поле "Сумма вклада" укажите сумму, которую Вы планируете разместить на депозитном счете.

    Примечание: данная сумма не должна быть меньше минимальной суммы вклада.

  • Поле "Минимальная сумма вклада" заполняется системой автоматически.

  • В поле "Списать со счета" выберите из выпадающего списка счет, с которого Вы хотите списать денежные средства.

    Примечание: валюта счета списания должна совпадать с выбранной Вами валютой вклада.

  • В поле "Действие по окончании срока вклада" необходимо выбрать одно из двух значений:

    1. "Автоматически пролонгировать" - система автоматически продлит действие договора вклада после завершения срока его действия;

    2. "Перевести на счет" - выберите из выпадающего списка счет, на который будут переведены денежные средства по окончании срока вклада.

  • В поле "Дата визита в отделение банка" введите дату, когда Вы сможете прийти в отделение банка.

  • Поле "Ф.И.О." заполняется автоматически, но его можно отредактировать.

  • В поле "Гражданство" укажите Ваше гражданство.

  • В поле "Контактный телефон" укажите номер телефона, по которому с Вами может связаться сотрудник банка.

Вы можете просмотреть и скачать для ознакомления типовой договор по интересующему Вас виду вклада. Для этого щелкните по ссылке Посмотреть договор.

После заполнения полей формы нажмите на кнопку Сохранить. Заявка приобретет статус "Введена". Если Вы решили не делать заявку, с данной страницы можно вернуться к списку выполненных Вами заявок на открытие счета/вклада, щелкнув по кнопке Список, или на страницу выбора типа заявки, щелкнув по кнопке Заявки.

Подтверждение заявки

После сохранения заполненного документа Вы попадаете на страницу подтверждения. Здесь Вам будет показана заполненная форма заявки на открытие счета/вклада, в которой Вы должны проверить правильность указанных сведений, после чего выполнить одно из следующих действий:

  • Подтверждение заявки

    Процедура подтверждения зависит от используемого Вами способа регистрации в системе. Например, если Вы используете карту одноразовых паролей, то Вам необходимо будет ввести пароль с указанным номером с Вашей карты одноразовых паролей.

    • Когда все сведения указаны, щелкните по кнопке Подтвердить, чтобы документ был отправлен в банк на обработку.

  • Печать документа

    Если Вы хотите распечатать данный документ, щелкните по кнопке Печать.

  • Отказ от формирования заявки

    Если Вы передумали делать заявку, можно вернуться к списку сделанных Вами заявок на открытие счета/вклада, щелкнув по кнопке Список, или на страницу выбора типа заявки, щелкнув по кнопке Заявки.

Просмотр документа

После подтверждения заявки Вы попадаете на страницу просмотра, где отображается заполненный документ. О том, что документ передан в банк, свидетельствует отображаемый на форме документа штамп «Принято к исполнению». На страницу просмотра документа Вы можете также попасть из списка документов, щелкнув по номеру того документа, который хотите просмотреть. На странице просмотра Вы можете выполнить следующие действия:

  • Создание нового документа

    Вы можете создать еще один документ данного типа, щелкнув по кнопке Создать. В результате Вы перейдете к форме ввода нового документа. Данная возможность полезна при создании нескольких однотипных документов.

  • Печать документа

    Если Вы хотите распечатать данный документ, щелкните по кнопке Печать.

  • Возврат к списку заявок или выбору типа заявки

    Можно вернуться к списку сделанных Вами заявок на открытие счета/вклада, щелкнув по кнопке Список, или на страницу выбора типа заявки, щелкнув по кнопке Заявки.

Список заявок

В данном списке отображаются все сделанные Вами заявки на открытие счета/вклада. Вы можете зайти в список со страницы выбора типа заявки, щелкнув на выбранной заявке по кнопке История операций, или при создании заявки, щелкнув по кнопке Список. Для каждой заявки в списке отображаются дата создания, номер заявки, тип заявки и статус, по которому можно отследить состояние заявки в системе, и другие сведения. Вы можете выполнить следующие операции:

  • Просмотреть заявку

    Для этого щелкните по номеру интересующей Вас заявки в списке и Вы попадете на страницу просмотра заполненной формы заявки.

  • Создать заявку на открытие счета/вклада

    Для этого щелкните по кнопке Создать. Подробную информацию по данной операции Вы можете посмотреть в разделе контекстной справки Заявки - Заполнение заявки.

  • Удалить заявку

    Для этого установите флажок напротив заявки, которую Вы хотите удалить, и нажмите на кнопку Удалить.

  • Фильтр заявок

    Для поиска в системе интересующих Вас заявок предусмотрен фильтр, который позволяет осуществлять поиск по любому из полей либо их комбинации. Щелкните по кнопке Показать фильтр. Откроется страница с формой, в которой необходимо задать критерии поиска:

    • В поле "Статус" выберите из выпадающего списка статус Вашей заявки на открытие счета/вклада: "Введена", "Одобрена", "Исполнена", "Отказана" и др.;

    • В поле "Период" введите период времени, в течение которого была создана заявка на открытие счета/вклада;

    • В поле "Тип заявки" выберите из выпадающего списка тип созданной Вами заявки: "Все", "Открытие счета", "Открытие вклада".

    Вы можете фильтровать список по любой комбинации полей, например, можно отобрать все заявки на открытие счета/вклада, созданные в один период, со статусом "Принята".

    После того как критерии поиска заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас заявок.

Статусы заявок

Ход обработки заявок Вы можете наблюдать в списке заявок по статусу. Для заявлений по открытию счета/вклада в системе предусмотрены следующие статусы:

  • Введен - документ введен, сохранен в системе, но не подтвержден;

  • Принят - документ полностью заполнен клиентом банка, подтвержден и передан на обработку в банк;

  • Одобрен - документ одобрен сотрудником банка, но не исполнен;

  • Исполнен - документ исполнен банком;

  • Отказан - документ отказан банком с указанием причины.
