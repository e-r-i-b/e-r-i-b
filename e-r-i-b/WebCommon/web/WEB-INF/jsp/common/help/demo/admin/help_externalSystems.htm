Для подключения соответствующих подразделений банка к системе ИКФЛ необходимо завести одну или несколько внешних систем. Создание внешних систем осуществляется в справочнике, доступном в пункте главного меню Внешние системы.

Справочник внешних систем содержит подробную информацию о внешних системах, версиях RS-Retail, автоматизирующих одно или несколько подразделений банка. К внешним системам относятся RS-Retail v.5.1, RS-Retail v.5.5, RS-Retail V6r4, RS-Retail V6r20. Для каждой внешней системы создается шлюз для взаимодействия с ней, который представляет собой отдельное приложение на сервере приложений. Справочник внешних систем заполняется администратором системы вручную при первом входе в систему.

В пункте главного меню Внешние системы автоматически открывается список внешних систем. Для каждой внешней системы в списке отображается идентификатор (уникальный идентификатор внешней системы), наименование внешней системы и адрес шлюза.

Создание внешней системы

Для того чтобы добавить внешнюю систему, щелкните по кнопке Добавить, откроется страница редактирования, на которой заполните следующие поля:

  • В поле «Наименование» введите название внешней системы;

  • В поле «Адрес шлюза» укажите адрес шлюза для взаимодействия внешней системы (RS-Retail v.5.1, RS-Retail v.5.5, RS-Retail V6r4, RS-Retail V6r20 и т.п.) с системой ИКФЛ. Например, http://phizic-server:8888/PhizGateV5.1/services, где phizic-server – имя или IP-адрес сервера, на котором развернуто приложение шлюза, 8888 – порт приложения, который формируется при установке сервера приложений, PhizGateV5.1 – название шлюза для внешней системы RS-Retail v.5.1 (наименование, которое указывается при заполнении поля Application Name перед запуском приложения шлюза).

    Примечание: подробнее о настройках адреса шлюза смотрите в пользовательской документации.

    Поле «Идентификатор» заполнится автоматически после сохранения сведений о подразделении и будет недоступно для редактирования.

    Если Вы используете в качестве биллинговой систему ИСППН «Город», то заполните следующие поля:

  • В поле «Номер агентской карты «Город»» укажите номер карты данного филиала банка для взаимодействия ИКФЛ с системой ИСППН «Город».

  • В поле «Пароль карты» впишите пароль карты «Город».

После того как все поля заполнены, щелкните по кнопке Сохранить.

Фильтр/поиск внешних систем

Для поиска интересующих Вас внешних систем можно воспользоваться фильтром. Для этого щелкните по кнопке Показать фильтр и задайте критерии поиска:

  • "Идентификатор" - поиск по уникальному идентификатору внешней системы;

  • "Наименование" - поиск по наименованию шлюза;

  • "Адрес шлюза" - поиск по адресу шлюза.

Вы можете осуществлять поиск по подстроке в текстовых полях фильтра. Например, можно ввести в поля фильтра только несколько первых букв наименования внешней системы.

Можно отфильтровать список по любой комбинации полей. Например, можно отфильтровать все внешние системы по наименованию и адресу шлюза.

После того как критерии поиска заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас внешних систем.

Редактирование внешней системы

Для замены шлюза необходимо остановить приложение шлюза на сервере приложений, установить новую версию и запустить шлюз снова.

Примечание: процедура остановки и запуска приложения шлюза описана в пользовательской документации.

Чтобы изменить сведения о внешней системе, установите флажок напротив выбранной внешней системы и нажмите на кнопку Редактировать или щелкните по идентификатору внешней системы в списке. Откроется страница для редактирования, на которой внесите необходимые изменения и щелкните по кнопке Сохранить.

Удаление внешней системы

Для того чтобы удалить внешнюю систему, установите флажок напротив выбранной внешней системы и щелкните по кнопке Удалить.

Примечание: если с внешней системой связаны подразделения, то удаление данной внешней системы невозможно.
