Фильтр/поиск платежей

В журнал платежей включаются все выполненные Вами операции. В списке операций по каждому платежу отображается дата и время операции, дата приема к исполнению, номер документа, вид операции, счет списания, наименование получателя платежа, сумма платежа, сумма комиссии, сумма операции, валюта и статус, указывающий на текущее состояние операции в системе. В данном журнале Вы можете выполнить следующие действия:

  • Просмотр документа

    Вы можете просмотреть любой документ, для этого нужно щелкнуть по номеру интересующего Вас документа в списке.

  • Отзыв платежа

    Вы можете отозвать переданный на обработку в банк платеж. В этом случае переведенные средства (за вычетом комиссии банка) будут возвращены на Ваш счет. Чтобы отозвать документ, отметьте его в списке, установив напротив него галочку, и щелкните по кнопке Отозвать, в результате Вы перейдете к форме отзыва платежа. При необходимости измените значение в поле "Дата отзыва", информация в остальных полях документа недоступна для редактирования, затем щелкните по кнопке Сохранить. Откроется страница, на которой проверьте правильность заполнения полей документа и нажмите на кнопку Подтвердить, чтобы передать документ в банк на исполнение.

  • Редактирование документа

    Если документ еще не передан на обработку в банк, Вы можете отредактировать его. Для этого отметьте редактируемый документ в списке, установив напротив него галочку, и щелкните по кнопке Редактировать. На экран будет выведена форма ввода документа, в которой измените значения интересующих Вас полей и сохраните изменения.

  • Создание копии документа

    Вы можете создать копию документа, отметив документ, который служит оригиналом в списке, щелкнув по кнопке Создать копию. В результате Вы перейдете к форме ввода нового документа, причем в данную форму будут скопированы все сведения из документа, выбранного Вами в качестве оригинала. Данная возможность полезна при создании нескольких однотипных документов.

  • Печать списка

    Если Вы хотите распечатать список совершенных операций, щелкните по кнопке Печать.

Фильтр/поиск платежей

Также на этой странице Вы можете воспользоваться фильтром для поиска интересующих Вас операций.

Для того чтобы воспользоваться фильтром, щелкните по кнопке Показать фильтр и заполните поля фильтра:

"Сумма" - поиск по сумме, на которую совершен перевод;

"Вид операции" - поиск по виду операции: например, "Пополнение карты", "Отзыв перевода", "Перевод между счетами", "Перевод рублей РФ";

"Номер" - поиск по номеру платежа в системе;

"Дата совершения операции" - поиск по дате совершения операции;

"Счет" - поиск по номеру счета;

"Наименование получателя" - поиск по названию получателя, которому переведены денежные средства;

"Дата приема к исполнению" - поиск по дате, когда перевод принят к исполнению;

"Статус" - поиск по статусу операции в системе: "Введен", "Обрабатывается", "Исполнен", "Отозван", "Отказан".

Вы можете фильтровать список по любой комбинации полей, например, можно отобрать все операции, совершенные в один день со статусом "Обрабатывается".

После того как критерии поиска заданы, щелкните по кнопке Построить список, и система покажет список интересующих Вас операций.
