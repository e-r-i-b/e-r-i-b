Список счетов, карт, вкладов и кредитов

В пункте главного меню Мой банк Вы можете просмотреть список Ваших счетов, с которыми можно работать в системе "Интернет-клиент для физических лиц", список выпущенных для Вас пластиковых карт, список открытых Вами срочных вкладов, список кредитов, полученных в банке, получить выписку по счету, карте, вкладу, просмотреть информацию по любому счету, пластиковой карте, вкладу и кредиту, полученному в банке, и просмотреть новости банка.

Данная страница является главной в системе, она отображается сразу после регистрации, и Вы всегда можете вернуться к ней, щелкнув по пункту Мой банк в главном меню. На данной странице, как и на всех остальных, отображаются курсы валют ЦБ, установленные на сегодняшний день, и текущая дата.

Для каждого счета отображаются номер счета, тип счета (текущий, до востребования и т.д.), остаток (сколько средств в данный момент у Вас на счете), валюта счета и дата последней операции.

Под списком счетов располагается список банковских карт, в котором для каждой выпущенной для Вас банковской карты отображаются ее номер (обратите внимание, что в целях безопасности отображаются только первая и четыре последних цифры номера пластиковой карты), тип карты, остаток (сколько средств в данный момент у Вас на счете), валюта счета, статус карты (заблокирована она или нет) и срок действия карты.

Для каждого вклада отображаются вид вклада, сумма вклада, валюта вклада, срок вклада в днях, процентная ставка (% годовых), дата открытия вклада и дата окончания.

В том случае, если Вы получили в банке кредит или несколько кредитов, на главной странице для каждого кредита отображается номер договора, кредитный продукт, дата окончания, сумма кредита, валюта, дата погашения, сумма погашения, сумма просрочки.

По умолчанию в списке отображаются только открытые счета, карты, вклады и кредиты. Если Вы хотите просмотреть информацию обо всех счетах, картах, вкладах и кредитах, установите галочку напротив строки Показывать неактивные счета/карты/вклады/кредиты. Закрытые и заблокированные счета, карты и вклады будут отмечаться в списке значком .

Также на главной странице можно посмотреть новости системы. Для просмотра детальной информации по каждой новости щелкните по пункту левого меню Новости или нажмите кнопку Подробнее, и система покажет страницу с подробной информацией по выбранной новости.

Просмотр информации по счету

Вы можете получить подробную информацию по любому счету из списка счетов на главной странице. Для этого необходимо щелкнуть по типу того счета, о котором Вы хотите просмотреть подробные сведения. Информация по счету включает в себя номер счета, текущий остаток по счету, валюту счета, текущее состояние (открыт или закрыт), дату открытия, номер договора. В окне просмотра сведений о счете Вы можете выполнить следующие действия:

  • Печать карточки счета

    Чтобы распечатать карточку счета, щелкните по кнопке Печать.

  • Возврат к списку счетов, карт и вкладов

    Чтобы вернуться на главную страницу, где отображается список счетов и вкладов, щелкните по кнопке На главную.

Просмотр информации по карте

Вы можете получить подробную информацию по Вашим пластиковым картам. Для этого необходимо щелкнуть по типу карты или остатку денежных средств на карте, по которой Вы хотите просмотреть подробные сведения. Информация по карте включает в себя ФИО держателя карты, тип карты, номер и дату договора, номер карты, срок действия карты, номер спецкарточного счета, доступные средства, валюту и дату последней операции по карте. В окне просмотра сведений по карте Вы можете выполнить следующие действия:

  • Печать сведений по пластиковой карте

    Чтобы распечатать сведения по Вашей пластиковой карте, щелкните по кнопке Печать.

  • Возврат к списку счетов, карт и вкладов

    Чтобы вернуться на главную страницу, где отображается список счетов, карт и вкладов, щелкните по кнопке На главную.

Просмотр информации по вкладу

Вы можете получить подробную информацию по любому вкладу из списка Ваших вкладов на главной странице. Для этого необходимо щелкнуть по названию вида вклада, о котором Вы хотите получить подробную информацию. Сведения по вкладу включают в себя название вида вклада, сумму вклада, валюту вклада, срок вклада, процентную ставку, номер депозитного счета, признак пролонгации вклада на следующий срок (возможность его продлить), сумму неснижаемого остатка вклада, номер счета для перечисления причисленных процентов (на какой счет Вам будут перечислены проценты по вкладу), номер счета для перечисления средств по истечении срока вклада, текущее состояние (открыт или закрыт), дату открытия, дату закрытия и номер договора на предоставление услуг системы "Интернет-клиент для физических лиц". В окне просмотра сведений о вкладе Вы можете выполнить следующие действия:

  • Печать сведений о вкладе

    Чтобы распечатать сведения о вкладе, щелкните по кнопке Печать.

  • Возврат к списку счетов, карт и вкладов

    Чтобы вернуться на главную страницу, где отображается список счетов, карт и вкладов, щелкните по кнопке На главную.

Просмотр информации по кредиту

Вы можете получить подробную информацию по кредитам, выданным Вам банком. Чтобы получить подробные сведения по кредиту, щелкните на названии кредитного продукта. Информация по кредиту включает в себя текущее состояние кредита (открыт или закрыт), номер договора, название кредитного продукта, дату начала, дату окончания кредита, сумму кредита, валюту, процентную ставку, срок кредита, сумму остатка по кредиту, дату погашения, сумму погашения, сумму просроченной задолженности по кредиту, дату последнего платежа, сумму последнего платежа. Далее располагается график погашения кредита. В окне просмотра сведений по кредиту Вы можете выполнить следующие действия:

  • Печать сведений по кредиту

    Чтобы распечатать сведения по кредиту, щелкните по кнопке Печать.

  • Возврат к списку счетов, карт и вкладов

    Чтобы вернуться на главную страницу, где отображается список счетов, карт и вкладов, щелкните по кнопке Список.

Выписка по счетам

Просмотреть выписку (список совершенных операций) по счету Вы можете несколькими способами:

  1. Отметьте флажком интересующий Вас счет в списке (можно выбрать несколько счетов) и щелкните по кнопке Выписка. Система сформирует и отобразит выписку, которую Вы сможете распечатать.

  2. Если Вы хотите просмотреть выписку по одному счету, щелкните в списке счетов на номере счета или на сумме остатка средств на счете, и Вы перейдете к окну просмотра выписки по выбранному счету.

  3. В левом меню щелкните по ссылке Выписка по счетам. Сформируется выписка по всем Вашим счетам. Выберите из списка нужный счет или карту, задайте дату и время, за которое хотите получить выписку по счету. Затем нажмите кнопку Применить, система сформирует и отобразит выписку, которую Вы сможете распечатать.

В каждой выписке по счету отображается следующая информация: заголовок с номером счета, датой и периодом, за который создана выписка, валюта счета, входящий остаток по счету, исходящий остаток, а также таблица операций по данному счету, которые совершены за указанный в выписке период.

На странице просмотра выписки Вы можете выполнить следующие действия:

  • Фильтр выписки

    В окне просмотра выписки Вы можете выбрать счет, изменить параметры выписки (например, период, за который распечатывается выписка, он установлен в системе по умолчанию с первого числа текущего месяца до дня выписки, но Вы можете выбрать любой интересующий период и дату), затем нажать кнопку Применить, и система сформирует выписку по выбранному счету.

  • Печать выписки

    Чтобы распечатать выписку по счетам, щелкните по кнопке Печать.

  • Возврат к списку счетов, карт и вкладов

    Чтобы вернуться на главную страницу, где отображается список счетов, вкладов и пластиковых карт, щелкните по кнопке Список счетов/карт.

Выписка по картам

Просмотреть выписку (список совершенных операций) по карте Вы можете несколькими способами:

  1. Отметьте флажком интересующую Вас карту в списке (можно выбрать несколько карт) и щелкните по кнопке Выписка. Система сформирует и отобразит выписку, которую Вы сможете распечатать.

  2. Если Вы хотите просмотреть выписку по одной карте, щелкните в списке карт на номере карты и Вы перейдете к окну просмотра выписки по выбранной карте.

  3. В левом меню щелкните по ссылке Выписка по картам, сформируется выписка по всем Вашим пластиковым картам. Выберите из списка нужную карту, задайте дату и время, за которое хотите получить выписку по карте. Затем нажмите кнопку Применить, система сформирует и отобразит выписку, которую Вы сможете распечатать.

В выписке по каждой карте отображается следующая информация: заголовок с периодом и датой предоставления выписки, ФИО держателя карты, номер карты, номер спецкарточного счета, входящий и исходящий баланс, а также таблица операций, которые совершены по данной карте за выбранный период.

На странице просмотра выписки Вы можете выполнить следующие действия:

    В окне просмотра выписки Вы можете выбрать карту/счет, изменить параметры выписки (например, период, за который распечатывается выписка, он установлен в системе по умолчанию с первого числа текущего месяца до дня выписки, но Вы можете выбрать любой интересующий период и дату), затем нажать кнопку Применить, и система сформирует выписку по выбранной карте/счету.

  • Печать выписки

    Чтобы распечатать выписку по картам, щелкните по кнопке Печать.

  • Возврат к списку счетов, карт и вкладов

    Чтобы вернуться на главную страницу, где отображается список счетов, вкладов и пластиковых карт, щелкните по кнопке Список счетов/карт.

Выписка по вкладам

В каждой выписке по вкладу отображается следующая информация: название вклада, дата и период, за который формируется выписка, ФИО вкладчика, документ, удостоверяющий личность, дата последней операции, входящий остаток по вкладу, исходящий остаток по вкладу, а также таблица операций по вкладу, которые совершены за указанный в выписке период.

На странице просмотра выписки Вы можете выполнить следующие действия:

  • Фильтр выписки

    В окне просмотра выписки Вы можете выбрать вклад, изменить параметры выписки (например, период, за который распечатывается выписка, он установлен в системе по умолчанию с первого числа текущего месяца до дня выписки, но Вы можете выбрать любой интересующий период и дату), затем нажать кнопку Применить, и система сформирует выписку по выбранному вкладу.

  • Печать выписки

    Чтобы распечатать выписку по вкладам, щелкните по кнопке Печать.

  • Возврат к списку счетов, карт и вкладов

    Чтобы вернуться на главную страницу, где отображается список счетов, вкладов и пластиковых карт, щелкните по кнопке Список счетов/карт.
