Пункт меню Подразделения позволяет добавлять, удалять и редактировать сведения о подразделениях.

Подразделения в справочнике имеют иерархическую структуру: они делятся на головные и подчиненные подразделения.

Список подразделений

Список подразделений открывается автоматически в главном меню в пункте Подразделения. В нем содержится список подразделений банка, которые подключены или не подключены к системе ИКФЛ. Все сотрудники и клиенты банка привязываются к одному из данных подразделений.

Создание подразделения

Если Вы хотите создать в системе новое подразделение, щелкните по кнопке Добавить подразделение, откроется страница редактирования подразделения ИКФЛ. Порядок добавления нового подразделения см. в разделе контекстной справки Регистрация нового подразделения.

Редактирование подразделения

Для того чтобы отредактировать сведения о подразделении, в списке выберите подразделение, установите флажок напротив наименования интересующего Вас подразделения и щелкните по кнопке Изменить . Откроется страница редактирования подразделения банка, на которой измените значения интересующих Вас полей, затем щелкните по кнопке Сохранить , и система покажет данное подразделение в списке в измененном виде.

Репликация подразделений

Для того чтобы загрузить из внешних систем подчиненные подразделения, выберите в списке интересующие головные подразделения, установив напротив них флажки, и щелкните по кнопке Репликация . После загрузки подразделений система Вам покажет протокол, в котором содержится информация о количестве загруженных, измененных подразделений.

Если не реплицировалось ни одного подразделения, то возможно необходимо выполнить настройку репликации для шлюза внешней системы, из которой осуществлялась репликация. Описание данной настройки смотрите в пользовательской документации.

Примечание: репликация подчиненных подразделений доступна только для тех подразделений, для которых установлен признак головного.

В результате для каждого подразделения, которое является головным для внешней системы, Вы можете просмотреть список подчиненных подразделений. Для этого щелкните по наименованию головного подразделения, и система покажет Вам список подчиненных подразделений.

Удаление подразделения

Если необходимо удалить запись о подразделении, установите флажок напротив интересующего Вас подразделения и щелкните по кнопке Удалить . Система попросит Вас подтвердить операцию, Вы подтверждаете удаление, и подразделение будет удалено из списка.

Примечание: подразделения, за которыми закреплены сотрудники или клиенты, не могут быть удалены из системы.

Фильтр/поиск подразделений

Для поиска интересующих Вас подразделений Вы можете воспользоваться фильтром. Для этого щелкните по кнопке Показать фильтр, введите значение в поле "Наименование", затем нажмите на кнопку Применить.

В результате система покажет список интересующих Вас подразделений банка.

Общие сведения

Вы можете добавить в справочник подразделения разных уровней иерархии.

Примечание: сотрудник банка может работать с системой на определенном уровне иерархии подразделения. Настройка уровня иерархии подразделений в системе ИКФЛ задается встроенным администратором в пункте меню Настройки – Параметры системы.

Для того чтобы добавить подразделение верхнего уровня, нажмите на кнопку Добавить.

Для создания подчиненного подразделения установите флажок напротив интересующего Вас подразделения верхнего уровня и щелкните по кнопке Добавить. Откроется страница редактирования подразделения банка, на которой в полях формы необходимо указать сведения о добавляемом подразделении. Система экспортирует поля данной формы в заявление на подключение к услуге ИКФЛ. Для того чтобы создать подразделение, заполните следующие поля формы:

  • В поле "Наименование" укажите название нового подразделения.

  • Если в создаваемом подразделении Вы планируете подключать клиентов к системе ИКФЛ, то установите флажок в строке «Обслуживается в системе ИКФЛ».

    Примечание: сотрудники банка могут быть закреплены за любым подразделением, а клиенты только за теми, которые подключены к системе ИКФЛ.

  • При создании головного подразделения для внешней системы установите флажок в поле «Головное для внешней системы». Данный признак означает, что можно реплицировать подчиненные подразделения из внешней системы (RS-Retail v 5.1, RS-Retail v 5.5, RS-Retail V6r20).

    Примечание: для одной внешней системы может быть только одно головное подразделение.

  • Если создаваемое подразделение будет подключено к системе ИКФЛ, то Вам необходимо указать внешнюю систему (например, RS-Retail v.5.1), которая автоматизирует это подразделение, и подразделение (офис) внешней системы. Для этого в поле «Внешняя система» щелкните по кнопке (...), откроется страница со списком внешних систем, выберите из списка интересующую систему, установив напротив нее галочку и щелкнув по кнопке Выбрать. На данной странице можно воспользоваться фильтром.

    Фильтр/поиск внешних систем

    Для поиска интересующих Вас внешних систем можно воспользоваться фильтром. Для этого щелкните по кнопке Показать фильтр и задайте критерии поиска:

    • "Идентификатор" - поиск по уникальному идентификатору внешней системы;

    • "Наименование" - поиск по наименованию шлюза;

    • "Адрес шлюза" - поиск по адресу шлюза.

    Вы можете осуществлять поиск по подстроке в текстовых полях фильтра. Например, можно ввести в поля фильтра только несколько первых букв наименования внешней системы.

    Можно отфильтровать список по любой комбинации полей. Например, можно отфильтровать все внешние системы по наименованию и адресу шлюза.

    После того как критерии поиска заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас внешних систем.

    Примечание: если подразделение не будет обслуживаться в системе ИКФЛ, то указывать внешнюю систему и подразделение (офис) внешней системы не нужно.

  • В поле «Офис» укажите название соответствующего подразделения во внешней системе (RS-Retail v 5.1, RS-Retail v 5.5, RS-Retail V6r20). Для этого щелкните по кнопке (...), откроется страница со справочником подразделений внешней системы, на которой выберите название подразделения, установив напротив него флажок, и щелкните по кнопке Выбрать.

    Фильтр подразделений

    Для поиска интересующих Вас подразделений щелкните по кнопке Показать фильтр и задайте значение в поле "Наименование".

    После того как параметры фильтра заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас подразделений.

    После выбора из справочника подразделения внешней системы автоматически заполнятся поля "Город", "Почтовый адрес", "Местонахождение", "Телефон".

  • Далее укажите из выпадающего списка часовой пояс города, в котором находится подразделение. В системе по умолчанию указывается значение часового пояса «Москва (MSK)». Данная настройка необходима для того, чтобы правильно учитывалось время приема документов во всех подразделениях банка.

  • В поле «Тариф за организацию расчетного обслуживания» укажите тариф.

  • В поле «Ежемесячная плата» введите размер оплаты в месяц за использование услуг системы ИКФЛ.

  • В поле "Тариф за восстановление возможности доступа в систему" укажите размер тарифа.

После того как все сведения о подразделении указаны, щелкните по кнопке Сохранить , и подразделение будет добавлено в справочник.

Время приема документов

Вкладка Время приема документов предназначена для настройки режима приема платежных документов каждого подразделения.

На данной вкладке для каждого вида документа задается время начала и окончания приема и обработки документов определенного вида сегодняшним операционным днем, а также календарь рабочих и праздничных дней. Выбрать календарь Вы можете только для подразделений, подключенных к внешним системам RS-Retail v.5.1 и RS-Retail v.5.5. Для подразделений, подключенных к внешней системе RS-Retail V6r20, используется календарь внешней системы.

Примечание: управление рабочими календарями выполняется в пункте меню Сервис – Рабочие календари.

Вы можете посмотреть и при необходимости изменить настройки. Для изменения настройки выберите документ, введите в поля новые значения, время должно быть в формате ЧЧ:ММ, укажите рабочий календарь и щелкните по кнопке Сохранить. В результате время приема выбранного Вами типа документа будет изменено в системе. Например, если для перевода между счетами Вы укажете время окончания приема документов 16:00, то документы, полученные позже 16:00, будут обрабатываться системой следующим операционным днем.

На данной вкладке Вы также можете для каждого вида документа задать настройку времени приема документов вышестоящего подразделения, для этого выберите вид документа и напротив него установите флажок в поле «Использовать настройку вышестоящего». Например, когда для вышестоящего подразделения задано значение времени приема документов по оплате товаров и услуг «с 9:00 по 15:00», и Вы для создаваемого подразделения установите флажок напротив строки «Использовать настройку вышестоящего», то поле станет недоступно для редактирования, и для данного подчиненного подразделения будет использоваться настройка времени приема документов по оплате товаров и услуг вышестоящего подразделения, т.е. значение «с 9:00 по 15:00».
