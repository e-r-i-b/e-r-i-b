Сменить пароль

На данной странице можно изменить пароль, который Вы используете при регистрации в системе. Чтобы попасть на эту страницу с других страниц системы, щелкните по пункту главного меню Сервис.

Для смены пароля в полях предложенной формы надо ввести старый пароль в поле "Введите старый пароль" и вписать новый пароль в поля "Введите новый пароль" и "Повторите новый пароль". Чтобы пароль был заменен на новый, щелкните по кнопке Изменить.

Для защиты Ваших конфиденциальных данных, пожалуйста, выполняйте следующие рекомендации:

  • Рекомендуется сменить пароль при первом сеансе работы с системой — это гарантирует, что новый пароль будете знать только Вы.

  • Регулярно меняйте пароль.

  • Используйте пароли, которые трудно угадать. Например, не вводите пароли с очевидной ассоциацией с именами других членов семьи.

  • Используйте в паролях символы как верхнего, так и нижнего регистра, цифры и прочие допустимые символы.

Параметры оповещений

Для того чтобы оперативно получать информацию о состоянии Ваших счетов и выполняемых операций, Вы можете использовать систему оповещений. Вы сами можете выбрать, о каких событиях Вас будут оповещать, а также в какой форме будут приходить оповещения.

На данной странице Вы можете задать параметры оповещений. Чтобы попасть на эту страницу с других страниц системы, щелкните по пункту Сервис, затем выберите в левом меню пункт Параметры оповещений. Вам доступно два типа оповещений — по электронной почте и SMS. Данные типы оповещений могут приходить одновременно по обоим каналам: по e-mail и в виде SMS.

  • Оповещения по электронной почте

    Чтобы получать оповещения по электронной почте, Вам необходимо указать в поле "Присылать сообщения по электронной почте на адрес (E-mail)" адрес Вашей электронной почты и в списке оповещений установить флажки в столбце E-mail для тех оповещений, которые Вы хотите получать.

  • Оповещения через SMS

    Чтобы получать оповещения через SMS, Вам необходимо указать в поле "Присылать сообщения на мобильный телефон номер" номер мобильного телефона, на который Вы хотите получать оповещения, и в списке оповещений установить флажки в столбце SMS для тех оповещений, которые Вы хотите получать. Если Ваш телефонный аппарат или поставщик услуг мобильной связи не обеспечивают корректной доставки SMS, написанных кириллицей, установите флажок напротив строки "Транслитерация".

Для некоторых оповещений можно задавать параметры, например их периодичность или номера счетов, к которым эти оповещения относятся.

Чтобы сохранить сформированную Вами схему оповещений, щелкните по кнопке Сохранить. Если Вы передумали менять схему оповещений, щелкните по кнопке Отменить.

Сертификаты

Фильтр/поиск сертификатов

С помощью сертификата Вы можете осуществлять безопасный вход в систему в защищенном режиме и подписывать документы (платежи, заявки) ЭЦП (электронно-цифровой подписью), которая позволит сотруднику банка убедиться в аутентичности отправленных Вами сведений, а также проверить, была ли информация изменена (искажена) в процессе передачи, гарантируя целостность данных.

В пункте меню Сертификаты Вы можете просмотреть и распечатать выпущенные для Вас сертификаты, а также отфильтровать список сертификатов. Для каждого сертификата в списке отображается идентификатор, дата начала действия, дата окончания действия и статус, по которому Вы можете определить состояние Вашего сертификата: статус "Активный" (сертификат активен, и его можно использовать для защищенной работы в системе), "Неактивный" (сертификат надо установить, активировать), "Заблокирован" (сертификат заблокирован), "Истек" (закончился срок действия сертификата).

Для того чтобы просмотреть сертификат, установите галочку напротив интересующего Вас сертификата и щелкните по кнопке Просмотр.

Если Вы хотите распечатать выбранный сертификат, установите галочку напротив данного сертификата и нажмите кнопку Печать.

Порядок создания запроса на выпуск сертификата описан в разделе контекстной справки Запрос на сертификат.

Фильтр/поиск сертификатов

В пункте меню Сертификаты Вы можете воспользоваться фильтром/поиском сертификатов.

Для того чтобы воспользоваться фильтром, щелкните по кнопке Показать фильтр и заполните поля фильтра:

"Дата начала действия" - поиск по дате начала действия сертификата;

"Дата окончания действия" - поиск по дате окончания действия сертификата;

"Статус" - поиск по статусу сертификата в системе: "Активный" (сертификат активен, и его можно использовать для защищенной работы в системе), "Неактивный" (сертификат надо установить, активировать), "Заблокирован" (сертификат заблокирован), "Истек" (закончился срок действия сертификата).

Вы можете фильтровать список по любой комбинации полей, например, можно отобрать все сертификаты со статусом "Неактивный".

После того как критерии поиска заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас сертификатов.

Запрос на сертификат

С помощью данного пункта меню Вы можете создать запрос на сертификат и просмотреть список Ваших заявок на выпуск сертификата. Для каждого запроса в списке отображается его номер, дата создания и статус запроса.

  • Создание запроса

    Чтобы создать запрос на сертификат, щелкните по кнопке Создать.

    Если у Вас нет этой кнопки, значит в списке уже есть запрос на сертификат, который в данный момент обрабатывается банком (об этом говорит его статус "Обрабатывается" или "Отправлен"). В таком случае Вам надо дождаться завершения обработки предыдущего запроса.

    Откроется форма с Вашей заявкой на выпуск сертификата, которую Вам нужно сохранить, щелкнув по кнопке Сохранить.

    Если Вы не хотите создавать запрос, щелкните по кнопке Список, и система вернет Вас в список запросов на сертификат.

  • Подтверждение запроса

    Для того чтобы отправить запрос, щелкните по кнопке Подтвердить. Сформированный запрос будет отправлен в банк, и его статус изменится на "Отправлен".

    В банке запрос обрабатывается в Центре сертификации и формируется ответ, который отправляется Вам.

    Полученный от банка сертификат необходимо установить. Для этого щелкните по номеру заявки на сертификат в списке (у нее должен быть статус "Сертификат выдан"), а затем щелкните по кнопке Установить. При этом статус заявки на сертификат меняется на "Сертификат установлен", а сам сертификат можно увидеть в списке сертификатов, для доступа к которому надо выбрать пункт Сертификаты из меню Сервис.

    Если запрос на сертификат отказан банком (об этом свидетельствует статус "Отказан"), Вы можете просмотреть причину отказа, щелкнув по номеру отказанной заявки в списке.

    Если Вы решили не создавать запрос, щелкните по кнопке Список, и система вернет Вас в список запросов на сертификат.

  • Просмотр запроса на сертификат

    Для того чтобы просмотреть интересующий Вас запрос на сертификат, щелкните на номере запроса и откроется страница просмотра.

    Если Вы хотите просмотреть печатную форму запроса на сертификат открытого ключа, то щелкните по кнопке Просмотр.

    Чтобы распечатать запрос на сертификат, нажмите на кнопку Печать.

    Если Вам необходимо вернуться в список запросов на сертификат, щелкните по кнопке Список.

Активировать карту ключей

С помощью данного пункта меню Вы можете активировать новую карту одноразовых паролей.

Для активации карты ключей укажите требуемый системой ключ с действующей карты и в поле "№ новой карты паролей" впишите номер карты, которую Вам необходимо активировать, и щелкните по кнопке Активировать.

Пункт меню SMS-банкинг предназначен для предоставления услуги SMS-банкинга, т.е. Вы сможете с помощью мобильного телефона получить информацию о счетах, картах и вкладах, открытых в банке, а также совершить любую платежную операцию.

SMS-шаблоны

С помощью данного пункта меню можно управлять списком SMS-шаблонов, которые используются при совершении операций через мобильный телефон.

SMS-шаблоны делятся на шаблоны банка и шаблоны клиента. Шаблоны банка создаются сотрудником банка и недоступны для редактирования. Шаблоны клиента Вы можете добавлять, редактировать, удалять.

Создание SMS-шаблона

Для того чтобы создать SMS-шаблон, щелкните по кнопке Добавить. После этого откроется окно, и система предложит Вам выбрать вид платежа, шаблон для которого Вы хотите создать. В данном окне установите галочку напротив интересующего Вас вида платежа и щелкните по кнопке Продолжить. Откроется форма заполнения выбранного Вами платежа, в которой в поле "название шаблона" введите наименование шаблона, заполните нужные поля и щелкните по кнопке Сохранить, и новый шаблон появится в списке Ваших SMS-шаблонов.

Изменение SMS-шаблона

Для того чтобы изменить SMS-шаблон, выберите интересующий Вас шаблон, установите напротив него галочку и щелкните по кнопке Изменить. Откроется форма для редактирования шаблона, в которую внесите изменения и щелкните по кнопке Сохранить.

Удаление SMS-шаблона

Для удаления SMS-шаблон из списка выберите интересующий Вас шаблон, установите напротив него галочку и щелкните по кнопке Удалить, и шаблон будет удален из списка.

Печать списка SMS-шаблонов

Если Вам нужно распечатать список SMS-шаблонов, щелкните по кнопке Печать.

SMS-псевдонимы

SMS-псевдонимы - это уникальные для каждого клиента имена объектов: счетов, карт, вкладов, которые можно использовать при совершении операций через мобильный телефон. При подключении к системе автоматически генерируются банковские псевдонимы, т.е псевдонимы по умолчанию, которые Вы можете изменить.

Редактирование SMS-псевдонима

Если Вы хотите изменить SMS-псевдоним, щелкните на интересующем Вас псевдониме, откроется окно редактирования псевдонима, в котором в поле "Псевдоним" введите название псевдонима и щелкните по кнопке Сохранить.

Отказ от редактирования SMS-псевдонима

Для того чтобы отменить внесенные изменения, щелкните по кнопке Отменить.

Печать списка SMS-псевдонимов

Если Вам нужно распечатать список SMS-псевдонимов, щелкните по кнопке Печать.

Журнал операций

Для работы с журналом операций, совершенных через SMS-банкинг, предназначен пункт меню SMS-банкинг – Журнал операций. Также просмотреть список операций, выполненных через SMS-банкинг, Вы можете в пункте меню Платежи – Журнал операций, в котором отображаются все созданные Вами платежные и неплатежные документы.

В журнале по каждой операции отображается дата и время операции, дата приема к исполнению, номер, вид операции, счет списания, наименование получателя платежа, сумма платежа, сумма комиссии, сумма операции, валюта и статус, по которому Вы можете отследить состояние операции в системе.

Подробную информацию по работе с журналом смотрите в разделе контекстной справки Журнал операций.

Фильтр/поиск операций

Для поиска интересующих Вас операций можно воспользоваться фильтром. Для этого щелкните по кнопке Показать фильтр и задайте критерии поиска.

  • "Сумма" - поиск по сумме, на которую совершен перевод;

  • "Вид операции" - поиск по виду операции: например, "Пополнение карты", "Отзыв перевода", "Перевод между счетами", "Перевод рублей РФ";

  • "Номер" - поиск по номеру платежа в системе;

  • "Дата совершения операции" - поиск по дате совершения операции;

  • "Счет" - поиск по номеру счета;

  • "Наименование получателя" - поиск по названию получателя, которому переведены денежные средства;

  • "Дата приема к исполнению" - поиск по дате, когда перевод принят к исполнению;

  • "Статус" - поиск по статусу операции в системе: "Введен", "Обрабатывается", "Исполнен", "Отозван", "Отказан".

  • Вы можете фильтровать список по любой комбинации полей, например, можно отобрать все операции, совершенные в один день со статусом "Обрабатывается".

После того как критерии поиска заданы, щелкните по кнопке Построить список, и система покажет список интересующих Вас операций.

О программе

Пункт меню О программе предназначен для просмотра основной информации о программе "Интернет-клиент для физических лиц". В данном пункте меню Вы можете просмотреть следующие сведения: версию программы, номер сборки и наименование разработчика системы.
