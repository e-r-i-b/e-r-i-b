Работа с платежами

Пункт меню Аудит (Журнал заявок и платежей) предназначен для управления заявками, поступившими от клиентов банка, просмотра платежных документов, для формирования статистической информации (количество операций, суммы по платежным операциям и т.п.). А также данный функционал может быть использован для печати твердых копий платежных документов (квитанций об оплате), которые клиент может получить в офисах банка.

Отличие платежей от заявок состоит в том, что заявки - это неплатежные документы, которые клиенты подают на обработку в банк.

Примечание: сотрудник банка в журнале заявок и платежей может просмотреть и обработать заявки и просмотреть, отфильтровать, распечатать платежные документы, поступившие от клиентов банка всех подразделений, к которым имеет доступ данный сотрудник.

В данном пункте меню автоматически открывается список всех заявок и платежей, поступивших от клиентов банка.

Фильтр/поиск документов

В данном пункте меню по умолчанию показаны операции, выполненные пользователями за текущий день.

В журнале над списком операций отображается общее количество заявок и платежных операций.

Ниже располагается таблица с итоговым количеством платежных операций, в которой для каждого вида валюты по всем платежам отображается сумма операций, сумма комиссий и общая сумма платежей с учетом комиссии.

По любому документу в списке отображается следующая информация:

  • Дата и время совершения операции;

  • Номер документа;

  • Вид операции;

  • Статус документа;

  • ФИО клиента;

  • Счет списания;

  • Счет зачисления;

  • Сумма;

  • Валюта платежа;

  • Наименование получателя платежа.

В журнале заявок и платежей Вы можете выполнить следующие действия:

  • Просмотр документа

    Для того чтобы просмотреть заявку или платеж в списке, щелкните по дате, времени или номеру интересующего Вас документа, в результате откроется страница просмотра платежа/заявки.

  • Просмотр анкеты клиента

    В журнале заявок и платежей Вы можете просмотреть анкету клиента банка, оформившего документ, для этого щелкните по ссылке ФИО клиента. Из анкеты клиента банка можно обратно вернуться в Журнал заявок и платежей, щелкнув по кнопке Назад. Причем в журнале сохранятся заданные Вами настройки фильтрации списка операций.

  • Печать журнала

    Вы можете распечатать журнал в соответствии с настройками фильтра. Для этого нажмите на кнопку Печать журнала, откроется печатная форма списка заявок и платежей, которую можно распечатать.

  • Печать документов по платежам/заявкам

    Для того чтобы распечатать документы по платежу/заявке или сразу по нескольким операциям, установите флажки напротив интересующих Вас заявок и платежей, щелкните по кнопке Печать. Откроется окно, в котором выберите нужные шаблоны документов и/или пакеты шаблонов и нажмите на кнопку Напечатать. В результате система покажет документы с заполненными реквизитами для предварительного просмотра. Данные документы Вы можете распечатать на принтере.

    Если для выбранного платежа/заявки печатные формы не настроены, то система покажет соответствующее сообщение.

    Примечание: настройка печатных форм документов осуществляется в пункте меню Сервис – Формы платежей. Для того чтобы настроить печатные формы, установите флажок напротив выбранной формы платежа или заявки, щелкните по кнопке Печатные формы, откроется страница со списком шаблонов документов и пакетов. Вам нужно выбрать в списке необходимые пакеты шаблонов и/или отдельные шаблоны, отметив их флажками, и щелкнуть по кнопке Сохранить изменения. В результате в списке форм платежей, платежи, для которых назначены печатные формы, будут отмечены специальной пиктограммой . Предварительно у Вас должны быть созданы шаблоны и пакеты шаблонов документов (см. разделы контекстной справки Шаблоны документов банка, Пакеты шаблонов документов банка).

Фильтр/поиск документов

В данном пункте меню Вы можете воспользоваться фильтром для поиска интересующих Вас заявок и платежей по любым из указанных полей либо их комбинации. Для этого щелкните по кнопке Показать фильтр и задайте критерии поиска:

  • "ФИО" - поиск по фамилии, имени и отчеству клиента банка;

  • "Сумма" - поиск по сумме, на которую совершен платеж;

  • "Номер" - поиск по номеру документа;

  • "Вид операции" - поиск по виду операции;

  • "Период" - поиск по дате и времени совершения операции;

  • "Статус" - поиск по статусу документа в системе;

  • "Наименование получателя" - поиск по названию получателя платежа.

Вы можете фильтровать список по любой комбинации полей. Например, можно отобрать все документы, созданные клиентами банка по одному виду операции со статусом "Исполнен".

После заполнения полей фильтра нажмите на кнопку Применить, и система покажет список интересующих Вас документов.

Работа с заявками

В журнале заявок и платежей Вы можете просмотреть и обработать заявку клиента банка.

Чтобы просмотреть заявку, щелкните по дате, времени или номеру заявки в списке. Открывается страница с подробной информацией по данной заявке.

В зависимости от способа обработки заявки можно разделить на три группы:

  1. Заявки, которые существуют только в ИКФЛ (ни в какие внешние системы они не передаются). К данной группе заявок относятся все заявки, которые связаны с пластиковыми картами. К ней также примыкают заявки, которые можно исполнить, но они не передаются на дальнейшую обработку во внешние системы, например, заявка на аренду сейфовой ячейки, заявка на совершение операций с ценными бумагами и т.п.

  2. Заявки, которые попадают в RS-Retail после того как Вы нажмете на кнопку Исполнить. Такая процедура предусмотрена только для заявки на открытие счета/вклада в соответствии с законодательством.

  3. Заявки, которые автоматически попадают в RS-Retail (список отложенных документов). К данной группе относятся все заявки, кроме заявок первой группы. Заявки данной группы Вы не можете обработать в системе ИКФЛ.

Заявки первой и второй группы Вы можете обработать следующим образом:

  • Если Вам необходимо отклонить заявку, щелкните по дате, времени или номеру интересующей Вас заявки, откроется страница просмотра заявки, на которой щелкните по кнопке Отклонить, затем в поле "Причина отклонения" выберите причину отказа из специального справочника или введите в ручную другую причину и щелкните по кнопке Отклонить. В результате система присвоит документу статус "Отказан".

  • Если Вы хотите одобрить заявку, щелкните по дате, времени или номеру интересующей Вас заявки, откроется страница просмотра заявки, и щелкните по кнопке Одобрить. В результате система присвоит заявке статус "Одобрена", т.е. данная заявка может быть исполнена.

  • Чтобы исполнить заявку, т.е. отправить заявку на обработку в RS-Retail (только для заявок второй группы), щелкните на номере интересующей Вас заявки, откроется страница просмотра заявки, и щелкните по кнопке Исполнить. В результате система присвоит заявке статус "Исполнена".

    Примечание: аналогично Вы можете исполнить заявки первой группы, например, заявку на аренду сейфовой ячейки, заявку на совершение операций с ценными бумагами. Отличие состоит в том, что данные заявки не передаются на дальнейшую обработку во внешние системы, происходит только изменение их статуса.

При необходимости Вы можете распечатать документы по любой заявке. Для этого щелкните по дате, времени или номеру выбранной заявки, откроется страница просмотра, нажмите на кнопку Печать. Откроется окно, в котором необходимо выбрать из списка шаблонов документов и/или пакетов интересующие Вас документы и щелкнуть по кнопке Напечатать. Если для данной заявки печатные формы не настроены, то система покажет соответствующее сообщение.

Также если Вам нужно вернуться в список платежей и заявок, щелкните по кнопке Назад.

Работа с платежами

Для того чтобы просмотреть интересующий Вас платеж в списке, щелкните по дате, времени или номеру интересующего Вас платежного документа. Откроется страница с подробной информацией по выбранной Вами платежной операции. На данной странице Вы можете выполнить следующие действия:

  • Вы можете распечатать документ, для этого щелкните по кнопке Печать. Откроется окно, в котором необходимо выбрать из списка шаблонов документов и/или пакетов интересующие Вас документы и щелкнуть по кнопке Напечатать. Если для выбранного платежа печатные формы не настроены, то система покажет соответствующее сообщение.

  • Вы можете вернуться в список платежей и заявок, для этого щелкните по кнопке Назад.
