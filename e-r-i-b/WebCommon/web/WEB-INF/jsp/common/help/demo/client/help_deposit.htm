В пункте меню Вклады Вы можете просмотреть список Ваших вкладов, с которыми можно работать в системе ИКФЛ (пополнить вклад, списать средства со вклада и др.), а также просмотреть список депозитных продуктов банка и воспользоваться депозитным калькулятором.

Список вкладов

Фильтр/поиск вкладов

В пункте главного меню Вклады автоматически открывается страница, на которой отображается список имеющихся у Вас вкладов. По каждому вкладу можно просмотреть следующую информацию: вид вклада, сумму вклада, валюту, в которой открыт вклад, срок вклада, процентную ставку по вкладу, дату открытия и дату окончания, статус вклада и номер договора.

На данной странице Вы можете выполнить следующие операции:

  • Открыть вклад. Для того чтобы подать заявку на открытие нового вклада, нажмите на кнопку Открыть вклад. Подробную информацию по выполнению данной операции Вы можете посмотреть в разделе справки Заявки - Открытие счета/вклада.

  • Пополнить вклад. Для пополнения вклада установите флажок напротив вклада, который Вы желаете пополнить, и нажмите на кнопку Пополнить вклад. Подробную информацию по выполнению данной операции Вы можете посмотреть в разделе справки Заявки - Пополнить вклад.

  • Просмотреть информацию по депозитным продуктам банка. Для этого щелкните по кнопке Продукты. Для подробного описания данной операции перейдите в раздел справки Заявки - Депозитные продукты банка.

  • Закрыть вклад. Для того чтобы подать заявку на закрытие вклада, установите флажок напротив вклада, который желаете закрыть, и нажмите на кнопку Закрыть вклад. Подробную информацию по выполнению данной операции Вы можете посмотреть в разделе справки Заявки - Закрытие счета/вклада.

Фильтр/поиск вкладов

Для поиска в системе интересующих Вас вкладов предусмотрен фильтр, который позволяет осуществлять поиск по любому из полей либо их комбинации. Щелкните по кнопке Показать фильтр. Откроется страница с формой, в которой необходимо задать критерии поиска:

  • В поле "Сумма" укажите сумму депозита.

  • В поле "Дата открытия" введите период времени, в течение которого был открыт вклад.

  • В поле "Дата закрытия" укажите период времени, в течение которого был закрыт интересующий Вас вклад.

  • В поле "Срок вклада" из выпадающего списка выберите тот период времени, на который открыт вклад: "Все", "От 1 до 3 лет" и др.

  • В поле "Вид вклада" выберите из выпадающего списка вид интересующего Вас вклада: "Все", "Зарплатный", "Срочный пролонгируемый", "Универсальный" и др.

  • В поле "Статус" выберите из выпадающего списка статус Вашего депозита: "Все", "Открытый" и др.

  • В поле "Валюта" из выпадающего списка выберите валюту, в которой открыт вклад: "Все", "RUB", "USD", "EUR".

После заполнения полей фильтра щелкните по кнопке Применить.

При необходимости редактирования полей фильтра щелкните по кнопке Очистить, внесите необходимые изменения и вновь щелкните по кнопке Применить, после чего система отобразит отфильтрованный список вкладов по заданным Вами параметрам.

Депозитные продукты банка

Щелкните по пункту левого меню "Депозитные продукты банка". Откроется страница, на которой приведен полный список предлагаемых Вам банком депозитных продуктов.

Примечание: к данному пункту меню можно также перейти со страницы списка вкладов, щелкнув по кнопке Продукты.

Если Вы хотите получить более подробную информацию о каком-нибудь из перечисленных видов депозитов, щелкните по названию вида вклада, либо по ссылке Подробнее. Откроется страница, на которой будет отображена таблица с подробной информацией по интересующему Вас депозитному продукту банка. По каждой валюте вклада будут указаны условия по выбранному Вами депозитному продукту: срок вклада, сумма вклада, процентная ставка, периодичность выплаты процентов и другие сведения.

Со страницы просмотра подробной информации по депозитному продукту Вы можете перейти к депозитному калькулятору для оценки доходности интересующего Вас депозита. Для этого щелкните по кнопке Калькулятор. Подробную информацию по работе с депозитным калькулятором смотрите в разделе контекстной справки Вклады - Депозитный калькулятор. Для возврата со страницы просмотра к списку депозитных продуктов банка нажмите на кнопку Список.

Из списка депозитных продуктов банка можно вернуться в список Ваших вкладов, для этого щелкните по кнопке Вклады.

Депозитный калькулятор

Данный пункт левого меню предназначен для использования калькулятора, предоставленного системой, который поможет Вам оценить доходность депозита по размеру вклада и выбрать наиболее подходящий для Вас депозитный продукт, предлагаемый банком.

Примечание: к данному пункту меню можно также перейти со страницы просмотра информации по депозитному продукту банка, щелкнув по кнопке Калькулятор.

Заполнение полей депозитного калькулятора осуществляется следующим образом:

  • В поле "Вид вклада" выберите интересующий Вас вид вклада из выпадающего списка.

  • В поле "Валюта вклада" выберите из выпадающего списка валюту, в которой Вы хотите открыть вклад. Список валюты зависит от выбранного Вами вида вклада.

  • В поле "Срок вклада" выберите из выпадающего списка срок вклада, который также будет зависеть от выбранного Вами вида вклада.

  • В поле "Сумма вклада" укажите сумму, которую Вы планируете разместить на депозитном счете.

    Примечание: данная сумма должна быть не меньше минимальной суммы вклада.

После проверки внесенных Вами данных щелкните по кнопке Рассчитать, после чего калькулятор покажет Вам сумму денежных средств в конце срока вклада, сумму дохода по вкладу и размер процентной ставки по выбранному Вами виду вклада.
