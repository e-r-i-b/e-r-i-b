Список сотрудников

Фильтр/поиск

Пункт меню Сотрудники предназначен для управления учетными записями сотрудников банка, которые работают с системой. Он позволяет регистрировать новых сотрудников, удалять, блокировать и разблокировать сотрудников, управлять доступом сотрудников к системе.

В списке для каждого сотрудника отображается следующая информация: ФИО, подразделение, дополнительная информация, логин и схема прав доступа.

Добавление сотрудника

Для того чтобы добавить нового сотрудника в список, щелкните по кнопке Добавить, откроется анкета сотрудника. Порядок добавления сотрудника см. в разделе контекстной справки Регистрация сотрудников.

Редактирование сведений о сотруднике

Если Вы хотите отредактировать сведения о сотруднике, щелкните по его фамилии, имени, отчеству, откроется анкета для редактирования, в которой измените нужные сведения и нажмите на кнопку Сохранить.

Удаление сотрудника

Если Вам необходимо удалить сотрудника из списка, установите галочку напротив интересующего Вас сотрудника и нажмите кнопку Удалить, система попросит подтвердить операцию, и после подтверждения сотрудник будет удален из списка.

Фильтр/поиск

Для просмотра списка всех интересующих Вас сотрудников банка щелкните по кнопке Показать фильтр и задайте критерии поиска:

  • "ФИО"- поиск по фамилии, имени, отчеству сотрудника;

  • "Номер отделения"- поиск по номеру отделения банка, в котором зарегистрирован сотрудник;

  • "Номер филиала"- поиск по номеру филиала банка, в котором числится сотрудник;

  • "Идентификатор пользователя (ID)" - поиск по идентификатору пользователя в системе;

  • "Дополн. информация"- поиск по дополнительной информации, которая известна о сотруднике;

  • "Статус"- поиск по статусу сотрудника в системе.

Вы можете осуществлять поиск по подстроке в текстовых полях фильтра. Например, можно ввести в поля фильтра только несколько первых букв фамилии, имени или отчества сотрудника банка.

Также Вы можете фильтровать список по любой комбинации полей. Например, можно отобрать всех активных сотрудников с фамилией на букву "А" из одного отделения банка.

После того как искомые параметры заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас операций пользователя системы.

В пункте меню Группы Вы можете просмотреть список групп сотрудников банка, создать, отредактировать или удалить группу. Принадлежность сотрудника банка к определенной группе позволит Вам отправлять письма сотрудникам, объединенным в группы. По каждой группе система показывает название группы и ее описание.

Фильтр/поиск групп сотрудников

Чтобы воспользоваться фильтром, щелкните по кнопке Показать фильтр и задайте в поле "Наименование" - поиск по названию группы сотрудников.

После того как параметры заданы, нажмите кнопку Применить, и система покажет список интересующих Вас сотрудников, входящих в состав группы.

Если Вы не хотите редактировать сведения о группе, щелкните по кнопке Список, и система вернет Вас в список групп сотрудников банка.

Список сотрудников

В пункте меню Список сотрудников Вы можете просмотреть список всех сотрудников, входящих в состав группы. Для поиска интересующего сотрудника банка Вы можете воспользоваться фильтром с помощью кнопки Показать фильтр и правильно заполнить поля фильтра:

  • "ФИО" - поиск по фамилии, имени, отчеству сотрудника, входящего в состав группы;

  • "Номер подразделения" - поиск по номеру отделения банка, в котором зарегистрирован сотрудник;

  • "Номер филиала" - поиск по номеру филиала банка, в котором числится сотрудник;

  • "Доп. информация" - поиск по дополнительной информации, которая известна о сотруднике банка;

  • "Статус" - поиск по статусу сотрудника в системе: "Активный", "Заблокирован".

    • Вы можете осуществлять поиск по подстроке в текстовых полях фильтра. Например, можно ввести в поля фильтра только несколько первых букв фамилии, имени или отчества сотрудника.

    Также Вы можете фильтровать список по любой комбинации полей. Например, можно отобрать всех активных сотрудников с фамилией на букву "А" из одного отделения банка.

    После того как параметры заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас сотрудников банка, входящих в группу.

    Чтобы включить сотрудника банка в группу, щелкните по кнопке Добавить, откроется страница со списком сотрудников, из которого выберите интересующего Вас сотрудника, установите напротив него галочку и щелкните по кнопке Выбрать. В результате система включит данного сотрудника банка в состав группы.

    Удаление группы

    Если Вы хотите удалить группу сотрудников, поставьте флажок напротив той группы, которую будете удалять, и щелкните по кнопке Удалить. Система попросит Вас подтвердить удаление, и после подтверждения группа будет удалена из списка групп сотрудников банка.

    Общие сведения

    В анкете сотрудника Вам необходимо правильно заполнить все поля, отмеченные звездочкой:

    • Подразделение банка, за которым будет закреплен новый сотрудник. Чтобы заполнить поле "Подразделение", щелкните по кнопке (...), откроется страница со списком подразделений, на которой выберите интересующее Вас подразделение банка и щелкните по кнопке Выбрать. На данной странице Вы также можете воспользоваться фильтром.

      Фильтр подразделений

      Для просмотра списка интересующих подразделений нажмите на кнопку Показать фильтр и введите в поле "Наименование" название подразделения. Затем щелкните по кнопке Применить, и система покажет список интересующих Вас подразделений банка.

      Примечание: поле анкеты "Подразделение" должно быть заполнено обязательно, чтобы новый сотрудник имел доступ к информации о клиентах, закрепленных за данным подразделением.

    • Кредитный офис, кредитные заявки которого сотрудник банка сможет обрабатывать. Для этого щелкните по кнопке (...), откроется страница со списком кредитных офисов банка. На данной странице установите галочку напротив интересующего Вас офиса и нажмите на кнопку Выбрать. Если офис головной (он выделяется жирным шрифтом), то сотрудник банка будет видеть заявки всех кредитных офисов банка.

    • Фамилию, имя и отчество нового сотрудника.

    • Логин, пароль и подтверждение пароля. Новый сотрудник при входе в систему должен будет ввести логин и пароль. При первом входе система потребует сменить пароль.

    Права доступа

    Чтобы назначить схему прав доступа сотруднику банка, выберите из списка интересующую Вас схему прав и щелкните по кнопке Сохранить.

    Если Вы хотите назначить сотруднику банка индивидуальные права, т.е. права доступа к операциям в системе, не связанным ни с одной схемой прав, то в приведенном списке операций установите флажки рядом с теми операциями, которые будет выполнять данный сотрудник. После того как список прав сформирован, щелкните по кнопке Сохранить.

    Доступ к подразделениям

    Для того чтобы сотруднику задать дополнительный доступ к соответствующим подразделениям банка, выберите вкладку Доступ к подразделениям в анкете сотрудника. Откроется справочник подразделений определенного уровня иерархии, в котором в поле «Доступ» установите флажки напротив выбранных подразделений и щелкните по кнопке Сохранить .

    Примечание: сотрудник банка может работать с системой на определенном уровне иерархии подразделения. Настройка уровня иерархии подразделений в системе ИКФЛ задается встроенным администратором в пункте меню Настройки – Параметры системы.

    Вы можете вернуться в список сотрудников, для этого щелкните по кнопке Список.

    Настройки оповещений

    Каждому сотруднику необходимо задать настройки оповещений. Оповещения отсылаются системой сотруднику в виде SMS или письма о том, что в системе произошли события, например, клиент банка пополнил пластиковую карту или окончился срок действия сертификата и т.д. Чтобы настроить сотруднику банка рассылку оповещений, нужно из списка возможных оповещений выбрать интересующие Вас виды рассылок. Рядом с каждой рассылкой находятся два флажка: SMS и E-mail. Чтобы указать рассылки, установите флажки напротив каждого выбранного оповещения и щелкните по кнопке Сохранить. Для подписки на оповещения о совершении платежей и заявок клиентами банка Вы можете щелкнуть по кнопке (...) (для вызова справочника), откроется список платежей и заявок, из которого Вы можете выбрать рассылку на определенные платежи и заявки, установив флажок напротив интересующего Вас события и щелкнув по кнопке Выбрать.

    • Если ни один из флажков не установлен, то данное оповещение сотруднику не отправляется.

    • Если установлен флажок SMS, то в случае возникновения данного события сотруднику будет отправлено SMS на указанный в его анкете номер мобильного телефона.

    • Если установлен флажок E-mail, то в случае возникновения данного события сотруднику будет отправлено электронное письмо на указанный в его анкете адрес.

    • Можно устанавливать оба флажка - в этом случае оповещения будут отправляться по обоим каналам.

    После того как указаны параметры оповещений, щелкните по кнопке Сохранить.

    Группы

    Вы можете включить сотрудника в состав группы, для этого установите флажок напротив той группы, в которую хотите включить сотрудника банка, и щелкните по кнопке Сохранить . Включение сотрудника в группу дает возможность отправлять письма сотрудникам, объединенным в группу.

    Назначение схемы прав

    Назначение схемы прав доступа выполняется следующим образом:

    1. В списке выберите тех сотрудников, которым будет назначаться выбранная схема прав доступа, установив флажки с именами сотрудников.

    2. В поле "Схема прав" выберите ту схему прав, которая будет назначена выбранным сотрудникам, после чего щелкните по кнопке Назначить права.

    Можно также назначить схему прав в анкете сотрудника банка: во вкладке анкеты Права доступа выберите ту схему прав, которая будет назначена сотруднику, после чего щелкните по кнопке Сохранить.

    Вы можете создать новую схему прав доступа, либо назначить сотруднику индивидуальные права, т.е. Вы вручную разрешаете доступ к различным услугам банка, не связанным ни с одной схемой доступа.

    Задать индивидуальные права можно также во вкладке анкеты Права доступа: в приведенном списке операций установите флажки рядом с теми операциями, которые будет выполнять данный сотрудник. После того как список прав сформирован, щелкните по кнопке Сохранить.

    Блокировка и разблокировка сотрудников

    Учетные записи сотрудников можно блокировать. Это позволяет запретить сотруднику доступ в систему (например, на время отпуска), не удаляя его учетную запись.

    Блокировка выполняется двумя способами:

    1. В главном меню системы щелкните по пункту Сотрудники. На экран будет выведен список всех зарегистрированных в системе сотрудников. В списке выберите тех сотрудников, учетные записи которых будут заблокированы, установив флажки рядом с именами сотрудников. Щелкните по кнопке Заблокировать. Откроется окно, в котором необходимо указать срок и причину блокировки, и снова нажать на кнопку Заблокировать.

    2. Также блокировку сотрудника можно выполнить из анкеты сотрудников, для этого нужно открыть анкету выбранного сотрудника и нажать на кнопку Заблокировать.

    Разблокировка учетных записей выполняется точно также, только отметить надо заблокированные учетные записи и щелкнуть по кнопке Снять блокировку.

    Заблокированных сотрудников в списке можно отфильтровать с помощью Фильтра/поиска сотрудников. Для этого в фильтре надо задать статус сотрудника "Заблокирован", и система покажет в списке только заблокированных сотрудников банка.

    Визуально отличить заблокированного сотрудника можно по пиктограмме , которая появляется рядом с фамилией заблокированного сотрудника банка.
