В системе предусмотрены справочники, которые облегчают заполнение документов и позволяют избежать возникновения опечаток в реквизитах документов.

Справочник банков

Справочник банков содержит реквизиты банков.

Для каждого банка в справочнике хранятся его код БИК, наименование, номер корреспондентского счета и краткое наименование банка. При выборе банка из справочника все эти сведения автоматически подставляются в документ. Справочник банков предоставляется уже заполненным, но Вы можете добавлять в него новые записи или редактировать информацию о банках.

Создание банка

Чтобы добавить в список реквизиты нового банка, щелкните по кнопке Добавить, откроется страница редактирования сведений о банке, на которой заполните следующие поля: в поле "Наименование" впишите название банка, в поле "БИК" укажите банковский идентификационный код, в поле "Корр. счет" введите корреспондентский счет банка, при необходимости заполните остальные поля и щелкните по кнопке Сохранить. В результате созданный Вами банк появится в списке банков.

Редактирование банка

Для редактирования сведений о банке установите напротив него галочку и щелкните по кнопке Редактировать или щелкните на наименовании банка, откроется страница редактирования, на которой внесите необходимые изменения и щелкните по кнопке Сохранить.

Удаление банка

Если Вы хотите удалить из списка банк или несколько банков, установите напротив него/них галочку и щелкните по кнопке Удалить.

Для поиска интересующего Вас банка в системе предусмотрен фильтр.

Для того чтобы воспользоваться фильтром, щелкните по кнопке Показать фильтр и задайте критерии поиска:

  • "Наименование" - поиск по названию банка;

  • "БИК" - поиск по банковскому идентификационному коду:

  • "Город" - поиск по названию города, в котором зарегистрирован банк.

Вы можете фильтровать список по любой комбинации полей. Например, можно отобрать все банки, название которых начинается с буквы "А", и из одного города.

После того как параметры поиска заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас банков.

Загрузка справочников

С помощью пункта меню Загрузка справочников Вы можете загрузить в систему ИКФЛ из внешних систем (RS-Retail v.5.1, RS-Retail v.5.5, RS-Retail V6r20) справочники, которые отображены на экране.

Примечание: предварительно необходимо задать путь до внешней системы, из которой будут загружаться справочники. Управление данной настройкой выполняется в пункте меню Настройки - Справочники.

В систему можно загрузить следующие справочники:

  • Справочник валют, в котором содержатся коды валют.

  • Справочник иностранных банков, содержащий реквизиты иностранных банков.

  • Справочник банков, в котором содержится информация о реквизитах банков.

  • Справочник стран системы CONTACT, в котором содержится информация о странах системы CONTACT.

  • Справочник стран, содержащий коды стран.

  • Справочник КБК, в котором содержатся коды бюджетной классификации.

  • Справочник кредитных офисов банка, содержащий информацию о дополнительных офисах, в которых предоставляются кредиты клиентам банка.

Чтобы загрузить справочники, выделите галочкой нужный справочник или все справочники и щелкните по кнопке Загрузить. Когда справочники загрузятся, система покажет Вам сообщение "Справочники загружены", а также рядом с каждым справочником отобразится дата и время последнего обновления справочника.
