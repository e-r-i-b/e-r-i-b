Назначение платежа

Сеть Contact позволяет Вам быстро осуществить перевод денежных средств в пользу физических лиц без открытия счета получателя. Чтобы создать перевод, в главном меню выберите пункт Платежи, а затем пункт левого меню Перевод по сети Contact.

Заполнение документа

В форме перевода по сети Contact Вам надо указать дату (в это поле автоматически будет подставлена текущая дата), выбрать из выпадающего списка валюту, в которой будут переводиться средства, сумму перевода и выбрать из списка счет списания (счет, с которого будут списаны денежные средства).

Далее необходимо заполнить реквизиты получателя: фамилию, имя, отчество и дату рождения. В нижней части формы надо указать реквизиты того банка, в котором получатель будет получать перевод.

Для быстрого заполнения реквизитов получателя можно воспользоваться предусмотренным в системе справочником получателей сети Contact. Для этого щелкните по кнопке (...), расположенной рядом с полем "Фамилия". Откроется справочник получателей, в котором необходимо выбрать интересующего Вас получателя и щелкнуть по кнопке Выбрать. Все реквизиты получателя будут скопированы из справочника в создаваемый документ.

Для того чтобы заполнить реквизиты банка получателя, щелкните по кнопке (...), расположенной рядом с полем "Код". Откроется справочник банков, выберите интересующий Вас банк и щелкните по кнопке Выбрать. В результате все реквизиты банка получателя скопируются в создаваемый документ.

После того как все поля заполнены, щелкните по кнопке Сохранить. Вы также можете добавить частично заполненный документ в свой список шаблонов, щелкнув по кнопке Сохранить как шаблон, чтобы в дальнейшем использовать его для быстрого формирования аналогичных документов.

Если Вы решили не делать перевод, с данной страницы можно вернуться к списку выполненных Вами переводов по сети Contact, щелкнув по кнопке Список, или на страницу выбора типа платежа, щелкнув по кнопке Платежи.

Подтверждение перевода

После сохранения заполненного документа Вы попадаете на страницу подтверждения. Здесь Вам будет показана заполненная форма перевода, в которой Вы должны проверить правильность указанных сведений, после чего выполнить одно из следующих действий:

  • Подтверждение перевода

    Процедура подтверждения зависит от используемого Вами способа регистрации в системе. Например, если Вы используете карту одноразовых паролей, то Вам необходимо будет ввести пароль с указанным номером с Вашей карты одноразовых паролей.

    • Когда все сведения указаны, щелкните по кнопке Подтвердить, чтобы документ был отправлен в банк на обработку.

  • Возврат к редактированию

    Если Вы хотите изменить какие-либо реквизиты перевода, щелкните по кнопке Редактировать, чтобы вернуться к форме ввода документа для его редактирования.

  • Печать документа

    Если Вы хотите распечатать данный документ, щелкните по кнопке Печать.

  • Отказ от платежа

    Если Вы передумали осуществлять перевод, можно вернуться к списку выполненных Вами переводов по сети Contact, щелкнув по кнопке Список.

Просмотр документа

После подтверждения перевода Вы попадаете на страницу просмотра, где отображается заполненный документ. О том, что документ передан в банк, свидетельствует отображаемый на форме документа штамп «Принято к исполнению». На страницу просмотра документа Вы можете также попасть из списка документов, щелкнув по номеру того документа, который хотите просмотреть. На странице просмотра Вы можете выполнить следующие действия:

  • Создание нового документа

    Вы можете создать еще один документ данного типа, щелкнув по кнопке Создать. В результате Вы перейдете к форме ввода нового документа. Данная возможность полезна при создании нескольких однотипных документов.

  • Отзыв документа

    Вы можете отозвать ошибочно отправленный документ. Для этого щелкните по кнопке Отозвать, в результате Вы перейдете к форме отзыва платежа. На экран будет выведена подробная информация по отзываемому документу. При необходимости измените значение в поле "Дата отзыва", информация в остальных полях документа недоступна для редактирования. Для отзыва платежа щелкните по кнопке Сохранить. Откроется страница, на которой проверьте правильность заполнения полей документа и нажмите на кнопку Подтвердить, чтобы передать документ в банк на исполнение.

  • Печать документа

    Если Вы хотите распечатать данный документ, щелкните по кнопке Печать.

  • Возврат к списку переводов

    Можно вернуться к списку выполненных Вами переводов по сети Contact, щелкнув по кнопке Список.

Список переводов по сети Contact

В данном списке отображаются все выполненные Вами переводы по сети Contact. Вы можете зайти в список операций со страницы выбора типа платежа, щелкнув на выбранном платеже по кнопке История операций, или при создании платежа, щелкнув по кнопке Список. Для каждого платежа в списке отображается дата, номер документа, счет списания, сумма и статус, отображающий текущее состояние операции. Вы можете выполнить следующие операции:

  • Просмотр документа

    Вы можете просмотреть любой документ, для этого нужно щелкнуть по номеру интересующего Вас документа в списке.

  • Отзыв документа

    Вы можете отозвать ошибочно отправленный документ. Для этого щелкните по кнопке Отозвать, в результате Вы перейдете к форме отзыва платежа. На экран будет выведена подробная информация по отзываемому документу. При необходимости измените значение в поле "Дата отзыва", информация в остальных полях документа недоступна для редактирования. Для отзыва платежа щелкните по кнопке Сохранить. Откроется страница, на которой проверьте правильность заполнения полей документа и нажмите на кнопку Подтвердить, чтобы передать документ в банк на исполнение.

  • Создание копии документа

    Вы можете создать копию документа, для этого отметьте документ, который служит оригиналом в списке, и щелкните по кнопке Создать копию. В результате Вы перейдете к форме ввода нового документа, причем в нее будут скопированы все данные из документа, выбранного Вами в качестве оригинала. Данная возможность полезна при создании нескольких однотипных документов.

  • Удаление платежа

    Вы можете удалить перевод из списка, для этого установите галочку напротив интересующего документа и щелкните по кнопке Удалить.

  • Фильтр переводов

    Для того чтобы найти интересующий Вас платеж, щелкните по кнопке Показать фильтр и заполните поля фильтра:

    "Статус" - поиск по статусу платежа в системе: "Введен", "Выдан", "Исполнен", "Отказан", "Отозван" и др.;

    "Период" - поиск по времени создания платежа;

    "Сумма" - поиск по размеру суммы платежа;

    "Счет списания" - поиск по номеру счета, с которого списаны средства.

    Вы можете фильтровать список по любой комбинации полей, например, можно отобрать переводы по сети Contact на одну сумму со статусом "Исполнен".

    После того как критерии поиска заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас переводов.

Статусы платежа

Ход обработки платежа Вы можете наблюдать в списке переводов или в журнале операций по статусу. Для перевода по сети Contact предусмотрены следующие статусы:

  • Введен - перевод создан и сохранен в системе, но не подтвержден;

  • Получен банком получателем - перевод успешно принят банком получателем. Получатель может приходить в банк за переводом. Данный статус для некоторых точек может отсутствовать, т.к. они не передают информацию о нем в клиринговый центр.

  • Выдан - перевод выдан получателю платежа в банке получателя. Данный статус для некоторых точек может отсутствовать, т.к. они не передают информацию о нем в клиринговый центр.

  • Аннулирование - перевод аннулирован на стороне банка;

  • Изменение - перевод был изменен на стороне банка;

  • Отозван - перевод передан в банк на обработку, но затем Вами отозван по какой-либо причине;

  • Возвращен - перевод возвращен банком получателем:

  • Обрабатывается - перевод полностью заполнен клиентом банка, подтвержден и передан на обработку в банк;

  • Ожидание обработки - ожидается передача документа в банк или получение информации о результате его исполнения;

  • Исполнен - форма перевода полностью заполнена клиентом банка, подтверждена ключом из карты ключей или ЭЦП и передана на обработку в банк;

  • Отказан - перевод отказан банком с указанием причины, например, по причине нехватки денежных средств на счете клиента на момент их списания.
