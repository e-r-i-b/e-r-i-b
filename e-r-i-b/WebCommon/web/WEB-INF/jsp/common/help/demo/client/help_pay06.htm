Переводы

Пункт меню Переводы объединяет следующие платежные операции:

  1. Перевод между счетами - перевод денежных средств между Вашими счетами;

  2. Рублевый перевод - перевод денежных средств с Вашего счета на счет физического или юридического лица;

  3. Перевод иностранной валюты - перевод денежных средств с Вашего счета на счет получателя в иностранной валюте;

  4. Налоговый платеж - перечисление денежных средств с Вашего счета в счет оплаты налогов и сборов;

  5. Перевод по сети Contact - быстрый перевод денежных средств с Вашего счета в счет физических лиц без открытия счета получателя;

  6. Пополнение пластиковой карты - перевод денежных средств с Вашего счета для пополнения пластиковой карты.

Назначение перевода

Данный документ позволяет Вам переводить средства с одного своего счета на другой. Чтобы создать перевод, в главном меню выберите пункт Платежи, а затем пункт левого меню Перевод между счетами.

Заполнение документа

В форме перевода между счетами Вам надо указать дату (в это поле автоматически будет подставлена текущая дата), счет списания (счет, с которого будут списаны денежные средства), счет зачисления (счет, на который будут переведены денежные средства) и сумму перевода.

Примечание: валюта счета списания должна совпадать с валютой счета зачисления.

После того как все поля заполнены, щелкните по кнопке Сохранить. Вы также можете добавить частично заполненный документ в свой список шаблонов, щелкнув по кнопке Сохранить как шаблон, чтобы в дальнейшем использовать его для быстрого формирования аналогичных документов.

Если Вы решили не делать перевод, с данной страницы можно вернуться к списку выполненных Вами переводов между счетами, щелкнув по кнопке Список, или на страницу типов платежей, щелкнув по кнопке Платежи.

Подтверждение перевода

После сохранения заполненного документа Вы попадаете на страницу подтверждения. Здесь Вам будет показана заполненная форма перевода, в которой Вам нужно проверить правильность указанных сведений, после чего выполнить одно из следующих действий:

  • Подтверждение перевода

    Процедура подтверждения зависит от используемого Вами способа регистрации в системе. Например, если Вы используете карту одноразовых паролей, то Вам необходимо будет ввести пароль с указанным номером с Вашей карты одноразовых паролей.

    • Когда все сведения указаны, щелкните по кнопке Подтвердить, и документ будет отправлен в банк на обработку.

  • Возврат к редактированию

    Если Вы хотите изменить какие-либо реквизиты перевода, щелкните по кнопке Редактировать, и Вы вернетесь к форме редактирования документа.

  • Создание перевода

    Если Вы хотите создать новый перевод между счетами, щелкните по кнопке Создать, и откроется форма для создания нового перевода между счетами.

  • Печать документа

    Если Вы хотите распечатать данный документ, щелкните по кнопке Печать.

  • Отказ от платежа

    Если Вы передумали осуществлять перевод, можно вернуться к списку выполненных Вами переводов между счетами, щелкнув по кнопке Список.

Просмотр документа

После подтверждения перевода Вы попадаете на страницу просмотра, где отображается заполненный документ. О том, что документ передан в банк, свидетельствует отображаемый на форме документа штамп «Принято к исполнению». На страницу просмотра документа Вы можете также попасть из списка документов, щелкнув по номеру того документа, который хотите просмотреть. На странице просмотра Вы можете выполнить следующие действия:

  • Создание нового документа

    Вы можете создать еще один документ данного типа, щелкнув по кнопке Создать. В результате Вы перейдете к форме ввода нового документа. Данная возможность полезна при создании нескольких однотипных документов.

  • Отзыв документа

    Вы можете отозвать ошибочно отправленный документ. Для этого щелкните по кнопке Отозвать, в результате Вы перейдете к форме отзыва перевода. На экран будет выведена подробная информация по отзываемому документу. При необходимости измените значение в поле "Дата отзыва", информация в остальных полях документа недоступна для редактирования. Для отзыва перевода щелкните по кнопке Сохранить. Откроется страница, на которой проверьте правильность заполнения полей документа и нажмите на кнопку Подтвердить, чтобы передать документ в банк на исполнение.

  • Печать документа

    Если Вы хотите распечатать данный документ, щелкните по кнопке Печать.

  • Возврат к списку переводов

    Можно вернуться к списку выполненных Вами переводов между счетами, щелкнув по кнопке Список.

Список переводов между счетами клиента

В данном списке отображаются все выполненные переводы средств между счетами. Вы можете зайти в список операций со страницы выбора типа платежа, щелкнув на выбранном платеже по кнопке История операций, или при создании платежа, щелкнув по кнопке Список. Для каждого перевода отображаются дата, номер документа, номера счета списания и счета зачисления, сумма перевода и статус, отображающий текущее состояние операции. Вы можете выполнить следующие операции:

  • Просмотр документа

    Вы можете просмотреть любой документ, для этого нужно щелкнуть по номеру интересующего Вас документа в списке.

  • Отзыв документа

    Вы можете отозвать ошибочно отправленный документ. Для этого щелкните по кнопке Отозвать, в результате Вы перейдете к форме отзыва перевода. На экран будет выведена подробная информация по отзываемому документу. При необходимости измените значение в поле "Дата отзыва", информация в остальных полях документа недоступна для редактирования. Для отзыва перевода щелкните по кнопке Сохранить. Откроется страница, на которой проверьте правильность заполнения полей документа и нажмите на кнопку Подтвердить, чтобы передать документ в банк на исполнение.

  • Создание копии документа

    Вы можете создать копию документа, для этого отметьте документ, который служит оригиналом в списке, и щелкните по кнопке Создать копию. В результате Вы перейдете к форме ввода нового документа, причем в нее будут скопированы все данные из документа, выбранного Вами в качестве оригинала. Данная возможность полезна при создании нескольких однотипных документов.

  • Удаление платежа

    Вы можете удалить перевод из списка, для этого установите галочку напротив интересующего документа и щелкните по кнопке Удалить.

  • Фильтр переводов

    Для того чтобы найти интересующий Вас перевод в списке, щелкните по кнопке Показать фильтр и заполните поля фильтра:

    "Статус" - поиск по статусу перевода в системе: "Введен", "На исполнении", "Исполнен", "Отказан", "Отозван";

    "Период" - поиск по времени создания перевода;

    "Сумма" - поиск по размеру суммы перевода. Рядом с данным полем можно указать валюту, в которой совершен перевод;

    "Счет списания" - поиск по номеру счета, с которого списаны средства;

    "Счет зачисления" - поиск по номеру счета, на который зачислены средства.

    Вы можете фильтровать список по любой комбинации полей, например, можно отобрать все переводы между счетами на одну сумму со статусом "Исполнен".

    После того как критерии поиска заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас переводов.

Статусы платежа

Ход обработки платежа Вы можете наблюдать в списке платежей или в журнале операций по статусу. Для переводов между Вашими счетами в системе предусмотрены следующие статусы:

  • Введен - платеж введен, сохранен в системе, но не подтвержден;

  • Обрабатывается - платеж полностью заполнен клиентом банка, подтвержден и передан на обработку в банк;

  • Ожидание обработки - ожидается передача документа в банк или получение информации о результате его исполнения;

  • Отозван - платеж передан в банк на обработку, но затем Вами отозван по какой-либо причине;

  • Исполнен - платеж проведен в банке: денежные средства со счета клиента поступили на указанный в форме счет зачисления;

  • Отказан - платеж отказан банком с указанием причины, например, по причине нехватки денежных средств на счете клиента на момент их списания.
