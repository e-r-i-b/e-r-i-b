Список депозитных продуктов

Пункт меню Депозиты предназначен для управления депозитными продуктами банка, которые будут доступны для клиентов в системе ИКФЛ. При выборе данного пункта меню открывается список депозитных продуктов банка всех подразделений, доступ к которым имеет сотрудник.

В списке депозитных продуктов по каждому из них показывается следующая информация:

  • Наименование вклада. Наименование депозитного продукта банка.

  • Сумма вклада. Отображается перечень диапазонов суммы вклада в зависимости от валюты и срока размещения средств.

  • Валюта вклада. Отображается перечень валют вклада для данного депозитного продукта.

  • Срок вклада. Отображается перечень возможных сроков вклада для данного депозитного продукта.

  • Процентная ставка по вкладу. Отображается процентная ставка (% годовых) для данного депозитного продукта (вида вклада) в зависимости от валюты, срока и суммы размещаемых денежных средств.

Управление депозитными продуктами

Для работы со списком депозитных продуктов банка Вам необходимо их завести в системе ИКФЛ. Существует два способа заведения депозитных продуктов:

  1. Автоматически, т.е. можно реплицировать депозитные продукты из системы RS-Retail. Для этого щелкните по кнопке Репликация, и в систему ИКФЛ реплицируется из системы RS-Retail информация о всех вкладах банка.

  2. Ручное добавление депозитных продуктов, т.е. Вы самостоятельно вводите все депозитные продукты банка при помощи кнопки Добавить. Вы сами определяете, какие виды вкладов входят в состав депозитных продуктов банка.

Вы можете управлять депозитными продуктами банка следующим образом:

  • Добавление депозитного продукта

    Вы можете добавить в список депозитный продукт. Для этого щелкните по кнопке Добавить, откроется справочник подразделений, в котором выберите из список интересующее Вас подразделение и щелкните по кнопке Выбрать. Далее откроется страница редактирования депозитного продукта. На данной странице в поле "Имя депозитного продукта" впишите название создаваемого депозитного продукта, в поле "Описание депозитного продукта" введите комментарий депозитного продукта. В форме ввода депозитных продуктов по каждому вкладу в системе показывается: идентификатор (уникальное имя вклада в системе), название (название вклада), валюта (валюта, в которой производится вклад). Затем укажите перечень видов вкладов, входящих в состав депозитного продукта, установите флажки напротив интересующих Вас видов вкладов и щелкните по кнопке Сохранить, и новый депозитный продукт появится в списке депозитных продуктов банка.

  • Редактирование депозитного продукта

    Если Вам необходимо отредактировать депозитный продукт банка, в списке продуктов выберите интересующий Вас депозитный продукт, установите напротив него флажок и нажмите на кнопку Редактировать или щелкните по наименованию депозитного продукта. Откроется страница редактирования депозитного продукта банка, на которой Вы можете изменить имя и описание продукта, а также перечень видов вкладов, входящих в состав депозитного продукта, затем щелкните по кнопке Сохранить, и депозитный продукт появится в списке в измененном виде. Для того чтобы вернуться к списку депозитных продуктов, щелкните по кнопке Список.

  • Удаление депозитного продукта

    Если Вам нужно удалить депозитный продукт банка из списка, установите флажок напротив выбранного депозитного продукта и щелкните по кнопке Удалить.
