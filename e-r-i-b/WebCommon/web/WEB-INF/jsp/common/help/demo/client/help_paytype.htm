Выбор платежа

На данной странице Вы можете выбрать тип создаваемого платежа. Чтобы попасть на эту страницу, щелкните по пункту Платежи в главном меню системы. Рядом с каждым типом платежа приведено его краткое описание.

Если Вы щелкнете по значку платежа или по его названию, то перейдете к заполнению формы платежа.

Если Вы щелкнете по ссылке История операций, то на экран будет выведен список платежей данного типа, которые Вы совершили в системе.
