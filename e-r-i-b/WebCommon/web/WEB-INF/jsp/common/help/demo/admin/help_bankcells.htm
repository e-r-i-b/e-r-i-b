Пункт меню Сейфы предназначен для управления сведениями об условиях аренды сейфовых ячеек банка.

Свободные сейфы

Вы можете просмотреть информацию о свободных сейфовых ячейках в офисах банка.

На экране отображается список подразделений банка, в которых предоставляется услуга аренды ячеек. Для каждого подразделения отображается перечень типов ячеек (согласно их размерам), которые клиент банка может забронировать.

Размеры сейфов

В пункте меню Размеры сейфов Вы можете просмотреть, изменить, добавить, проверить наличие и удалить любой размер сейфа, аренда которого предоставляется банком.

Добавление размера сейфа

Чтобы добавить новый размер сейфа, нужно в поле "Описание" ввести его параметры и щелкнуть по кнопке Добавить, и система добавит в список размеров сейфов новый размер.

Изменение размера сейфа

Чтобы изменить размер сейфовой ячейки, установите галочку напротив выбранного размера в списке и нажмите на кнопку Изменить или щелкните по наименованию интересующего Вас размера сейфа. Затем в поле "Описание" измените размер сейфовой ячейки и при необходимости установите галочки напротив интересующих Вас сроков аренды сейфа. После того как параметры изменены, щелкните по кнопке Сохранить, и измененный размер сейфа появится в списке размеров сейфов. Если Вы не хотите менять размер сейфовой ячейки, щелкните по кнопке Список, и система вернет Вас в список размеров сейфов.

Проверка наличия размера сейфа

Для того чтобы проверить наличие свободного сейфа в банке, установите галочку напротив интересующего Вас размера сейфовой ячейки и щелкните по кнопке Проверить наличие. В результате на экран будет выведен список подразделений, в котором данный размер сейфа доступен для аренды.

Удаление размера сейфа

Если Вы хотите удалить размер сейфа из списка, установите галочку напротив интересующего Вас размера и щелкните по кнопке Удалить, и выбранный размер сейфовой ячейки будет удален системой из списка.

Сроки аренды

В пункте меню Сроки аренды Вы можете добавить и удалить интересующий Вас срок аренды сейфовой ячейки.

Добавление срока аренды

Чтобы добавить новый срок аренды сейфа, в поле "Описание" введите новый срок аренды, в поле "Порядок сортировки" укажите числовое значение, определяющее последовательность пунктов в списке, причем пункты с меньшим номером появляются в списке выше, с большим номером - ниже, затем щелкните по кнопке Добавить, и система добавит в список сроков аренды новый период.

Удаление срока аренды

Если Вы хотите удалить интересующий Вас срок аренды сейфовой ячейки, установите галочку напротив интересующего Вас срока и щелкните по кнопке Удалить, и выбранный срок будет удален из списка.

Сейфы в офисах

В пункте меню Сейфы в офисах Вы можете самостоятельно настраивать перечень размеров сейфов, доступных в каждом офисе банка. В данном пункте автоматически открывается список офисов банка, в которых клиент банка сможет арендовать сейфовую ячейку определенного размера.

Если Вам нужно указать интересующую Вас сейфовую ячейку, которая доступна клиентам банка в конкретном офисе для аренды, установите галочку напротив выбранной ячейки и щелкните по кнопке Сохранить, и ячейка появится в списке свободных сейфовых ячеек в пункте меню Свободные сейфы.

Если Вы хотите удалить из списка свободных сейфов ячейку, уберите галочку напротив выбранной ячейки и щелкните по кнопке Сохранить, и интересующая Вас сейфовая ячейка будет удалена из списка свободных сейфов.
