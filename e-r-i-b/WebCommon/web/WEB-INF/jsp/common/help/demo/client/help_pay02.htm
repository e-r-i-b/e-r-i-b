Назначение перевода

Данный перевод позволяет Вам перевести средства со своего рублевого счета на счет другого юридического или физического лица. Чтобы создать перевод, в главном меню выберите пункт Платежи, а затем пункт левого меню Рублевый перевод.

Заполнение документа

В форме перевода рублей РФ Вам надо указать дату (в это поле автоматически будет подставлена текущая дата), выбрать из списка доступных Вам счетов счет списания (счет, с которого будут сняты деньги) и указать сумму переводимых средств.

Далее необходимо указать, кому Вы переводите средства. Сначала укажите является получатель юридическим или физическим лицом, установив галочку напротив соответствующего получателя. После этого необходимо заполнить реквизиты получателя: название (для юридического лица) или ФИО (для физического лица), ИНН и КПП (для физического лица КПП не нужно указывать), номер счета, на который будут перечислены средства, и реквизиты банка, в котором этот счет открыт. Не забудьте указать назначение платежа.

Для быстрого заполнения реквизитов можно воспользоваться предусмотренными в системе справочником получателей и справочником банков. Для доступа к справочнику получателей щелкните по кнопке (...), расположенной рядом с полем Наименование (ФИО). Откроется страница со справочником получателей, на которой надо выбрать необходимого получателя, после чего щелкнуть по кнопке Выбрать. На данной странице Вы можете воспользоваться фильтром.

Фильтр получателей

Для того чтобы воспользоваться фильтром, щелкните по кнопке Показать фильтр и заполните поля фильтра:

"Наименование" - поиск по названию получателя;

"Счет" - поиск по номеру счета получателя;

"Банк" - поиск по названию банка получателя.

Вы можете фильтровать список по любой комбинации полей, например, можно отобрать всех получателей, наименование которых начинается на одну и ту же букву и счета которых открыты в одном банке.

После того как критерии поиска заданы, щелкните по кнопке Построить список, и система покажет список интересующих Вас получателей.

После выбора получателя все сведения будут скопированы в создаваемый документ.

Для доступа к справочнику банков щелкните по кнопке (...), расположенной рядом с полем БИК. Откроется страница со справочником банков, на которой надо выбрать банк получателя, после чего щелкнуть по кнопке Выбрать. Также на этой странице Вы можете воспользоваться фильтром.

Фильтр банков получателей

Для того чтобы воспользоваться фильтром, щелкните по кнопке Показать фильтр и заполните поля фильтра:

"Наименование" - поиск по названию банка получателя;

"БИК" - поиск по банковскому идентификационному коду;

"Город" - поиск по городу, в котором зарегистрирован банк.

Вы можете фильтровать список по любой комбинации полей, например, можно отобрать все банки, название которых начинается на одну и ту же букву и из одного города.

После того как критерии поиска заданы, щелкните по кнопке Построить список, и система покажет список интересующих Вас банков.

После выбора банка получателя все реквизиты банка будут скопированы в создаваемый документ.

На данной странице Вы можете перейти к форме заполнения налогового платежа, для этого щелкните по ссылке Осуществить налоговый платеж.

После того как все поля заполнены, щелкните по кнопке Сохранить. Вы также можете добавить частично заполненный документ в свой список шаблонов, щелкнув по кнопке Сохранить как шаблон, чтобы в дальнейшем использовать его для быстрого формирования аналогичных документов.

Если Вы решили не делать перевод, с данной страницы можно вернуться к списку выполненных Вами переводов, щелкнув по кнопке Список, или на страницу типов платежей, щелкнув по кнопке Платежи.

Подтверждение перевода

После сохранения заполненного документа Вы попадаете на страницу подтверждения. Здесь Вам будет показана заполненная форма перевода, в которой Вы должны проверить правильность указанных сведений, после чего выполнить одно из следующих действий:

  • Подтверждение перевода

    Процедура подтверждения зависит от используемого Вами способа регистрации в системе. Например, если Вы используете карту одноразовых паролей, то Вам необходимо будет ввести пароль с указанным номером с Вашей карты одноразовых паролей.

    • Когда все сведения указаны, щелкните по кнопке Подтвердить, и документ будет отправлен в банк на обработку.

  • Возврат к редактированию

    Если Вы хотите изменить какие-либо реквизиты перевода, щелкните по кнопке Редактировать, и Вы вернетесь к форме редактирования документа.

  • Создание нового документа

    Вы можете создать еще один документ данного типа, щелкнув по кнопке Создать. В результате Вы перейдете к форме ввода нового документа. Данная возможность полезна при создании нескольких однотипных документов.

  • Печать документа

    Если Вы хотите распечатать данный документ, щелкните по кнопке Печать.

  • Отказ от выполнения перевода

    Если Вы передумали осуществлять рублевый перевод, можно вернуться к списку выполненных Вами переводов, щелкнув по кнопке Список.

Просмотр документа

После подтверждения платежа Вы попадаете на страницу просмотра, где отображается заполненный документ. О том, что документ передан в банк, свидетельствует отображаемый на форме документа штамп «Принято к исполнению». На страницу просмотра документа Вы можете также попасть из списка документов, щелкнув по номеру того документа, который хотите просмотреть. На странице просмотра Вы можете выполнить следующие действия:

  • Отзыв документа

    Вы можете отозвать ошибочно отправленный документ. Для этого щелкните по кнопке Отозвать, в результате Вы перейдете к форме отзыва перевода. На экран будет выведена подробная информация по отзываемому документу. При необходимости измените значение в поле "Дата отзыва", информация в остальных полях документа недоступна для редактирования. Для отзыва перевода щелкните по кнопке Сохранить. Откроется страница, на которой проверьте правильность заполнения полей документа и нажмите на кнопку Подтвердить, чтобы передать документ в банк на исполнение.

  • Создание нового документа

    Вы можете создать еще один документ данного типа, щелкнув по кнопке Создать. В результате Вы перейдете к форме ввода нового документа. Данная возможность полезна при создании нескольких однотипных документов.

  • Печать документа

    Если Вы хотите распечатать данный документ, щелкните по кнопке Печать.

  • Возврат к списку переводов

    Можно вернуться к списку выполненных Вами рублевых переводов, щелкнув по кнопке Список.

Список рублевых переводов

В данном списке отображаются все выполненные Вами рублевые переводы на счета юридических или физических лиц. Вы можете зайти в список операций со страницы выбора типа платежа, щелкнув на выбранном платеже по кнопке История операций, или при создании платежа, щелкнув по кнопке Список. В списке для каждого перевода рублей РФ отображается его дата, номер документа, счета списания и зачисления, сумма перевода и статус, отображающий текущее состояние операции. Вы можете выполнить следующие операции:

  • Просмотр документа

    Вы можете просмотреть любой документ, для этого нужно щелкнуть по номеру интересующего Вас документа в списке.

  • Отзыв документа

    Вы можете отозвать ошибочно отправленный документ. Для этого щелкните по кнопке Отозвать, в результате Вы перейдете к форме отзыва перевода. На экран будет выведена подробная информация по отзываемому документу. При необходимости измените значение в поле "Дата отзыва", информация в остальных полях документа недоступна для редактирования. Для отзыва перевода щелкните по кнопке Сохранить. Откроется страница, на которой проверьте правильность заполнения полей документа и нажмите на кнопку Подтвердить, чтобы передать документ в банк на исполнение.

  • Создание копии документа

    Вы можете создать копию документа, для этого отметьте документ, который служит оригиналом в списке, и щелкните по кнопке Создать копию. В результате Вы перейдете к форме ввода нового документа, причем в нее будут скопированы все данные из документа, выбранного Вами в качестве оригинала. Данная возможность полезна при создании нескольких однотипных документов.

  • Удаление платежа

    Вы можете удалить перевод из списка, для этого установите галочку напротив интересующего документа и щелкните по кнопке Удалить.

  • Фильтр переводов

    Для того чтобы найти интересующий Вас перевод в списке, щелкните по кнопке Показать фильтр и заполните поля фильтра:

    "Статус" - поиск по статусу перевода в системе: "Введен", "На исполнении", "Исполнен", "Отказан", "Отозван";

    "Период" - поиск по времени создания перевода;

    "Сумма" - поиск по размеру суммы перевода;

    "Счет списания" - поиск по номеру счета, с которого списаны средства;

    "Счет получателя" - поиск по номеру счета получателя перевода.

    Вы можете фильтровать список по любой комбинации полей, например, можно отобрать все рублевые переводы на одну сумму со статусом "Исполнен".

    После того как критерии поиска заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас переводов.

Статусы платежа

Ход обработки платежа Вы можете наблюдать в списке платежей или в журнале операций по статусу. Для рублевых переводов предусмотрены следующие статусы:

  • Введен - платеж введен, сохранен в системе, но не подтвержден;

  • Обрабатывается - платеж полностью заполнен клиентом банка, подтвержден и передан на обработку в банк;

  • Ожидание обработки - ожидается передача документа в банк или получение информации о результате его исполнения;

  • Отозван - платеж передан в банк на обработку, но затем Вами отозван по какой-либо причине;

  • Исполнен - платеж проведен в банке: денежные средства со счета клиента поступили на указанный в форме счет зачисления;

  • Отказан - платеж отказан банком с указанием причины, например, по причине нехватки денежных средств на счете клиента на момент их списания.
