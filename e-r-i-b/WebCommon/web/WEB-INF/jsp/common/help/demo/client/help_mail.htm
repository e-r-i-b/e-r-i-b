В пункте меню Почта Вы можете просмотреть письма, присланные из банка, ответить на них, а также создать новые письма.

Список полученных писем

Фильтр полученных писем

В пункте главного меню Почта автоматически открывается страница со списком полученных писем. По каждому письму в списке отображаются: "Тема" и "Дата".

Примечание:

  1. непрочитанные письма будут отображаться в списке входящих писем с пиктограммой ;

  2. письма, обязательные для прочтения, будут отображаться в списке со значком .

На данной странице Вы можете выполнить следующие операции:

Просмотреть письмо

Вы можете просмотреть текст любого письма. Для этого необходимо установить галочку напротив выбранного Вами письма и нажать на кнопку Просмотр или щелкнуть по теме письма.

Для возврата к списку входящих писем щелкните по кнопке Список.

Ответить на полученное письмо

Вы можете ответить на полученное письмо. Для этого установите флажок напротив интересующего Вас письма и нажмите на кнопку Ответить.

Примечание: выполнить данную операцию Вы можете также на странице просмотра письма, щелкнув по кнопке Ответить.

Создать новое письмо

Вы можете создать новое письмо. Для этого щелкните по кнопке Новое письмо, откроется страница редактирования письма, на которой Вам необходимо заполнить все поля:

  • В поле "Тип письма" выберите из выпадающего списка интересующий Вас тип письма: "Письмо", "Вопрос", "Жалоба", "Проблема". По умолчанию система задает значение "Письмо".

  • Значение в поле "Номер" заполняется системой автоматически.

  • В поле "Тема" укажите тему письма.

  • В поле "Текст" впишите полный текст письма.

После заполнения всех полей формы создания письма щелкните по кнопке Отправить, и письмо будет отправлено в банк. Если Вы не хотите отправлять письмо, щелкните по кнопке Отмена.

Фильтр полученных писем

Вы можете воспользоваться фильтром/поиском для осуществления отбора входящих писем в списке по любым из указанных в нем полей либо их комбинации. Для этого щелкните по кнопке Показать фильтр.

Откроется страница с формой, в которой необходимо задать критерии поиска:

  • "Тема" - поиск по теме полученного письма.

  • "Период с...по..." - поиск по периоду времени, в течение которого письмо было получено.

  • "Обязательно для прочтения" - поиск по наличию у письма признака обязательности прочтения.

  • "Прочтено" - поиск по признаку прочтения письма: прочитали Вы полученное письмо или нет.

После заполнения полей фильтра нажмите на кнопку Применить.

Для редактирования полей фильтра нажмите на кнопку Очистить, внесите необходимые изменения и вновь щелкните по кнопке Применить, после чего система отобразит отфильтрованный список входящих писем по заданным Вами параметрам.

Список отправленных писем

Фильтр отправленных писем

В данном пункте левого меню Вы можете просмотреть список отправленных писем, создать новое письмо, а также воспользоваться фильтром для поиска интересующего письма. По каждому письму в списке отображаются: "Номер", "Тема", "Дата" и "Тип письма".

На данной странице Вы можете выполнить следующие операции:

Просмотреть исходящее письмо

Вы можете просмотреть текст любого отправленного Вами письма. Для этого установите флажок напротив интересующего Вас письма и нажмите на кнопку Просмотр или щелкните по номеру письма в списке.

Для возврата к списку отправленных писем щелкните по кнопке Список.

Создать новое письмо

Вы можете создать новое письмо. Для этого нажмите на кнопку Создать.

  • В поле "Тип письма" выберите из выпадающего списка интересующий Вас тип письма: "Письмо", "Вопрос", "Жалоба", "Проблема". По умолчанию система задает значение "Письмо".

  • Значение в поле "Номер" заполняется системой автоматически.

  • В поле "Тема" укажите тему письма.

  • В поле "Текст" впишите полный текст письма.

После заполнения всех полей формы создания письма щелкните по кнопке Отправить, и письмо будет отправлено в банк. Если Вы не хотите отправлять письмо, щелкните по кнопке Отмена.

Фильтр отправленных писем

Вы можете воспользоваться фильтром/поиском для осуществления отбора исходящих писем в списке по любым из указанных в нем полей либо их комбинации. Для этого щелкните по кнопке Показать фильтр.

Откроется страница с формой, в которой необходимо задать критерии поиска:

  • "Тема" - поиск по теме отправленного письма.

  • "Тип" - поиск по типу письма: "Все", "Письмо", "Вопрос", "Жалоба", "Проблема".

  • "Период с...по..." - поиск по периоду времени, в течение которого письмо было отправлено в банк.

После заполнения полей фильтра нажмите на кнопку Применить.

Для редактирования полей фильтра нажмите на кнопку Очистить, внесите необходимые изменения и вновь щелкните по кнопке Применить, после чего система отобразит отфильтрованный список отправленных писем по заданным Вами параметрам.
