Список сертификатов

В пункте меню Сертификаты ведется работа с выдаваемыми клиентам сертификатами СКЗИ (Системы криптозащиты информации). С помощью сертификата клиент банка может осуществлять безопасный вход в систему в защищенном режиме и подписывать документы (платежи, заявки) ЭЦП (электронно-цифровой подписью), которая позволит сотруднику банка убедиться в аутентичности отправленных клиентом данных, а также проверить, была ли информация изменена (искажена) в процессе передачи, гарантируя целостность данных.

Список запросов на сертификат

Фильтр запросов

Вы можете просмотреть список поступивших от клиентов запросов на сертификат, принять запрос на создание сертификата или отказать клиенту в создании сертификата.

В пункте главного меню Сертификаты автоматически открывается страница со списком запросов на сертификат, поступивших от клиентов банка.

По каждому запросу отображается следующая информация: идентификатор запроса, ФИО клиента, дата создания запроса и статус запроса на сертификат.

Примечание: если клиент делает запрос на перевыпуск сертификата при наличии действующего на данный момент сертификата, то в списке запросов данный запрос на сертификат будет отображаться со специальной пиктограммой.

Если Вы хотите просмотреть запрос на сертификат, поступивший от клиента банка, щелкните на номере сертификата и Вам откроется форма просмотра информации о запросе.

Для создания сертификата Вы должны выполнить следующие действия:

  1. Выгрузить запрос на сертификат в центр сертификации.

    Для того чтобы выгрузить запрос на сертификат в центр сертификации, щелкните на номере интересующего Вас сертификата и нажмите кнопку Выгрузить. Откроется стандартное окно Windows Сохранить, и Вы сохраняете файл с запросом на сертификат на диск.

  2. Загрузить ответ из центра сертификации.

    Центр сертификации выдает файл с ответом на запрос и файл с сертификатом. Для того чтобы загрузить данные файлы, щелкните на номере интересующего Вас сертификата и нажмите кнопку Загрузить. Откроется страница загрузки данных о сертификате, на которой Вы должны указать "Файл с ответом на запрос" и "Файл с сертификатом".

    Чтобы указать файл с ответом, щелкните по кнопке Обзор, выберите интересующий Вас файл с ответом, нажмите кнопку Открыть, и в поле появится название файла с ответом на запрос.

    Чтобы указать файл с сертификатом, щелкните по кнопке Обзор, выберите файл с сертификатом и нажмите кнопку Открыть.

    После заполнения полей щелкните по кнопке Загрузить. Откроется страница подтверждения загрузки сертификата. Для регистрации сертификата нажмите на кнопку Зарегистрировать, после чего запрос регистрируется как принадлежащий клиенту со статусом "Сертификат выдан".

    Загруженный в систему сертификат теперь будет отображаться в списке сертификатов.

Чтобы вернуться в список запросов на сертификат, щелкните по кнопке Список, и система вернет Вас на страницу списка запросов на сертификат.

Если Вам необходимо отказать клиенту банка в запросе на сертификат, установите флажок напротив интересующего запроса и щелкните по кнопке Отказать. Система попросит Вас подтвердить операцию, Вы подтверждаете отказ, открывается страница отказа запроса на сертификат, на которой в поле "Причина отказа" Вы должны вписать причину отказа запроса и щелкнуть по кнопке Отказать. В результате система присвоит данному запросу в списке статус "Отказан".

Фильтр запросов

Для просмотра списка всех интересующих Вас запросов нужно щелкнуть по кнопке Показать фильтр и правильно заполнить все поля фильтра:

  • "Фамилия, имя, отчество" - поиск по фамилии, имени и отчеству клиента банка;

  • "Дата с.. по.." - поиск по дате запроса;

  • "Статус" - поиск по статусу запроса на сертификат. Вы можете выбрать статус запроса на сертификат: "Отправлен" (запрос отправлен на обработку в банк), "Обрабатывается" (запрос обрабатывается системой), "Сертификат выдан" (сертификат уже выдан клиенту банка), "Отказан" (банк отказал клиенту в создании сертификата), "Сертификат установлен" (сертификат установлен в системе для защиты операций конкретного клиента), "Сертификат удален" (сертификат удален из системы).

  • "Подписан" - поиск по электронно-цифровой подписи, т.е. нужно указать подписан или нет интересующий Вас сертификат. Сертификат подписывается в том случае, если клиент банка запрашивает перевыпуск сертификата при истечении срока действия предыдущего сертификата.

Вы можете осуществлять поиск по подстроке в текстовых полях фильтра. Например, можно ввести в поля фильтра только несколько первых букв фамилии, имени или отчества клиента банка.

Вы можете фильтровать список по любой комбинации полей, например, по первой букве фамилии клиента банка и по статусу сертификата в системе.

После того как искомые параметры заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас запросов на сертификат.

Список сертификатов

Фильтр сертификатов

В пункте меню Список сертификатов Вы можете просмотреть список всех выданных клиентам банка сертификатов, активировать, заблокировать и удалить сертификат из списка.

В данном пункте меню автоматически открывается страница со списком сертификатов. По каждому сертификату в списке отображается следующая информация: идентификатор сертификата, ФИО клиента, дата начала действия, дата окончания действия, номер ПИН-конверта и статус сертификата.

С каждым сертификатом Вы можете выполнить следующие действия:

  • Просмотреть информацию по сертификату

    Для того чтобы просмотреть информацию по сертификату, щелкните по идентификатору выбранного сертификата в списке. Откроется страница просмотра, на которой Вы можете просмотреть следующую информацию по сертификату: идентификатор, ФИО клиента, дату начала и дату окончания действия сертификата и его статус. Если сертификат "Неактивный", то на странице просмотра Вы можете активировать сертификат, щелкнув по кнопке Активировать.

    Если Вам необходимо вернуться в список сертификатов, нажмите на кнопку Список.

  • Активировать сертификат

    Если Вам необходимо активировать сертификат, установите флажок напротив сертификата со статусом "Неактивный" и щелкните по кнопке Активировать. Система присвоит сертификату статус "Активный", и клиент банка может использовать данный сертификат для входа в систему в защищенном режиме и для подписания документов.

  • Блокировать сертификат

    Если Вам нужно заблокировать сертификат в случае утери или компрометации секретного ключа клиента, установите напротив сертификата со статусом "Активный" флажок и щелкните по кнопке Заблокировать. Система присвоит сертификату статус "Заблокирован", и клиент банка не сможет использовать сертификат с таким статусом для работы в системе.

  • Удалить сертификат

    Если Вам нужно удалить сертификат из списка, установите флажок напротив сертификата со статусом "Неактивный" и нажмите на кнопку Удалить, после чего система удалит выбранный Вами сертификат из списка сертификатов.

Фильтр сертификатов

Для просмотра списка всех интересующих Вас сертификатов нужно щелкнуть по кнопке Показать фильтр и правильно заполнить все поля фильтра:

  • "Фамилия, имя, отчество" - поиск по фамилии, имени и отчеству клиента банка.

  • "Дата начала действия" - поиск по дате начала действия сертификата;

  • "Дата окончания действия" - поиск по дате окончания действия сертификата;

  • "Номер ПИН-конверта" - поиск по номеру ПИН-конверта, привязанного клиенту, которому выдан данный сертификат;

  • "Статус" - поиск по статусу сертификата: "Истек" (время действия сертификата закончилось), "Активный" (сертификат действителен и может использоваться клиентом банка для авторизации и подписания документов) и др.

Точно указать дату создания и дату окончания действия сертификата сложно, поэтому осуществлять поиск сертификата лучше либо только по дате начала действия сертификата, либо только по дате окончания действия сертификата, либо вообще не указывая дату.

Вы можете осуществлять поиск по подстроке в текстовых полях фильтра. Например, можно ввести в поля фильтра только несколько первых букв фамилии, имени или отчества клиента банка.

Вы можете фильтровать список по любой комбинации полей, например, по первой букве фамилии клиента банка и по статусу сертификата в системе.

После того как искомые параметры заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас сертификатов.
