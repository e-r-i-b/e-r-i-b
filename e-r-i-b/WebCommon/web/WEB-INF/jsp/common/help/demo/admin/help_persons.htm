Список клиентов

Фильтр/поиск клиентов

Пункт меню Клиенты предназначен для управления пользователями системы. Он позволяет регистрировать новых клиентов (импортировать данные клиентов из установленной в банке системы розничного обслуживания), удалять, блокировать и разблокировать клиентов, управлять доступом клиентов к счетам и картам, назначать клиентам схемы доступа, определяющие доступные им операции, управлять картами ключей клиентов и др.

В данном пункте меню автоматически открывается список всех клиентов, закрепленных за определенным подразделением банка. В списке для каждого клиента отображается следующая информация: ID (идентификатор пользователя в системе), ФИО, номер ПИН-конверта, номер договора, схема прав доступа для упрощенного и для защищенного доступа в систему и статус клиента в системе.

Добавление клиента

Для того чтобы завести в системе нового клиента, щелкните по кнопке Добавить. Дальнейшая процедура регистрации клиентов банка описана в разделе контекстной справки "Регистрация клиентов".

Редактирование клиента

В случае, если изменились сведения о клиенте или нужно подключить новые счета пользователя, Вы можете отредактировать сведения о клиенте. Для этого щелкните на фамилии, имени, отчестве клиента, откроется анкета клиента банка, внесите изменения на интересующих Вас вкладках анкеты и нажмите на кнопку Сохранить.

На данной странице Вы можете назначить права доступа клиентов к операциям в системе, для этого установите флажки напротив интересующих Вас клиентов, в поле "Схема прав" выберите название схемы прав и щелкните по кнопке Назначить права.

Удаление клиента/прекращение обслуживания

Прекращение обслуживания клиента в системе ИКФЛ осуществляется с клиентом, который имеет статус "Активный". Данную операцию можно выполнить из списка клиентов банка с помощью кнопки Удалить и из анкеты клиента.

Чтобы прекратить обслуживание клиента в системе ИКФЛ, в анкете клиента банка введите в поле "Дата расторжения договора" дату расторжения, в поле "Причина расторжения договора" выберите причину из списка и щелкните по кнопке Сохранить. Далее распечатайте заявление о прекращении обслуживания, щелкнув по кнопке Прекращение обслуживания. Затем нажмите на кнопку Удалить и подтвердите операцию. В результате клиент будет удален из списка клиентов банка и появится в списке архива клиентов банка.

Вы можете полностью удалить из системы клиента, имеющего статус "Подписание договора" и "Подключение". Для этого установите галочку напротив интересующего Вас клиента и щелкните по кнопке Удалить.

Фильтр/поиск клиентов

Для облегчения поиска требуемого клиента в системе предусмотрен фильтр. Чтобы воспользоваться фильтром, щелкните по кнопке Показать фильтр и задайте в появившихся в верхней части списка полях критерии поиска:

  • "ФИО клиента" - поиск по фамилии, имени и отчеству клиента;

  • "Вид документа" - поиск по виду документа, удостоверяющего личность клиента;

  • "Серия" - поиск по серии паспорта клиента банка;

  • "Номер" - поиск по номеру паспорта клиента банка;

  • "Номер ПИН-конверта" - поиск по номеру ПИН-конверта клиента банка;

  • "Номер договора" - поиск по номеру договора клиента банка;

  • "Статус" - поиск по статусу клиента банка в системе ("Активный", "Заблокирован", "Подключение");

  • "Идентификатор пользователя (ID)" - поиск по идентификатору пользователя в системе.

Вы можете осуществлять поиск по подстроке в текстовых полях фильтра. Например, можно ввести в поля фильтра только несколько первых букв фамилии, имени или отчества клиента.

Также Вы можете фильтровать список по любой комбинации полей. Например, можно отобрать всех активных клиентов с фамилией на букву "А" с паспортом.

После того как искомые параметры заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас клиентов банка.

Общие сведения

В данной вкладке анкеты клиента банка вводятся сведения о клиенте. Основную информацию о клиенте Вы импортировали из САРБУ. Остальные сведения Вы вводите самостоятельно. Обязательны для заполнения все поля, отмеченные звездочкой: дата рождения, гражданство, серия, номер, дата выдачи, кем выдан документ, удостоверяющий личность клиента, кодовое слово клиента, номер ПИН-конверта, дата начала обслуживания клиента.

Примечание: если Вы не имеете ПИН-конверт для данного клиента, его необходимо создать в системе.

Чтобы создать и привязать клиенту банка ПИН-конверт, выберите пункт главного меню ПИН-конверты, введите количество ПИН-конвертов в запросе и подразделение банка, за которым закреплен клиент, выгрузите запрос на генерацию ПИН-конверта, щелкнув по кнопке Выгрузить.

Откроется стандартное окно Windows Сохранить. Сохраните файл на сменный носитель (диск, "флешку"), отнесите его на сервер, где установлена утилита печати ПИН-конвертов, скопируйте файл в каталог XML_in, запустите утилиту печати и распечатайте ПИН-конверты.

Чтобы загрузить ответ из утилиты печати, Вам нужно на сервере, где установлена утилита печати, из каталога XML_out скопировать файл ответа на сменный носитель, отнести его на компьютер, где запущена система ИКФЛ, указать путь до файла с ответом и щелкнуть по кнопке Выполнить импорт.

Вы имеете возможность сохранить частично заполненную анкету. Для этого щелкните по кнопке Сохранить .

Если Вы хотите выйти из анкеты клиента, щелкните по кнопке Назад.

Счета и карты

Для каждого пользователя необходимо создать список счетов и пластиковых карт, к которым он будет иметь доступ. Обратите внимание, что при импорте анкеты клиента банка из системы розничного обслуживания при загрузке автоматически загружаются все имеющиеся там счета и пластиковые карты данного клиента. Причем клиент будет иметь доступ ко всем этим счетам и картам, так что может потребоваться удалить лишние счета и/или карты из списка.

Чтобы добавить или удалить счета/карты клиента из системы, выполните следующие действия:

  • В анкете клиента выберите пункт Счета и карты, на экран будет выведен список счетов и карт, к которым клиент имеет доступ.

  • Чтобы добавить в список счета (карты), щелкните по кнопке Добавить счет (если Вы хотите добавить сведения о пластиковых картах, то надо щелкнуть по кнопке Добавить карту).

  • Если Вы хотите удалить счет/карту клиента из списка, установите галочки напротив интересующих счетов/карт и щелкните по кнопке Удалить.

  • На экран будет выведен список счетов (или пластиковых карт) клиента. Выберите те счета (или карты), сведения о которых необходимо загрузить, чтобы клиент имел к ним доступ через систему ИКФЛ. Для этого проставьте рядом с ними флажки и щелкните по кнопке Добавить.

Права доступа

Затем задайте тип доступа клиента к системе: через Интернет, назначив клиенту схемы прав доступа для каждого способа аутентификации, и при необходимости через SMS-банкинг, т.е. для того чтобы клиент получил доступ к системе через SMS-обслуживание.

Схемы прав определяют возможности Интернет-обслуживания, которые будут доступны клиенту после входа в систему заданным способом.

Для доступа клиента через интернет Вам необходимо для клиента задать способы аутентификации в системе ИКФЛ: "Упрощенный" с помощью логина/пароля и "Защищенный" с аутентификацией по сертификату ключа ЭЦП и для каждого из них указать:

  1. Способ подтверждения входа.

    • При упрощенном доступе Вы можете установить способ "Без подтверждения", т.е. клиент банка входит в систему только с помощью логина/пароля, или с подтверждением: одноразовым паролем с карты ключей, SMS-паролем.

      Примечание: SMS-пароль – это одноразовый пароль, формируемый системой ИКФЛ.

      Если Вы назначите клиенту подтверждение входа sms-паролем, то данному клиенту необходимо будет ввести на странице регистрации логин и пароль, затем нажать на кнопку Продолжить. Система ИКФЛ сформирует одноразовый SMS-пароль и отправит его клиенту на номер мобильного телефона, указанный в анкете. В результате откроется специальная страница, на которой клиенту необходимо ввести одноразовый SMS-пароль.

    • При защищенном доступе вход клиента в систему осуществляется с помощью сертификата (ЭЦП).

  2. Способ подтверждения операций в системе.

    • При защищенном доступе подтверждение операций возможно только с помощью ЭЦП.

    • При упрощенном доступе Вы можете выбрать следующие способы подтверждения операций клиентом банка:

      • "Одноразовый пароль" - подтверждение операций одноразовым паролем с карты ключей;

      • "SMS" - подтверждение операций клиентом банка с помощью SMS-пароля.

  3. Схему прав доступа, т.е. перечень операций, доступных клиенту при входе в систему данным способом. Например, при аутентификации по сертификату пользователь получает права на выполнение всех операций, включая платежные, а при входе в систему по логину и паролю он получает только информационный доступ.

    Чтобы назначить клиенту схему прав доступа, в поле "Схема прав" выберите из выпадающего списка схему доступа или задайте индивидуальные права, установив флажки у тех операций, которые будет осуществлять данный клиент, и щелкните по кнопке Сохранить .

Для доступа данного клиента к услуге SMS–банкинга установите флажок в поле «SMS-банкинг» и выберите способ подтверждения операций: без подтверждения (только по логину и паролю) или с помощью одноразового пароля (дополнительно клиенту необходимо будет указывать пароль с карты ключей).

Список представителей

На вкладке анкеты Представители Вы можете просмотреть, добавить или изменить по желанию клиента представителей, которые имеют доступ к выбранным счетам, картам и операциям клиента в системе.

Регистрация представителя клиента

Подключить нового представителя клиенту банка Вы можете на вкладке анкеты клиента Представители, щелкнув по кнопке Добавить.

Откроется анкета создания нового представителя, в которой Вам нужно заполнить ФИО, дату рождения, гражданство, серию и номер документа, удостоверяющего личность представителя клиента, дату выдачи, кем выдан документ, номер ПИН-конверта и дату начала обслуживания. Обязательны для заполнения все поля, отмеченные звездочкой.

Примечание: если Вы не имеете ПИН-конверт для представителя клиента банка, Вам необходимо создать ПИН-конверт в системе и привязать его представителю. Для этого выберите пункт главного меню ПИН-конверты, введите количество ПИН-конвертов в запросе и подразделение банка, за которым закреплен клиент, выгрузите запрос на генерацию ПИН-конверта, щелкнув по кнопке Выгрузить.

Откроется стандартное окно Windows Сохранить. Сохраните файл на сменный носитель (диск, "флешку"), отнесите его на сервер, где установлена утилита печати ПИН-конвертов, скопируйте файл в каталог XML_in, запустите утилиту печати и распечатайте ПИН-конверты.

Чтобы загрузить ответ из утилиты печати, Вам нужно на сервере, где установлена утилита печати, скопировать из каталога XML_out файл ответа на сменный носитель, отнести его на компьютер, где запущена система ИКФЛ, указать путь до файла с ответом, и щелкнуть по кнопке Выполнить импорт.

Далее Вам необходимо выбрать доступные представителю счета и карты клиента банка, для этого поставьте флажок напротив пункта Перечень доступных счетов или Перечень доступных карт.

Затем Вам нужно указать список доступных данному представителю операций в системе. Для этого выберите тип доступа представителя в систему (упрощенный, защищенный), установив флажок напротив интересующего типа доступа. Далее укажите способ подтверждения входа в систему: при упрощенном типе доступа Вы можете задать подтверждение входа с помощью SMS, при защищенном доступа подтверждение осуществляется с помощью ЭЦП. А также выберите способ подтверждения операций: для упрощенного доступа можно задать без подтверждения или с подтверждением SMS-паролем, для защищенного типа доступа подтверждение операций осуществляется с помощью ЭЦП. Затем для предоставления представителю клиента доступа к определенным операциям поставьте флажки напротив выбранных операций и щелкните по кнопке Сохранить. Если Вы хотите разрешить представителю клиента выполнять все операции, то установите флажок в поле "Доступ" и нажмите на кнопку Сохранить.

Примечание: представителю необходимо предоставить доступ хотя бы к одному счету или карте клиента и к одной операции в системе.

Чтобы активировать представителя клиента банка, в анкете представителя щелкните по кнопке Подключить представителя. Если все поля анкеты со звездочкой заполнены правильно, то представителю клиента система присвоит статус "Активный".

Вы можете заблокировать представителя по желанию клиента. Для этого в анкете представителя щелкните по кнопке Заблокировать, укажите срок и причину блокировки и нажмите на кнопку Заблокировать.

Если Вам нужно распечатать анкету представителя клиента, щелкните по кнопке Печать анкеты.

Группы

Вы можете включить клиента в состав группы, для этого установите флажок напротив той группы, в которую хотите включить клиента, и щелкните по кнопке Сохранить. Включение клиента в группу дает возможность отправлять письма группе клиентов.

Карты ключей

С помощью карт ключей клиент банка может подтвердить доступ и совершение операций в системе.

В пункте левого меню Карты ключей автоматически открывается страница со списком карт ключей, которые привязаны к данному клиенту банка. По каждой карте ключей отображается следующая информация: номер карты ключей, дата создания, дата выдачи, дата активации, количество ключей на карте, статус карты, тип блокировки, причина блокировки.

На данной вкладке анкеты клиента банка Вы можете выполнить следующие операции:

Добавить и активировать карту ключей.

Чтобы привязать и активировать карту ключей клиенту банка, щелкните по кнопке Добавить, введите номер карты, которую нужно добавить клиенту банка, и снова нажмите кнопку Добавить. Карта ключей будет добавлена клиенту. Если у клиента нет в данный период активной карты, то карте система сразу присвоит статус "Активна". Если добавленная карта ключей неактивна, чтобы ее активировать, щелкните по кнопке Активировать. У клиента может быть активна только одна карта ключей, если Вы хотите активировать другую карту ключей, то предыдущая карта автоматически заблокируется.

Примечание: если у клиента нет карты ключей, ее надо создать. Подробную информацию по созданию карты ключей смотрите в разделе контекстной справки пункта главного меню Карты ключей.

Заблокировать карту ключей.

Блокировка карты может выполняться следующим образом:

  • Карта автоматически блокируется при неверном вводе пароля определенное количество раз подряд. Количество попыток для ввода пароля до автоматической блокировки доступа задается с помощью пункта меню Настройки - Клиенты и сотрудники. См. соответствующие разделы контекстной справки.

  • Карта может быть заблокирована сотрудником банка по телефонному или личному обращению клиента (например, в случае утери карты). Идентификация клиента сотрудником банка в этом случае может осуществляться, например, по кодовому слову, которое клиент должен сообщить сотруднику банка. Кодовое слово содержится в анкете клиента банка.

    Для блокировки карт ключей необходимо выбрать клиента, кликнуть на его имени, откроется анкета клиента банка. Выберите вкладку анкеты Карты ключей, в ней найдите карту, которую необходимо заблокировать, и щелкните по кнопке Заблокировать.

Разблокировать карту ключей.

Вы можете снять блокировку с карты ключей клиента. Разблокировка карты выполняется в случае, если карта была заблокирована автоматически, но карта осталась при клиенте и может использоваться по назначению дальше. В этом случае, обращаясь в банк по телефону или лично, клиент должен назвать сотруднику банка очередной пароль из карты. Сотрудник банка вводит этот пароль в системе, и, если пароль верный, карта будет разблокирована. Для этого нужно выбрать клиента, кликнуть на его имени, откроется анкета клиента банка, выберите вкладку анкеты Карты ключей, в ней найдите карту, которую необходимо разблокировать, и щелкните по кнопке Снять блокировку.

Фильтр карт ключей.

Вы можете воспользоваться фильтром для поиска карт ключей. Для этого щелкните по кнопке Показать фильтр и задайте критерии поиска:

  • "Номер" - поиск по номеру карты ключей;

  • "Статус" - поиск по статусу карты ключей: "Все", "Активна", "Неактивна", "Использована", "Заблокирована";

    • Примечание: подробную информацию о значении каждого статуса смотрите в разделе контекстной справки Карты ключей - Статусы карт ключей.

  • "Количество ключей" - поиск по количеству одноразовых паролей в карте;

  • "Выдана с...по..." - поиск по дате выдачи интересующей Вас карты ключей;

  • "Тип блокировки" - поиск по типу блокировки карты ключей.

После заполнения полей фильтра нажмите на кнопку Применить, после чего система отобразит отфильтрованный список карт ключей по заданным Вами параметрам.

Сертификаты

На вкладке анкеты клиента Сертификаты открывается список всех сертификатов клиента. По каждому сертификату Вы можете просмотреть идентификатор сертификата, дата начала действия сертификата, дата окончания действия сертификата, статус сертификата в системе: "Активный", "Неактивный", "Истек" и др.

С каждым сертификатом клиента Вы можете выполнить следующие действия:

  • Активировать сертификат

    Если Вам необходимо активировать сертификат, установите флажок напротив сертификата со статусом "Неактивный" и щелкните по кнопке Активировать. Система присвоит сертификату статус "Активный", и клиент банка может использовать данный сертификат для входа в систему в защищенном режиме и для подписания документов.

  • Заблокировать сертификат

    Если Вам нужно заблокировать сертификат в случае утери или компрометации секретного ключа клиента, установите напротив сертификата со статусом "Активный" флажок и щелкните по кнопке Заблокировать. Система присвоит сертификату статус "Заблокирован", и клиент банка не сможет использовать сертификат с таким статусом для работы в системе.

  • Удалить сертификат

    Если Вам нужно удалить сертификат из списка, установите флажок напротив сертификата со статусом "Неактивный" и нажмите на кнопку Удалить, после чего система удалит выбранный Вами сертификат из списка сертификатов.

  • Фильтр/поиск сертификатов

    Для просмотра списка всех интересующих Вас сертификатов нужно щелкнуть по кнопке Показать фильтр и задать критерии поиска:

    • "Дата начала действия" - дата начала действия сертификата;

    • "Дата окончания действия" - дата окончания действия сертификата;

    • "Статус" - статус сертификата в системе: "Активный", "Неактивный", "Истек" и др. Статус "Активный" обозначает, что клиент может использовать данный сертификат для защиты информации, которую он передает в банк. Статус "Неактивный" обозначает, что сертификат создан в системе, но не привязан данному клиенту банка. Статус "Истек" обозначает, что время действия сертификата закончилось. Статус "Заблокирован" означает, что сертификат заблокирован по запросу клиента (утеря, компрометация и пр.). Сертификат с таким статусом не может быть использован клиентом для работы в системе.

    Также Вы можете фильтровать список по любой комбинации полей. Например, можно ввести в поля фильтра только статус сертификата.

    После того как параметры заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас сертификатов.

Пакет документов банка

При необходимости назначьте пользователю пакет услуг, который определяет перечень шаблонов документов (договоров) банка, которые печатаются и подписываются клиентом при подключении к системе. Пакет документов формируется в пункте главного меню Сервис - Пакет шаблонов документов банка.

Назначение пакета осуществляется через пункт меню Пакет документов банка. При его выборе на экран выводится список существующих в системе пакетов услуг, из которых требуется выбрать тот, который назначается данному клиенту.

Щелкнув по кнопке Печать пакета документов, Вы можете распечатать пакет документов для подключения клиента к системе в соответствии с назначенным пользователю пакетом услуг.

Если Вам необходимо распечатать анкету клиента банка, нажмите кнопку Печать анкеты.

Блокировка и разблокировка клиентов

Учетные записи клиентов можно блокировать. Это позволяет запретить клиенту доступ в систему (например, из-за того, что данная услуга не была своевременно оплачена), не удаляя его учетную запись.

Блокировка выполняется следующим образом:

  1. В главном меню системы щелкните по пункту Клиенты. На экран будет выведен список всех зарегистрированных в системе клиентов.

  2. В списке выберите тех клиентов, учетные записи которых будут заблокированы, и установите флажки напротив их фамилии, имени и отчестве.

  3. Щелкните по кнопке Заблокировать. Откроется окно, в котором необходимо указать срок и причину блокировки, и снова нажать на кнопку Заблокировать. Система присвоит такому клиенту статус "Заблокирован", и рядом с его именем появится пиктограмма . При наведении курсора мыши на данное изображение система покажет всплывающее окно, в котором указаны тип блокировки (например, системная) и причина блокировки клиента.

Разблокировка учетных записей выполняется точно также, только нужно отметить заблокированные учетные записи и нажать на кнопку Снять блокировку.

Заблокированных клиентов в списке можно отфильтровать с помощью Фильтра/поиска клиентов. Для этого в фильтре надо задать статус клиента "Заблокирован", и система покажет в списке только заблокированных клиентов.

Визуально отличить заблокированного клиента можно по пиктограмме , которая появляется рядом с фамилией заблокированного клиента банка.
