Справочник получателей

В данном справочнике хранятся реквизиты получателей, которые Вы можете использовать при заполнении различных переводов, например, при оплате налогов. Для каждого получателя в справочнике хранятся его наименование (или ФИО для физического лица), ИНН, КПП (для юридического лица), номер счета и реквизиты банка получателя, в котором открыт данный счет. При выборе получателя из справочника все эти сведения автоматически подставляются в документ. Справочник является индивидуальным для каждого пользователя системы.

Вы вручную формируете свой справочник получателей. На данной странице Вы можете выполнять следующие действия:

  • Добавление получателя

    Чтобы добавить получателя в справочник, щелкните по кнопке Добавить. В результате Вы перейдете к форме ввода нового получателя. В данной форме в поле "тип получателя" выберите, какой тип получателя Вы заводите (юридическое или физическое лицо), установив напротив интересующего типа галочку; в поле наименование (ФИО) впишите название организации для юридического лица или ФИО получателя для физического лица; в поле "ИНН" укажите ИНН получателя, в поле "КПП" введите КПП организации (данное поле заполняется только для юридических лиц), в поле "счет" укажите счет получателя, затем заполните реквизиты банка, в котором открыт счет получателя. Для заполнения реквизитов банка в поле "банк получателя, наименование" щелкните по кнопке (...), откроется страница со справочником банков. На данной странице установите галочку напротив интересующего Вас банка получателя и нажмите кнопку Выбрать. Также на этой странице Вы можете воспользоваться фильтром.

    Фильтр банков получателей

    Для того чтобы воспользоваться фильтром, щелкните по кнопке Показать фильтр и заполните поля фильтра:

    "Наименование" - поиск по названию банка получателя;

    "БИК" - поиск по банковскому идентификационному коду;

    "Город" - поиск по городу, в котором зарегистрирован банк.

    Вы можете фильтровать список по любой комбинации полей, например, можно отобрать все банки, название которых начинается на одну и ту же букву и из одного города.

    После того как критерии поиска заданы, щелкните по кнопке Построить список, и система покажет список интересующих Вас банков.

    После выбора банка получателя все реквизиты банка будут скопированы в создаваемый документ.

    После заполнения полей щелкните по кнопке Сохранить.

  • Редактирование получателя

    Вы можете отредактировать любую запись справочника, для этого установите галочку напротив интересующего Вас получателя в списке, щелкните по кнопке Изменить. В результате Вы перейдете к форме редактирования записи получателя, в которой измените необходимые сведения и щелкните по кнопке Сохранить.

  • Удаление получателя

    Вы можете удалить получателя из справочника, для этого Вам надо отметить галочкой удаляемые записи (их может быть несколько) и щелкнуть по кнопке Удалить. Подтвердите удаление, и получатель будет удален из списка.

Фильтр/поиск получателей

Для поиска интересующего Вас получателя в системе предусмотрен фильтр. Чтобы воспользоваться фильтром, щелкните по кнопке Показать фильтр и задайте критерии поиска:

  • "Наименование" - поиск по наименованию получателя;

  • "Счет" - поиск по номеру счета получателя;

  • "Банк" - поиск по названию банка, в котором открыт счет получателя.

Вы можете фильтровать список по любой комбинации полей. Например, можно отобрать всех получателей, наименование которых начинается с буквы "В" и счета которых открыты в одном банке.

После того как параметры поиска заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас получателей.

Справочник банков

В системе предусмотрены справочники, которые облегчают заполнение документов и позволяют избежать возникновения опечаток в реквизитах документов.

В данном справочнике хранятся реквизиты банков, которые Вы можете использовать при заполнении переводов, например, при оплате электроэнергии. Для каждого банка в справочнике хранятся его код БИК, наименование, номер корреспондентского счета и краткое наименование банка. При выборе банка из справочника все эти сведения автоматически подставляются в документ. Справочник банков предоставляется Вам уже заполненным, Вы не можете добавлять в него новые записи или редактировать информацию о банках.

Фильтр/поиск банков

Для поиска интересующего Вас банка в системе предусмотрен фильтр. Чтобы воспользоваться фильтром, щелкните по кнопке Показать фильтр и задайте критерии поиска:

  • "Наименование" - поиск по наименованию банка;

  • "БИК" - поиск по номеру банковского идентификационного кода;

  • "Город" - поиск по названию города, в котором банк зарегистрирован.

Вы можете фильтровать список по любой комбинации полей. Например, можно отобрать все банки, название которых начинается с буквы "А", и из одного города.

После того как параметры поиска заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас банков.
