Назначение заявки

С помощью данного пункта меню Вы можете перевести денежные средства с депозитного счета вклада на интересующий Вас счет. Для этого в главном меню выберите пункт Заявки или Вклады, а в открывшемся списке пункт Списать средства со вклада. После заполнения заявки система попросит проверить правильность заполнения полей и подтвердить созданную заявку, после чего Ваша заявка будет отправлена в банк. Ход обработки заявки можно отслеживать в списке заявок по статусу.

Заполнение заявки

Для заполнения формы заявки на списание средств со вклада Вам необходимо заполнить следующие поля:

  • Поля "Номер документа", "Номер договора", "Дата открытия", "Дата окончания" и "Неснижаемый остаток вклада" заполняются системой автоматически и недоступны для редактирования.

  • Поле "Дата документа" также автоматически заполняется системой, но его можно изменить. По умолчанию проставляется текущая дата.

  • В поле "Вклад для снятия" выберите из выпадающего списка вклад, с которого Вы хотите списать денежные средства.

  • В поле "Сумма для снятия" укажите сумму, которую Вы желаете снять со вклада.

  • В поле "Перевести на счет" выберите из выпадающего списка счет, на который будут зачислены денежные средства.

После заполнения полей формы нажмите на кнопку Сохранить. Заявка приобретет статус "Введена". Если Вы решили не делать заявку, с данной страницы можно вернуться к списку выполненных Вами заявок на списание средств со вклада, щелкнув по кнопке Список, или на страницу выбора типа заявки, щелкнув по кнопке Заявки.

Подтверждение заявки

После сохранения заполненного документа Вы попадаете на страницу подтверждения. Здесь Вам будет показана заполненная форма заявки на списание средств со вклада, в которой Вы должны проверить правильность указанных сведений, после чего выполнить одно из следующих действий:

  • Подтверждение заявки

    Процедура подтверждения зависит от используемого Вами способа регистрации в системе. Например, если Вы используете карту одноразовых паролей, то Вам необходимо будет ввести пароль с указанным номером с Вашей карты одноразовых паролей.

    • Когда все сведения указаны, щелкните по кнопке Подтвердить, чтобы документ был отправлен в банк на обработку.

  • Создание новой заявки

    Вы можете создать еще один документ данного типа, щелкнув по кнопке Создать. В результате Вы перейдете к форме ввода нового документа. Данная возможность полезна при создании нескольких однотипных документов.

  • Редактирование заявки

    Для того чтобы отредактировать созданную заявку, щелкните по кнопке Редактировать, внесите необходимые изменения и щелкните по кнопке Сохранить.

  • Печать документа

    Если Вы хотите распечатать данный документ, щелкните по кнопке Печать.

  • Отказ от формирования заявки

    Если Вы передумали делать заявку, можно вернуться к списку сделанных Вами заявок на списание средств со вклада, щелкнув по кнопке Список, или на страницу выбора типа заявки, щелкнув по кнопке Заявки.

Просмотр документа

После подтверждения заявки Вы попадаете на страницу просмотра, где отображается заполненный документ. О том, что документ передан в банк, свидетельствует отображаемый на форме документа штамп «Принято к исполнению». На страницу просмотра документа Вы можете также попасть из списка документов, щелкнув по номеру того документа, который хотите просмотреть. На странице просмотра Вы можете выполнить следующие действия:

  • Создание нового документа

    Вы можете создать еще один документ данного типа, щелкнув по кнопке Создать. В результате Вы перейдете к форме ввода нового документа. Данная возможность полезна при создании нескольких однотипных документов.

  • Печать документа

    Если Вы хотите распечатать данный документ, щелкните по кнопке Печать.

  • Возврат к списку заявок или выбору типа заявки

    Можно вернуться к списку сделанных Вами заявок на списание средств со вклада, щелкнув по кнопке Список, или на страницу выбора типа заявки, щелкнув по кнопке Заявки.

Список заявок

В списке заявок отображаются все созданные Вами заявки на списание средств со вклада. Вы можете зайти в список со страницы выбора типа заявки, щелкнув на выбранной заявке по кнопке История операций, или при создании заявки, щелкнув по кнопке Список. Для каждой заявки в списке отображаются дата создания, номер заявки, название вклада и статус, по которому можно отследить состояние заявки в системе, и другие сведения. На данной странице Вы можете выполнить следующие операции:

  • Просмотреть заявку

    Для этого щелкните по номеру интересующей Вас заявки в списке и Вы попадете на страницу просмотра заполненной формы заявки.

  • Создать копию документа

    Вы можете создать копию документа, отметив документ, который служит оригиналом в списке, щелкнув по кнопке Создать копию. В результате Вы перейдете к форме ввода нового документа, причем в нее будут скопированы все данные из документа, выбранного Вами в качестве оригинала. Данная возможность полезна при создании нескольких однотипных документов.

  • Удалить заявку

    Для этого установите флажок напротив заявки, которую Вы хотите удалить, и нажмите на кнопку Удалить.

  • Фильтр заявок

    Для поиска в системе интересующих Вас заявок предусмотрен фильтр, который позволяет осуществлять поиск по любому из полей либо их комбинации. Щелкните по кнопке Показать фильтр. Откроется страница с формой, в которой необходимо задать критерии поиска:

    • В поле "Статус" выберите из выпадающего списка статус заявки на списание средств со вклада: "Введена", "Одобрена", "Исполнена", "Отказана" и др.;

    • В поле "Период" введите период времени, в течение которого была создана заявка на списание средств со вклада;

    • В поле "Вклад" выберите из выпадающего списка вклад, с которого были сняты денежные средства: например, "Универсальный".

    Вы можете фильтровать список по любой комбинации полей, например, можно отобрать все заявки на списание средств со вклада, созданные в один период, со статусом "Принята".

    После того как критерии поиска заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас заявок.

Статусы заявок

Ход обработки заявок Вы можете наблюдать в списке заявок по статусу. Для заявлений по списанию средств по вклада в системе предусмотрены следующие статусы:

  • Введен - документ введен, сохранен в системе, но не подтвержден;

  • Обрабатывается - документ полностью заполнен клиентом банка, подтвержден и передан на обработку в банк;

  • Ожидание обработки - ожидается передача документа в банк или получение информации о результате его исполнения;

  • Исполнен - документ исполнен банком;

  • Отказан - документ отказан банком с указанием причины.
