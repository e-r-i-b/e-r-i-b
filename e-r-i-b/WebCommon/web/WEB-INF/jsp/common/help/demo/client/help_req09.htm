Назначение заявки

Если Вы хотите, чтобы банк выпустил дополнительную пластиковую карту, присоединенную к уже выпущенной основной карте, необходимо заполнить и отправить в банк заявку на изготовление дополнительной пластиковой карты. Для этого в главном меню выберите пункт Заявки, а затем пункт Изготовление дополнительной карты. Ход обработки заявки можно отслеживать в списке заявок по статусу.

Заполнение заявки

В заявке на изготовление дополнительной пластиковой карты Вам надо сначала выбрать из списка ту карту, для которой должна быть изготовлена дополнительная, в поле "Номер СКС" система автоматически введет номер спецкарточного счета. Затем надо в поле "Карта" выбрать из списка тип выпускаемой карты, в поле "Срочность изготовления" указать срочность изготовления, определяющую сколько времени займет выпуск карты.

В группе полей Данные пользователя введите в отдельные поля имя и фамилию латинскими буквами (так как они должны быть указаны на выпускаемой пластиковой карте), остальные сведения о владельце карты: ФИО, пол, дата рождения, гражданство, статус, ИНН заполняются автоматически, данные поля недоступны для редактирования. Помимо этого впишите кодовое слово, которое Вам необходимо будет называть при обращениях в банк.

В группе полей Паспорт автоматически заполняются все поля, которые также недоступны для редактирования.

В группе полей Адрес прописки введите адрес регистрации, указанный в паспорте того лица, для которого выпускается дополнительная карта, а в группе полей Адрес проживания укажите адрес его фактического проживания и контактную информацию, по которой с ним может связаться сотрудник банка.

После того как все поля заполнены, щелкните по кнопке Сохранить. Если Вы решили не делать заявку, с данной страницы можно вернуться к списку выполненных Вами заявок на изготовление дополнительной пластиковой карты, щелкнув по кнопке Список, или на страницу выбора типа заявки, щелкнув по кнопке Заявки.

Подтверждение заявки

После сохранения заполненного документа Вы попадаете на страницу подтверждения. Здесь Вам будет показана заполненная форма заявки на изготовление дополнительной пластиковой карты, в которой Вы должны проверить правильность указанных сведений, после чего выполнить одно из следующих действий:

  • Подтверждение заявки

    Процедура подтверждения зависит от используемого Вами способа регистрации в системе. Например, если Вы используете карту одноразовых паролей, то Вам необходимо будет ввести пароль с указанным номером с Вашей карты одноразовых паролей.

    • Когда все сведения указаны, щелкните по кнопке Подтвердить, чтобы документ был отправлен в банк на обработку.

  • Печать документа

    Если Вы хотите распечатать данный документ, щелкните по кнопке Печать.

  • Отказ от формирования заявки

    Если Вы передумали делать заявку, можно вернуться к списку сделанных Вами заявок на изготовление дополнительной пластиковой карты, щелкнув по кнопке Список, или на страницу выбора типа заявки, щелкнув по кнопке Заявки.

Просмотр документа

После подтверждения заявки Вы попадаете на страницу просмотра, где отображается заполненный документ. О том, что документ передан в банк, свидетельствует отображаемый на форме документа штамп «Принято к исполнению». На страницу просмотра документа Вы можете также попасть из списка документов, щелкнув по номеру того документа, который хотите просмотреть. На странице просмотра Вы можете выполнить следующие действия:

  • Создание нового документа

    Вы можете создать еще один документ данного типа, щелкнув по кнопке Создать. В результате Вы перейдете к форме ввода нового документа. Данная возможность полезна при создании нескольких однотипных документов.

  • Печать документа

    Если Вы хотите распечатать данный документ, щелкните по кнопке Печать.

  • Возврат к списку заявок или выбору типа заявки

    Можно вернуться к списку сделанных Вами заявок на изготовление дополнительной пластиковой карты, щелкнув по кнопке Список, или на страницу выбора типа заявки, щелкнув по кнопке Заявки.

Список заявок

В данном списке отображаются все сделанные Вами заявки на изготовление дополнительной пластиковой карты. Вы можете зайти в список со страницы выбора типа заявки, щелкнув на выбранной заявке по кнопке История операций, или при создании заявки, щелкнув по кнопке Список. Для каждой заявки в списке отображаются дата создания, номер заявки, номер карты и статус, по которому можно отследить состояние заявки в системе, и другие сведения. Вы можете выполнить следующие операции:

  • Просмотр документа

    Вы можете просмотреть любой документ, для этого нужно щелкнуть по номеру интересующего Вас документа в списке.

  • Создание новой заявки

    Вы можете создать новую заявку на изготовление дополнительной пластиковой карты, щелкнув по кнопке Создать. В результате Вы перейдете к форме ввода нового документа.

  • Фильтр заявок

    Для того чтобы найти интересующую Вас заявку в списке, щелкните по кнопке Показать фильтр и заполните поля фильтра:

    "Карта" - поиск по номеру и типу карты;

    "Номер" - поиск по номеру заявки в списке;

    "Статус" - поиск по статусу заявки в системе: "Введена", "Одобрена", "Исполнена", "Отказана" и др.;

    "Период" - поиск по времени создания заявки.

    Вы можете фильтровать список по любой комбинации полей, например, можно отобрать все заявки на изготовление дополнительной карты, созданные в один период, со статусом "Принята".

    После того как критерии поиска заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас заявок.

Статусы заявок

Ход обработки заявок Вы можете наблюдать в списке заявок по статусу. Для заявлений по изготовлению дополнительной пластиковой карты в системе предусмотрены следующие статусы:

  • Введен - документ введен, сохранен в системе, но не подтвержден;

  • Принят - документ полностью заполнен клиентом банка, подтвержден и передан на обработку в банк;

  • Одобрен - документ одобрен сотрудником банка, но не исполнен;

  • Исполнен - документ исполнен банком;

  • Отказан - документ отказан банком с указанием причины.
