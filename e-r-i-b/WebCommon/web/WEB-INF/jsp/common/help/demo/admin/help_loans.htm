Пункт меню Кредиты предназначен для обработки кредитных заявок клиентов, для работы с кредитными продуктами, видами кредитов, кредитными офисами, а также с сообщениями о результатах рассмотрения заявки в банке.

Заявки

В пункте главного меню Кредиты автоматически открывается страница со списком кредитных заявок клиентов банка. По каждой заявке можно просмотреть следующую информацию: номер заявки, номер заявки в системе RS-Loans, дату передачи заявки клиентом в банк, ФИО клиента, сумму/срок запрашиваемого клиентом кредита, сумму/срок утвержденного банком кредита, валюту кредита и статус заявки на кредит.

  • Для просмотра кредитной заявки нажмите на номер интересующей Вас заявки. Откроется страница с подробной информацией по данной кредитной заявке, на которой Вы можете выполнить следующие операции:

  • Для отклонения заявки установите флажок напротив интересующей Вас заявки и щелкните по кнопке Отказать. Заявка приобретает статус "Отказана".

    Отказать заявки можно только со статусами "Принята" и "Требуется доработка".

  • Для того чтобы отправить заявку в центральную базу RS-Loans, установите флажок напротив интересующей Вас заявки и нажмите на кнопку Отправить. Заявка приобретает статус "Принята".

  • При необходимости отправления заявки клиенту на доработку выберите нужную Вам заявку, установите напротив нее флажок и щелкните по кнопке Требуется доработка, после чего заявка приобретает соответствующий статус "Требуется доработка".

    Редактировать заявки можно только со статусом "Принята".

  • При отправлении заявки клиенту на доработку Вы можете прокомментировать заявку, указав, например, какие поля клиенту необходимо отредактировать. Для этого нажмите на кнопку Комментарий. Откроется страница с полем для ввода комментария клиенту. Впишите текст комментария и нажмите на кнопку Сохранить. Прокомментированная заявка будет отображаться в списке заявок со специальной пиктограммой.

    Добавлять комментарий можно для заявок со всеми статусами кроме статуса "Утверждена".

    Примечание: в клиентском приложении Ваш комментарий отобразится в пункте меню Кредиты-Результаты рассмотрения заявки (во всплывающем окне при наведении на статус заявки на кредит).

  • Для того чтобы поместить заявку в архив, установите флажок напротив выбранной Вами заявки и щелкните по кнопке В архив.

    Заявки, переведенные в архив, отображаются только в списке архивных заявок.

    Примечание: в архив Вы можете поместить заявки только со статусами "Отказана", "Утверждена" и "Выдан кредит".

Для осуществления отбора заявок в списке Вы можете воспользоваться фильтром по любым из указанных в нем полей, либо их комбинации. Для этого щелкните по кнопке Показать фильтр и задайте критерии поиска:

  • "Фамилия, имя, отчество" - поиск по фамилии, имени и отчеству клиента;

  • "Тип клиента" - поиск по типу клиента: "Все", "Анонимный", "Зарегистрированный";

  • "Серия и номер документа" - поиск по данным документа клиента;

  • "Номер заявки" - поиск по порядковому номеру заявки;

  • "Сумма запрашиваемого клиентом кредита" - поиск по размеру суммы кредита, которую клиент желает получить;

  • "Статус заявки" - поиск по статусу заявки;

    • Примечание: подробную информацию о значении каждого статуса смотрите в пункте меню Кредиты-Сообщения о результатах рассмотрения заявки в банке.

  • "Вид кредита" - поиск по виду кредита: "Все", "Автокредит" и др.;

  • "Период с...по..." - поиск по периоду времени, в течение которого была создана заявка на кредит клиентом банка.

Вы можете осуществлять поиск по подстроке в текстовых полях фильтра. Например, можно ввести в поля фильтра только несколько первых букв фамилии, имени или отчества клиента.

Также Вы можете фильтровать список по любой комбинации полей. Например, можно отобрать все заявки от анонимных клиентов на получение потребительского кредита на одну и ту же сумму.

После того как параметры поиска заданы, нажмите на кнопку Применить, и система покажет список интересующих Вас заявок на кредит.

Архив заявок

Данный пункт левого меню предназначен для просмотра заявок клиентов банка со статусами "Утверждена", "Отказана" и "Выдан кредит".

По каждой заявке на кредит в списке отображается номер заявки, номер заявки в системе RS-Loans, дата подачи заявки в банк, ФИО клиента, сумма запрашиваемого кредита, срок запрашиваемого кредита (в месяцах), сумма и срок утвержденного банком кредита, валюта кредита и статус заявки.

Просмотр заявки

Для того чтобы просмотреть заявку на кредит, щелкните по номеру интересующей Вас заявки. Откроется страница просмотра, на которой Вы можете просмотреть поля заявки и при необходимости вернуть данную заявку из архива.

Для возврата заявки из архива установите флажок напротив интересующей Вас заявки и щелкните по кнопке Вернуть из архива. Данная заявка отобразится в пункте меню Заявки в списке кредитных заявок.

Со страницы просмотра Вы можете вернуться в список архивных заявок, для этого нажмите на кнопку Список.

Фильтр/поиск заявок

На данной странице Вы можете воспользоваться фильтром, для того чтобы отобрать интересующие Вас заявки по любому из полей или их комбинации. Для этого нажмите на кнопку Показать фильтр и задайте критерии поиска:

  • "Фамилия, имя, отчество" - поиск по фамилии, имени, отчеству клиента;

  • "Тип клиента" - поиск по типу клиента: "Все", "Анонимный", "Зарегистрированный";

  • "Серия и номер документа" - поиск по данным документа клиента;

  • "Номер заявки" - поиск по порядковому номеру заявки;

  • "Сумма запрашиваемого клиентом кредита" - поиск по размеру суммы кредита, которую клиент запросил у банка;

  • "Статус заявки" - поиск по статусу заявки: "Утверждена", "Отказана", "Кредит выдан";

    • Примечание: подробную информацию о значении каждого статуса смотрите в пункте меню Кредиты-Сообщения о результатах рассмотрения заявки в банке.

  • "Вид кредита" - поиск по виду кредита: "Все", "Потребительский" и др.;

  • "Период" - поиск по периоду времени, в течение которого была создана заявка на кредит клиентом банка;

Вы можете осуществлять поиск по подстроке в текстовых полях фильтра. Например, можно ввести в поля фильтра только несколько первых букв фамилии, имени или отчества клиента.

Также Вы можете фильтровать список по любой комбинации полей. Например, можно отобрать все заявки на получение потребительского кредита со статусом "Отказана".

После того как параметры поиска заданы, нажмите на кнопку Применить, и система покажет список интересующих Вас заявок на кредит.

Кредитные продукты

Пункт меню Кредитные продукты предназначен для управления кредитными продуктами банка. На данной странице Вы можете просмотреть список доступных кредитных продуктов банка.

Кредитный продукт - это конкретное предложение (продукт) в рамках определенного вида кредита (типа кредитования).

Просмотр детальной информации по кредитному продукту

Для просмотра детальной информации по кредитному продукту щелкните по его наименованию. Откроется страница, на которой будут отображены имя кредитного продукта, анонимная заявка, вид кредита, пакет документов, а также описание интересующего Вас кредитного продукта.

Для возврата к списку кредитных продуктов нажмите на кнопку Список.

Создание кредитного продукта

На странице со списком кредитных продуктов Вы можете создать новый кредитный продукт. Для этого необходимо выполнить следующие операции:

  • Нажмите на кнопку Добавить. Откроется страница для создания кредитного продукта.

  • В поле "Имя кредитного продукта" введите наименование кредита, которое будет отображаться в клиентском приложении.

  • В поле "Вид кредита" введите вид кредита, соответствующий данному кредитному продукту. Например, для кредитного продукта «Авто+» вид кредита будет «Автокредит».

  • В поле "Анонимная заявка" укажите, будет ли данный кредитный продукт доступен для анонимных пользователей, не зарегистрированных в системе ИКФЛ.

  • В поле "Пакет документов" выберите из выпадающего списка пакет документов для создаваемого Вами кредитного продукта.

    • Примечание: создание шаблонов документов и пакета шаблонов смотрите в разделе контекстной справки пункта меню Сервис - Пакеты шаблонов документов банка и Шаблоны документов банка.

  • В поле "Описание" введите дополнительную информацию по кредитному продукту.

  • В таблице со списком офисов банка и видами кредитов необходимо указать, для какого офиса доступен выбранный Вами кредитный продукт, а также вид кредита, соответствующий данному кредитному офису. Для каждой из доступных валют выпадает список с наименованием видов кредитов из RS-Loans. Для каждой валюты можно выбрать только одно значение вида кредита. В данной таблице напротив офиса, которому доступен данный кредитный продукт, выберите вид кредита, соответствующий виду кредита в системе RS-Loans.

    Примечание: Поля "Идентификатор" и "Сроки кредита" заполняются системой автоматически при выборе вида кредита, кроме значения "Недоступен".

После заполнения полей формы нажмите на кнопку Сохранить.

Для возврата к списку кредитных продуктов щелкните по кнопке Список.

Редактирование кредитного продукта

Для редактирования сведений о кредитном продукте банка щелкните по его наименованию. Откроется страница редактирования кредитного продукта, аналогичная странице создания кредитного продукта. Внесите необходимые изменения и нажмите на кнопку Сохранить.

Для возврата к списку кредитных продуктов без внесения изменений щелкните по кнопке Список.

Удаление кредитного продукта

Для удаления кредитного продукта установите флажок напротив интересующего Вас кредита и нажмите на кнопку Удалить.

Примечание: удаление кредитного продукта подразумевает невозможность дальнейшей работы с ним в системе: он исчезает из списка кредитных продуктов банка, кредитного калькулятора, а также прочих списков. При этом, если у клиента есть действующие кредиты по данному продукту, то информация по нему (продукту) в рамках текущего кредита должна быть доступна пользователю.

Виды кредитов

Данный пункт меню предназначен для управления сотрудником банка видами кредитов. В пункте левого меню Виды кредитов автоматически открывается страница со списком видов кредитов.

Вид кредита - это тип кредитования, которому соответствуют несколько кредитных продуктов. Например, виду кредита «Автокредит» соответствуют кредитные продукты «Выгодный» и "Авто+".

Создание нового вида кредита

Для создания нового вида кредита щелкните по кнопке Добавить. Откроется страница для создания вида кредита, аналогичная странице редактирования вида кредита, на которой необходимо заполнить следующие поля:

Примечание: поля со звездочкой обязательны для заполнения.

  • В поле "Наименование" введите название вида кредита.

  • В поле "Описание" укажите дополнительную информацию по виду кредита.

  • В поле "Описание заявки" щелкните по кнопке Обзор и выберите из списка необходимый xml-файл с описанием заявки.

  • В поле "XSLT шаблон детальной информации" щелкните по кнопке Обзор и укажите из списка XSLT-файл с подробной информацией по данному виду кредита.

  • В поле "Кредитный калькулятор" установите флажок, для того чтобы данный вид кредита отобразился в кредитном калькуляторе.

  • В поле "Целевой кредит" установите флажок для отображения в кредитном калькуляторе поля "Стоимость товара" для расчета суммы кредита.

Для сохранения созданного вида кредита нажмите на кнопку Сохранить.

Для возврата к списку видов кредитов щелкните по кнопке Список.

Редактирование вида кредита

Для редактирования сведений о виде кредита щелкните по его наименованию. Откроется страница редактирования вида кредита, на которой внесите необходимые изменения и нажмите на кнопку Сохранить.

Вы можете выгрузить из базы файл заявки по выбранному виду кредита и изменить описание полей заявки. Для этого щелкните по наименованию вида кредита, откроется страница для редактирования. На данной странице нажмите на кнопку Выгрузить описание заявки, сохраните xml-файл на диск, внесите в данный файл изменения, затем снова загрузите его, щелкнув по кнопке Обзор в поле "Описание заявки".

Удаление вида кредита

Для удаления вида кредита установите флажок напротив интересующего Вас вида кредита и нажмите на кнопку Удалить.

Кредитные офисы

Пункт меню Кредитные офисы предназначен для работы с кредитными офисами, к которым прикрепляются кредитные продукты. В данном пункте меню автоматически откроется страница со списком кредитных офисов банка.

Просмотр информации по кредитному офису

Для просмотра подробной информации по кредитному офису щелкните по его наименованию. Откроется страница со следующими данными: ID кредитного офиса, его название, адрес, телефон и примечание.

Для возврата к списку кредитных офисов нажмите на кнопку Список.

Головной офис

Для назначения кредитного офиса банка головным офисом установите напротив него флажок и нажмите на кнопку Головной офис.

Примечание: сотрудник головного офиса будет иметь возможность видеть заявки клиентов всех остальных офисов.

Редактирование информации по кредитному офису

Для редактирования сведений о кредитном офисе либо установите флажок напротив интересующего Вас офиса и щелкните по кнопке Редактировать, либо кликните на наименовании офиса. Откроется страница редактирования кредитного офиса. Внесите необходимые изменения и щелкните по кнопке Сохранить.

Для возврата к списку кредитных офисов без внесения изменений щелкните по кнопке Список.

Удаление кредитного офиса

Для удаления кредитного офиса установите флажок напротив выбранного Вами офиса и нажмите на кнопку Удалить.

Примечание: для того чтобы удалить кредитный офис, необходимо сначала удалить сотрудников, прикрепленных к данному офису.

Сообщения о результатах рассмотрения заявки в банке

Данный пункт меню предназначен для просмотра статусов, которые присваиваются заявкам. Статус описывает ход обработки заявки на кредит клиента банка.

На данной странице отображается таблица с сообщениями о результатах рассмотрения заявки в банке: статус заявки и соответствующий ему текст сообщения.

Вы можете отредактировать текст сообщения. Для этого установите флажок напротив интересующего Вас сообщения и нажмите на кнопку Редактировать или щелкните по наименованию статуса заявки. Откроется страница редактирования текста сообщения. Внесите необходимые изменения и щелкните по кнопке Сохранить.

Для возврата к списку сообщений о результатах рассмотрения заявки в банке щелкните по кнопке Список.

Примечание: в клиентском приложении данное сообщение отобразится в пункте меню Кредиты-Результаты рассмотрения заявки (во всплывающем окне при наведении на статус заявки).
