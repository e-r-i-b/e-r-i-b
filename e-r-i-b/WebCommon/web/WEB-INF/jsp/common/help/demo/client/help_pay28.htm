Оплата товаров и услуг через платежную систему Rapida

Пункт меню Оплата товаров и услуг через платежную систему Rapida включает в себя следующие операции по оплате товаров и услуг через платежную систему Rapida:

  1. Оплата сотовой связи - перечисление денежных средств с Вашего счета для оплаты услуг сотовой связи;

  2. Услуги телефонии - перевод денежных средств с Вашего счета для оплаты услуг операторов телефонной связи;

  3. Интернет и IP-телефония - перечисление денежных средств с Вашего счета для оплаты услуг операторов IP-телефонии;

  4. Коммерческое телевидение - перевод денежных средств с Вашего счета для пополнения счета операторов спутниковой связи и ТВ;

  5. Погашение кредитов - перечисление денежных средств с Вашего счета для погашения кредитов и процентов по нему;

  6. Пополнение счета в платежной системе - перевод денежных средств с Вашего счета для пополнения счета (электронного кошелька) платежных систем;

  7. Оплата услуг ЖКХ для г.Москва - перечисление денежных средств с Вашего счета для оплаты услуг ЖКХ для г.Москва;

  8. Оплата электроэнергии - перевод денежных средств с Вашего счета для оплаты электроэнергии.

Назначение платежа

Данный платеж позволяет Вам перечислить денежные средства для оплаты услуг сотовой связи. Чтобы создать платеж, в главном меню выберите пункт Платежи, а затем пункт левого меню Оплата товаров и услуг через платежную систему Rapida - Оплата сотовой связи.

Заполнение документа

Сначала Вам надо выбрать поставщика услуг сотовой связи, установив напротив него галочку, и щелкнуть по кнопке Продолжить, чтобы перейти к заполнению формы платежа. В форме платежа Вам надо выбрать из списка счет списания (счет, с которого будут списаны денежные средства), указать сумму платежа и номер телефона.

После того как все поля заполнены, щелкните по кнопке Сохранить. Вы также можете добавить частично заполненный документ в список шаблонов, щелкнув по кнопке Сохранить как шаблон, чтобы в дальнейшем использовать его для быстрого формирования аналогичных документов.

Если Вы решили не делать платеж, с данной страницы можно вернуться к списку выполненных Вами платежей, щелкнув по кнопке Список, или на страницу выбора типа платежа, щелкнув по кнопке Платежи.

Подтверждение платежа

После сохранения заполненного документа Вы попадаете на страницу подтверждения. Здесь Вам будет показана заполненная форма платежа, в которой Вам необходимо проверить правильность указанных сведений, после чего выполнить одно из следующих действий:

  • Подтверждение платежа

    Процедура подтверждения зависит от используемого Вами способа регистрации в системе. Например, если Вы используете карту одноразовых паролей, то Вам необходимо будет ввести пароль с указанным номером с Вашей карты одноразовых паролей.

    • Когда все сведения указаны, щелкните по кнопке Подтвердить, чтобы документ был отправлен в банк на обработку.

  • Возврат к редактированию

    Если Вы хотите изменить какие-либо реквизиты перевода, щелкните по кнопке Редактировать, чтобы вернуться к форме ввода документа для его редактирования.

  • Печать документа

    Если Вы хотите распечатать данный документ, щелкните по кнопке Печать.

  • Отказ от выполнения платежа

    Если Вы передумали осуществлять платеж, можно вернуться к списку выполненных Вами платежей, щелкнув по кнопке Список.

Просмотр документа

После подтверждения платежа Вы попадаете на страницу просмотра, где отображается заполненный документ. О том, что документ передан в банк, свидетельствует отображаемый на форме документа штамп «Принято к исполнению». На страницу просмотра документа Вы можете также попасть из списка документов, щелкнув по номеру того документа, который хотите просмотреть. На странице просмотра Вы можете выполнить следующие действия:

  • Печать документа

    Если Вы хотите распечатать данный документ, щелкните по кнопке Печать.

  • Возврат к списку платежей

    Можно вернуться к списку выполненных Вами платежей, щелкнув по кнопке Список.

Список платежей

В данном списке отображаются все выполненные Вами платежи. Вы можете зайти в список операций со страницы выбора типа платежа, щелкнув на выбранном платеже по кнопке История операций, или при создании платежа, щелкнув по кнопке Список. Для каждой операции в списке отображается дата совершения операции, номер перевода, счет списания, получатель, сумма платежа и статус, по которому можно отследить состояние операции в системе. Вы можете выполнить следующие операции:

  • Просмотр документа

    Вы можете просмотреть любой документ, для этого нужно щелкнуть по номеру интересующего Вас документа в списке.

  • Создание копии документа

    Вы можете создать копию платежа, отметив документ, который служит оригиналом в списке, щелкнув по кнопке Создать копию. В результате Вы перейдете к форме ввода нового документа, причем в нее будут скопированы все данные из документа, выбранного Вами в качестве оригинала. Данная возможность полезна при создании нескольких однотипных документов.

  • Удаление платежа

    Вы можете удалить платеж из списка, для этого установите галочку напротив интересующего документа и щелкните по кнопке Удалить.

  • Фильтр платежей

    Для того чтобы найти интересующий Вас платеж, щелкните по кнопке Показать фильтр и заполните поля фильтра:

    "Статус" - поиск по статусу платежа в системе: "Введен", "На исполнении", "Исполнен", "Отказан" и т.д.

    "Период" - поиск по времени создания платежа;

    "Сумма (RUB)" - поиск по размеру суммы платежа;

    "Счет списания" - поиск по номеру счета, с которого списаны средства.

    Вы можете фильтровать список по любой комбинации полей, например, можно отобрать все платежи для оплаты товаров и услуг на одну сумму со статусом "Исполнен".

    После того как критерии поиска заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас платежей.

Статусы платежа

Ход обработки платежа Вы можете наблюдать в списке платежей или в журнале операций по статусу. Для оплаты товаров и услуг через платежную систему Rapida в системе предусмотрены следующие статусы:

  • Введен - платеж введен, сохранен в системе, но не подтвержден;

  • Обрабатывается - платеж полностью заполнен клиентом банка, подтвержден и передан на обработку в банк;

  • Ожидание обработки - ожидается передача документа в банк или получение информации о результате его исполнения;

  • Исполнен - платеж проведен в банке: денежные средства со счета клиента поступили на указанный в форме счет зачисления;

  • Отказан - платеж отказан банком с указанием причины, например, по причине нехватки денежных средств на счете клиента на момент их списания.
