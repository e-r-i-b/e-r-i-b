Назначение заявки

С помощью данного пункта меню Вы можете подать в банк заявку на закрытие счета или вклада. Для этого в главном меню выберите пункт Заявки, а в открывшемся списке пункт Закрытие счета/вклада, или пункт меню Вклады - Заявка на закрытие вклада. После заполнения заявки система попросит проверить правильность заполнения полей и подтвердить созданную заявку, после чего Ваша заявка будет отправлена в банк. Ход обработки заявки можно отслеживать в списке заявок по статусу.

Заполнение заявки

Заполнение заявки на закрытие счета
Заполнение заявки на закрытие вклада

Для заполнения заявки необходимо определить, какой тип заявки Вы собираетесь закрыть. Для этого в поле "Закрыть" выберите из выпадающего списка интересующий Вас тип заявки: счет или вклад. От этого зависит набор полей в заявке.

Заполнение заявки на закрытие счета

Для заполнения формы заявки на закрытие счета Вам необходимо заполнить следующие поля:

  • В поле "Счет для закрытия" выберите из выпадающего списка счет, который желаете закрыть.

  • Поля "Номер договора", "Дата открытия" и "Дата закрытия" заполняются автоматически и недоступны для редактирования.

  • В поле "Остаток средств перевести на счет" из выпадающего списка выберите счет, на который Вам будут переведены оставшиеся денежные средства.

Если Вы хотите перевести денежные средства с закрываемого счета на счет в другой банк, то из выпадающего списка выберите значение "В другом банке". В зависимости от валюты закрываемого счета Вам будут добавлены новые поля:

  1. При переводе денежных средств с рублевого счета дополнительно отображаются для заполнения поля формы «Перевод рублей РФ».

    В данной форме необходимо указать, кому Вы переводите средства. Сначала укажите является получатель юридическим или физическим лицом, установив галочку напротив соответствующего получателя. После этого заполните реквизиты получателя: название (для юридического лица), ФИО (для физического лица), коды ИНН и КПП (для физического лица КПП не нужно указывать), номер счета, на который будут перечислены средства, и реквизиты банка, в котором этот счет открыт.

    Для быстрого заполнения реквизитов Вы можете воспользоваться предусмотренными в системе справочником получателей и справочником банков. Для доступа к справочнику получателей щелкните по кнопке (...), расположенной рядом с полем Название организации или рядом с полем ФИО (если получатель - физическое лицо). Откроется страница со справочником получателей, на которой надо выбрать необходимого получателя, после чего щелкнуть по кнопке Выбрать. На данной странице Вы можете воспользоваться фильтром.

    Фильтр/поиск получателей

    Для того чтобы воспользоваться фильтром, щелкните по кнопке Показать фильтр и задайте критерии поиска:

    "Наименование" - поиск по названию получателя;

    "Счет" - поиск по номеру счета получателя;

    "Банк" - поиск по названию банка получателя.

    Вы можете фильтровать список по любому из полей либо их комбинации.

    После заполнения полей фильтра щелкните по кнопке Построить список.

    После выбора получателя все сведения будут скопированы в создаваемый документ.

    Для доступа к справочнику банков щелкните по кнопке (...), расположенной рядом с полем БИК. Откроется страница со справочником банков, на которой надо выбрать банк получателя, после чего щелкнуть по кнопке Выбрать. Также на этой странице Вы можете воспользоваться фильтром.

    Фильтр/поиск банков получателей

    Для того чтобы воспользоваться фильтром, щелкните по кнопке Показать фильтр и задайте критерии поиска:

    "Наименование" - поиск по названию банка получателя;

    "БИК" - поиск по банковскому идентификационному коду;

    "Город" - поиск по городу, в котором зарегистрирован банк.

    Вы можете фильтровать список по любому из полей либо их комбинации.

    После заполнения полей фильтра щелкните по кнопке Построить список.

    После выбора банка получателя все реквизиты банка будут скопированы в создаваемый документ.

  2. При переводе денежных средств с валютного счета дополнительно отображаются для заполнения поля формы "Перевод иностранной валюты".

    В данной форме в поле "Название получателя платежа" введите наименование получателя, в поле "Код страны получателя" укажите код, для этого щелкните по кнопке (...), откроется страница со справочником стран получателей, в котором выберите интересующий Вас код, установив напротив него галочку, щелкните по кнопке Выбрать. На данной странице Вы можете воспользоваться фильтром.

    Фильтр/поиск стран

    Для того чтобы воспользоваться фильтром, щелкните по кнопке Показать фильтр и задайте критерии поиска:

    "Код" - поиск по буквенному коду страны;

    "Числовой код" - поиск по числовому коду страны;

    "Наименование" - поиск по названию страны.

    Вы можете фильтровать список по любому из полей либо их комбинации.

    После заполнения полей фильтра щелкните по кнопке Построить список.

    После выбора банка получателя все реквизиты банка будут скопированы в создаваемый документ.

    Затем нужно ввести счет получателя, город, адрес получателя. Далее нужно ввести реквизиты банка получателя. Для этого в поле "SWIFT" щелкните по кнопке (...), откроется страница со справочником банков, на которой выберите SWIFT банка, установив напротив него галочку, щелкните по кнопке Выбрать. На данной странице Вы можете воспользоваться фильтром/поиском банков получателей.

    Фильтр/поиск банков получателей

    Для того чтобы воспользоваться фильтром, щелкните по кнопке Показать фильтр и задайте критерии поиска:

    "SWIFT" - поиск по коду SWIFT банка;

    "Наименование" - поиск по названию банка получателя;

    "Город" - поиск по городу, в котором зарегистрирован банк.

    Вы можете фильтровать список по любому из полей либо их комбинации.

    После заполнения полей фильтра щелкните по кнопке Построить список.

    После выбора банка получателя все реквизиты банка: город, код страны, адрес будут скопированы в создаваемый документ.

    Далее нужно ввести корр. счет банка получателя и выбрать из выпадающего списка условия перевода.

После заполнения полей формы нажмите на кнопку Сохранить. Заявка приобретет статус "Введена". Если Вы решили не делать заявку, с данной страницы можно вернуться к списку выполненных Вами заявок на закрытие счета/вклада, щелкнув по кнопке Список, или на страницу выбора типа заявки, щелкнув по кнопке Заявки.

Заполнение заявки на закрытие вклада

Для заполнения формы заявки на закрытие вклада Вам необходимо заполнить следующие поля:

  • В поле "Вклад для закрытия" выберите из выпадающего списка вклад, который желаете закрыть.

  • Поля "Номер договора", "Дата открытия" , "Дата окончания" и "Дата закрытия" заполняются автоматически и недоступны для редактирования.

  • "Перечислить на счет" - выберите из выпадающего списка счет, на который будут переведены денежные средства по окончании срока вклада.

После заполнения полей формы нажмите на кнопку Сохранить. Заявка приобретет статус "Введена". Если Вы решили не делать заявку, с данной страницы можно вернуться к списку выполненных Вами заявок на закрытие счета/вклада, щелкнув по кнопке Список, или на страницу выбора типа заявки, щелкнув по кнопке Заявки.

Подтверждение заявки

После сохранения заполненного документа Вы попадаете на страницу подтверждения. Здесь Вам будет показана заполненная форма заявки на закрытие счета/вклада, в которой Вы должны проверить правильность указанных сведений, после чего выполнить одно из следующих действий:

  • Подтверждение заявки

    Процедура подтверждения зависит от используемого Вами способа регистрации в системе. Например, если Вы используете карту одноразовых паролей, то Вам необходимо будет ввести пароль с указанным номером с Вашей карты одноразовых паролей.

    • Когда все сведения указаны, щелкните по кнопке Подтвердить, чтобы документ был отправлен в банк на обработку.

  • Создание новой заявки

    Вы можете создать еще один документ данного типа, щелкнув по кнопке Создать. В результате Вы перейдете к форме ввода нового документа. Данная возможность полезна при создании нескольких однотипных документов.

  • Редактирование заявки

    Для того чтобы отредактировать созданную заявку, щелкните по кнопке Редактировать, внесите необходимые изменения и щелкните по кнопке Сохранить.

  • Печать заявки

    Если Вы хотите распечатать данный документ, щелкните по кнопке Печать.

  • Отказ от формирования заявки

    Если Вы передумали делать заявку, можно вернуться к списку сделанных Вами заявок на закрытие счета/вклада, щелкнув по кнопке Список, или на страницу выбора типа заявки, щелкнув по кнопке Заявки.

Просмотр документа

После подтверждения заявки Вы попадаете на страницу просмотра, где отображается заполненный документ. О том, что документ передан в банк, свидетельствует отображаемый на форме документа штамп «Принято к исполнению». На страницу просмотра документа Вы можете также попасть из списка документов, щелкнув по номеру того документа, который хотите просмотреть. На странице просмотра Вы можете выполнить следующие действия:

  • Создание нового документа

    Вы можете создать еще один документ данного типа, щелкнув по кнопке Создать. В результате Вы перейдете к форме ввода нового документа. Данная возможность полезна при создании нескольких однотипных документов.

  • Печать документа

    Если Вы хотите распечатать данный документ, щелкните по кнопке Печать.

  • Возврат к списку заявок или выбору типа заявки

    Можно вернуться к списку сделанных Вами заявок на закрытие счета/вклада, щелкнув по кнопке Список, или на страницу выбора типа заявки, щелкнув по кнопке Заявки.

Список заявок

В списке заявок отображаются все созданные Вами заявки на закрытие счета/вклада. Вы можете зайти в список со страницы выбора типа заявки, щелкнув на выбранной заявке по кнопке История операций, или при создании заявки, щелкнув по кнопке Список. Для каждой заявки в списке отображаются дата создания, номер заявки, тип заявки и статус, по которому можно отследить состояние заявки в системе, и другие сведения. На данной странице Вы можете выполнить следующие операции:

  • Просмотреть заявку

    Для этого щелкните по номеру интересующей Вас заявки в списке и Вы попадете на страницу просмотра заполненной формы заявки.

  • Создать заявку на закрытие счета/вклада

    Для этого щелкните по кнопке Создать. Подробную информацию по данной операции Вы можете посмотреть в разделе контекстной справки Заявки - Заполнение заявки.

  • Удалить заявку

    Для этого установите флажок напротив заявки, которую Вы хотите удалить, и нажмите на кнопку Удалить.

  • Фильтр заявок

    Для поиска в системе интересующих Вас заявок предусмотрен фильтр, который позволяет осуществлять поиск по любому из полей либо их комбинации. Щелкните по кнопке Показать фильтр. Откроется страница с формой, в которой необходимо задать критерии поиска:

    В поле "Статус" выберите из выпадающего списка статус заявки на закрытие счета/вклада: "Введена", "Одобрена", "Исполнена", "Отказана" и др.;

    В поле "Период" введите период времени, в течение которого была создана заявка на закрытие счета/вклада;

    В поле "Тип заявки" выберите из выпадающего списка тип созданной Вами заявки: "Все", "Закрытие счета", "Закрытие вклада".

Вы можете фильтровать список по любой комбинации полей, например, можно отобрать все заявки на закрытие счета/вклада, созданные в один период, со статусом "Принята".

После того как критерии поиска заданы, щелкните по кнопке Построить список, и система покажет список интересующих Вас заявок.

Статусы заявок

Ход обработки заявок Вы можете наблюдать в списке заявок по статусу. Для заявлений по закрытию счета/вклада в системе предусмотрены следующие статусы:

  • Введен - документ введен, сохранен в системе, но не подтвержден;

  • Обрабатывается - документ полностью заполнен клиентом банка, подтвержден и передан на обработку в банк;

  • Ожидание обработки - ожидается передача документа в банк или получение информации о результате его исполнения;

  • Исполнен - документ исполнен банком;

  • Отказан - документ отказан банком с указанием причины.
