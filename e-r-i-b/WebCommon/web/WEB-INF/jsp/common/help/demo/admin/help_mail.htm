Пункт меню Письма предназначен для работы с письмами: для просмотра входящих писем от клиентов банка и отправки им писем.

Список входящих писем

Фильтр входящих писем

При выборе пункта меню Письма на экран выводится список писем, полученных от клиентов банка. Вы можете создать новое письмо, просмотреть и ответить на письмо клиента банка. По каждому письму в нём отображаются номер, отправитель, тема, дата, тип письма.

Примечание: в списке входящих писем непрочитанные письма будут отображаться со специальной пиктограммой .

Вы можете просмотреть текст любого письма. Для этого установите флажок напротив интересующего Вас письма от клиента банка и щелкните по кнопке Просмотр.

При необходимости Вы можете вернуться к списку входящих писем, для этого щелкните по кнопке Список.

Вы можете создать новое письмо, для этого щелкните по кнопке Создать, откроется страница создания письма, заполните все необходимые поля и нажмите на кнопку Отправить. Подробную информацию по созданию нового письма смотрите в разделе контекстной справки Письма - Список отправленных писем.

Если Вы хотите ответить на письмо клиента банка, установите флажок напротив выбранного письма или кликните на номере письма и щелкните по кнопке Ответить. Откроется страница редактирования письма. Поля письма "Номер" и "Получатель" автоматически заполняются системой. Вам нужно заполнить поля "Тема" (можно оставить тему письма клиента банка) и "Текст" сообщения.

Вы можете указать признак обязательности прочтения сообщения. Для этого установите флажок напротив поля Обязательно для прочтения.

Примечание: при входе в систему клиенту принудительно отображается данное сообщение, для дальнейшей работы с системой ему нужно нажать на кнопку Продолжить.

Для отправки письма клиенту щелкните по кнопке Отправить. Если Вы не хотите отвечать на письмо, щелкните по кнопке Отмена.

Кроме того, Вы можете осуществлять фильтрацию сообщений в списке.

Фильтр входящих писем

Для просмотра списка всех интересующих Вас входящих писем нужно щелкнуть по кнопке Показать фильтр и задать критерии поиска:

  • "Фамилия, имя, отчество" - поиск по фамилии, имени, отчеству клиента банка, отправившего письмо;

  • "Тема" - поиск по тематике письма;

  • "Тип" - поиск по типу сообщения;

  • "Номер письма" - поиск по номеру письма;

  • "Период с.. по.." - поиск по дате и времени получения сообщения сотрудником банка,

Вы можете осуществлять поиск по подстроке в текстовых полях фильтра. Например, можно ввести в поля фильтра только несколько первых букв фамилии, имени или отчества клиента банка.

Также Вы можете фильтровать список по любой комбинации полей. Например, можно отобрать письма всех клиентов с фамилией на букву "В" с одинаковой тематикой письма.

После того как параметры поиска заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас писем.

Список отправленных писем

Фильтр отправленных писем

Сотрудник банка может создавать и отправлять новые письма, адресованные определенному клиенту или группе клиентов.

Вы можете просмотреть текст любого письма. Для этого установите флажок напротив интересующего Вас письма, отправленного клиенту или группе клиентов банка, и нажмите на кнопку Просмотр или щелкните по номеру письма.

Чтобы отправить клиенту банка новое письмо, щелкните по кнопке Создать, откроется страница редактирования письма, на которой Вам нужно заполнить все поля:

  • Значение в поле "Номер" проставляется системой автоматически.

  • В поле "Получатель" Вы должны указать клиента или группу клиентов, которым хотите отправить сообщение. Щелкните по кнопке (...), откроется страница, на которой установите галочку напротив выбранного вида получателя и щелкните по кнопке Продолжить. Откроется страница со списком клиентов или групп клиентов, на которой Вам необходимо установить флажок напротив интересующего Вас клиента или группы клиентов, и щелкнуть по кнопке Выбрать.

  • В поле "Тема" введите тему письма.

  • В поле "Текст" впишите полный текст письма.

  • Вы можете указать признак обязательности прочтения сообщения. Для этого установите флажок напротив поля Обязательно для прочтения.

    • Примечание: при входе в систему клиенту принудительно отображаются сообщения, которые ему необходимо открыть для продолжения работы с системой..

После заполнения всех полей формы создания письма щелкните по кнопке Отправить, и письмо будет отправлено выбранному Вами получателю. Если Вы не хотите отправлять письмо клиенту или группе клиентов банка, щелкните по кнопке Отмена.

Фильтр отправленных писем

Для просмотра списка всех интересующих Вас отправленных писем нужно щелкнуть по кнопке Показать фильтр и задать критерии поиска:

  • "Фамилия, имя, отчество" - поиск по фамилии, имени, отчеству клиента банка, которому отправлено письмо;

  • "Тема" - поиск по тематике письма;

  • "Имя группы " - поиск по названию группы, в которой состоит клиент банка;

  • "Тип получателя" - поиск по типу получателя, например, клиент, группа и т.д.

  • "Период с.. по.." - поиск по дате и времени получения сообщения сотрудником банка.

Вы можете осуществлять поиск по подстроке в текстовых полях фильтра. Например, можно ввести в поля фильтра только несколько первых букв фамилии, имени или отчества клиента банка.

Также Вы можете фильтровать список по любой комбинации полей. Например, можно отобрать письма всех клиентов с фамилией на букву "В" с одинаковой тематикой письма.

После того как искомые параметры поиска заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас писем.
