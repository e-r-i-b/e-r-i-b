Назначение перевода

Данный платеж позволяет Вам перевести денежные средства с текущего счета для пополнения пластиковой карты. Чтобы создать перевод, в главном меню выберите пункт Платежи, а затем пункт левого меню Пополнение пластиковой карты.

Заполнение документа

В форме пополнения пластиковой карты Вам надо указать дату (в это поле автоматически будет подставлена текущая дата), выбрать из списка доступных Вам счетов счет списания (счет, с которого будут перечислены денежные средства), выбрать из списка карт ту, на счет которой будут зачислены средства, и указать сумму переводимых средств.

Примечание: валюта счета списания должна совпадать с валютой пополняемой карты.

После того как все поля заполнены, щелкните по кнопке Сохранить. Вы можете данный платеж сохранить как шаблон, для этого щелкните по кнопке Сохранить как шаблон.

Если Вы решили не делать перевод, с данной страницы можно вернуться к списку выполненных Вами переводов средств на карту, щелкнув по кнопке Список, или на страницу типов платежей, щелкнув по кнопке Платежи.

Подтверждение перевода

После сохранения заполненного документа Вы попадаете на страницу подтверждения. Здесь Вам будет показана заполненная форма перевода средств на спецкарточный счет, в которой Вам нужно проверить правильность указанных сведений, после чего выполнить одно из следующих действий:

  • Подтверждение перевода

    Процедура подтверждения зависит от используемого Вами способа регистрации в системе.

    Для подтверждения платежа Вам необходимо воспользоваться назначенным для Вас способом подтверждений. Например, если Вы используете карту одноразовых паролей, то Вам необходимо будет ввести пароль с указанным номером с Вашей карты одноразовых паролей.

    • Когда все сведения указаны, щелкните по кнопке Подтвердить, и документ будет отправлен в банк на обработку.

  • Возврат к редактированию

    Если Вы хотите изменить какие-либо реквизиты перевода, щелкните по кнопке Редактировать, и Вы вернетесь к форме ввода документа для его редактирования.

  • Создание перевода

    Если Вы хотите создать новый перевод, щелкните по кнопке Создать, и откроется форма для создания нового перевода для пополнения пластиковой карты.

  • Печать документа

    Если Вы хотите распечатать данный документ, щелкните по кнопке Печать.

  • Отказ от выполнения перевода

    Если Вы передумали осуществлять перевод, можно вернуться к списку операций пополнения пластиковых карт, щелкнув по кнопке Список.

Просмотр документа

После подтверждения платежа Вы попадаете на страницу просмотра, где отображается заполненный документ. О том, что документ передан в банк, свидетельствует отображаемый на форме документа штамп «Принято к исполнению». На страницу просмотра документа Вы можете также попасть из списка документов, щелкнув по номеру того документа, который хотите просмотреть. На странице просмотра Вы можете выполнить следующие действия:

  • Отзыв документа

    Вы можете отозвать ошибочно отправленный документ. Для этого щелкните по кнопке Отозвать, в результате Вы перейдете к форме отзыва перевода. На экран будет выведена подробная информация по отзываемому документу. При необходимости измените значение в поле "Дата отзыва", информация в остальных полях документа недоступна для редактирования. Для отзыва перевода щелкните по кнопке Сохранить. Откроется страница, на которой проверьте правильность заполнения полей документа и нажмите на кнопку Подтвердить, чтобы передать документ в банк на исполнение.

  • Печать документа

    Если Вы хотите распечатать данный документ, щелкните по кнопке Печать.

  • Создание нового документа

    Вы можете создать еще один документ данного типа, щелкнув по кнопке Создать, в результате Вы перейдете к форме ввода нового документа. Данная возможность полезна при создании нескольких однотипных документов.

  • Возврат к списку переводов

    Можно вернуться к списку переводов - пополнение пластиковых карт, щелкнув по кнопке Список.

Список переводов - пополнение пластиковой карты

В данном списке отображаются все выполненные Вами операции пополнения пластиковых карт. Зайти в список переводов можно со страницы выбора типа платежа, щелкнув по кнопке История операций, или при создании перевода, щелкнув по кнопке Список. Для каждой операции отображаются дата, номер документа, номер пополняемой карты, сумма операции и статус, отображающий текущее состояние операции. Вы можете выполнить следующие операции:

  • Просмотр документа

    Вы можете просмотреть любой документ, для этого нужно щелкнуть по номеру интересующего Вас документа в списке.

  • Отзыв документа

    Вы можете отозвать ошибочно отправленный документ. Для этого щелкните по кнопке Отозвать, в результате Вы перейдете к форме отзыва перевода. На экран будет выведена подробная информация по отзываемому документу. При необходимости измените значение в поле "Дата отзыва", информация в остальных полях документа недоступна для редактирования. Для отзыва перевода щелкните по кнопке Сохранить. Откроется страница, на которой проверьте правильность заполнения полей документа и нажмите на кнопку Подтвердить, чтобы передать документ в банк на исполнение.

  • Создание копии документа

    Вы можете создать копию документа, для этого отметьте документ, который служит оригиналом в списке, и щелкните по кнопке Создать копию. В результате Вы перейдете к форме ввода нового документа, причем в нее будут скопированы все данные из документа, выбранного Вами в качестве оригинала. Данная возможность полезна при создании нескольких однотипных документов.

  • Удаление платежа

    Вы можете удалить перевод из списка, для этого установите галочку напротив интересующего документа и щелкните по кнопке Удалить.

  • Фильтр переводов

    Для того чтобы найти интересующий Вас перевод в списке, щелкните по кнопке Показать фильтр и заполните поля фильтра:

    "Статус" - поиск по статусу перевода в системе: "Введен", "Исполнен", "Отказан" и др.;

    "Период" - поиск по времени создания перевода;

    "Сумма" - поиск по размеру суммы перевода. Рядом с данным полем можно указать валюту, в которой совершен платеж;

    "Карта" - поиск по номеру карты, счет которой пополнен.

    Вы можете фильтровать список по любой комбинации полей, например, можно отобрать все переводы на одну сумму со статусом "Исполнен".

    После того как критерии поиска заданы, щелкните по кнопке Применить, и система покажет список интересующих Вас переводов.

Статусы платежа

Ход обработки платежа Вы можете наблюдать в списке платежей или в журнале операций по статусу. Для перевода на пополнение пластиковой карты предусмотрены следующие статусы:

  • Введен - платеж введен, сохранен в системе, но не подтвержден;

  • Обрабатывается - платеж полностью заполнен клиентом банка, подтвержден и передан на обработку в банк;

  • Ожидание обработки - ожидается передача документа в банк или получение информации о результате его исполнения;

  • Отозван - платеж передан в банк на обработку, но затем Вами отозван по какой-либо причине;

  • Исполнен - платеж проведен в банке: денежные средства со счета клиента поступили на указанный в форме счет зачисления;

  • Отказан - платеж отказан банком с указанием причины, например, по причине нехватки денежных средств на счете клиента на момент их списания.
