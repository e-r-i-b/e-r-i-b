Для быстрого осуществления часто совершаемых платежных операций Вы можете использовать шаблоны платежей. Каждый шаблон представляет собой форму некоторого платежа с частично заполненными реквизитами. Шаблоны платежей в системе делятся на шаблоны банка и шаблоны клиента. Шаблоны банка создаются сотрудником банка, они выделяются в списке жирным шрифтом и недоступны для редактирования. С помощью данного пункта меню Вы можете создать платеж по шаблону, создать, изменить, удалить шаблон, создать копию шаблона любого платежа.

Создание платежа

Для создания платежа по шаблону выберите в главном меню пункт Платежи, а затем щелкните по пункту левого меню Шаблоны платежей. Вам будет предоставлен список имеющихся шаблонов, выберите интересующий Вас шаблон, установив напротив него галочку, а затем щелкните по кнопке Создать платеж.

Создание шаблона

Для того чтобы создать шаблон, щелкните по кнопке Создать шаблон. После этого откроется окно, и система предложит Вам выбрать вид платежа, шаблон для которого Вы хотите создать. В данном окне установите галочку напротив интересующего Вас вида платежа и щелкните по кнопке Продолжить. Откроется форма заполнения выбранного Вами платежа, в которой в поле "Название шаблона" введите наименование шаблона, заполните нужные поля и щелкните по кнопке Сохранить , и новый шаблон появится в списке Ваших шаблонов платежей.

Изменение шаблона

Для того чтобы изменить шаблон платежа, выберите интересующий Вас шаблон, установите напротив него галочку и щелкните по кнопке Изменить. Откроется форма для редактирования шаблона, в которую внесите изменения и щелкните по кнопке Сохранить .

Удаление шаблона

Для удаления шаблона платежа из списка выберите интересующий Вас шаблон, установите напротив него галочку и щелкните по кнопке Удалить, и шаблон будет удален из списка.

Создание копии шаблона

Для того чтобы создать копию шаблона, выберите интересующий Вас шаблон, установите напротив него галочку и щелкните по кнопке Создать копию. Откроется форма для ввода/редактирования шаблона, в которой в поле "Название шаблона" впишите наименование шаблона и щелкните по кнопке Сохранить . Если Вы решили не создавать копию шаблона, нажмите кнопку Список, и система вернет Вас в список шаблонов.
