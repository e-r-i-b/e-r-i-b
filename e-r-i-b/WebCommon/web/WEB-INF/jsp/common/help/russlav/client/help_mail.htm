Письма

Система позволяет Вам отправлять в банк и получать из банка сообщения свободного формата — письма. Для работы с письмами выберите в расположенном в левой части экрана меню пункт Почта. На экран будет выведен список писем. В нем вы можете выполнить следующие действия:

  • Просмотреть список своих сообщений. Можно сортировать сообщения в списке по дате, времени и номеру.

  • Просмотреть текст сообщения. Для этого просто щелкните по заголовку сообщения в списке.

  • Отправить сообщение в банк. Для этого щелкните по кнопке Создать и заполните поля предложенной формы.
