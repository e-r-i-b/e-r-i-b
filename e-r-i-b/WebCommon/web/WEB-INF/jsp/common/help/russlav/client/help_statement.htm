Выписки

На данной странице Вы можете просмотреть и распечатать выписку по Вашим счетам, содержащую информацию об операциях, выполненных по данному счету за указанный период. Параметры выписки задаются в верхней части страницы.

В поле Счет вы выбираете те счета, по которым хотите получить выписку. Можно выбрать несколько счетов. Для этого нажмите клавишу Ctrl и, не отпуская ее, щелкните по требуемым счетам.

В полях Период с ... по ... вы указываете интересующий Вас период (даты начала и конца); в выписку попадут только те операции, которые были выполнены в заданный период. Не рекомендуется указывать слишком большой период, так как в этом случае формирование выписки может занять много времени.

В поле Тип выписки Вы выбираете из списка тип получаемой выписки, определяющий какая информация будет включена в выписку. В системе есть обычные и расширенные выписки. В обычной выписке содержится следующая информация:

  • Заголовок с номером счета и датой (или периодом) выписки;

  • Валюта счета;

  • Входящий остаток по счету;

  • Дата, сумма прихода/расхода и наименование получателя по каждой операции;

  • Исходящий остаток по счету.

В расширенной выписке дополнительно выводится номер счета получателя, банк получателя и назначение платежа для каждой операции.

После того, как Вы задали параметры выписки, для ее формирования необходимо щелкнуть по кнопке Выписка. Если щелкнуть по кнопке Очистить , то будут очищены все поля с параметрами выписки.

Щелкнув по расположенной над списком кнопке Список счетов/карт , Вы вернетесь к главной странице системы со списком счетов и вкладов.

Чтобы распечатать показанную на экране выписку щелкните по кнопке Печать , расположенной в верхней части страницы.
