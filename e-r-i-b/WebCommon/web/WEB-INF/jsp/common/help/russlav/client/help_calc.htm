Депозитный калькулятор

Депозитный калькулятор позволяет Вам быстро оценить доходность депозитов и выбрать подходящий для вас продукт из предлагаемых банком. Сначала выберите интересующий вас вклад из предлагаемого списка в поле Тип вклада. Затем выберите валюту вклада (список валют зависит от выбранного типа вклада) и укажите срок вклада. В поле Сумма вклада укажите ту сумму, которую вы планируете разместить на депозитном счете.

После щелчка по кнопке Рассчитать калькулятор покажет вам процентную ставку по выбранному вами типу вклада, сумму, которая будет на депозитном счету по окончании срока вклада и сумму полученного дохода.
