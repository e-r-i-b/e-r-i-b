Справочники

В системе предусмотрены справочники, которые облегчают заполнение документов и позволяют избежать возникновения опечаток в реквизитах документов.

Справочник получателей

В данном справочнике хранятся реквизиты получателей, которые Вы можете использовать при заполнении рублевых переводов. Для каждого получателя в справочнике хранятся его название (или ФИО для физического лица), коды ИНН и КПП, адрес, номер счета и реквизиты банка получателя, в котором открыт данный счет. При выборе получателя из справочника все эти сведения автоматически подставляются в документ. Справочник является индивидуальным для каждого пользователя системы.

Справочник Contact

В данном справочнике хранятся реквизиты получателей, которым Вы отправляете средства через сеть Contact (без открытия счета получателя). Для каждого получателя в справочнике хранятся его фамилия, имя, отчество, адрес и реквизиты банка в котором он будет получать перевод. При выборе получателя из справочника все эти сведения автоматически подставляются в документ. Справочник является индивидуальным для каждого пользователя системы.
