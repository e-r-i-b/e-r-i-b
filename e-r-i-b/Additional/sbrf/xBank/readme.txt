	При тестировании взаимодействия с xBank необходимо внести изменения в БД. Для этого нужно под SYS-ом выполнить скрипт "скрипт изменения БД для тестирования xBank.sql". Скрипт создаёт новую схему "A_XBANK", хранимую процедуру "AccountPRS_P(iXMLDocument_IN XMLTYPE, iXMLDocument_OUT OUT XMLTYPE)", которая имитирует ответ внешней системы и вспомогательную таблицу (A_XBANK.XMLDATA), содержащая примеры ответов xBank (см. "analytics\Проекты ИКФЛ\Клиентские\ЕРИБ\1-Аналитика\Интеграция\COD_format.doc"): "xml_204" - тип ответа 204, "xml_297" - тип ответа 297, "xml_error" - ответ с ошибкой. При необходимости данные в этой таблице можно менять, соблюдая формат, описанный в "COD_format.doc".
