Для получения для формирование отчета по попыткам изменения варианта уплаты процентов в формате .csv, необходимо выполнить "create_report.bat", предварительно определив указанные в нем параметры.

По умолчанию отчеты формируются за текущий месяц. 
Для получения отчета за конкретный период, необходимо в файле "create_report.bat" заполнить значения параметров: 
 - "SQLPLUS_HOME" - путь до sqlplus.exe
 - "SCHEME_NAME" - название схемы БД
 - "SCHEME_PASS" - пароль к схеме БД
 - "SERVER_NAME" - сервер
 - "SERVER_PORT" - порт
 - "DB" - имя БД
 - "SCRIPT_HOME" - путь к выполняемому sql-скрипту
 - "OUTPUT_PATH" - путь куда будет помещен результат выполнения запроса (csv-файл)
 - "REPORT_BEGIN_DATE" - дата начала формирования отчета (в формате "dd.mm.yyyy") включительно,
 - "REPORT_END_DATE" - дата окончания формирования отчета (в формате "dd.mm.yyyy") НЕ включительно,
 Рекомендуется указывать период не более месяца, например: 01.10.2013 - 01.11.2013
 - "SEPARATOR_CHARACTER" - служит для указания символа-разделителя для файла отчета (*.csv).
 - "REPORT_FILE_NAME" - название файла отчета

Для формирования отчета также можно выполнить в среде разработки скрипт "query.sql" и определив параметры:
 - "date_begin" - дата начала формирования отчета;
 - "date_end" - дата окончания формирования отчета;
 - "acc_opening" - ИД формы открытия вклада
 - "acc_opening_with_close" - ИД формы открытия вклада с закрытием счета списания
 - "change_interest_dest" - ИД формы доп. соглашения к договору о вкладе
