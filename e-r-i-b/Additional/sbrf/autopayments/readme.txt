Для того, чтобы обновить ПУ для промо-блока Автоплатежей, необходимо запустить updatePromoBlock.bat, предварительно определив указанные в нем переменные.

Фармат данных в файле для загрузки:
регион1:поставщик1,поставщик2,поставщик3
регион2:поставщик4,поставщик5,поставщик6

Результаты загрузки будут записаны в файл result.txt
