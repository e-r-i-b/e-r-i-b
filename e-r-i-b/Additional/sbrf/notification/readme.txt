Использование скриптов.

====== Количество зарегистрированных клиентов, у которых введены email-адреса за период =========================

Скрипт: enterEmailReport.sql
В блоке SELECT проставляем даты начала и конца периода. Запускаем в среде разработки запросов.

====== Количество клиентов, которые настроили себе email-уведомления в разрезе каждого типа за период. ===========

Скрипт: notificationSettingsReport.sql
В блоке SELECT проставляем даты начала и конца периода. Запускаем в среде разработки запросов.

====== Общее количество отправленных клиентам из ЕРИБ email-уведомлений за период. ===============================

Скрипт: notificatinSentReport.sql
В блоке SELECT проставляем даты начала и конца периода. Запускаем в среде разработки запросов.
