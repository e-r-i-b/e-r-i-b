BUG084691  ЕРМБ. При подключении услуги МБ, делать регистрацию у разных ОСС, как сейчас у мегафона 

Таблица ERMB_MOBILE_OPERATOR задает настройки интеграции с ОСС. ОСС идентифицируется по MNC (см. справочник DEF кодов). Если для ОСС включена интеграция (USE_INTEGRATION), необходимо задать URL, логин, пароль веб-сервиса
