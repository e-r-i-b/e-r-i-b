Для получения отчета в формате csv, необходимо запустить "create_report.bat", предварительно определив указанные в нем переменные.
По умолчанию отчет формируется за текущий месяц. 
Для получения отчета за конкретный период, необходимо в "create_report.bat" заполнить значения переменных (в формате dd.mm.yyyy): 
 - "SQLPLUS_HOME" - путь до sqlplus.exe
 - "SCHEME_NAME" - название схемы БД
 - "SCHEME_PASS" - пароль
 - "SERVER_NAME" - сервер
 - "SERVER_PORT" - порт
 - "DB" - имя БД
 - "SCRIPT_HOME" - путь к выполняемому sql-скрипту
 - "OUTPUT_PATH" - путь куда будет помещен результат выполнения запроса (csv-файл)
 - "REPORT_BEGIN_DATE" - дата начала формирования отчета,
 - "REPORT_END_DATE" - дата окончания формирования отчета.
 - "SEPARATOR_CHARACTER" - служит для указания символа-разделителя для файла отчета (*.csv).

Для формирования отчета также можно выполнить в среде разработки скрипт "query.sql" и определив параметры:
 - "report_begin_date" - дата начала формирования отчета
 - "report_end_date" - дата окончания формирования отчета.
